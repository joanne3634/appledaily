20歲女打雜供4弟妹求學

爺死無依 日掙600元薄薪 愁生活費無著

2008年10月17日

20歲的小萍是家中的老大，四個弟妹分別就讀國、高中，母親自幼離家，父親十幾年前經商失敗，跑路躲債。自幼小萍與弟妹由阿公撫養長大，但今年7月，小萍79歲的阿公因心肺衰竭而去世，家庭頓失依靠。報導．攝影╱韓旭爾

想起阿公對他們姊弟的付出，小萍感傷地說：「我覺得阿公很偉大也很可憐，從年輕到老沒過好日子，爸爸把祖產土地都敗光了，他還是很認命，辛苦養育我們五姊弟，真是老歹命。」阿公往生後，現在家中僅靠小萍在廟裡打雜的微薄工資，難敷家中開銷及弟妹教育費用。
小萍無奈表示，身為家裡長女，從小她就到處打工貼補家用，「別看我才20歲，卻已有7年的工作經歷。」她苦笑說，上國中時在學校福利社打工，國中畢業後本想放棄升學，在老師勸導下，就讀高職夜間部，白天就在冷氣工廠當作業員，每月掙得一萬元協助家計。高中畢業後，因居住地區偏遠，工作機會少，又不可能拋下弟妹到外地工作，只能到附近廟宇擔任打雜臨時工，每天工資600元。

嘆「不知阿公怎籌到」

小萍說，自從阿公去世後，她才知道持家不容易，「8月份收到電費帳單，兩個月的電費要繳3515元，一下子，瓦斯沒了，一桶就要800元，」小萍說，「看到這些帳單，我終於了解阿公生前為什麼生活得如此節省，省到每天三餐吃的都是白飯，配上一點鹹魚滷豬肉的醬汁，原來省下的錢都用來繳水電、瓦斯費了。」
小萍說，她很希望弟妹將來能繼續念書，「我們雖因低收入戶可減免學費，但雜費、書籍費、校車費等這些還是好多錢。」小萍說，這學期四個弟妹的學雜費就要1萬2000元，她不知以前阿公是怎麼籌到的。

大妹缺電腦開不了口

大妹小卿目前就讀高職二年級，學的是機械製圖，她說：「這學期學校開始教導電腦製圖，家裡沒電腦，只能利用上課時間學習軟體操作，但練習的時間實在不夠，現在老師在台上講課，我就好像鴨子聽雷一樣，大部分都聽不懂。」但想起一台電腦要二、三萬元，她實在開不了口向姊姊要，「姊姊為了我們三餐、學費已經夠累了，不能再增加她負擔。」
當地村長湯全壁說，小萍阿公去世時，是靠農保15萬元的死亡給付才完成，雖還有伯、叔住在同縣內，但都很貧困，現在依靠政府的低收補助和她微薄的薪水總共2萬多元收入，要維持五口生活不容易，蘋果基金會獲悉後已先撥款紓困。
