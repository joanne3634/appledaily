經費減 夜光天使無處去

40生沒人顧 「憂出問題」

2011年07月07日

晚上6時，一大群孩子吃飽晚餐，在課輔教室寫功課，在旁的「台灣築心全人生命教育關懷協會」（以下簡稱築心協會）祕書長莊淑真，她看著孩子的純真笑容，但內心卻有極大擔憂。報導．攝影╱向高彬

莊淑真跟記者說，近日接到新竹縣家庭教育中心通知，由於教育部課輔經費縮減，「100學年度『夜光天使』經費告停，現在我擔憂這些辛苦家庭孩子，若暑假後經費用罄，可能就得先停供晚餐，再來只好減收學生，很擔心以後孩子晚上若沒人顧，功課不寫、下課在外遊蕩去網咖，容易出問題。」

擴大據點計劃將停擺

莊淑真說，協會3年前由一群熱心社區民眾捐資創立，「這裡隔代教養及外配家庭多，經濟及教育上相對弱勢，目前近40位孩子接受課輔，不少單親家長忙做工，孩子在這裡吃晚餐、把功課做好，並加強較弱的英數科目。」
新竹地區現有竹東教會、迦恩教會及築心協會等3所民間單位，與教育處合作課輔方案，莊淑真說，協會2年來連獲評鑑特優，目前教育處年度方案補助約40萬元，「只是我們協會沒像其他單位，有母機構或教會的部分場地經費支持，經費籌措原就較吃力，我們擔心不夠力量再好好顧孩子。」
莊淑真說，原本協會還規劃新學年再籌200萬元，擴大新據點服務鄰近關東橋學區約50名學童，但如今協會自身服務已受衝擊，新計劃恐將停擺。

上課一年調皮孫變乖

桂香阿嬤是隔代教養家庭，她有小三孫子接受課輔，她說，孫子之前好調皮，但來協會上課1年，功課好也乖多了，「聽說協會經費不夠，好擔心。」嫁自印尼的阿恬是低收家庭，先生身障待業，家有2個智能障礙幼子，「我跟小孩一起上課，跟孩子說，一起寫功課學寫字。這裡老師還幫我找零工，賺點錢幫家用。」
築心協會表示，今年3月起，博幼基金會有協助指定補助年度師資費約35萬元，蘋果基金會訪視後，也拋轉引玉撥款協力。
新竹縣家庭教育中心主任許瑜庭說：「夜光天使計劃因經費縮減，對相關課輔單位造成衝擊，深感為難。」
