病妻顧癌夫 憂撐不下去

盲子撿蝸牛 日掙200元生活苦

2012年11月13日

採訪時，63歲的李英圳正住院進行肝癌化療，他的老伴李潘金花到院外買了兩個便當走進來，約100公斤的身軀緩慢跛步，一到床旁，急忙坐在躺椅上喘氣休息，李英圳眉頭緊蹙地說：「老伴有糖尿病，她顧我，我擔心她也病倒！」報導．攝影╱韓旭爾

李英圳說，6月開始他體重驟降，1個月內體重掉了10多公斤，只剩50公斤，也感手腳無力，老伴感覺不對勁，8月底時，要他到醫院檢查，「我的身體一向健康，沒想到一病就是肝癌3期。」李英圳說，他婚後與妻子生3子1女，其中2子分別因車禍、燙傷而去世，女兒已出嫁，同住45歲長子阿明右眼失明，工作機會少，多年來家計主要依靠他到山上做除草工每日掙得800元維持，「本來就難過日子，現在更慘，我病成這樣，以後家裡怎麼辦？」

嘆「命中多折磨」

一旁61歲的李潘金花邊喘邊接說，她因糖尿病併發症導致心臟病、腳部循環不良，視力也不好，已10多年無法外出工作，「我飯雖吃得少，但還是這麼胖，走沒幾步路就喘。」訪談過程中，李潘金花見掛在床頭點滴已滴完，站了起身想按鈴呼叫護理師，卻是一陣頭暈目眩襲來，單膝跪在躺椅緩了一緩，過了1分鐘才完全站起，她苦笑說：「這2個月擔心老伴病情，加上我在醫院睡不好，害怕自己撐不下去，想來想去，是我命中注定多折磨，老天決定這樣，也沒辦法。」

僅領4700元津貼

接著記者來到李潘金花家中，只見李英圳中度視障的兒子阿明正在把早上到山上撿拾的蝸牛敲打去殼取出蝸牛肉、清洗裝袋，打算明天拿到市場販賣。阿明說，自己只剩左眼視力，「看東西沒法判斷距離，到了晚上視力更差，不能騎車出門，怕發生車禍。」阿明說，他因視力不佳覓職四處碰壁，平時會到附近山區撿蝸牛販賣，每日約可掙2、300元，「只是冬天到了，蝸牛要冬眠，也沒得撿了。」
李英圳出嫁的女兒阿鈴說，她遺傳媽媽糖尿病，除了視力不佳外，每周還需洗腎3次，家計只靠丈夫在資源回收場做粗工，手頭也緊，沒法拿錢幫爸媽。當地里長說，李英圳夫妻均計工作人口，所以難列低收，目前政府補助只有阿明殘障津貼每月4700元，屬貧困邊緣家庭，現家中主要經濟支柱罹癌，日子過得更苦，已協助申請「馬上關懷」急難金，並轉介蘋果慈善基金會協助。基金會訪視後，也撥款暫紓李英圳一家醫病生活急困。
