單親媽失明 憂過動幼子

臨時工丟了 拭淚「未來在哪」

2012年11月11日

「看得清楚，對我來說是種奢求。」48歲的單親媽阿敏左眼剛動完眼角膜手術，但視線仍舊模糊，「右眼看不到，左眼是白濛濛一片，我的世界是灰白的，醫生說，半年後還要再開右眼，但我現在左眼只能看得到光，未來到底在哪裡？」
報導．攝影╱江品璁

阿敏說到傷心處，眼淚撲簌簌流下，「今年5月中一覺醒來，10歲兒子小海站在我面前叫媽媽，我卻看不到他的臉，忍不住就哭了，這要怎麼辦？我那時在工地當臨時工，每月才賺1、2萬，根本沒錢就醫。」阿敏擦了下眼淚接著說，「還好前夫阿清念及舊情，就帶我去眼科診所看病，醫生說我雙眼失明沒救了，我絕望到想跳樓輕生，後來被阿清勸回，家扶社工也介入幫忙，送到大醫院就診，才知道是角膜炎併發角膜結疤，因為等無國內角膜移植，只好安排美國眼庫的角膜，光移植左眼就要6萬多元，我付不起，後來張榮發基金會願意協助，才解決這個難題。」

工地髒亂染眼疾

阿敏無奈的說，以前視力就不好，但今年開始視力變更差，騎機車常要靠近到1公尺內才看得到東西，發生了十幾次車禍，這才知道是雙眼失明的前兆，「醫生跟我說，我的狀況可能是因為罹患疱疹，又在髒亂的工地工作，常有揉眼睛的習慣，才導致角膜潰瘍，醫生說一次只能開一眼，只好先移植較嚴重的左眼，右眼要等半年後才能再動刀。」

「下個月怎麼辦」

訪視時，阿敏把左眼瞇成一線，努力許久僅依稀看到記者模糊身影，她氣餒的說：「開完刀散光嚴重，要很費力才能看個大概，報紙上的字更完全看不到。」阿敏緊接著催促兒子小海去寫作業，但小海坐在書桌前沒半分鐘，又起身在屋內亂跑，一頁的作業寫了1小時都寫不完，阿敏眼眶泛紅的說，小海原本是輕度智障，上資源班後程度有提升，2年前鑑定後取消智障身分，但還是過動兒，「現在我眼睛看不清，又是單親，兒子還那麼小，以後日子要怎麼過？」
阿敏嘆道，開刀後無法工作，靠「救命錢」馬上關懷急難金3萬元，和每月家扶、政府兒少補助共4千多生活，但還要負擔房租7千元，「過了這個月，下個月該怎麼辦？」阿敏又說，自己排行老大，與1弟1妹少有聯繫，還好有表嫂阿珍關心，表嫂阿珍則說，知道阿敏開刀後生活辛苦，但自己有5個小孩，經濟也困難，幫忙有限。
醫院社工則說，阿敏左眼需再觀察3個月，看有無排斥反應，成功率約6成。當地公所說，已協助阿敏申請租屋補助、低收，另轉介蘋果基金會。基金會訪視後，已撥款暫紓困境。
