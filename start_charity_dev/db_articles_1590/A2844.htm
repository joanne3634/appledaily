夫突逝 臨盆婦掛心幼子

心疼孩子沒了爸 「我要堅強」

2012年08月08日

7歲的長女小慧比手劃腳擺出拳擊手姿勢，5歲的弟弟在旁也懵懵懂懂學著，兩人憑著印象模仿爸爸生前打拳擊的英姿，小慧30歲的媽媽劉麗銀見狀，滿臉心疼孩子這麼小就沒了爸。報導．攝影╱仲芝蓉

8月中下旬將產下第4胎的劉麗銀嘆了口氣說，丈夫生前身體一向健壯，還是拳擊好手，就只有痛風老毛病。今年4月他因細菌感染壞死性筋膜炎，發病3周病逝，「一切發生得太突然，他才37歲，留下長女小慧才要上小學，長子要念幼稚園大班，次女才3歲，以及肚裡來不及見到爸爸的胎兒」。

5歲長子輕度自閉

劉麗銀回想丈夫的發病經過，難忍悲慟紅著眼眶說，一開始丈夫只說被蟲咬，腫了一個禮拜覺得不對勁才去看醫生，醫生說他可能肝不好抵抗力差，前後開刀10幾次清瘡，卻仍導致敗血症、多重器官衰竭不治。
當地家婦中心社工說，劉麗銀和丈夫原本都從事清潔零工維生，孩子陸續出生後，劉麗銀還去超商做代班工作，一個月賺500、600元貼補家計，夫妻倆生活單純，日子勉強可過，不幸丈夫突然病故，公所正協助申請低收入戶等相關補助，孩子學校也募款3萬多元協助，已處理妥喪葬後事，但劉麗銀即將臨盆，暫時無法工作，故轉介蘋果慈善基金會，基金會訪視後，也先撥款暫紓困境。
劉麗銀接著說，從沒想過有一天會讓人接濟，很感謝大家即時伸出援手，「以後不管發生什麼事，這4個孩子都只能靠我，我一定要堅強振作」。記者訪視時，劉麗銀的3歲次女因年幼尚不懂事，直嚷著要媽媽抱不停尖叫，但劉麗銀挺著大肚子，實在難安撫次女情緒，一旁5歲的長子卻似乎沒聽見妹妹吵鬧，劉麗銀無奈地說，長子經診斷發現輕度自閉症，正接受早療，「丈夫和我都學歷不高，不求孩子有多優秀，只希望長子能跟其他孩子一樣活潑長大就好」。

友送白包助辦後事

曾與劉麗銀夫妻倆一起做清潔工的同事回述，劉麗銀的丈夫工作勤奮，為了迎接第4個孩子，每天都分外努力，「沒想到卻英年早逝，當時劉麗銀挺著5個月身孕到加護病房探望，拚命哭求丈夫一定要好起來，但最後她丈夫還是走了，真的很擔心她會撐不住，大家只能送點白包幫忙後事，希望平安度過難關」。
