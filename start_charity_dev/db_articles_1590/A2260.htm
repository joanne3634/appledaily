癱漢失語 兒：想聽爸罵我

一家三口野菜果腹「不餓肚子就好」

2010年09月07日

走進陳設簡陋的家屋裡，遠遠望見45歲中風癱瘓失語的沈信光，在他瘦弱的軀體上，有著十多隻蒼蠅盤旋不去，他連揮手趕蒼蠅都沒辦法，空洞洞的眼裡，盡是力不從心的悲哀。報導．攝影╱游竹君

重擔落71歲老母

71歲的排灣族人沈金蓮說，兒子沈信光今年6月初腦中風成半癱，且他因痛風石腫大變形的關節，在細瘦僵硬的四肢上更顯突兀，稍一移動就痛苦地發出「嗚…嗚…」的聲音。金蓮嬤說，兒子當年與女友生下一子小宏，小宏兩歲時生母就過世，十多年來就是3人相依生活，其他3女1子也都有家計重擔，「兒子倒了，住工寮我一個人顧不來，只好暫時搬到女兒家，村落裡的親戚才能就近幫忙。」

兒貼心幫趕蒼蠅

就讀國二的小宏，拿著餐桌蓋幫爸爸揮趕蒼蠅，「我不知道能幫爸爸做什麼，就是盡量幫奶奶照顧他。」
小宏說，以前自己愛玩、不聽話，總是讓爸爸跑著四處找人，「以前遠遠就能聽見他催我回家的聲音，但是他現在沒辦法說話，很想再聽到爸爸罵我的聲音，很怕他就會這樣離開我。」

年金難抵復健費

金蓮嬤說，每月6000元老人年金，實在不夠負擔兒子每周包車復健兩次開銷，無論晴雨她總是在工寮空地摘野菜，金蓮嬤滿意地望著手裡的野菜，不改樂觀天性說：「能省就省，只要不餓肚子就好！」蘋果基金會訪視後，也已撥款協助醫療及生活開銷，當地社工也進行「馬上關懷」急難金及「單親兒少」補助申請。
