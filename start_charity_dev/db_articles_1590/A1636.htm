單親父 日賺不到50元

拖中風病體拾荒「孩子不能餓啊」

2008年07月17日

高雄市狹小巷弄裡，陳振華拖著中風後的身軀，沿路撿拾寶特瓶、鋁罐等回收物，「有時撿一整天，也賣不到50元，加減貼補買菜錢啦。」他單親照料13歲的兒子，3月份中風後，生活更辛苦。報導 攝影╱孫敏聿

「我…自己…沒關係，孩子不能餓啊。」陳振華中風後伴隨語言障礙，說起話來上氣不接下氣，還往往詞不達意。「以前扛著瓦斯桶爬上5樓都沒問題，我中風後卻連起身都很吃力，沒有收入，不知道怎麼辦。」一旁13歲的兒子阿信則接著說，「爸爸，不然我去打工幫忙賺錢，就有飯吃了。」阿信幫爸爸端來泡麵果腹，讓陳振華好不心疼。

房租尚積欠8千元

憶及過往，陳振華說，6年前與妻子離婚，協議各自監護一子女，「我沒讀書，只會送瓦斯、送一桶賺40元，一個月好的話有2萬多，差的時候才1萬多元，勉強養活自己、母親和孩子。」
不料屋漏偏逢連夜雨，3年前母親竟也中風倒下，終日癱臥，大小便失禁，陳振華為照料母親常請假，收入銳減，2年前雖由政府公費安置於機構，每月仍須自費2千元尿布耗材等費用。
陳振華的妹妹說，哥哥是家中獨子，一直以來努力打拼持家，「我是養女，嫁到嘉義後，現還要幫忙女兒照料女婿自殺後留下的孫子，只能偶爾幫忙2、3千元媽媽的安養費，沒法長期資助。」房東林先生則表示，以往陳振華還能夠按時繳交每月6千元房租，自從生病沒了收入後，只能靠親友偶爾接濟，每月勉強擠出1、2千元交租，至今房租累計尚積欠8千元。

校方協助減免餐費

阿信就讀國一的江姓導師也表示，阿信在班上功課雖然不好，但認真學習，是很乖的孩子，原本父親只靠送瓦斯收入不豐，學校自上學期開始，就申請愛心基金減免午餐費，也商請參考書出版商贈送阿信教科書，省下書籍費，蘋果基金會獲悉，已先行撥款暫紓生活急難。
