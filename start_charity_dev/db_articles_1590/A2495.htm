癌父鬱母舉債 日坐愁城

少女打工分憂 「她是唯一希望」

2011年06月21日

57歲的阿尚去年罹患鼻咽癌，讓他喪失了聽覺，聽不見世界紛擾，卻無法逃避眼前現實。阿尚嘆：「我得這種難纏病，太太又患憂鬱症，18歲女兒是家中唯一希望，盼老天多保佑。」
報導．攝影╱仲芝蓉

記者進入馬路邊一處老公寓，穿過狹長樓梯來到阿尚家，只見阿尚面色枯黃，穿著大如布袋的衣服，戴助聽器扯著喉嚨告訴記者，放射線療把唾液腺燒壞，什麼都難吞，2個月體重掉25公斤，醫生說要喝管灌營養品補充熱量，但一罐50、60元，1個月要1萬元，去年起已欠親友10幾萬元，真的難再借。

歎日子平順是福

阿尚說完，拿起水杯喝了一口又吐出，55歲妻子阿蓮見狀說，丈夫無法分泌唾液容易口乾，所以要一直漱口，加上他原本就有鼻竇炎，鼻塞嚴重，只能用嘴巴呼吸，「好擔心他晚上睡覺噎住，愈想愈害怕就睡不著，煩惱太多得了憂鬱症。」阿蓮緊蹙眉頭說著。
阿尚潤完口回述過往，與妻子是在餐廳工作認識，其實她本來個性開朗，當時自己剛結束漁船工作，年近40想成家安定生活，阿蓮則剛離開一段不合適姻緣，兩人婚後共育1女，靠經營餐廳為生，「雖生意平淡，但平順就是福。」

厄運連連店關門

阿蓮在旁嘆了口氣說，沒想到3年前開始，家中厄運連連，「我連遇2次車禍，左臉頰、手肘粉碎性骨折，丈夫為了照顧我，只好把餐廳關了。」
阿尚接著說，去年初兩人想再拼看看，租店面賣羹湯、滷肉飯，卻只做了5個月，就因高血壓身體不適，左眼漸漸看不到、雙耳也聽不見，只好歇業，9月就醫才發現罹患鼻咽癌第4期，接著做化、放療。阿尚的女兒小芸說，爸爸重聽，媽媽把辛苦和煩惱都往肚裡吞，常看媽媽半夜掉淚，「我會趁課餘時間打工，幫爸媽分擔一點。」
當地里長廖先生說，去年11月已申請「馬上關懷」急難金1萬元協助，目前一家3口靠殘障津貼共6000元過活，但管灌開銷每月就要1萬元，阿蓮又因前段婚姻子女成年，恐難通過低收入戶補助，「謝謝蘋果基金會訪視，提撥急難金先幫忙他們家度過難關。」

暖流見證 女兒成績優可上大學

小芸的導師陳老師說，3年前小芸的媽媽車禍重傷，家中遭逢變故，但小芸仍努力兼顧家庭與課業，高中3年都維持前5名，學校也申請獎學金減免學費，沒想到這次她爸爸病倒，還好小芸的大學統測成績理想，有希望錄取志願學校，鼓勵她繼續升學。
