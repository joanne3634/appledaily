癌男咬牙養家 怎奈病發

心繫妻兒殘父老母「很害怕我倒了」

2013年04月09日

38歲的阿雄（謝進雄）3年多前發現罹患2期結腸癌，治療後還能挺著身子做板模工人養家，無奈去年5月癌症復發，阿雄頂著蒼白面容無奈說，他不怕自身癌病痛楚，而是擔憂沒有氣力養妻兒、顧父母，「很害怕我倒了。」報導．攝影╱韓旭爾

阿雄說，去年4月出現腹瀉不止情況，「心裡就覺得怪怪的」，5月回診追蹤時，醫生告知癌症復發噩耗，「心情難免低落，但事情碰到了，只能面對，我絕不放棄。」一旁30歲的妻子阿思（胡氏思）心疼地說，老公知道家裡沒錢，常強撐身體自己騎車1個小時到醫院治療，「阿雄平時在家常喊頭暈，很擔心他騎車危險，但包車每趟要800元，實在付不起，想到心好酸。」
阿雄與妻子育有10歲及8歲的2子女，還有63歲的媽媽及重殘繼父，阿雄說：「現太太不僅要照顧我，還要到處做工，每日賺600元貼補家用，真的辛苦她了。」

妻做工洗碗苦撐

阿思的朋友阿蘭說，阿思除陪伴阿雄住院外，「她所有的時間都在掙錢，不是在工廠做臨時工，就是在餐廳洗碗端菜，常每周工作7天，實在擔心她累倒，才打電話到蘋果基金會求助。」
阿思說：「我不怕吃苦，只是最近阿雄常住院，工作時間有限，賺到的錢還是不夠應付一家6口吃穿、阿雄營養品開銷，還得跟親友東籌西借，不知一個人怎麼扛起這個家。」蘋果基金會訪視後，已先撥急難金暫紓困境。
與阿雄一家同住的媽媽玉珠婆說，她要照顧59歲、因中風行動不便老伴，對於家裡經濟困境也出不了力，「知道兒子癌症復發時，心裡真的很難受，常半夜就驚醒，擔心兒子就走了，只能念阿彌陀佛，才能再入睡。」阿雄大姊表示，家中共有5兄弟姊妹，大家都只念到國中，賺的都是勞力錢，生活不好過，「我平時買些營養品回家看弟弟，但實在無力長期協助。」

女兒主動做家事

阿雄讀小四的女兒小靜說，常看媽媽工作累到回家後不想講話，「我會盡量幫忙家事，讓她輕鬆些。」小靜的導師表示，學校會盡力協助減免學費、營養午餐等費用，減輕負擔。當地村長說，阿雄一家補助僅有繼父殘障津貼4700元，但阿雄尚無殘障證明，玉珠婆也列工作人口，一家無法通過低收入戶審查。「一家都是老幼病弱，阿思拚命找工賺錢，真的很讓人肯定。」
