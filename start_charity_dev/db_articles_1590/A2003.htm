腳傷硬上工 為養洗腎妻

不顧人工關節磨損 「日子過下去才重要」

2009年10月16日

早上起床，54歲的塗彼得拿熱毛巾為自己腫脹、疼痛的膝蓋熱敷，但一見村長上門告知，今天山上有除草零工可做，他顧不得雙腿不適，急忙換上工作服，拿起除草機準備上工，他說：「腿痛不痛不重要，日子過下去才重要。」報導．攝影╱韓旭爾

「在部落裡，有工作就是撿到的機會，要把握！」塗彼得說，他與44歲的妻子阿英育有一名現年14歲兒子小文，3年前阿英患腎衰竭後，每天須在家洗腎，無法工作，家計全靠他一人擔任建築綁鐵零工獨撐，不料2年前他雙膝罹患退化性關節炎，陸續更換人工關節，行動大不如前，工作機會大減，「我已無法在鷹架爬上、爬下工作，現在只能在部落從事除草、農務零工，每天收入剩下8、9百元。」塗彼得說，每月頂多工作5、6天，「連家裡的吃飯錢都不夠！」

兒盼考軍校擔家計

記者採訪時，阿英剛在房間裡洗腎完走到客廳，腹部還露出管子，她聽了先生述說家裡情況，低頭嘆道：「最近常聽見先生抱怨膝蓋痠痛，人工關節好像磨損，萬一要再動刀，那就麻煩大了。」塗彼得一邊揉著紅腫的雙膝一邊說：「做完手術時，醫生就交代不能做粗活，人工關節才能維持久一點，但為了妻兒三餐溫飽，做牛做馬也要去呀！」
塗彼得就讀國二的兒子小文早熟懂事，「看爸爸走路一跛一跛，還是硬撐工作，我看了好不忍心，希望快點長大，賺錢幫家裡解決難題。」小文接著說，現在他所能做的，就是努力讀書，「將來如果能考上軍校，讀書不用錢，每個月還能有薪水可領，爸爸就能輕鬆一些了。」

「希望能栽培孩子」

塗彼得看見小文貼心，心裡欣慰地說，從小因書讀得不多，只能打零工，辛苦賺血汗錢，「所以不管多辛苦，我希望能好好栽培孩子，教育程度高一些，將來不要像我們這一代，只能這樣過。」
當地村幹事歸曉惠說，塗彼得一家雖有低收入戶每月1萬8200元的補助，但僅靠政府補助實在不敷開銷，蘋果基金會訪視後，已撥款暫紓塗彼得一家生活、醫病困窘。

後記
虔誠教徒命名「彼得」

採訪時，塗彼得告訴記者說，父母親都是虔誠的基督徒，特別以耶穌基督門徒「彼得」為他命名，期盼上帝能祝福他一切平安，自己現在一家也信教，每周上教堂作禮拜，他說：「上帝一定會賜福讓我們一家度過疾病災難。」
