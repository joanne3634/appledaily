癌母苦撐拾荒 「為女兒努力」

籌不出掛號費 長女心疼打工貼補

2009年07月15日

罹患舌癌的46歲單親媽媽紀寶春，因拔除全口牙齒，只能靠軟質食物維生，她在悶熱窄小的樓梯間費力整理拾來的瓶瓶罐罐，語帶感恩地說：「一袋寶特瓶能賣個40、50塊，兩個女兒的早餐錢就有著落了。」報導．攝影╱游竹君

紀寶春原本白天擔任清潔工、模板零工，晚上在小吃店工作，身兼2職養子，2年前發現罹患舌癌，切除部分病灶後，去年底又發現癌細胞轉移導致下顎骨壞死，摘除所有牙齒，今年3月因疼痛就醫發現舌癌復發，她口齒略含糊地說：「醫生說要開刀切除全舌，但我實在不想讓女兒為了照顧我請假、沒辦法上課，所以可能在暑假期間動刀。」

「這是命 只能接受」

紀寶春說，年輕時被愛沖昏頭，與已婚男友育有2女，現國中畢業及小六，10年前兩人分手後自己便獨力扶養2女，罹癌後，她整天以淚洗面，不知將何去何從，去年她找到精神寄託，每天吃藥前總會虔誠跪求禱告：「希望主耶穌幫助我的病情好轉，有勇氣走出病魔的陰影，照顧兩個女兒成年。」
面對舌癌病情日益惡化，紀寶春說：「我不怨任何人，這是命，只能接受，然後為了女兒繼續努力下去。」長女小翎說：「有時掛號費湊不出來，媽媽就沒定時回診了，我看了心裡很難過，從國三開始我就利用假日去火鍋店打工，一個月賺個1、2千元，全交給媽媽。」

只靠補助難敷開銷

從旁協助關懷的圓愛協會劉督導說，由於小翎和妹妹戶籍上生母登記的是生父元配，因此難以請領政府相關兒少補助，在社會局、慈濟、家扶及圓愛協會協助下，已完成親子DNA（Deoxyribonucleic acid，去氧核醣核酸）鑑定及確認親子關係訴訟，近期將可取得女兒共同監護權；里長邱昭金表示，紀寶春很堅強，若身體狀況允許，還會炒花生去市場賣，等她取得女兒監護權後，將再協助申辦低收。
紀寶春說，這1年多來都靠慈濟及家扶補助度日，扣除房租5000元，每月僅剩8400元實在難敷生活開銷；今年3月初拾荒時被巡邏警員發現生活窘境，經《聯合報》地方版披露後，獲讀者捐贈消費券及善款共約3萬多元，已陸續支應開銷，「只祈禱病情快點穩定，我才能靠雙手工作養大女兒。」蘋果基金會訪視後，也先撥款暫紓紀寶春困境，並以「克蘭詩．蘋果助學專戶」協助2女完成學業。
