單親母恐截肢 難養3兒

已動14次手術 子女貼心助擦澡

2012年05月21日

46歲單親媽吳惠媚因車禍傷腿，且因術後復元不佳導致血管栓塞、組織壞死，至今左小腿已動了14次手術，她無奈說，當初為了不截肢努力保住左腿，未料如今「恢復時間遙遙無期，還要靠兒女們把屎把尿。」報導．攝影╱張嘉恬

採訪時吳惠媚只能坐臥在椅子上，下半身難以動彈，她說去年10月底因車禍重傷，期間因血管接回復元不佳，進行了血栓清除、筋膜切開、血管繞道、皮瓣移植等手術，4月中又因內植的鋼板外露動了第14次刀，她看著約30公分長的傷口說：「不能縫合，因為肉一直沒長好，如果整個壞死得要截肢。」

憂孩子生活無著

她說罷不過3秒，淚水倏地從臉頰滑落，情緒漸緩和後，她提及前夫因投資失利負債，兩人去年離婚後不久，她便發生車禍，而前夫早年也因車禍導致肢障，現已自身難保，「我很甘願擔起照顧3個孩子責任，只是原以為這意外只是粉碎性骨折，沒想到復元一延再延，皮肉痛能忍，但孩子生活沒著落……」
吳惠媚的趙姓好友說，吳惠媚子女最大念高三、最小國三，都是最花錢年紀，「單親媽本就辛苦，她也一向堅強撐家，面對未知，她忍著不在孩子面前哭，心裡擔心的是以後腿殘還能不能工作。我也有自己家庭，只能盡力協助入出院事宜、幫忙處理日常事務。」
採訪時恰逢吳惠媚的子女放學回家，17歲次女放下書包後，立即端來臉盆替母親擦澡，15歲長子也在旁幫忙，次女說，媽媽辛苦賺錢撫養她和姊弟，「全家平時勉強過活，沒想到媽媽出車禍重傷，我們又都還在念書，能盡力的就是好好照顧媽媽」。吳惠媚看著子女心感安慰說：「一場車禍把我弄得很慘，但也讓孩子成長獨立了。」

需休養半年以上

吳惠媚次女學校的學務處組長李建德說，吳惠媚單親養育3名子女，平時家計緊絀，現又遭逢意外，生計雪上加霜，學校已協助申請學產基金、張榮發基金會急難金共約3萬元，但資源有限僅能應急，所以轉介蘋果基金會。吳惠媚需至少半年以上休養復健，屆時也才能進行強制險殘廢給付認定，她已獲「馬上關懷」急難金2萬5000元，蘋果基金會訪視後已撥款紓困。
