「祖孫合吃一碗麵也要捱」

爸媽都走了 3童只剩7旬祖父母相依

2009年08月17日

「爸爸，我會照顧弟妹。」將升國三的小偉和弟、妹，7月中送走猝逝爸爸；不過才4年前，3兄妹送走患病自殺的媽媽，如今3兄妹只能與7旬爺爺奶奶相依偏僻鄉間。報導 攝影╱向高彬

小偉和他兩個現將讀小五、小三的弟妹過了個超傷心暑假，「畢業旅行也沒去，一想到爸媽都走了，就想哭。」採訪當天，是小偉的爸爸張建文出殯日，小偉手捧爸爸骨灰罈，最後一次感受爸爸溫度，他與弟妹眼泛淚光，隨師姑誦念佛號引靈進塔，協助法事的陳姓師姑說，看到孩子幼弱身影也心不捨，「我祈求孩子爸媽在天之靈，保佑長大。」

父扛家累壞身體

小偉說，爸爸生前在北部做綁鐵工，4月回家待工，但6月中見他病懨懨、脾氣煩躁，爺爺曾勸去看醫生，但都說沒事，直到病逝前兩天，突然全身癱軟，「一句話沒留給我們就走了。」小偉說，爸爸後事靠鄰里親友、慈善會和叔伯幫忙辦好。
小偉的72歲爺爺張誠助說，他發現44歲的兒子建文走後留下一堆藥包，才知他早患肝炎、腎結石等病，「他話少脾氣硬，從不跟親友訴苦，自妻子走後一肩扛起養孩子責任，醫師說他累壞身體了。」
爺爺說，之前都靠兒子做工寄錢來讓他和老伴養孫，近年景氣差，兒子4月回來後到處問工，好不容易一個月才有三、四天工，一家吃飯都不夠，「我以前也做綁鐵，現在老了雖做不動，但祖孫5人合吃一碗麵也要捱。」

思親哭到掉下床

小偉回憶爸爸生前最愛和他看職棒轉播，父子都是兄弟象隊彭政閔的球迷，「爸爸知道我讀不好書，但他說，一定要正正當當做人，還說努力打棒球一樣能有成就。」10歲的大弟小浩曾邊睡邊哭想爸媽，哭到太傷心掉下床，他和8歲小妹小瑛記得以前爸媽最愛騎車載他們在田野兜風。
小偉的二伯阿發說，兄弟都做綁鐵，如今景氣差，自家又有就學孩子，「照顧是可以，但現在每月沒5天工，時機很歹。」小偉的學校已申辦學產基金協助，當地公所正申辦「馬上關懷」救助金和相關社福補助，蘋果基金會訪視後，也撥款暫紓鄉間老弱生活窘困。
