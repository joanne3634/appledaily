癌末男偕病妻 種菜餬口

女兒打工擔家計 「不會讓爸擔心」

2011年11月25日

47歲的邱顯勝面色蒼白，虛弱地提著水壺幫陽台上幾盆青菜澆水說：「我身體一向健康少生病，怎麼也沒想到今年2月發現胃癌末期，妻子有糖尿病9月又摘除子宮，傷口難癒合也要休養，兩人都沒辦法工作賺錢，只能種點菜勉強餬口。」報導．攝影╱仲芝蓉

邱顯勝說，自己10多年前曾離婚，單親撫養兩名現年20、18歲的女兒，6年前再婚，未再生養子女，「我開計程車在外奔波慣了，雖然三餐不定時，但從沒鬧過胃病，直到今年初突然胃痛，以為脹氣吃胃散，硬撐一個多禮拜。」

除夕凌晨掛急診

邱顯勝40歲的妻子蔡毓芬接著說，除夕那天丈夫凌晨跑完計程車回來，家門都還沒進，就直接掛急診，醫生說胃部腫瘤17公分大已經破了，緊急開刀切掉一半的胃，「化療做了半年多還沒結束，體重也掉20幾公斤，連爬樓梯都喘。」
蔡毓芬繼續說，兩女兒在外租屋半工半讀，分別念大二和大一，自己則在汽車用品廠做計時工，收入都不穩，原先房租和生活開銷就常透支，蔡毓芬嘆了口氣接著說，怎知自己經痛的老毛病突然發作，9月看醫生才知得子宮肌腺瘤，開刀摘除子宮。

流質營品開銷大

公所急難救助承辦員說，醫院已協助邱顯勝辦理健保紓困，並以重大傷病減免醫療開銷，目前邱家僅領有民間補助每月7000元，但他需喝流質營養品每月開銷約3000元，房租也已積欠3萬多元，公所已申請急難救助共2萬2000元，不料他妻子又開刀需休養，再轉介蘋果慈善基金會，基金會訪視後已先撥款紓困。
邱顯勝的姊姊說，以前家境不好4兄弟姊妹學歷都不高，現多靠零工養家，「弟弟兩個女兒從小就懂事，高中開始打工分擔家計。」
18歲的次女小瑩說，自己原想放棄升學去工作，但爸爸堅持自己要上大學，「姊姊成績比較好，我也會努力兼顧課業不讓爸爸擔心。」

我有話要說

邱顯勝說，妻子曾有過一段婚姻，得過憂鬱症差點自殺，6年前開計程車載到妻子，兩人有緣分才會再走入家庭，「當初覺得陪她是我的責任，沒想到我卻生這種病，我們之間特別有默契，常想到同一件事，結婚6年對我們來說不夠，所以無論如何我一定要好起來。」
