母子蜷破厝 苦籌修繕費

臉盆接漏水 「房塌被壓成半死怎辦」

2008年04月18日

時序入春，夜裡突然下起滂沱大雨，85歲的罔市婆（許罔市）才剛躺在床上準備睡覺，只能再起身，拖著蹣跚步伐拿幾個臉盆放在房間、客廳與廚房來接屋頂漏水，她無奈地說：「這房子跟我一樣都是老伙子（台語，意為老人）了，不中用了！」報導．攝影╱韓旭爾

目前罔市婆與67歲未婚的長子阿太同住，阿太原本是排油煙機清洗臨時工，每月掙得一萬多元維持家計，但現年老體衰，加上雙腿患有退化性關節炎，已無法勝任工作，阿太說：「客人看見我這把年紀，還要爬上、爬下拆卸排油煙機，總是會擔心地問：『阿伯，你有法度做嗎？不要勉強喔！』要是我受傷了，會替客人帶來麻煩，現在沒人敢找我清洗了！」

晚上才敢進屋睡覺

因為房子破損嚴重，罔市婆與阿太擔心房子倒塌，每天起床後就到屋外坐著，晚上才會返家睡覺，活在恐懼之中。現在母子兩人的老人年金、中低老人津貼補助每月共9千元，但因阿太之前替同母異父的弟弟作保，現房子已遭設定抵押，每月須繳約7千元的貸款，剩下的2千元就是生活費。阿太說：「弟弟因車禍受傷，雙眼幾乎已失明，哪能負擔債務！我這個做保的大哥只好含淚扛債。」

靠慈善會救濟度日

罔市婆表示，她曾有二段婚姻，阿太是她與第一任丈夫所生，她原本與第二任丈夫同住，十多年前，丈夫因病去世，所生的四子經濟情況不好，只好搬去與未婚的阿太同住。阿太表示，他因工作收入不好，一直沒有結婚打算，「自己都吃不飽了，哪會有小姐要嫁給我！」阿太說，這二年來，全無工作機會，只能靠鄰里與慈善會救濟辛苦度日，沒有錢修屋或搬遷。
對於生活，罔市婆安貧知命，目前苗栗協和慈善會每周一至周五中午皆會為罔市婆母子免費送餐；但對於破舊不堪的屋子，她十分擔心。
罔市婆說：「我活了這麼久，已經夠本了，要是房子塌了把我壓死就算了，要是把我壓成了半死不活，怎麼辦？」目前估價約須工程款16萬元，罔市婆母子無力負擔，蘋果慈善基金會訪視後，已先撥款暫紓生活困難。

無法計算的人生

罔市婆說：「我吃到這把歲數，就只想要一個安穩的窩能終老，沒想到，現在淪落到不管天氣熱、天氣冷都只能在樹下坐著發呆，有家不敢回，真的好感慨！」
