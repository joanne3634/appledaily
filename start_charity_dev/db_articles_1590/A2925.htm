病婦無藥治 嘆「像死刑」

腰椎長瘤難開刀 耗盡積蓄養病育兒

2012年11月18日

「活著就像在受罪。」44歲的單親媽阿燕四肢乾瘦，憔悴的躺在病床上，阿燕說：「我的腰椎長了好多顆小腫瘤，一整排的，只能靠嗎啡減輕疼痛，但嗎啡不就跟鴉片一樣是毒嗎？每天『吸毒』，就像慢性自殺一樣。」報導．攝影╱江品璁

阿燕虛弱的說，2年前腰部抽痛，到醫院檢查才知是腰椎腫瘤，「醫生跟我說治不好，因為神經上長滿像水泡的腫瘤，看起來像一整串葡萄，不能開刀、吃藥治療，試過中西醫都沒效，我說這不就是判我死刑嗎？」阿燕接著說：「醫生說這些腫瘤從小就有，隨著年齡跟著變大，痛起來的時候，常壓迫其他器官，今年9月中，就因此引發腸阻塞住院，打了太多嗎啡，導致器官衰竭昏迷了2天。」

每月剩1900度日

阿燕說，生病前有買醫療險，前年發病，還可到餐廳做端盤子零工，並靠著理賠金和積蓄約20萬度日，「但今年9月身體愈來愈差，現在連碗麵都端不了，保險金和積蓄也用盡，只靠政府兒少補助每月1900生活，但房貸就要7千元。」麵店同事阿敏說，知道阿燕身體不好，之前就不會勉強她做太粗重工作，現在她什麼都不能做了，希望她的病能趕快好轉。

子遭霸凌懼上學

阿燕難過的說，14歲的兒子小鴻有學習障礙，今年4月份，他在學校被霸凌，學校通知家長到校，「我因為生病精神不好，也搞不清楚狀況，當場就跟對方父母和解，1個月後我看到監視器畫面，兒子被人賞耳光，又捶又打，心都碎了。」
一旁的小鴻想起遭霸凌過程，邊流淚邊開口說：「我沒朋友，想交朋友，跟同學借錢請客，但沒錢還，隔壁班同學就打我，打了半個多月，但我只是想交朋友而已。」小鴻王姓班導師說，小鴻個性內向，被霸凌後畏懼上學，現在每周做2次心理輔導，打人的同學也已記過懲處。
阿燕說，與前夫共育1女1子，6年前因家暴離婚，大女兒監護權歸前夫，小鴻由自己扶養。22歲的女兒小靜說，平常很少與媽媽聯絡，「這次因為收到媽媽病危通知，才知道她的病況，但我在餐廳打工每月賺1萬8，離婚生1子，只拿得出4千元應急。」阿燕大哥阿堂則說，自己兒子也殘障，知道阿燕生病，有拿3千多元協助，其他3個弟弟都各有家庭，收入不好難多幫忙。當地公所說，阿燕的腰椎腫瘤不符殘障鑑定，且大女兒已成年，無法通過低收資格。蘋果基金會獲悉訪視後已先撥款紓困。
