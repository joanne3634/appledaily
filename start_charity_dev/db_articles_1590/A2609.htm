女險癱 殘父：願代受病痛

窩住廢工寮 「盼讓她快樂」

2011年11月04日

44歲的古新德四肢關節因痛風嚴重變形，無奈嘆道，和17歲的女兒小華只能寄人籬下，小華兩年前又心臟病發差點中風癱瘓，「我沒辦法給她完整的家，但願能代替承受病痛，讓小華至少擁有健康和快樂。」報導．攝影╱仲芝蓉

古新德因關節變形，四肢不甚靈活，他說，大概9年前發病，剛開始只是小硬塊，不影響行動，還能做模板、砍草等零工，但4年前右手和右腳各截1指，之後病況加劇惡化，「腳背和腳跟也長出痛風石，之前勉強穿最大號的雨鞋去工作，但後來什麼鞋都穿不住，怎麼去工作。」

同學慶生給鼓勵

古新德回述，和前妻生有分別18、17和15歲的2女1子，家計靠自己在水泥廠打零工加上低收入戶補助，7年前貸款整修老舊祖宅，但3、4年前痛風越來越嚴重，工作漸漸不穩，積欠房貸房子遭查封，和妻子更常因經濟問題產生口角，「今年初房子被法拍，和妻子也離了婚，長女和么子跟妻子，小華身體不好，願意體諒我的苦處，跟著借住親戚家的廢棄工寮。」小華接著說：「媽媽工作也不穩定，帶姊姊和弟弟在外面租房子，萬一我生病怕會拖累他們，所以我跟爸爸，不然也沒人照顧他。」
小華說：「今年暑假我生日時，同學知道我們家被賣掉，不得不搬家，十幾位同學來家裡幫我慶生，我知道她們是想鼓勵我。」

就醫交通開銷重

小華的學校護理師說，小華兩年前在學校昏倒，才發現罹患大動脈炎症候群，病因為不明血管發炎物質導致，但她身形單薄，已加強健康飲食教育，「兩年來陸續開刀裝了4個支架，耗材開銷已由瑞信兒童醫療基金會和心臟病友協會全額減免，學生平安保險也陸續理賠15萬元。」護理師說，但因她家居住偏遠深山，每月回診坐火車交通開銷約2000元，今年8月左臂支架肌肉發炎住院，醫療開銷由醫院社工協助減免，但坐救護車單趟要5600元，就醫交通開銷沉重。
學校導師說：「學校知道她家境困難，已申請教育部學產基金1萬元協助。」公所低收入戶承辦員說，小華因留級重讀，僅能以中低收入戶減免學費開銷，另申請公所急難金減免雜費和餐費，古新德2兄2姊1弟都已成家各有負擔，故轉介蘋果慈善基金會，基金會訪視後已撥款暫紓困境。
