春節前喪夫 跛婦養4老小

5歲女親吻遺照 夢中喊「要找爸爸」

2010年02月24日

39歲的巴梅芳想起38歲的丈夫阿明經過1年多痛苦抗癌療程，最後還是不敵病魔在1月中去世，不禁哭紅了眼，巴梅芳說，想不到夫妻的緣分只有12年，「家裡沒了男主人，這個新年過得好孤單！」報導．攝影╱韓旭爾

巴梅芳說，丈夫生前靠著建築綁鐵零工粗活，每月約2萬元薪水養家，自己也擔任行政助理每月掙得1萬多元貼補家用，日子還算過得去，但1年多前，阿明開始感覺吞嚥困難、咳嗽不止，「當時阿明也不在意，還是隨著工地包商四處工作，這一拖就是半年。」巴梅芳回憶說，後來阿明痛到連水都沒法喝，才到大醫院檢查，確定罹患第3期食道癌，雖立即進行化療及放射性治療，但病情還是無法控制，去年底，癌細胞已轉移到胃部及肝臟，最後回天乏術，巴梅芳無奈地說：「阿明雖身體不適，但為了養家活口，硬是咬牙苦撐，結果把自己命都賠掉了。」
新春期間，看到左鄰右舍舉家出遊，巴梅芳更是感傷，她說，以往過年時，雖然家境不富裕，阿明還是會帶著她與現年9歲及5歲的子女到遊樂場遊玩，讓孩子們有個歡樂年節，5歲的女兒小珈說：「爸爸以前常帶我到公園溜滑梯、放風箏，我好想爸爸喔！」巴梅芳說，現在阿明走了，小珈常拿起爸爸的遺照嘟起嘴巴親一下，晚上睡覺作夢時，也會喊著要找爸爸。

車禍腳傷仍未好

巴梅英感嘆說，家裡災厄一件接著一件，去年10月時，她前往醫院探視丈夫路上，竟發生車禍，導致左腿脛骨開放性骨折，現傷口尚未痊癒，仍走路微跛，「當時為了照顧丈夫，沒法按時回診，車禍都過了這麼久，傷口還是紅腫流膿，也不知還要多久才會好？」而阿明病逝後，帶給巴梅英的除了喪夫之痛外，現在還得為了喪葬費用發愁，「雖國民年金可補助喪葬費用8萬多元，但還缺少7萬多元，只能先向親友借，將來找到工作再慢慢還。」蘋果基金會訪視後，已撥款協助部分喪葬欠費。

母曾中風嬤痴呆

然而更讓巴梅芳擔心的是家裡媽媽及外婆兩位老人家的健康情況，巴梅英說，63歲的媽媽6年前曾發生中風，現須使用拐杖才能走路，而外婆已年屆90，除了有輕微老人痴呆外，還有嚴重駝背、高血壓，巴梅英說：「兩人三不五時就到醫院報到，包車每次就要1千多元，負擔實在很大。」
當地村長賴立誠說，現巴梅芳一家5口就靠政府每月補助1萬4400元度日，因巴梅芳的腿傷未癒，在山區部落覓職困難。巴梅芳說，現在只希望自己的腿傷快癒，「之後找份穩定工作，再怎麼辛苦，也要努力撫養小孩長大。」
