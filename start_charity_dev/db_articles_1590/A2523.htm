癌男養癱兄 尚有3子待哺

老母四處問神 「求多些時間」

2011年07月21日

45歲楊鑫難過地說，去年中發作鼻咽癌，今年初癌瘤轉移壓迫腦部致視力眼花昏茫，自家3子年幼，家中8旬老母得顧胰臟癌又中風的二哥，雖經里幹事協助於6月通過低收資格，但每月社福補助共1萬多元，難敷一家7口老病弱開銷。蘋果基金會訪視後，已撥款暫紓困境。報導 攝影／張嘉恬

楊鑫說，和28歲妻子阿欣育有現各4、3及1歲幼子，罹癌前家計靠他水泥工維持。楊鑫的10多年吳姓同事說，楊鑫工作認真盡責，為妻小常加班拚命賺錢，在他病後更感嘆曾說：「這輩子沒機會有大成就，沒能力給妻兒過好生活，現在中年罹癌，只祈求多些時間陪家人。」

老母願用命換癌兒

「如果命運能改，希望用我老命換兒子的命。」楊鑫的8旬母阿專婆老淚縱橫說，二兒子多年前患胰臟癌又中風癱瘓，平日由她照顧，但未料先是三兒子楊鑫罹鼻咽癌，緊接在外生活的小兒子近日也患口腔癌。

四兄弟3人均罹癌

阿專婆說：「求神、問神，心都不安，只有幫媳婦照顧孫子時候，才暫時不操煩。」阿專婆的4子中有3子已罹癌，剩長子在外靠打零工養活自己，她說，現在最讓她掛心的是，楊鑫還有3個幼子，想到就「心肝就不甘。」（台語：心中很不捨）
「下輩子再做夫妻」
楊鑫妻子阿欣說：「不後悔嫁給他！」她說，與先生6年前透過網路認識，從香港嫁來台灣，婚後雖生計緊絀，又遇夫罹癌，但「他處處替我設想，讓我不覺得苦，他說下輩子還要再做夫妻，照顧我們。」
得知先生罹癌時，阿欣已懷么兒4月，看到先生診斷書寫「鼻咽癌」，腦海只浮現「他千萬不要死掉！」阿欣最感無力的是「不能替先生承受癌痛，不能幫他什麼。」阿欣看著3個嗷嗷待哺幼子，她拭淚勇敢地說：「好好把孩子扶養長大，是我最重要的任務。」
