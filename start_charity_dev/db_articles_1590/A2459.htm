癌男灌食 掛心7口家計

「我沒去工作 這個家不知道怎麼過」

2011年05月04日

37歲的阿娥（潘素娥）看著罹患口咽癌3期的先生阿全（陳義全）說：「癌細胞蔓延了整個口腔與喉嚨，先生幾乎無法說話與吞嚥，只能靠鼻胃管灌食營養品，他卻仍掛心要趕緊工作，我勸他為了家人要專心養病。」報導．攝影╱郭恩新

記者訪視時，42歲的阿全臉頰與脖子均被電得焦黑，他挺起僅49公斤的瘦弱身子，咿咿嗚嗚地說著話，說了幾遍記者還是不明白，透過一旁的妻子阿娥解釋才知道，阿全說：「沒有我去工作，這個家不知道要怎麼過。」阿娥說：「先生10月初便發現身體不適、吞嚥有困難，因一直忙於工作而沒有立即就醫，還硬撐了一個多月，沒想到11月時就醫檢查，居然發現得了癌症。」

月掙萬元不敷開銷

阿娥回述，自己與先生育有3子女，並與70歲重度聽障的公公與90歲的祖母一家7口同住，夫妻倆就以水泥工賺取4萬元維持生活。17歲的長女小米原就讀餐飲科，但去年3月因不適應實習環境休學，現在家照顧先生，16歲長子小吉就讀高一夜間部，白天在製香工廠當學徒，么子小誠就讀國二。阿娥說，雖有小米幫忙照顧先生，但自己卻因常要帶先生去醫院電、化療，一個月做不到10天工，現一家就靠她與長子每月僅掙約1萬6千元，不敷先生的醫療管灌品開銷。
阿娥說，阿全是獨子，雖還有1姊1妹，但大姑嫁人後少有聯絡，且本身也有家庭重擔，婆婆於2年前中風後，便由單身的小姑照顧，難有餘力協助。「得知先生罹癌時，我只能鼓勵他樂觀抗癌，只要配合治療都有希望，但面對家裡的生活困境，實在不知道該怎麼辦。」阿娥苦惱的說。

么子憂「爸變好瘦」

么子小誠的班導胡老師說，小誠在學校表現乖巧，對老師所交代的事總認真負責，看見小誠在聯絡簿上寫道：「爸爸變好瘦」、「爸媽都不在家（住院）」等字句，令人鼻酸。小誠還曾請假一周到醫院幫忙照顧爸爸，但他在學校仍把自己的擔心藏在心裡，還是幫同學帶來歡笑，大家封他為班上的吉祥物。小誠的學費已透過家長慈善會支付，教育部學產基金急難金也已申請中。

「為養家工作勤奮」

當地村長陳國誠說，阿全、阿娥夫妻倆平時為人憨直，為養一家老小在工作上總勤奮努力。公所去年12月已核撥「馬上關懷」2萬元急難金協力，慈濟於2月核撥5000元急難金相助，蘋果基金會訪視後也已提撥急難救助金暫紓阿全一家醫病、生活困境。
