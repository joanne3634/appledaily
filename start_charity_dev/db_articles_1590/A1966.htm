一家3罹癌 弟逝兄苦撐

吸口氣越來越難 獨居男「要再拼」

2009年08月31日

採訪時，患咽喉癌末期的楊勝一突然不停咳嗽，血痰從氣切口噴出，他脹紅臉幾乎喘不過氣，護理人員趕緊急救，幫他抽痰、穩住呼吸，約20分鐘後逐漸回復，他指頸部說，「氣管被血塊卡住了。」報導 攝影╱向高彬

吸口氣對病況日益嚴重的楊勝一來說，越來越難，46歲的他發病至今1年半，去年10月腫瘤壓迫頸部、靠近聲帶，手術氣切後，他現須從腹部胃造瘻管灌營養品，說話費力，發出夾帶「嘶…嘶…」的模糊語聲，多數他透過點頭及手勢表達，很多事得透過一旁的外甥女阿鳳協助表達。

顧雙胞胎弟耗盡積蓄

病床上楊勝一瘦如皮包骨，他表示，去年2月吞嚥困難，一檢查竟是咽喉癌，他相當震驚，因他當時正在照顧早他一年發病，同樣患咽喉癌的雙胞胎弟弟楊勝二，他說，單身弟弟前年發病後由他照料，沒想到自己也發病。楊勝一說，弟弟已於去年10月底病逝，而他自己現也情況不樂觀。
楊勝一的媽媽早年因肺癌病逝，醫院李姓社工說，楊家可能有癌症遺傳，雙胞胎兩兄弟同樣發生病變，「先後服務兩位兄弟病友，讓我很同情，他獨居老家生活困難，這次住院前沒錢喝營養品，病餓了2天，打電話求助，我趕緊要他來住院，醫院協助部分醫費，我也自掏腰包提供點營養品。」李姓社工說，目前楊勝一每月領輕度殘津3000元，但平常營養品約6000元，還有紗布等醫療耗材及回診車資開銷，難敷生活所需。蘋果基金會訪視後，先撥款暫紓楊勝一急窘。
阿鳳代舅舅楊勝一回述，從小與雙胞胎弟弟長相相似、感情也好，一起上學，連長大後工作也都以開卡車為業，在外也租屋同住，不過早先兩人對人生沒規劃，總賺多少花多少，也沒想安定結婚生子，還想反正老了，有兩兄弟相互照顧就好，但萬萬沒想到，癌病先奪走弟弟一命，當時楊勝一見弟弟病苦，才醒悟凡人生命有限，而為負擔弟弟醫護開銷，用罄他約10萬元積蓄。

有最壞打算但不認輸

楊勝一病後至今，親友生活也都不好過，幫助有限。阿鳳說，舅舅6年前曾因工作不順缺錢，而與一大陸女子非法假結婚，但該女4年前已遭逮遣返，卻因此女仍在戶籍內影響低收申辦，「已透過法院和警方協助，9月中再申覆辦理離婚，若可，再申辦低收資格。」當地鄉公所已撥「馬上關懷」救助金1萬元協助醫病。採訪時，楊勝一費力地說：「遇到了，只能面對。」他說雖已有最壞打算，但絕對不就此認輸，「也許還有贏的機會，要再拼一拼。」
