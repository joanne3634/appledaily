洗腎父截肢 貧兒求學坎坷

5口家計無著 兄：不知要找多久才有工作

2008年10月05日

16歲小星今年國中剛畢業，因父親洗腎又截肢，家貧讓小星求學路受阻，蘋果基金會訪視後，已先撥款助小星重返校園；但現家中由大哥一人扛五口生活，又要照顧病父，正急覓大夜班工作。報導 攝影╱向高彬

小星家住台東鄉間一處簡陋平房，57歲爸爸耿田興坐輪椅，他左小腿於8月中動完截肢手術，傷口仍裹紗布，小星說，爸爸預計兩個月後裝義肢，又是一筆開銷。耿田興感嘆，讓孩子受苦，「要趕緊復健，不要拖累他們。」

「擔心下一餐在哪」

小星說，爸爸原從事看護工作，但9年前嚴重糖尿病無力工作，8年前洗腎，媽媽外遇離家，大哥阿明國二時中輟做工養家；現年22歲的大哥2年前結婚生下1女，但大嫂離家，無人可顧小女兒，「於是17歲的姊姊小怡去幫哥哥顧女兒，家裡剩我顧老爸，可是現在爸爸身體太差，我顧不來，所以他們都回家來幫忙。」
大哥阿明在8月初與前妻離婚，他說，前妻回來時，挺著與外面男友有的五個月身孕，只好離婚，「貧賤夫妻百事哀，媽媽和前妻都這樣，很無奈。」正是不景氣時節，阿明回台東發現工作難覓，採訪時他說，身上剩不到一千元，卻不知還得多少天才有工可作。而為兼顧爸爸每周三天洗腎復健和照料幼女，他正覓職大夜班工作，「不怕吃苦，把家裡顧好最重要。」
小星說，他常像小媽媽一樣煩家裡，「擔心下一餐在哪裡？還好今天教會送米，鄰居阿婆送絲瓜、高麗菜，晚餐有著落了。」

少上課成績仍優異

曾任小星國中班導師的陳莉平說，小星有責任感、抗壓性好，「做飯、料理家務，還照顧爸爸住院，有時一周只上課兩、三天，但成績仍名列前茅，是很乖的孩子。」
當地鄉公所社會課許小姐說，阿明一家五口正值多事之秋，他單親育幼女又為父病扛家計，「我們正申辦急難救助，10月後由村幹事協辦低收，希望幫忙讓家裡生活慢慢穩定下來。」

後記
「想消失在地表」令人心酸

小星家計窘迫，洗冷水澡省瓦斯，住家鄰近草叢多，不時有眼鏡蛇、南蛇報到，小星和阿明練就抓蛇好手藝，「算牠們倒楣，來讓我們家加菜補一補。」
又因小星自己正在發育中、食量大，他自嘲說：「我常希望自己消失在地球表面好了，不要吃家裡那麼多飯。」「消失在地球表面」，這句對白在電影情節中聽來趣味，但從弱勢家庭孩子口中說出，讓記者聽得真是心酸。
