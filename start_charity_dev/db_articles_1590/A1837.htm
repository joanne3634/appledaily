顧腦麻雙胞胎 父辭職照料

6歲兄弟腳裹石膏痛到哭 「想走路那樣難」

2009年03月29日

走進25歲泰雅族人田雅玲的家裡，見剛動完腳部矯正攣縮手術的6歲腦麻雙胞胎兄弟文文和豪豪，兩人的雙腳緊裹石膏、趴臥在地，他們因疼痛不時大聲哭號，田雅玲心疼地說：「為了他們的未來，只能咬牙熬過眼前的痛苦。」報導 攝影╱游竹君

田雅玲與31歲的丈夫阿昌（孫正昌）育有3名子女，7歲長女小愉讀幼稚園大班，腦麻雙胞胎兄弟則在智能發展中心接受早療，阿昌原在外地擔任消防管線工廠作業員，為了幫忙照顧孩子辭工回家，靠工地及山間零工維持家計。

出生時腦缺氧致肢障

田雅玲說，還沒開刀前哥哥文文要靠助行器才能墊著腳尖走個幾步，弟弟豪豪雙腳萎縮狀況更嚴重，只能靠站立架才能支撐身體。兄弟倆看到姊姊行動自如地跑跳，便會拖著身體在後面費力爬追玩耍，哥哥有時也會生氣地哭嚷著：「為什麼我不能走路？我要學走路！」這話讓媽媽心疼落淚：「我從沒想過孩子想走路是那樣難與辛苦！」
田雅玲說，當年懷有三胞胎，4個月後發現有一個胚胎是死胎，因此懷孕7個月就剖腹產，文文和豪豪出生時才900多公克，她說，文文、豪豪快1歲還不會翻身講話，「我以為是早產兒發展比較慢，直到1歲多找醫生檢查，才發現是出生時腦部缺氧導致肢體障礙，當時聽到醫生診斷兩個孩子都是腦麻兒時，我根本不敢多想未來的日子，直到兩個孩子接受早療後，才發現有機會改變。」

復健做不好恐更嚴重

今年2月中旬，在專業建議下，兩兄弟同時接受矯正肢體攣縮手術，在雙腳膝蓋及踝關節後方動刀，再以石膏固定，後續還須輔以大量復健訓練雙腳功能。早療中心的李社工說，兩兄弟語言及學習能力不錯，但因肢體障礙經早療仍無法有效改善，只好接受手術矯正，「剛開完刀的疼痛對孩子和家長都是巨大的煎熬，但術後復健若做得不好，反而會比開刀前攣縮更嚴重，所以我們也叮嚀家長要積極復健、絕不能輕忽。」
田雅玲家有三款低收身分，兩兄弟領有的殘障津貼轉入早療中心作為學費，每月僅領有長女兒少補助2200元，里長彭誠吉將協助申請「馬上關懷」急難金應急，阿昌原以零工每月約2萬多元勉強維持一家五口開銷，接下來他們得帶兩兄弟奔波醫院接受復健，很難做零工，蘋果基金會訪視後已撥款協助。

後記 盼９月順利讀小學

訪視的3個多小時期間，雙腳裹石膏的兩兄弟只能趴臥或仰躺，稍一移動便疼痛地放聲號哭，只見田雅玲和丈夫阿昌兩人雙手不曾停下輕撫哄騙，阿昌面露疲態地說：「醫生說，尤其拆石膏後的復健期是關鍵，一點都不能馬虎，我們已經安排3間醫院每天接送復健，希望9月時孩子們可以順利讀小學。」在田雅玲夫妻心中，大人生活的苦可以捱，但兩兄弟可以走路的希望卻一刻都不能耽誤。
