夫猝逝翁癱子幼 婦扛重擔

7歲兒慰母：爸爸變星星保護我們

2010年05月27日

初夏天際繁星點點，7歲小元拉著3歲小妹，他指最亮一顆，「爸爸在那裡，他變成星星保護我們。」媽媽林心瑜聽孩子童言越聽越心酸，也知道往後得肩扛照料兩子和中風公公重擔，她默禱先生多保佑。報導．攝影╱向高彬

剛喪偶、36歲的林心瑜仍難掩悲慟，40歲的先生阿朋走得太突然，身後留下兩幼子和77歲重癱臥床的公公，讓她愁煩往後，她說，阿朋4月中傍晚下工返家途中，騎車不慎自撞路樹猝逝，「驗出他車禍前曾喝酒，可能他壓力大，兼做兩份工作，酒駕又精神不濟才出事。只是他來不及交代半句就走了，我一直很遺憾。」

丈夫生前兼差扛家

林心瑜說，先生後事靠鄰里親友和地方慈善救助辦妥，「幸好有先生生前好友張羅，很謝謝他們。」由於阿朋生前經營汽車鈑金生意不善，積欠上百萬元債務，林心瑜說，正辦債務拋棄繼承，但她仍有之前為生計而借的卡債約20萬元，如今她要顧幼女及公公，平日尿布及營養品費用開銷重，一時難工作，只能等待幼女長大些、安頓好公公後，再外出工作撐家計。
當地賴姓里幹事說，林心瑜家現為三款低收戶，每月共領補助8400元，正協辦調為二款低收，每月可多5000元補助，里幹事說：「阿朋生前對爸爸非常疼顧，鄰里都誇讚是孝子，壯年早逝，大家都惋惜。」蘋果基金會訪視後，先撥款暫紓生活窘迫。
林心瑜回述，她與先生讀夜校高職相識，交往時就知他家境不好，又是獨子，公公已中風17年，加上已歿6年的婆婆生前多病，多年來他一人一肩扛起家計。她欣賞先生肯打拼，9年前兩人結婚，生下子女，只是3年前生意虧損扛債，之後阿朋白天做鈑金工、晚上兼差送貨至午夜，盡力償債掙錢養家，但盼快把家境改善，無奈天不從人願。

娘家心疼盼女堅強

林心瑜的娘家媽媽也心疼女兒，但自家老伴中風體弱，只能「安慰女兒為孩子要快堅強起來。」

後記
「媽媽別哭 我會孝順爺爺」

阿朋家是間約20坪老瓦房，記者見阿朋的爸爸手腳蜷縮一團，只會嗚嗚瞪眼，採訪時，正是阿朋出殯前一天，大兒子小元模樣聰慧，他幫爺爺灌牛奶，還說長大後要孝順爺爺；又見媽媽哽泣，安慰媽媽別哭。我相信，小元遺傳了他父親的良善，雖然離他長大仍有時日，但相信天上父愛如星照耀，會保佑孩子平安長大。
