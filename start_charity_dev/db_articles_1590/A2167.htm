嬤搬百斤蛤仔 養癱兒幼孫

年邁身軀苦奔波「不知撐多久」

2010年05月09日

70歲玉華嬤（陳玉華）望著病榻上中風半身偏癱無力、失語的獨子阿義（丁義崇），不禁眼眶泛淚：「一輩子為兒孫勞碌，沒想到老來不僅要幫兒子把屎把尿，這把老骨頭不知還能撐多久。」報導．攝影╱游竹君

病房裡，44歲腦中風的阿義，咿咿啊啊發出難以成句的聲音，見人無法理解他的意思而焦急地脹紅臉。玉華嬷難過地說，2月初早上，阿義在家腦中風倒地不起後，意識時而清楚、時而模糊，「只有見到他兒子小豪時，他才會開心地笑咧了嘴，但沒多久又哭。」

自幼歹命做童養媳

玉華嬷因須在醫院照顧阿義的生活起居，阿義的兒子小豪只能暫託老家親戚照料，3個多月以來，玉華嬷總騎著摩托車往返相距30、40公里的醫院和老家，年邁身軀滿是疲憊。
玉華嬷育有一子一女，女兒早年遠嫁他處，獨子阿義12年前與印尼女子結婚4年後隨即離異，但因妻子離家打擊過大，阿義常借酒澆愁，就讀小四的孫子小豪自幼與玉華嬷相依生活。
小豪說，阿嬤很辛苦，不僅要種稻，有時還要很早就出門幫人洗蛤仔，「我想快點長大，幫阿嬤賺錢。」玉華嬷說，自幼就被收養做童養媳，從6歲開始養鵝、養牛，「我從不怨老天爺，因為知道是自己歹命；現在照顧癱兒幼孫，一把老骨頭，還要去做搬著一百多斤蛤仔的辛苦差事，為了生活，還是得咬牙做。」

醫院助兩萬看護費

醫院社工表示，因阿義已領有重度殘障手冊，但他體型魁梧，玉華嬷實在難勝任，因此院方協助支付10天共2萬元的看護費用，以減輕玉華嬷的負擔，但往後還有漫長復健路要走。村幹事廖錦男表示，玉華嬷家目前領有老農及單親兒少補助，將協辦低收及「馬上關懷急」急難金應急。蘋果基金會訪視後，也已撥款暫紓玉華嬤困境。

後記
孫真情字條 嬤滿眼驚喜

記者前往探訪暫住親戚家的小豪，告知小豪將到醫院探視阿嬤和爸爸，問他想不想寫些話給他們，只見他望著白紙搔頭思考許久，才以注音夾雜國字寫下：「爸爸你要加油，我會好好讀書」及給阿嬤的「母親節快樂，你要快點回家。」（左圖）
當記者將字條拿給玉華嬤時，不識字的玉華嬤聽記者念出小豪的字條，眼裡滿是驚喜終於露出笑容，讓她暫時忘卻眼前無情病痛的磨難。
