父猝逝 國三女顧癌母3妹

常請假陪回診 5口擠小通舖 「很心酸」

2009年11月25日

「媽媽很不舒服，今天我請假顧她。」讀國三的小敏跟3個妹妹說。小敏的媽媽張秀英患鼻咽癌7年，她跟大妹輪流每周請假兩、三天顧母已成常態，原本靠爸爸阿誠在外做拼模板工勉撐家計，未料爸爸卻在10月中心肌梗塞猝逝。報導 攝影╱向高彬

到訪時，享年僅41歲的阿誠剛出殯第2天，記者來到偏遠山間的小敏家，阿誠神主牌桌前香煙裊裊，38歲媽媽張秀英長期癌病變已右眼失明、雙耳重聽，說話含混聽不太懂，她慘哭泣不成聲，模糊語聲告訴記者：「孩子沒爸爸了！」

醫病家用已積欠40萬

小敏說，爸爸原做農雜工，但山間工太少，又因媽媽患三期鼻咽癌，長期抗癌醫病經濟壓力大，爸爸只得常年出外打拼做模板工，扣掉外地房租，省吃儉用全寄回家用，但爸爸工活不穩、媽媽病況惡時也得請假返鄉處理，微薄收入不敷醫病家用，這幾年積欠40多萬元債務。
小敏說，爸爸身高181公分、100公斤重，在她們心裡像家鄉高山般雄壯，而爸爸出事前一天才返家過，還跟她叮嚀好好照顧媽媽和妹妹，沒想到隔天下午在校上課時，二阿姨慌張到校通知爸爸猝逝，小敏和妹妹全嚇傻直哭，「爸爸同事說，事發中午他才吃過飯，突然一聲不響癱軟，半點徵兆都沒有，救護車來不及到醫院，他半路上就斷氣了。」
阿誠後事經農保理賠、鄰里親友和公所急難救助辦妥，當地家婦中心葉姓社工已介入關懷，正協辦低收，她感嘆癌母與4女往後生活漫長，「小敏常請假陪媽媽回診，交通來回近1000元；老家屋小，女兒和媽媽5人擠睡一小房通舖，很心酸。」蘋果基金會訪視後，已以「視障專戶」急難金暫紓癌母弱女生活窘困。

「想讀護校照顧媽媽」

如今小敏扛起父職照料家人，她說別無選擇只能逆來順受。但眼看困境，她曾想明年畢業不再升學，但師長都要她別放棄學業，「和大妹常請假，功課跟不太上，進教室看到同學會心虛自卑，但能讀書又覺得開心，爸爸生前也說要多念書，以後才有前途幫家裡。」讀國一的大妹小香說，想讀護校，可照顧媽媽也有理想工作。
小敏的二阿姨阿幸說，她嘆彷彿家人受魔咒，父親肺癌、母親胃癌均病逝，未料妹妹秀英也逃不過癌魔。原本對秀英惡病已有心理準備，但想孩子還有爸爸照顧，無奈竟是健壯的人先走，「孩子很不幸，想到她們以後，我也好煩惱。」

暖流見證
提早扛家務師不捨

小敏家生計長期困苦，未料父猝逝，學校已撥教育儲蓄專戶急難金5萬元協力後事。小敏很懂事，只是這年紀就被迫提早成熟承擔家務，很辛苦也讓人疼惜。
