癌末父抱病養家 癱躺入院

妻小伴身旁 女兒童語：快好起來 一起回家

2008年01月08日

黃添送罹患食道癌末期，全身疼痛難耐躺在床上，中國妻子阿鳳和兩幼子自去年3月起，便隨他多次化療進出醫院，幾乎以院為家。阿鳳將他扶坐起身時，聽到圍繞身邊的兒女要爸爸：「快好起來一起回家！」她再也止不住淚水。報導．攝影／張嘉恬

阿鳳語帶哽咽道，43歲的黃添送一直有胃潰瘍宿疾，前年9月因身體不適就醫時尚未發現腫瘤，直至去年2月產生吞嚥困難做切片檢查，未料才發現癌細胞卻已是末期。

「只想好好陪阿添」
32歲的阿鳳孤單無援，為長期斷炊的生計發愁。阿鳳說，她和阿添婚後共育兩個現年分別4歲、3歲的子女，並和阿添第一次婚姻的20歲長子同住，但阿添長子做計時工維生，收入僅足維持自己生計，因此她和兩幼子生活，均仰賴阿添每月開拖車約3萬元收入。「阿添倒下後沒任何積蓄，住院靠每日勞保給付約300元應付三餐，我只好向朋友借生活費，已欠5萬多元了。」
阿鳳憶起和阿添相識過程，「我和他不是靠婚姻仲介結婚，是透過中國的台灣朋友介紹。」阿添在某次赴中國觀光與阿鳳相識，此後密集往來兩岸追求阿鳳，「我是被他的誠意打動了。」阿鳳無奈說，阿添久病近一年來，外人對她能堅守婚姻多久，多抱持懷疑，她豁達地說：「現在只想好好陪阿添走下去，往後還有兩個孩子要拉拔，哪有心情想這麼多？」

公所將協助辦低收

阿添的三姊說，阿添很顧家，未罹癌前每日工作逾12小時，去年中阿添病情一度好轉，「但他說不工作不行，不然妻小要吃什麼？又抱病工作一陣子，沒想到後來愈來愈嚴重。」
採訪時阿添正想步行至廁所小解，但因全身僵硬，不到3公尺的來回距離，他緩步一趟就氣喘吁吁，這時天真的小女兒突然說：「我愛ㄅㄚˇ ㄅㄚˊ，ㄅㄚˇ ㄅㄚˊ生病快點好。」阿添聽到後奮力答了一聲「好！」雖然女兒的童語帶來力量，他卻難撐虛弱的身體而癱躺床上。
里幹事陳其琛說，因阿添長子將入伍，全家無收入來源，公所將協助阿鳳辦急難救助金、低收資格，阿鳳在取得本地永久居留權後，也會盡力協助覓職。
