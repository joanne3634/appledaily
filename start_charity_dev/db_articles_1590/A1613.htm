嬤智障 母罹癌 兒遲緩

苦命三代 誰助他們活下去

2008年06月24日

「馬麻，你的手痠痛，今天我煮飯給你吃。」11歲的阿弘洗米煮好飯，正準備打蛋下鍋，不小心卻讓蛋白流了一地。「他很貼心，只是他有學習遲緩，有些東西就算不斷重複，還是會出錯。」卓玟慧說，罹癌三年來，最放心不下就是這個孩子。報導 攝影╱孫敏聿

今年43歲的卓玟慧憶起12年前與男友分手後，發現懷孕，在去留之間掙扎，「那時我拿成衣回家車縫，每月收入近3萬元，加上哥哥離婚也未育子女，我擔心卓家無後，牙一咬，決心生下阿弘。」

硬撐病體打零工

卓玟慧說，父親在她國小時車禍早逝，母親患有中度智障，哥哥易暴怒、不時失蹤，她國中畢業後四處打零工，之後學會車縫手藝，在家車成衣。「我和男友交往數年，一談到結婚生子，他就擔心孩子會遺傳我媽的智障，不讓我生，我才驚覺他的嫌棄，而我無法拋下這個家，只好分手。」
玟慧再怎麼堅強，也擋不住接連的打擊。四年前阿弘剛上國小，就被老師發現學習遲緩，至今仍每周就讀部分資源班課程。三年前玟慧發現罹患乳癌二期，切除右乳、淋巴後，進行長達一年的化療與電療，還沒復原，兩年前哥哥又發生車禍，住院多日，才鑑定出有輕度智障，解答了多年來哥哥暴怒的奇異行為。
「我發現癌症後，領有保險理賠46萬元，但免疫牛奶等營養品的補充，就花了10多萬元，其餘的用在三年來的生活與醫療，早已用盡。」去年初，她苦於沒有收入，硬撐著病體到工廠包裝煞車，卻因淋巴切除後的右手不堪負重，三個月就被迫離職。

村里送物資紓困

近半年來好心鄰人王太太，不定時僱請她幫忙包裝竹筍、顧巡灌溉用水，半天薪水500元，王太太說：「玟慧生病後還要照顧媽媽、兒子，才決定僱她，玟慧很肯吃苦。」
卓玟慧一家四口現領有殘障津貼與單親兒少共8400元，村幹事說，因過去有保險金，需至今年10月才能提出低收申請，「玟慧家裡很苦，村辦公室常送米等物資，多少幫忙，明年低收若通過，四口補助即有1萬2800元。」蘋果基金會獲悉，先行撥款暫紓目前生活急難。
