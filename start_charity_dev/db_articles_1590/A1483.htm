翁顧憨兒女 5口擠陋屋

沒有廁所自來水 洗澡還得燒柴

2008年01月01日

訪問阿勝伯時，剛好鋒面來襲，氣溫驟降，台南縣郊山區下起豪雨，雨水傾盆而下，他簡陋的屋子在大雨中顯得搖搖欲墜，陣陣寒風從破裂的石綿板牆灌進，更讓一家五口冷得整夜無法入睡。報導．攝影╱韓旭爾

現年59歲的林文勝（阿勝伯）育有3子1女，妻子在4年前去世，34歲長子與18歲女兒各為中度與輕度智障，阿勝伯說，33歲次子腦筋也不好，只是沒法通過殘障手冊鑑定，右腳不良於行，走路微跛，僅31歲三子較正常，還能打零工。這幾年來，他與四名子女同住山上農舍，靠打零工與拾荒，一家五口清苦度日。

農忙打零工農閒拾荒

一想到子女多為智障，阿勝伯無奈表示：「真想不透，我跟老婆腦筋都正常，為何會生出3個憨憨的孩子？現在我還能看顧他們，要是將來我老了，他們要怎麼活下去？」目前阿勝伯只能拖著老邁的身子四處打零工，維持一家生計，他說：「農忙有工活可做，平均每月可收入1萬多元，但農閒時，只能撿些鐵鋁罐、寶特瓶賣，月收只有4、5千元。」
對於生活，阿勝伯安貧知命，但是對於破舊不堪的屋子，他擔心不已。阿勝伯的屋子十分陳舊、簡陋，僅為搭在果園中的鐵皮農舍，不到10坪大，四周堆滿撿來的廢紙箱、鋁鐵罐。屋子不但沒廁所、衛浴，連自來水都沒有，只能取用屋後灌溉溝渠的山泉水，阿勝伯說：「大小便在果園解決就好了，洗澡比較麻煩，夏天可以洗冷水，但冬天受不了，得燒水洗澡，比較花時間。」
村長施燕珠說，鄉下工作難覓，阿勝伯的子女腦筋又憨直，工作機會更少，「連水果是否成熟他們也分不清楚，根本沒人敢用他們。」尤其屋舍破爛不堪，連住都有問題。「只有要有颱風，都會要求阿勝一家到活動中心暫住。」目前估價約需工程款22萬元，蘋果慈善基金會已先撥款暫紓部分急需，期望在新的一年協助阿勝伯有個遮風避雨的家。

「補助審查不符現實」

中華民國智障者家長總會社工督導曾雅倫說：「當政府低收入補助審查將輕度智障者視為工作人口時，相對也應提供如職訓、媒合工作機會等配套服務，只憑醫學鑑定，就認定一個人是否具工作能力，也不衡量智障者是否能找到工作，實在不符現實情況。」

無法計算的人生

阿勝伯：「生育子女，是好、是壞都是命中註定，只要他們叫我爸爸，我就有責任要照顧他們。」
