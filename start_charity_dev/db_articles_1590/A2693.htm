稚女靠針保命 婦嘆殘忍

顧癱夫病女無力謀生 「怕撐不住」

2012年02月16日

晚餐前半小時，41歲的阿娟（黃淑娟）幫11歲的女兒小芬施打胰島素針劑，當針頭往小芬肚皮注射時，小芬害怕地閉眼轉頭。阿娟不捨地說：「小芬因先天糖尿病，須打胰島素控制血糖，老天好殘忍，她還這麼小就要天天打針來保命。」
報導．攝影╱韓旭爾

記者採訪時，初見小芬模樣，便覺得她嬌小孱弱，已國小五年級，身高卻僅128公分、體重26公斤，與一旁大1歲的姊姊小薇相比，足足矮一個頭。
阿娟說，小芬雖成長較緩、眼睛弱視，但一向健康，不料去年12月因重感冒住進加護病房，檢查出罹患先天性糖尿病併發酮酸血症，「醫生表示，小芬若沒好好控制血糖，將來視網膜剝離、腎衰竭這些糖尿病併發症都會發生，一定要好好保養身體。」阿娟說著不禁哽咽，「小芬的未來注定困難重重了。」

夫勤復健盼再起

回溯過往，阿娟說，她與48歲丈夫阿舜育有3子女，除了么女小芬外，另外兩個孩子分別為12歲及15歲，家計原靠夫妻打零工維持，去年7月，阿舜突發腦中風，住院治療1個月，半年來仍左半身癱瘓，腦筋、說話也不清楚了，「問他事情，他總是想很久才能開口回答。」當阿娟與記者說話時，阿舜正拄著四腳柺杖吃力地由房間往客廳走來。他說，自己還不算老，但卻像廢人一樣，心裡很不好受，「我會努力復健，希望能早日恢復工作能力。」

5口家入不敷出

阿娟輕嘆，現在家中有一大一小需她照顧，實在無法工作，娘家父母及4個兄弟姊妹大家都經濟緊絀，目前阿娟一家雖領有低收入戶補助、殘障津貼共1萬6000元過活，扣除每月3000元房租，入不敷出，蘋果慈善基金會已先撥款為阿娟一家生計紓困。

發願不讓母擔心

阿娟就讀國二的長子小翔說，這些年來媽媽兼3份工作養家，她的身體也不好，「妹妹生病後，媽媽常對我交代說，萬一她撐不住，要我負起照顧妹妹的責任，我聽了好心酸。」
小芬的學校導師說：「小芬身體出了問題，學校會特別注意，讓媽媽能安心讓孩子來學校上課。」早熟的小芬說，她正在努力克服對打針的恐懼，也學著自己打胰島素，希望她跟爸爸都能身體健康、不再住院，不讓媽媽擔心，就是她新年最大的願望了。
