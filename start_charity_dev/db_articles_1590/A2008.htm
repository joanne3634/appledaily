癌夫切肺 妻撐5口也病倒

「像3分球被蓋大火鍋 得分希望破滅」

2009年10月22日

40歲的阿銘（李振銘）說，真沒想到夫妻運氣這麼差，2年前發現自己罹患3期肺腺癌，經開刀、化療，病況才稍穩定，不料今年9月初，40歲的妻子阿鳳突然中風，目前還在醫院治療。
報導．攝影╱韓旭爾

病房內，中風癱瘓的阿鳳表情呆滯躺在病床上，嘴歪眼斜、不時流出口水，阿銘看了直嘆息，拿起桌上的面紙為她擦拭，阿銘說，阿鳳因先天性顱內動靜脈畸形，中風的機會比一般人高，「這2年來，阿鳳擔心我的病況，又操煩家裡3個小孩、生活開銷，可能壓力太大才會病倒。」

打籃球拼復元接零工

阿銘說，一家5口原靠他在工廠上班，每月3萬元度日，不料2年前竟發現患肺腺癌，家中無人工作，「只能靠著向親戚借錢過日子，但有借無還，現在也已無處可借了。」
阿銘說，妻子阿鳳為了家裡生活，顧不得自身畸形腦血管壓迫神經的問題，今年7月開始在家幫人顧嬰兒，每月能有1萬6000元收入，暫紓全家人生活費。沒想到才過2個月，阿鳳卻突然中風倒下，「這好像打籃球時，已出手的3分球被蓋了個大火鍋，得分希望一下子就破滅了。」嬰兒的媽媽說，孩子3月大時便交給阿鳳看顧，「她很細心，把我的孩子顧得很好，讓我能安心工作，希望她能趕快好起來。」
記者採訪過程中，阿銘沒講幾句便咳嗽不止，上氣不接下氣，193公分的身高，體重卻只有70公斤，阿銘說，因左肺2/3切除，時常出現呼吸不順，體力也大不如前。但能活到現在，也算是幸運。阿銘提及自己病發經過說道，2年前因左手痠麻、無力就醫，經診斷為肺腺癌。
阿銘說：「肺腺癌初期幾無症狀，等到發現時多已是晚期，很難治，自己是由於腫瘤壓迫神經，左手先感到不適，才能及早發現、治療。」但現在走幾步路便氣喘如牛，「為增加肺活量，只能努力打籃球，因為只要能恢復以往一半力氣，至少還有接零工的機會，掙些錢。」

補助還房貸所剩無幾

「能幫哥哥的有限，現只能先幫忙照顧他2個年幼女兒，減輕他奔波醫院、家裡的辛苦。」阿銘的弟弟這麼說。社工表示，阿銘一家現靠每月低收補助1萬4600元度日，但扣除每月1萬1000元房貸，剩下3600元，無法應付家用，蘋果慈善基金會已撥款暫助阿銘急困。
