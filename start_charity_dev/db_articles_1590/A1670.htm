癌病夫妻醫費無著 女棄學

積欠3個月房租 「真不知怎麼辦」

2008年08月22日

當生命開始倒數，那種無奈對患腎上腺癌已末期的鄭清輝來說，內心煎熬與身體病痛同樣難受，但他最放心不下精障太太阿鈴與兩個剛成年子女，「想用最後的幾個月時間多陪陪他們。」
報導．攝影╱向高彬

鄭清輝虛弱躺在輪椅上，臉色慘白，如今他舉步維艱，起身便溺或上床都得子女攙扶，53歲的他說，患腎上腺癌已6年，手術切除後控制穩定，但今年初體力衰退厲害，「我現在常喘不過氣，要吸氧氣瓶才吸得到氣。」

子剛退伍工作難覓

鄭清輝說，剛發病時腹部腫脹疼痛，原不以為意，後來日漸虛弱才就醫診治，竟得腎上腺癌；當時因妻病子弱，他得用盡力量抗癌，這幾年來除西醫外，還另外每天吃中藥，一天得二、三百元，加上兩個孩子之前仍在讀書，家裡靠微薄積蓄苦熬，如今耗罄，房租還欠了3個月付不出，還好「房東人不錯，我們是十年的房客，知道我和太太病了，讓我們拖一下，但不能一直欠下去，但家裡現在連坐車看病錢都東湊西借，真不知道怎麼辦？」
鄭清輝僅有一兄，已在兩個月前因口腔癌病逝，其他親友也能幫有限。蘋果基金會獲悉，已先撥款暫紓鄭清輝醫病困境。
鄭清輝原從事印刷及保全工作，與太太阿鈴生有一對現年22歲雙胞胎子女，長女阿琪去年大一下時休學在家照顧病殘雙親，兒子阿達今年7月剛退伍，由於得與姊姊合力照顧病重父親，加上他讀的是高職體育班，難符合就業市場所需，阿達說，不景氣加上工作經驗不足，一直都在努力找工作。

女盼來日重讀大學

採訪時，看到鄭妻阿鈴留著一頭醒目長紅髮、塗藍指甲油，她說是好友幫忙做的，「弄漂亮，心情好一點，不然我很煩很煩。」她患躁鬱症近20年，5年前先生病後，嚴重成精神分裂，採訪時，她盯著桌上彌勒佛像說，「你看祂在笑，在跟我說話，我也笑，我就跟祂說，你要保佑我們家。」
女兒阿琪說，媽媽雖有服藥控制，「但只有在她睡覺時，家裡才清靜一點，不然稍有不順她意，就像火山爆發一樣抓狂，非常恐怖。」還是雙十年華，阿琪為顧爸媽暫停學業，問想讀書嗎？阿琪點頭說：「很想，但家裡這樣，怎麼去？」阿琪只盼爸媽病況能好轉，弟弟順利工作，自己或許還有重回校園拾起書本機會。

暖流見證
熱心公益卻得重病

鄭清輝以前很熱心公益，義交和民防都參與，人也客氣，卻沒想到得重病。目前鄭家子女列計工作人口，暫難列低收，僅兩夫妻領每月殘津共8000元，有慈善會急難慰助時，我都協助申辦，也謝謝蘋果基金會能撥款暫助度過眼前急難。
