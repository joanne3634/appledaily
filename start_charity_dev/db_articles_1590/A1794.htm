愛女重癱 單親媽愁醫費

手術已耗20萬 還要復健一年

2009年02月05日

剛過的春節，是20歲的小馨第一次在醫院過年，她去年車禍身受重傷，學業中斷。春節前才剛接受胸椎修復手術，醫療費便達15至20萬元，媽媽陳思如單親育女，面對女兒未來漫長復健路，只能強打起精神。
報導．攝影╱向高彬

車禍發生至今大半年，但小馨一想到去年6月的意外仍心有餘悸，她說，當晚與友人騎車出門遭計程車撞擊，友人受輕傷，但坐後座的她飛出撞斷4顆門牙、氣血胸、胸椎碎裂4節，送醫急救後下半身癱瘓。

吃喝便溺需人照料

「但醫師也說以我胸椎的傷況，腳卻還有知覺能動作，是非常少見，算不幸中大幸，也說日後復健做好，有很大機會可以再站起來。」記者詢問小馨接受治療的醫院，院方說會盡最大努力，至少需時一年復健期。
只是小馨的51歲媽媽陳思如說，單親育女家計重，因小馨現在吃喝便溺靠她照料，情緒不穩也得在旁陪伴安撫，「我和第二任前夫經商失敗欠不少債務，10年前離婚後，我做事養小馨讓她讀書，母女租房子過活過得勉強，現在要整天顧她，根本沒辦法做事。」陳思如又說，過去大半年生活，靠各方的救助金暫撐，包括「馬上關懷」急難金1.6萬元、學校救助金3萬元、肇事方慰助5.9萬元再加上一名李姓工程師協助籌募5萬多元，前後共有善款約16萬元，但仍不足，又跟親友借了5萬元。
陳思如的前夫阿慶說，經商虧掉千萬元，「查封的查封、拍賣的拍賣，還欠外頭很多。但我看到孩子受傷也捨不得，前妻有心臟病和憂鬱症，這大半年吃不好、睡不好，一直在煩惱孩子的事，也快累倒。」

肇事司機無力賠償

記者多次電話聯絡肇事方運將，對方均未接聽，車行林姓經理說，他們目前也難聯絡上本人，據了解肇事責任在司機多些，但如今景氣差、肇事司機無力賠償和解，後將續循法律途徑處理。蘋果基金會訪視後，已先撥款暫紓小馨母女目前的醫療生活急難。

暖暖祈福
工程師網友 熱心募5萬

小馨養傷期間，無力行走僅能動手，所以經常上網瀏覽，去年11月意外結識一名在高雄工作的李姓工程師。李姓工程師因自己的三姊一家發生重大車禍，在部落格抒文為姊祈福，小馨留言祝福，後來，李姓工程師得知小馨困難，便在所服務的公司與教友間發起募捐5萬多元，去年12月初交給小馨媽媽。
李姓工程師本身是基督教會長老，他說：「主告訴我們去奉獻，愛親人、也愛需要幫助的陌生人。」而與李姓工程師同事多年的陳小姐，她說，李大哥一向熱心公益，「我們雖然能幫不多，但希望小馨得到誠摯祝福，早日康復。」
