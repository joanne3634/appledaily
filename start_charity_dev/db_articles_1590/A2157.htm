單親父腰傷 「不知怎養兒」

身穿鐵衣難覓工 小六兒懂事扛家務

2010年05月03日

42歲的單親父阿銘（楊建銘）因多次工作受傷導致腰椎滑脫開刀，必須身穿鐵衣固定，只見就讀小六的長子小宇，乖巧地幫忙洗衣、打掃，阿銘望著2子的瘦弱背影，堅定地說：「為了孩子，再苦再痛我也要熬過。」報導．攝影╱游竹君

阿銘原為預拌混凝土車的司機，10年前工作時曾從1樓高的車頂摔下，傷及腰椎，後來又發生2次車禍，整個人摔到擋風玻璃上，導致腰傷日益惡化，撐到1年前腰椎嚴重疼痛，同事阿史說：「那時看阿銘常歪著身子、拖著腳走路，卻還是咬牙幫人開車代班，勸他看醫生，他總說沒工作孩子就沒錢吃飯，只好拚命做，看了實在叫人不忍，朋友也只能多少送點白米青菜幫忙。」

下工回家兒已熟睡

阿銘說，這次已經痛到寸步難行，甚至手還得搭在小宇的肩上才能勉強走幾步，醫生診斷腰椎滑脫，3月中旬動刀切除凸出的椎間盤外，也打骨釘、以人工關節固定，「我不擔心手術，但想到需穿鐵衣固定半年、之後還得換穿軟背架，負重工作我都不能做，擔心房租，也不知該拿什麼養孩子。」已申請了政府的「馬上關懷」急難金，蘋果基金會訪視後，也已撥款暫紓生活及醫病困境。
阿銘曾有兩段婚姻，第一段婚姻育有1女由前妻照料，第二段婚姻離異後阿銘獨力照料就讀小四及小六的2子。阿銘說，起先孩子由北部的母親照顧，2年半前二哥工作也不穩定，不願再加重家中負擔，便把孩子帶回身邊照顧，「雖說是我照顧他們，但其實是哥哥小宇負責洗衣、打掃、照顧弟弟小麟，從不讓我多操心，尤其是生病後反而是靠他們照顧我。」
阿銘語帶愧疚地說，之前跑車為了節省時間多半半夜出門，作息日夜顛倒，「很多時候我只能看孩子熟睡的樣子，卻講不上話，就只能在桌上留字條叮嚀他們『加油』、『要用功念書』，紙條幾乎成為我們父子間唯一溝通的方式。」

一度苦撐婉拒援助

次子小麟的導師說，去年初小宇就讀5年級的導師（去年底已去世），發現小宇及小麟晚上因爸爸常不在家、總以泡麵果腹，曾自掏腰包買便當給兩個孩子當晚餐，但遭他們爸爸婉拒協助。總盼自食其力養子的阿銘說，直到去年身體實在熬不過了，才申請獲得三款低收資格，每月兒少補助4400元，阿銘現雖有女友，但她也失業、還需照顧自家臥床的奶奶，阿銘母親說，長子病逝、次子也單親育2子，實難再幫阿銘。
