女躲夫債養3子「像蟑螂」

僅靠醬油麵果腹 心痛孩子吃苦

2013年04月23日

晚餐時刻，39歲的阿慧水煮麵拌醬油，煎幾個荷包蛋，3個就讀國三到國一的孩子吃得津津有味，阿慧不捨地說，乾麵條是一家對抗苦日子的救星，「4人份的晚餐僅50塊錢，但孩子正在發育，擔心沒營養。」報導．攝影╱韓旭爾

阿慧說，12年前丈夫沉迷賭博又投資股票失利，竟挪用公款4、500萬元，為補缺口，以她名義向錢莊借貸200多萬元且跑了，留下她與3個子女面對討債者威脅。阿慧嘆，11年前雖已離婚，但債務關係無法終止，「很多債都是我簽名蓋章借的，丈夫失蹤，債都算我頭上，但我根本沒能力還！」
阿慧說，擔心孩子們受債主傷害，只能四處躲，「覺得自己像蟑螂，連累孩子跟著我吃苦，是我心裡的最痛。」阿慧國一的小女兒小臻心有餘悸地說：「現在聽到門鈴聲我心裡還會怕，怕有人來要錢。」

罹重度憂鬱難上工

當地衛生局進行自殺防治的社區關懷員說，阿慧3年前診斷出罹重度憂鬱症，曾2次自殺未遂，常勸她要定期到精神科就醫服藥。
阿慧說，這幾年靠臨時工養家，因服抗憂鬱藥物造成精神恍惚影響工作，月賺不到1萬元，雖有家扶、慈濟補助共1萬3100元，但扣掉房租和健保欠費等剩不到3000元生活。「晚餐只能吃乾拌麵或蛋炒飯。」阿慧說，學校老師曾讓孩子帶回營養午餐當晚餐，但因課後回家已晚上6時30分，餐食常餿掉而作罷。蘋果基金會訪視後已先撥款紓困。

長子盼工讀助撐家

當地村長說，阿慧因媽媽現住自宅價值超標，一家無法通過低收。阿慧媽媽阿蘭婆說，阿慧擔心債務連累娘家，不敢回家住，「現我有嚴重糖尿病沒法工作，阿慧其他3兄弟姊妹在經濟上都幫不上忙。」阿慧表示，家裡沒電視、網路，但孩子不曾抱怨說苦，尤其女兒小臻功課一直保持前3名。小臻導師說，小臻品學兼優，因家長曾有輕生念頭，學校已多關懷，也減免學費、營養午餐等費用。長子小維說，今年上高中後會半工半讀，「什麼工作都願做，想賺錢，不要讓媽媽、弟妹再沒飯吃！」
法律扶助基金會律師張智剛說，夫妻彼此的債務並無關聯，但阿慧為解決前夫挪用公款才出面借貸，「誰借的，就由誰負責還債。」張智剛強調，借貸涉及重利、暴力討債，已是刑事案件，應聯繫司法警察機關協助處理。
