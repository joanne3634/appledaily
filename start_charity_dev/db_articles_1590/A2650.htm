母洗腎 15歲孝女把屎尿

發憤當護士 「長大要養媽媽」

2011年12月22日

48歲單親媽葉碧珍因糖尿病引發尿毒症及腎膿瘍，今年10月切除左腎，醫師又宣告她日後得終身洗腎，她和15歲女兒因生計窘迫無法支付房租，困境雪上加霜。報導 攝影／張嘉恬

目前葉碧珍切除左腎後得休養至少一年，每月僅賴低收、家扶相關補助約4000元生活。葉碧珍說，女兒3歲時她與夫離異，此後仰賴廚工收入獨力撫養女兒，她2年前曾因遺傳性糖尿病發作昏倒在家，但之後為撐生計未積極治療，「那時昏倒，還好女兒放學回來看到打119。長久來我們過得清苦但至少彼此相依靠，想到往後我因病難再穩定工作，『艱苦』（台語：難受）到的是她。」葉碧珍無奈說。

幫忙洗衣擦藥

採訪時正逢女兒小君放學，小君先把污衣用手洗過後，再使盡力氣一件件擰乾，接著再替媽媽進行血液透析的洗腎傷口擦藥，葉碧珍說，失婚不遺憾，有小君當女兒是最大的幸福。「我住院一個月期間，她每天放學騎鐵馬一小時到醫院照顧我、把屎把尿，我的人生有她就滿足了。」

貧苦樂觀進取

小君說，一開始清理媽媽的排泄物時「雖然覺得怪怪的，但也只有我能做這些事，久了就習慣。」小君也說，媽媽發病時身體不適，加上擔憂經濟的雙重壓力，哭了很久，「我安慰媽媽說長大後我會養她，也希望能當護士照顧她。」小君的姚姓導師說，小君與母親往常因生計窘迫經常搬遷，「生活不安定加上得承攬家務，在家讀書時間少，她立下作護士目標後，積極利用在校時間補上進度，是個樂觀進取的孩子。」
葉碧珍上有7名姊姊皆已年長或病弱，各有自家生計重擔，當地鎮公所專員表示，去年曾申請「馬上關懷」急難金1萬5000元協助，再轉介蘋果基金會關懷，基金會訪視後已撥款暫紓困境。

信耶穌獲寄託

葉碧珍說， 離婚獨力養女辛苦，後又重病發作，幾度遇上絕境，幸好在住院期間，透過病友認識主耶穌。「有了信仰，我覺得很受眷顧，住院時擔憂出院後沒地方住，連吃飯錢都沒有，但我一直禱告，都迎刃而解，我覺得很富足，也有信心度難關、再站起來。」
