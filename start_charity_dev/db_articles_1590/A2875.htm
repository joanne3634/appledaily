跛父獨養7女 忍傷打工

「剩條腿也要做 不然沒飯吃」

2012年09月20日

6月中時，35歲單親爸阿福（陳亞福）在家中不慎跌倒造成右腳骨折，儘管醫生叮嚀需休養半年，但天一亮，他還是跛著腳、騎著車四處找尋零工機會，他說：「就算只剩1條腿，還是要想辦法工作，不然7口沒飯吃。」報導．攝影╱韓旭爾

阿福說，他是越南華僑，約10年前爸爸與5兄姊相繼來台，他在7年前先來台做粗工掙錢寄回家用，4年多前陸續將妻女接來台灣，但去年3月妻子要生下第7個女兒時，因難產而去世。阿福接著哽咽地說，原本只想生個男孩傳香火，「沒想到卻讓老婆沒了性命，後悔也來不及了！」
目前阿福一家領有政府低收入戶、家扶補助共約3萬元，他的7個女兒，最大的14歲，最小的才1歲多，么女暫時託付越南的岳父母照顧，每月寄交1.2萬元養育費，加上每月3000元房租，所剩無法應付一家7口吃穿。近來阿福因腿傷影響工作，即便各社福資源皆已介入協助，但因阿福腳傷及耳疾影響工作量，生計仍有不足，蘋果基金會訪視後，也撥款助阿福先行醫療。

漸失聽力不願就醫

伊甸基金會社工蘇姓社工說，去年阿福么女出生時，原本幫忙找合法保母照顧，但每月相關費用達2萬元，阿福無力負擔，「阿福認為么女是妻子用生命換來的，也捨不得讓她離開親人身邊接受寄養，不得已先由岳父母帶回越南，等女兒3歲再接回台灣。」蘇姓社工說，阿福因慢性中耳炎逐漸喪失聽力，需及早開刀治療，但阿福擔心治病休養沒法賺錢養家，遲遲不願就醫。

長女代母職不喊累

家扶中心林姓社工說，阿福雖想生兒子傳宗接代，但他對女兒們十分疼愛。自從阿福妻子去世後，大女兒小燕姊代母職，「放學回家要洗全家衣服、打掃、幫妹妹洗澡，從不聽她喊累。」
林連宗說，阿福家總是一菜一湯加白飯解決一餐，孩子們也不以為苦，「全家有緊密的凝聚力」，目前家扶中心每周都有志工到家裡協力，並不定期提供生活物資。當地派出所劉所長說，有時阿福打零工下班較晚，家中全是年幼女孩，警員巡邏時會特別注意家裡情況，派出所也提供場地讓家扶志工幫孩子們課輔。
當地幼稚園張主任說，目前阿福有2個學齡前女兒在幼稚園上課，因低收入戶學費補助，無須繳交任何費用，其他4個孩子下課後，也會到幼稚園報到，幼稚園免費幫忙照顧至晚上6時再讓姊妹們一起回家，盡一點心力協助。
