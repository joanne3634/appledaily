中風夫患癲癇 妻兒失生計

神志不清迷走鄉間 村人接濟扶回家

2008年06月27日

「我要去跟老闆拿工錢，今天會比較晚下班。」45歲的羅其清自言自語，搖晃著往村裡廟口走去。越籍妻子裴氏美仙無奈的說：「他中風前，每天早上都在廟口等老闆載去綁鐵，中風3年來都沒法工作了，頭腦還不清楚，沒事就亂走。」報導 攝影╱孫敏聿

家住屏東鄉間的裴氏美仙操著不流利的國語說，8年前嫁到台灣時，丈夫還能工作，每個月賺1、2萬元，加上自己也幫忙鄰人撿雞蛋、包裝皮蛋，養活現年7歲的兒子。
今年3月，羅其清因癲癇發作，再度送醫急診，「那晚他昏迷不醒，躺在床上褲子都是大便，我和孩子都嚇死了。」裴氏美仙說，丈夫出院後狀況更糟，晚上得包尿布，白天四處亂走、說些聽不懂的話，常靠村人扶回來，「我現在得照顧丈夫，沒法去工作，平常鄰居會送我們菜，不然就是跟姊姊（大姑）借1、2千元吃飯。」

不知母過世2年

羅其清的大姊羅素琴表示，自己20年前喪夫後，靠著低收補助，加上麵攤生意，照養三個孩子長大，但近兩年來物價飛漲，生意難做，無力長期資助弟弟。「現在一桶瓦斯要690元，擺攤真的只能『賺吃』（台語，意指養活自己），賺不到工錢。」她說，大哥負債跑路、大弟夫妻做建築零工養兩子，生計也很辛苦。
巷口機車行老闆羅先生指出，羅其清3年來多次中風後，又患有癲癇症，連2年前母親過世也不知道，還老說要回家吃媽媽煮的飯。「前兩天他還騎腳踏車摔倒在路邊不醒人事，我趕快把他扶回家，這樣子每個月要2、3次，有時還要叫救護車。」羅先生說，鄰人偶爾會給幾百元接濟，「大家知道他沒錢，家裡還有老婆小孩要養，也不會跟他討回來。」
由於羅其清的低收尚在申請中，目前僅領有3千元殘障津貼，難敷生活。鄉民代表林光華指出，羅母過世前，送到機構安置多年，家中積蓄用盡，只剩現居房舍勉強棲身，經濟確實辛勞。

須復健服藥控制

林光華說：「羅其清現須回診積極復健、按時服藥控制癲癇症，待病況穩定，羅妻才能工作持家。」蘋果基金會獲悉，已先撥款紓困。
