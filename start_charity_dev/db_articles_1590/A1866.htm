恐活不過2年 癌父憂6歲女

病妻無力工作 3口貧家難為繼

2009年05月05日

「爸爸，我畫很多白雲和綠草，希望你快病好，帶我們去玩。」6歲的小珊揚起天真笑容對爸爸說。46歲陳水源3月初發現患腦神經母細胞瘤，正努力抗癌。
報導．攝影╱向高彬

陳水源指著頭部一圈手術痕，他說2月常感暈眩，沒想到3月晚間突頭痛欲裂、全身抽搐，24歲太太阿玲發現送醫，因有腦出血，一度以為是中風，後經詳細檢查才知患腦瘤，「開刀取出一公分腫瘤，化驗結果是惡性的，幸好早期發現，太太哭到不行，我安慰她，事情既然發生，就好好接受治療。」陳水源現正進行電療，又說，醫師告知這類型腦瘤發展複雜，不易清除，甚至患者多半活命不超過2年，「我的態度就是堅強面對。」

妻：不能讓他離開我

陳水源病前以開挖土機為業，他說，8年前曾有一段短暫婚姻，離異後結識小22歲的太太阿玲，並生下小珊，「當時她娘家反對，但她認為我能給他安全感，所以在一起。」一旁的阿玲擔憂說：「他曾說不管發生什麼，一定會保護我們，婚後至今他一直有做到，所以我只想，一定不能讓他離開我。」

沒收入付不出房貸

採訪時，見阿玲手腳皆有嚴重紅疹，她說，生小珊後至今，從60公斤胖至108公斤，每天腫癢難癒，一照陽光發疼，疹處夏天流湯、冬天乾裂，嚴重時得吃類固醇，不僅身心難受，也影響她求職遭遇僱主恐懼不敢僱用。阿玲的皮膚專科林姓主治醫師說，患者因體質問題，難以根治，僅能以藥物減緩腫癢。
而陳水源中年喜悅得女，也以為一家3口能安穩過日，未料卻遭癌魔突襲，阿玲說，家裡靠先生做工支撐，婚後這幾年，慢慢償清先生先前債務200萬元，3年前向親友借款及貸款才購置現屋，「現在他病了，需要休養，我要顧他也沒辦法工作，而家裡只靠他吃穿，眼看這個月房貸就付不出了。」
陳水源的僱主陳光耀說，合作10多年，「他老實認真，知道他生病，同事們都很難過，我們會等他康復回來再做。」陳水源的妹妹阿月說，自己先生中風癱瘓，又有2個幼女須照料，「很遺憾沒有能力多幫哥哥。」當地社福中心趙姓督導說，已助陳家申辦「馬上關懷」急難金，後續由社工再協辦相關社福補助。蘋果基金會訪視後，也撥款暫紓陳水源生活急窘。
