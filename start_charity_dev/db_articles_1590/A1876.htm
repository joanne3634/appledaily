貼心女 病榻吟歌慰癱父

「爸要忍耐 你一定很快就會好了」

2009年05月11日

38歲的泰雅族人阿平（江發平）因腦傷、神智不清，四肢無意識地踢動著，長女江夢芃和二女兒小寒細心地抓著爸爸纏著紗布的手腳，紅著眼眶輕聲安慰道：「爸爸，你不要亂動，會受傷，要忍耐，你一定很快就會好了。」報導．攝影╱游竹君

病榻旁，3姊妹輕聲唱起詩歌和爸爸最愛的原住民歌曲，突然間，阿平安靜下來，停下奮力揮動的手腳，彷彿女兒們熟悉的歌聲能讓他忘卻疼痛，回到過去全家團聚的美好時光。
3月底，阿平和31歲的妻子阿晴下班搭同事的車回家，不幸發生車禍，導致阿晴肋骨及胸腔多處挫傷，須以背架固定傷勢。

一場車禍妻傷夫昏迷

阿晴說，自己與丈夫育有就讀國二、小六及小四的3個女兒，平日兩人在市場幫人殺鴨維生，靠兩人每月共5、6萬的收入維持家計，阿晴說：「車子突然在高速公路翻車，不僅我受傷得休養，丈夫還昏迷不醒，想到家裡有3個孩子，還有房租和健保分期就要1萬多元，我擔心地不知如何是好。」
同事周小姐表示，車禍那天開廂型車載大夥人回家，為閃避超車，突然間就撞上高速公路護欄，「車禍後我腦震盪，女兒骨盆腔碎裂，短期間無法工作，只剩丈夫帶著護頸咬牙持家，我們也很擔心阿平家的情況，但眼前能力實在有限，只能盡力分期賠償，但仍沒有辦法全額負擔阿平的安養費用跟生活開銷。」
妻子阿晴說：「看著眼前家裡全無收入，心裡很著急，還好醫療費用可以用乘客險及強制險實支實付，眼前只能每晚禱告傷勢快點好，才能再工作養家。」

「5口生活讓人擔心」

目前阿平家中每月生活及養護中心費用至少要5萬5000元，現僅獲「馬上關懷」急難金3萬元及蘋果基金會急難金3萬5000元，村長游有連表示，阿晴至少要靜養半年左右，一家5口往後生活讓人擔心。

後記
長女曾上《星光》5拼歌喉

採訪時，就讀國二的長女江夢芃表示自己很愛唱歌，曾在3月中旬參加歌唱節目《超級星光大道》5的比賽，但爸爸阿平還來不及在電視上看到她的演出，就意外發生車禍。
江夢芃說，當時自己在比賽上唱道：「愛什麼稀罕，不要再等待……」，心裡卻惦著在加護病房昏迷的父親，後悔沒有早些把心中深藏的愛說出口，「雖暫時被淘汰，但我不會放棄，希望爸爸早日清醒，聽我為他唱歌。」
超級星光大道節目製作人劉安華表示，當時主持人陶晶瑩才跟夢芃提到家裡的變故，夢芃就難過地說不出話來，眼眶泛紅，「她很有潛力， 7、8月份的PK賽將邀請她，希望幫夢芃圓夢，好好唱一首歌獻給爸爸。」
