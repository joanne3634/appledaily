憨父罹癌憂兒女 「要賺錢」

15歲兒懂事：想趕快畢業撐家

2012年02月19日

「爸，你安心養病，別擔心我和姊姊。」15歲的小慶安慰罹患鼻咽癌的爸爸阿章（李寶章），但現年52歲的阿章智能只有10歲，不了解自己罹患重病，喃喃念著：「什麼時候可以出院，我要賺錢給姊姊治病。」
報導．攝影╱仲芝蓉

小慶說，自己和21歲患小兒麻痺症又智能障礙的姊姊小宜，從小就跟爸爸生活，雖然爸爸反應比較慢，但他一直幫人種田養家，很少生病、也沒聽他說不舒服，直到去年底因脖子腫大，就醫才發現罹患鼻咽癌第四期。小慶表示，醫生說爸爸要做化療和放射線療，暫時不能工作，「我跟爸爸說先把病養好，但他放心不下姊姊，說要工作賺錢，才能幫姊姊付安養費。」

僅領4700元津貼

當地公所里幹事說，阿章至少得住院到今年3月，待出院返回戶籍地，後續才能申請低收入戶補助，而他的醫療開銷雖以重大傷病身分減免，但目前一家僅領有他中度智能障礙津貼4700元，小宜的安養差額就要3000元，公所正協助申請「馬上關懷」急難金，並轉介蘋果慈善基金會，基金會訪視後已先撥款紓困。

女兒送到教養院

阿章的三姊阿玉回述，阿章自幼智能障礙，年近30經人介紹才娶小兒麻痺妻，但兩人生下女兒小宜後離婚，「父親生前擔心阿章憨傻，不會照顧小宜，耗盡積蓄帶他去印尼娶妻，手續沒辦完父親就肺癌往生，沒想到第二任妻子生下小慶後，也跟他離婚。」阿玉說，小宜智能障礙不會說話，到4歲還不會走路，尿布包到10幾歲，「阿章連自己都顧不好，小孩只好由我們幫忙照料，直到10年前才把小宜送去教養院。」
阿玉說起家中往事不禁嘆氣，8個兄弟姊妹中阿章排行老么，母親病逝時阿章才2歲，父親務農拉拔兒女長大，但大哥也憨傻、二哥已鼻咽癌過世、二姊出養。

走10公里路探女

阿玉說：「我和3姊弟各有家計負擔，輪流幫忙顧小慶，好不容易小慶去年上高職建教班，再2年畢業就可撐家了，弟弟不抽菸、不吃檳榔，突然生病，實在措手不及。」
小慶說，去年4月姊姊跌倒住院，爸爸不會坐公車，憑印象走了10幾公里去醫院照顧姊姊，「爸爸雖然腦筋不好，但我知道他盡全力保護我和姊姊，我很想趕快畢業，代替他撐起這個家。」
