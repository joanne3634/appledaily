少年顧癌母 「想快賺錢」

2口月靠9千補助 立志考軍校扛家

2012年08月27日

17歲的小文拿著牛奶替罹患舌癌第四期的媽媽灌食，一邊安慰她說：「媽您安心養病，不用擔心我，明年我畢業就去考軍校。
報導．攝影╱仲芝蓉

小文53歲的媽媽林玉鳳忍著鼻胃管灌食的不適，一會兒鼻涕直流、一會兒嘔咳不止，斷斷續續勉強把牛奶灌完。林玉鳳因舌癌切除一半舌頭，口齒不清地說，自己單親撫養小文長大，他從小就成績好，高中還考上第一志願，「我連國中都沒有讀，國字只認得自己的名字，從來沒教過兒子功課，但他喜歡讀書，如果有機會，也希望他上大學，但我卻這時候病倒，讓他被迫提早撐家」。

房租月付3000元

採訪時，記者看到林玉鳳母子倆租住的古厝中央的神桌上供奉了幾尊熏黑的佛像，林玉鳳告訴記者，去年春天搬進來後，7月就發生嚴重車禍，今年5月又發現得癌症，「朋友說是房子風水不好，說神桌上的佛像已沒有神，才讓我身體一直出狀況，但我不信邪，這裡房租一個月才3000元，我現在又沒辦法工作，還能搬去哪裡」。
林玉鳳無奈地繼續說，自己原本在大理石場、木材場做臨時工，去年7月自己騎摩托車摔倒，左肩脫臼、斷了7根肋骨，休養半年多，好不容易今年初可以外出做摘菜零工，每月賺4000、5000元，加上民間慈善團體每月補助9250元，生活勉強還過得去，怎知又罹癌。小文接著說，媽媽動手術後一度氣切，整整72天沒辦法說話，「大舅身體不好、小舅要照料80歲的肢障祖母，3個阿姨和二舅都已過世，媽媽不識字，那72天比手畫腳只有我懂，真希望趕緊畢業開始賺錢，讓她不要愁煩生活」。
林玉鳳回述，自己早年曾有一段婚姻，但無生育子女，之後也因個性不合離婚，才結識小文的爸爸，但沒過多久，他就中風過世了，「當時小文才要上小學，他成年前都是我的責任，哪有時間迷信風水，只希望病趕快好能看小文成家立業」。

管灌食品開銷大

當地公所承辦員說，林玉鳳母子倆目前僅領有民間補助每月9250元，但她的管灌食品開銷每月就要9000元，還須負擔房租3000元，公所正協助申請「馬上關懷」急難金，和辦理低收入戶等補助，並轉介蘋果慈善基金會，基金會訪視後，已先撥款暫紓母子倆就醫生活急難。小文的導師許育菁說，學校也正申請學產基金，盼小文家早日度過難關。
