癌男拭淚 只求全家在一起

手術摘除牙齦骨 4口家借錢度日

2009年02月27日

罹患牙齦癌的49歲張唯勝（阿勝），牙齒及牙齦骨手術全數摘除後，凹陷的頰唇更顯得蒼老、虛弱，疼痛地幾乎無法言語，問他出院後想做什麼事，勉強口齒含混說：「一家人在一起就好。」教妻子阿珠聽了眼眶一陣泛紅。報導．攝影╱游竹君

阿勝與45歲的阿珠育有就讀高一、小五的一對子女，夫妻平日一起在工地搭配做工，兩人上工一天可賺2500元，一家四口生活勉強可過。
不料去年8月阿勝開始感覺牙痛，但因健保欠費3年多，選擇吃止痛藥先止痛，直到兩個多月後幾乎無法進食，才辦理健保紓困分期，就醫才發現罹患牙齦癌，今年1月手術將牙齒及牙齦全部拔除，日後僅能以流質食物灌食。

4兄弟就有3人罹癌

病榻上的阿勝，雙眉始終緊蹙，平常話不多的他，接受廓清手術後，說話時口水便不自主地從嘴角滴下，讓他更加寡言沉默；太太阿珠說：「阿勝很愛孩子，一直很擔心他們自己在家，但他住院行動不便期間，我只得家裡、醫院兩頭跑。」聽到兩個孩子獨自在家生活時，阿勝難忍思念地瞥過頭去偷偷拭淚。
阿珠說，同住離婚的大伯兩年前罹患氣管癌去年9月已病逝，8個月前小叔阿郎罹患食道癌現仍陸續接受化療、電療，「阿勝家4兄弟，就有3人罹癌，大家經濟都不好過，阿勝一路陪著大哥從罹癌到死亡，早知對抗病魔的痛苦，聽到自己也罹癌時他幾乎被擊垮。」阿勝也說：「若不是因為這兩個孩子，我早就沒有活下去的勇氣。」
問到就讀高一的長子阿杰要怎麼用消費券，他毫不猶豫地說：「留給爸爸繳醫藥費。」

兒女消費券助父醫病

阿珠語帶歉意地說，兒子從小都只穿兩、三百塊的布鞋，喜愛運動打球的他，布鞋常常穿個兩個月就壞了，之前打算用消費券買雙好點耐穿的球鞋給阿杰，也幫女兒買盞新檯燈，「沒想到爸爸一住院後，兄妹倆知道我擔心付不出醫藥費，主動說不用買東西，要留給爸爸看病，爸爸比較重要，看孩子這麼懂事，讓我們好心疼。」
阿珠說，自從阿勝發病到現在，已經有3個月無法工作了，房租每月12000元也已積欠了兩個月，生活費先向親友借款。
里長林添丁說，阿勝家的兩個孩子之前給親戚報扶養人口，今年低收恐無法通過，已先協助申請地方慈善會兩萬元應急，蘋果基金會訪視後，也撥款協助生活及房租欠費，並以「克蘭詩．蘋果助學專戶」協助阿勝子女完成學業。
