撐家漢腿癱 4口陷難關

賣屋償債 2子在學 妻嘆：這考驗太苦

2013年01月08日

「爸，你用力抬起右腳。」讀高二的小聖撐扶52歲爸爸邱明全身子，只見邱明全出力想挪動右腿，難以動彈。小聖跟記者說，一場突來意外變故，爸爸右腿癱了、家裡生活亂了，「老天爺給了我們一個難關考驗」。報導．攝影╱向高彬

小聖的46歲媽媽陳麗虹滿臉焦慮，她說，與先生這兩年在夜市擺攤賣現切水果，但近年景氣差，有時整晚掙不到600元，常連攤租錢都不夠，去年（2012）10月認賠收攤，11月她先找到包裝場計時工，做沒幾天，先生出門找工作卻出車禍，「先生頭撞傷縫合已無大礙，但他下半身右腿卻完全無力，轉診兩家醫院，做好多檢查，醫師判斷是左大腦中風造成」。目前邱明全意識清楚，左身手腳及右手可動，但右腿全無力，對車禍當時記憶一片空白。
陳麗虹說，還不知先生先腦中風、或出車禍再引發中風，醫師還在檢查，但已先開重大傷病，至少需復健休養半年以上，「我要照顧先生，大兒子讀高二、小女兒讀國三，房租家用開銷都困難，還得償還毀壞小貨車貸款，如今一個頭兩個大」。

「相信爸爸會好」

陳麗虹是印尼外配，她回述，先生曾做工廠機器維修，但產業外移關廠，7年前改做模板小包工，接連工程虧損，只得賣屋償債，「現在他撞車、又中風腿癱，老天給這考驗太苦了」。
採訪時，邱明全說，既然事情發生，不能怨天尤人，「現在只想趕緊復健，我沒有全身癱瘓，我還是可以工作的」。小聖說，爸一向盡力撐家，「相信爸爸會堅強好起來的」。
邱明全的大哥說，事故責任和弟弟傷病尚待警方及醫師評估鑑定，初步了解，弟弟這方車禍疏失多些，「我本來還能多幫忙，結果我心肌梗塞，裝心導管，兄弟住同家醫院。」邱明全的二嫂本身為肢障協會總幹事，她說，因明全居住及戶籍都非她服務區域，協會幫忙轉請邱明全所在區域的肢障協會關懷。

暫難獲車禍理賠

二嫂又說，「我們能力有限，之前已不時接濟，未料他又出意外，我們常去探望。」
肇事另方沈先生說，遺憾事故發生，交由法律裁決，盼邱明全早日康復。目前邱家公部門及學校救助申辦中，轉介邱家困境予蘋果基金會的肢體傷殘協會呂總幹事說，一家支柱傷病，尚須時日復健，有賴社會愛心協力，基金會訪視後已先撥款暫紓急窘。中華民國車禍關懷協會理事長沈志成律師認為，邱明全情況仍有待鑑定及調解事項，邱家恐難短期內得到車禍強制險理賠。
