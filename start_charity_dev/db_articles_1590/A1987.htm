心臟病嬤 帶孫上工拼下去

「只要有一口氣在 我不怕苦」

2009年09月27日

就讀國三的小萱，在鄰居眼中儼然像個「小媽媽」，早熟地打理家中大小瑣事，「阿婆脊椎不好，我幫忙洗衣服、拖地，加上阿婆不識字，弟弟妹妹的聯絡簿和功課我都會看，只希望她不要太操勞。」報導．攝影╱游竹君

小萱口中的「阿婆」，是她的58歲的阿嬤張秀嬌（阿嬌嬤），泰雅族人的阿嬌嬤住在山區偏遠部落，原與丈夫靠果園維生，2年多前丈夫病逝，1年多前兒子與村人離家私奔，媳婦也因重度憂鬱返回娘家休養，留下5個就讀高一、國三、小一、大班及4歲的孫子給她照顧。

沒工作憂養不起5孫

阿嬌嬤罹患心絞痛、心律不整等宿疾，加上脊椎壓迫神經，難以負重工作，今年初好不容易等到半年多元就業的工作機會，她每天帶著4歲的小孫子上工清掃步道，「我不怕吃苦，就只擔心沒有工作做、養不起5個孫子，只是工期到6月底已結束，還得等半年到一年才有工作機會，光想到這些，我每天晚上都擔心地睡不著。」多元就業的班長高小姐說，阿嬌嬤工作很認真，4歲的孫子也帶在身邊，「看到祖孫一起上工的樣子，實在很心酸。」
小萱說，阿婆曾經有一次掃地突然心絞痛，臉色發黑，還堅持要坐公車下山看醫生比較省錢，「才打針拿藥，顧不得休息，又趕回山上等弟弟妹妹放學，我聽了好難過，一直叮嚀阿婆，我和哥哥長大了，可以分擔。」阿嬌嬤堅定說：「這是天主給的十字架、祂給的考驗我都勇敢承受，只要有一口氣在，我還是會為了孫子拼下去。」

兒子失蹤媳娘家休養

校長王富美說，阿嬌嬤獨自努力工作養孫，讓村人都很心疼，她一直努力給孩子很多的愛，不希望孩子在父母婚姻破碎的陰影下長大。村長錢光輝說，由於兩個孩子監護權還在娘家休養的媽媽身上，但實際都由阿嬌嬤照料，也已將孫子父親通報失蹤人口，目前阿嬌嬤家領有老人年金、單親兒少及家扶共8900元，待改訂孫子監護權後，將再協助申辦低收。蘋果基金會訪視後，也已撥款暫紓生活困境。
