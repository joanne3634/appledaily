兒腦傷 嬤半夜挖蔥養家

照顧2幼孫 中風夫心疼妻

2009年11月23日

39歲阿典（蔡東典）12年前車禍癱瘓，長期臥床致褥瘡，腦傷嚴重智力剩10歲，3年前又患膀胱癌。67歲的母親陳欠嬤嘆說，去年老伴中風，家裡剩她和大兒子阿宗撐家計，但阿宗單親撫養兩名幼女，如今一家日子是越過越苦。報導 攝影╱仲芝蓉

陳欠嬤說，去年高齡89歲婆婆過世，小孫女開始上學，原以為可輕鬆些，不料老伴卻中風，「婆婆行動不便，那時我顧阿典又顧小孫女，光洗澡就要洗4人。」66歲的蔡格伯說：「妻子跟著我吃苦，現在我又變成她負擔。」

醫院家裡兩頭跑

阿典每2周得赴醫院換尿管，老伴中風後，右眼視力退化，右手無力，陳欠嬤說：「我一個人抱不動，和老伴合力扛，每次都累得心臟病快發作。」去年7月老伴住院，自己連兩周醫院家裡兩頭跑，回家餵阿典吃飯，再回醫院顧老伴，「連坐下吃飯時間都沒有。」

褥瘡嚴重可致命

陳欠嬤回述，阿典和阿宗當完兵就外出工作，阿宗擺攤賣錄音帶，阿典做水泥工，12年前冬至兩兄弟返家時，阿典騎車被撞，肇事者逃逸，事後領殘廢理賠100萬元，但當時住院2個月，醫護費就用掉一半，其餘這幾年已用罄。
阿宗說，弟弟受傷後，他返鄉種田顧家，9年前結婚生女，「卻因個性不合，在小女兒1歲時離婚。」阿宗嘆，為養育9歲和7歲幼女，到處打零工，但鄉下工少，每月勉強賺1萬多元養全家6口。
蔡格伯說，看到阿欠嬤幫人種蔥，半夜3點出門挖蔥苗，天亮去市場賣，每月勉強掙4、5千元，雖有老農津貼和阿典殘障補助共1萬6000元，但光阿典尿布、看護墊等耗材每月花近8000元。
醫院社工說，阿典褥瘡感染難處理，嚴重會引發敗血症致死，「勸送阿典去護理之家，但阿嬤說僅有的祖傳田地超過低收標準，家裡又付不起1萬多元自付額，只好自己顧。我們常多拿點尿布給阿嬤，減輕負擔。」當地李姓村幹事說，已安排居家服務員每周4次幫阿典擦澡，另協助減免阿宗女兒學雜費、申請單親兒少補助；蘋果基金會訪視後，也撥款暫紓困。
