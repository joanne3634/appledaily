「我要顧爸爸」6歲童餵癱父

母病無力工作 一家四口家計成愁

2010年04月30日

「爸爸，我跟你講話，你點頭或搖頭就好，啊，流口水了，我拿衛生紙擦擦。」6歲的小瑋活潑地跟中風癱瘓的爸爸說話。小瑋的媽媽楊芳妤跟記者說，孩子雖好動，但他對爸爸很貼心，有時鄰居拿碗麵來，他總想到先餵爸爸。報導．攝影╱向高彬

採訪時，楊芳妤聞到異味飄出，她問臥床無法說話的53歲先生林登雄說，是不是大便了，林登雄點了頭，51歲楊芳妤喚小瑋拿紙尿布幫忙，小瑋跑來衝去喊：「我要顧爸爸。」不過楊芳妤抬起先生臀腿換尿布，肩手力不從心，她說，去年底做完乳房切除手術，現在仍不時胸頸神經抽痛，「先生6年來3度中風，家裡又有兩個孩子，我能做工付房租掙家用的，都是洗碗端菜出力的工，現在我這情況，還不知道什麼時候才會好。」

母：夫癱我不能倒

楊芳妤說，去年底突發現兩側乳房有3、4顆小瘤硬塊，原本懷疑癌病，幸詳細檢查後為良性腫瘤，但瀕臨癌病變邊緣，「醫師評估仍有病變危險，就把乳房全切除。」
目前楊芳妤在家休養，每月回診追蹤，楊芳妤感嘆，算不幸中大幸，「要感謝神明保佑了，我都想，先生癱瘓需要我照顧，大女兒才讀國一，若我也病倒就糟了。」
只是先生重癱，楊芳妤苦惱原本生活已入不敷出，房租拖欠約5萬元，如今她病弱一時難做工，眼看復健家計成愁。當地公所社會課蕭小姐說，先前已撥「馬上關懷」3萬元協力醫病，「但因林登雄有家族共有持分難變賣田地，一時低收難過，我們已協辦兒少補助，現每月共領補助6800元。」蘋果基金會訪視後，也撥款暫紓林家病殘生活急窘。

出生兩個月父中風

楊芳妤述往，先生原以水電零工為業，晚婚也晚得子女，大女兒小珊現讀國一，就在小兒子出生後2個月，先生初次中風，原還能緩步行走，但前年底第三次中風後，先生雖意識清楚，但四肢全癱無法言語。
住對門老鄰居阿蓉婆說，眼看林家病弱，「我看了心疼，不時拿飯菜來幫忙，小瑋很乖，都說要先給爸爸吃。」當地里長謝文波說，先前阿雄經濟不錯，但7年前誤信好友借錢應急，卻跑路失聯，讓他周轉不靈致屋地查封，因此與家族失和沒往來，「很讓人同情，若有米油救助都先送來幫這家。」
楊芳妤的大姊租屋幫傭做工，「我也自身難保，但看妹妹家更難過，前幾天她騎車來跟我借500元，我說能力有限，也只有這些了。」
