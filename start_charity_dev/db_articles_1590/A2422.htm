癌婦截肢 為稚兒「活下去」

夫眼瞎失業 接連厄運 仍一路相伴

2011年03月22日

38歲的程月琴患骨癌，左大腿因癌擴散截肢，她懷裡攬著3歲幼子，「21年前爸媽接連死時，我曾一度想不開；現在，為了孩子，我怕沒命活下去。」報導 攝影╱向高彬

程月琴難過地說，前年底覺得腳踝腫脹，本想是不慎扭傷，先吃止痛藥，但都無效，一就醫發現竟是骨癌，醫師說可能免疫問題等不明原因造成，做了化療，「沒想到去年7月癌細胞迅速擴散至膝蓋，只得截大腿保命，目前持續回診治療。」

顧一家老幼難工作

程月琴與48歲先生謝先財育有3歲兒子小霖，她說，先生是獨子，家裡還與79歲公公，一家4口租屋同住，在她病後，先生為顧一家老病幼，一時無法工作，「雖然去年中開始有三款低收9200元，但原有微薄積蓄用光，現在下月房租不知在哪，有時我想不醫了，但孩子又還小。」蘋果基金會訪視後，也撥款暫紓醫病急困。

生活突變調幾崩潰

程月琴回述，自己曾有前段婚姻，前夫酗酒家暴離異，4年前她經介紹認識先生，再婚生了小霖，「他不酒不賭，對我和孩子很好。」
本以為找到幸福依靠，但未料災禍在前年接連襲來，先是前年中先生視網膜病變，右眼瞎了，隨即又因當時金融風暴波及被裁員失業，現在，換她自己病倒了，連番波折讓家裡生活變調，程月琴如今體力虛弱，甚至幾度心情瀕崩潰，還好先生一路陪伴。

「願照顧她一輩子」

先生謝先財說，太太命運坎坷，「跟她在一起時，就跟她說，自己沒錢沒屋，但願照顧她一輩子，只是這兩年變故太多，讓人受不了。」謝先財盼太太病快好轉，讓他可安心出外工作，把家重新撐起。
謝先財的大姊說，得知弟媳罹癌時很震驚，「我只能有時接濟小錢，弟媳住院時，幫忙帶孩子。」當地林姓里長說，謝先財夫妻與鄰居相處和善，公部門曾撥「馬上關懷」急難金3萬元，並協辦低收資格，也會持續關懷謝家。

後記
憶往事 天災奪走6親人

採訪時，程月琴哽咽憶21年前往事，1990年6月23日那天，中颱歐菲莉襲台發生土石流活埋30多人慘劇，其中她家9人有6人同時罹難，「那時爸爸遠洋捕魚，我在台北做工，大妹阿蘭在奶奶家逃過一劫，母親和3弟2妹都走了，後來爸爸太過悲傷，半年後也走了。」她和大妹一度也想輕生，但彷彿聽見媽媽喚「快回家」，打消念頭。只是無奈，逃過天災，如今沒躲開病魔，令人不勝唏噓。
