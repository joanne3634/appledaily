單親爸病垮 憂成兒負擔

靠國一子攙扶餵藥 嘆「顧不了家」

2012年02月28日

記者初見單親爸羅時濶時，52歲的他步履蹣跚，靠國一小兒子小岳攙扶緩走，訪後隔天，他再度住院，病床上他臉色憔悴，「醫師說我糖尿病併發一堆病，心臟不好，現在腎也出問題，再壞下去就洗腎了。」
報導．攝影╱向高彬

羅時濶表情黯然地說，這兩年糖尿病嚴重，把身體折磨壞光了，如今別說工作掙錢，連走路力氣都沒了，想到就著急，「小兒子讀國一，將滿18歲的大兒子做美髮設計助理，沒賺幾個錢，我這一年進出醫院，顧不了家，反而成了孩子負擔。」

糖尿病情況惡化

羅時濶說，18年前動胰臟手術，之後有糖尿病，近幾年惡化，6年前摔車傷右腿，前年心臟病發裝支架，如今腎不好，沒了力氣。醫院社工說，已協助醫費，轉介蘋果基金會，基金會經訪視後撥款暫紓窘迫。
羅時濶回述，年輕時曾做陶器製作，21年前至大陸幫友人做陶器廠工作，在當地娶大陸妻生兩子，後來工廠經營不善及與友人理念不合，13年前他返台後與妻離異，前妻已再婚另組家庭。之後羅時濶改送貨及當瓦斯工、加上低收補助過活，這1年身體已難出力做工，平常在家臥床休養，生活起居靠兩子照料，鄰居有時微薄接濟，親友能幫的不多。

向鄰居借錢度日

採訪時，大兒子說，自知不是讀書料，高一休學做美髮，已考取證照，剛可當設計師，但近期將準備服4個月兵役，「扣掉自己開銷，只能幫忙付水電費，爸最近一直進出醫院，家裡不夠用，爸爸還跟鄰居借錢。」小兒子雖調皮，但他勤快照顧爸爸的服藥飲食。
家扶社工說，羅家因羅時濶未達殘障標準、長子未就學，無法符合政府規定低收資格，目前每月領單親及家扶補助共5650元。

忍不了痛才就醫

社工說，「羅先生知道家裡沒錢，每回忍痛到受不了才就醫；他非常客氣也很顧孩子，最近病況加重，瘦好多，我們也持續關懷家裡。」

我有話要說

羅時濶說：「還在做事年紀，卻一身病，去年底我突然心肌梗塞，那時心想，老天就讓我走了吧，但又想，孩子小，擔子還不能放。做單親爸爸的，是有苦難開口，說了怕懦弱，只能往心裡吞。」
