婦熬病育子女 憂斷水電

家計靠兒薄薪 求神「保佑我堅強」

2009年04月13日

40歲的阿雲坐在客廳沙發上，許久不發一語，心想著，房貸繳不出來該怎麼辦？水電費沒繳，要被斷水、斷電了，種種生活困境湧現腦海，她急忙捻香向供奉在客廳的神明祈求：「王爺公，求你保佑我堅強活下去！」報導．攝影╱韓旭爾

記者踏進阿雲的住屋裡，對屋內的潔淨感到驚訝。阿雲說，自己罹患精神分裂症，沒法出門工作，幾乎整天待在家中，「天天在家裡不停地清掃、整理、拖地，如果不動手做事，我的腦子會胡思亂想。」

想賣屋也賣不出去

阿雲回溯說，她曾有過2次婚姻，分別生育現年22歲的兒子與12歲的女兒，第1次婚姻維持兩年丈夫便因肝硬化去世，13年前，經朋友介紹改嫁第2位丈夫，婚姻只維持了4年便因感情不和結束。離婚後，阿雲四處打零工維持生活，獨力撫養兩個子女。
提及精神分裂症發病經過，阿雲說，約在6年前，她開始覺得有人要害她，飯裡有人下毒，也感覺有人在監視她，「總想著結束生命後，這些煩惱就能一了百了，於是不斷地企圖自殺。」後經精神科醫生檢定，確認阿雲罹患精神分裂症，現每日定期服藥控制，阿雲狀況也跟著穩定下來。阿雲的長子阿豪說：「最近媽媽又因為家裡生活開銷問題常常情緒失控，我心裡又開始擔心。」
阿雲除患有精神疾病之外，1年多前發現兩邊腎臟各有2公分及6公分腫瘤，她說，醫生診斷6公分腫瘤已影響腎臟功能，建議割除，但往後恐需長期洗腎，「要是每周要洗腎3次，怎麼照顧小女兒？現在只能先吃草藥控制，能拖多久是多久，每周從台南到雲林拿藥，交通、藥費就需約1千多元，負擔實在很大。」阿雲說，為了應付生活、醫療開銷，已向友人借貸約10萬元，現也無力償還。
阿豪說，現在他在網咖打工，每月薪資約1萬7千元，連付每月1萬8千元的房貸都不夠，阿豪說：「我書讀的不高，又沒一技之長，也找不到穩定、收入更高的工作，真不知怎辦？」阿雲說，已想售屋另行租屋，但近來房市不佳，「想要賣也賣不出去。」
當地里長鄭峰鳴表示，阿雲因父母名下持有祖產農地，無法獲低收、單親兒少補助，連殘津也無法申請。

鄰里送食物解困窘

鄰里知道阿雲一家生活困窘，常拿來食物，安親班也免費幫阿雲女兒進行課輔，蘋果基金訪視後，已撥款暫紓困窘，減輕阿雲生活經濟壓力。
