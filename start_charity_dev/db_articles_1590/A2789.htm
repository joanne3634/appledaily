5口4病殘 婦「撐得好累」

打零工獨力扛家 國二女貼心分擔家務

2012年06月08日

「爸躁鬱症發作時會打媽，有次還拿刀，為保護媽我會把爸擋下。」阿萍的18歲長子說著，但他1個多月前因自殘行為就醫，疑因遺傳也診斷出躁鬱症，打零工持家的阿萍說：「我有糖尿病，全家老中青都有病，我撐得好累！」報導 攝影╱張嘉恬

「鏘鏘……鏘鏘」，阿萍家廚房傳來鏟子翻炒碰撞鍋子的聲音，阿萍就讀國二的女兒在廚房打理晚餐，女兒邊翻動鍋鏟邊說：「我還在念書，只能盡力分擔家務，以後想念高職，能工讀幫媽。」

丈夫不時要住院

採訪時，氣溫達攝氏35度，阿萍長子卻穿著厚外套，阿萍說，就讀夜校高一長子本在加油站打工助家計，但突出現割腕等自殘舉動，醫囑得長期服藥控制，同住的71歲婆婆也因心疾持有重度器障手冊，「就剩我撐著做，但包裝零工每月收入才1萬5000元，5口日子不知怎過下去。」
36歲的阿萍因與39歲先生倉促奉子成婚，婚後才漸了解先生罹病難工作，但堅信他會好轉才再生下女兒。先生急性發作時，得住院達3個月方可穩定，每月需自付伙食費5000多元，「長期以來都是靠公公支持，但公公去世後，很快也用完存款。因兒子也發作，先生又不時得住院……。」

每月都預支薪水

而阿萍也因糖尿病出現視網膜病變併發症，因視力模糊「只敢騎時速30公里，辛苦沒關係，只怕哪天我也倒了，家跟著垮」，阿萍全家生活依靠殘津、零工收入每月共約2萬5000元，難敷5口殘病生活，目前已積欠債務3萬多元，蘋果基金會訪視後已撥款紓困。
阿萍工作處老闆娘陳小姐表示，阿萍帶病又獨力撐家，家計入不敷出，「每月都得預支薪水，她自幼和外婆生活，小學沒畢業就工作，出嫁後更艱苦，工時常不夠扣抵預支的錢，我也能體諒。」

專家意見
躁鬱患者 治療後可工作

中市康復之友協會總幹事陳幗英表示，躁鬱症患者在精障者中具較高社會功能，「若服藥控制仍可工作，勞工局提供身心障礙者就業服務，可在就業輔導員協助下視個別交通、時間等可配合的情況，安排當地社福團體經營的庇護工廠做起，藉著從事清潔等工作獲得的獎勵金，漸回復自信，進行心理復健。」
