國二兒唱歌 盼癌母甦醒

夫顧妻難工作 子晚餐靠鄰接濟

2013年02月27日

春日初至，正是萬物甦醒時，53歲林文明喚著病床上的太太譚麗華，盼她從漫長冬眠睜眼，但47歲太太患腦部淋巴癌，癌細胞侵噬大腦，從去年10月發病昏睡至今，有時打呵欠或無意識夢囈幾句，沒再清醒過。報導．攝影╱向高彬

林文明說，太太去年9月不明原因暈眩摔車，後來常一坐下就昏睡，醫師查出淋巴癌，一顆4公分腫瘤在大腦深處、卵巢有兩顆大腫瘤，現先做化療，把腫瘤縮小，或許有3成清醒可能。「還好國二兒子小翰能照料自己，鄰居幫我盯著孩子，讓我少操心。只是這段時間沒工作，經濟已撐不下去」。多年鄰居阿城說，文明都在醫院陪太太，「做鄰居的幫點忙，晚上孩子來我家吃飯；看他家困難，於是幫忙打電話跟基金會求助」。蘋果基金會訪視後已先撥款紓困。

腦損嚴重若醒也殘癱

林姓主治醫師說，譚麗華神經中樞淋巴癌很少見，致病原因不明，可能曾受病毒或輻射感染，「病況或許可控制，但腦部受損嚴重，若醒仍可能殘癱，往後也需照顧」。
林文明說，他在小沙發工廠做木工師傅，工酬不定，與太太婚後生1子，有時太太做便當貼家用，雖欠銀行卡債，但日子尚可過，未料太太病倒。目前林家每月領第三類低收補助6600元，還不夠付房租，小翰的翁姓班導師說，知小翰母親重病，已協助申辦學產基金、張榮發基金會等救助金共約6萬元，用於之前醫病房租，「媽媽病前很照顧孩子，每天中午都親送便當到校」。醫院社工及當地社福中心皆已協力相關救助。採訪時，小翰說：「媽媽是1月生日，我和爸在病床邊唱生日快樂，相信她有聽見」。
與林文明合作20多年的老闆說：「文明的技術、態度都好，希望太太病好，他再回來做事」。林文明的兄姊就學子女多、難幫上忙。

「照顧自己不能病倒」

譚麗華的大姊有時微薄接濟，她說，妹妹早年曾有離異婚姻，與文明再婚後有平實家庭生活，「文明每回打電話給我，他一直掉眼淚哭，我安慰他，照顧好自己，不能病倒，妻兒還需要你」。

【我有話要說】

林文明：「太太病倒至今，看見鄰床癌病友陸續出院（指病逝）10個了，我常七上八下，也怕她怎麼了，只能求醫師盡力救。以前我和太太難免爭吵，現在我很後悔，怎麼不讓她，也寧願她再站起來，大聲跟我吵、罵我都好。」
