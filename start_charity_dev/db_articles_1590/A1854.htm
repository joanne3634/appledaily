夫罹癌 婦養家 存摺剩29元

一家四口 吃飯看病都成問題

2009年04月19日

近午時分，49歲許翠華煮好白粥，端到客廳讓罹患口咽癌的丈夫阿祥用餐，阿祥痛苦地張嘴，吞著稀飯，眉頭皺得化不開，喉頭癌症作祟，讓他進食有如針扎刺痛。報導．攝影╱韓旭爾

許翠華說，兩年前阿祥開始整天咳嗽，以為只是感冒，吃成藥也沒效，到醫院檢查才發現，原來是患開放性肺結核，經隔離治療約1個半月後才病情好轉，「回家後不到1個月，又覺喉嚨不舒服，結果發現罹患口咽癌第二期。」

「能有工作是萬幸」

許翠華說，經開刀切除腫瘤後，仍須進行化療、電療，「不知還要治療多久？」
許翠華說，為顧阿祥，她無法外出工作，只能靠手工貼補家用，但件量不定，近來每月僅剩約3000元工資，許翠華拿出存摺，本子上印著29元，她說，以前家裡雖不富裕，但省吃儉用還能過日子，實在想不到現在竟連吃飯、看病都走投無路，「最近我覺得心裡鬰悶、很怨嘆，一直好想哭。」
48歲的阿祥看見妻子為了家計發愁，內心愧疚不已，他說，妻子身體小毛病不斷，兩年多前騎車跌倒造成左肩韌帶斷裂，左手仍無力高舉，去年9月時因長了10公分大的子宮肌瘤而切除子宮，現不時喊肚子疼痛、脹氣，「可是她只要接到代工廠的電話，還是急忙騎車到工廠載貨，有多少就載多少，從沒考慮自己的身體能不能做得來，我覺得好對不起她。」許翠華說，景氣這麼差，能有工作就是萬幸，「工資雖低，但做整個下午也能賺到2、3百元，可以買幾個便當。」她接著苦笑說：「房租要錢、水電要錢、丈夫看病要錢，如果不工作，錢不會從天上掉下來。」

繳不起3000元房租

許翠華說，令她感到欣慰的是兩個分別就讀高二、國二的孩子都能體認家中困境，女兒小蓉讀高職建教合作班，每年6月至11月，學校安排半年時間的打工機會，扣除學費，每月約有1萬元收入來貼補家用。小蓉說：「我好希望能趕快畢業，早點有穩定工作幫家裡。」
當地里長盧萬敏說，目前許翠華一家領有低收入戶補助每月7200元，「房東同情處境，特別將房租減半，每月只收3000元租金，但他們還是繳不起。」蘋果基金會訪視後，先撥款協助許翠華繳清房租欠費以及一家生活、醫病急難。
