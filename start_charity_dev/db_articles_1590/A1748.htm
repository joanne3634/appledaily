小兄妹用功 撫慰腦瘤母

單親媽天黑眼更茫 「怕以後看不見孩子」

2008年11月30日

「媽媽，我幫妳搥背。」「哥哥，我先、我先啦。」現讀小二的小喆和讀小一的妹妹小歡，兩兄妹搶幫媽媽林春美按摩。41歲單親的林春美去年底患腦膜瘤，雖為良性，但術後仍常覺頭痛與視力漸模糊，還好有孩子貼心撫慰。報導．攝影╱向高彬

林春美看到孩子可愛模樣，反倒不安說，近兩個月視力不清，「天一暗，更模糊，醫師擔心腦瘤影響我的視力，我也怕以後看不見孩子，不能做事，怎麼養他們？」

丈夫胰臟癌病逝

林春美說，孩子的爸爸在3年前因胰臟癌病逝後，她便帶孩子回鄉間娘家附近租屋，以擺麵攤餬口，但前年起常頭痛，起初不以為意，去年底「連續三天左腦痛得要爆炸、又嘔吐，檢查才知長2公分良性腦膜瘤，我想到孩子還小，難過得一直哭。」
林春美說，今年3月進行放射治療，「醫師說腦瘤有控制住，但腦水腫要持續追蹤，不能太勞動和在陽光下曝曬，醫師說，若有變化就得開腦，不過我有高血壓，腦部手術有中風癱瘓可能，風險很高，孩子小，我怕有萬一，不敢隨便決定動刀。」
林春美曾從事會計工作，她回述，第一段婚姻生下1子1女，但因個性及財務問題離異，8年前她與高中老同學重逢相戀，兩人同居生小喆兄妹，未料相處時光短暫，小喆生父罹癌遽逝，「到11月剛好他過世滿三年，孩子到現在還常跟我說想爸爸，又天真講是不是把功課寫好、書讀好，爸爸會再從天上回來。」
採訪時，見有排灣族血統的小喆兄妹膚色黝黑、濃眉大眼，一會兒緊偎媽媽身邊，又不時調皮跟記者展示學校獎狀和漂亮美勞作品。
林春美單親撫養兩幼子，與小喆的中風癱臥奶奶少聯絡，如今她面對病痛，原本多靠娘家幫忙，但因7旬老父患心臟病、老母年初患血癌，仍進行化療中，如今兄妹能幫她有限；老母親秀鳳婆說，自己也病得渾身難受，「又擔心女兒生病，孫子還小，好煩惱。」
林春美第一段婚姻的女兒小淳現讀高三，她說，與媽媽仍保持聯繫，今年媽媽多次住院，是她與17歲的弟弟照顧，「爸媽雖不在一起，但她一直很關心我們；我也常去她家教小弟妹功課，我有打工，有時會給弟妹零用錢。」

尚欠卡債40萬元

當地鄉公所社會課表示，目前林春美與小喆兄妹母子三人，每月領補助共3900元；但林春美因之前有生活不足卡債約40萬元，需償債仍得強撐病體擺攤，讓她難靜養，蘋果基金會訪視後，也以「歐舒丹．蘋果視障專戶」撥款暫紓林春美醫病生活困窘。

暖流見證
在班上 是我得力助手

小喆在班上，是我得力小助手，只要需要幫忙，他搶著做。這次媽媽生病，小喆也很擔心。學校上個學期協助登錄「教育儲蓄專戶」獲5000元救助，這學期也協辦清寒獎助學金。
