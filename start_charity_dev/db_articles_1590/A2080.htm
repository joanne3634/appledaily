夫灼傷半殘 婦拼臨工養家

4口家月掙不到1萬 換藥耗費沉重

2010年01月24日

住家客廳內，30歲的阿姝（王津姝）正用繃帶幫忙嚴重燒傷的丈夫阿鈞包紮傷口，只見35歲的阿鈞低頭顯出痛苦表情。阿姝輕聲安慰說：「再忍一下就好了！」報導．攝影╱韓旭爾

阿姝說，去年9月，她帶著兩個分別7歲、4歲的孩子回娘家，只留罹患重度躁鬰症阿鈞一人在家中，「阿鈞雖精神狀況不穩，但平時還能自己照顧自己，沒想到會因吸菸不慎造成瓦斯氣爆。」阿姝接著說，氣爆造成阿鈞全身80％皮膚2至3度灼傷，現傷口仍未完全癒合，清潔、換藥好似人間煉獄，每天都必須承受撕裂傷痛。阿鈞說：「只要想到換藥心裡就皮皮顫（台語，意思為害怕），常痛到眼淚直流，痛得好想死掉，感覺力氣好像都被榨乾了。」

仍須長期復建與手術

阿鈞望著自己的雙手，手指因燒傷截指只剩4指，他無奈地說：「完了！這輩子什麼都不能做了，我已經廢了。」回想氣爆當時，阿鈞餘悸猶存，他說：「才剛點燃打火機要點菸，馬上就聽到『轟』的一聲，之後什麼事都不記得了，醒來時整個人包得像木乃伊，全身好痛。」阿鈞在燒燙傷中心住2個多月，直到去年11月中病況才趨穩定，因燒傷部位遍布全身，疤痕的攣縮使他的手臂抬舉困難、手指也難以抓握，後續還需長期復健及進行植皮手術。
阿姝說，丈夫原擔任作業員，5年前，阿鈞因晚上難以入眠而前往身心科就醫，經診斷發現罹患躁鬰症，之後病情每下愈況，根本無法上班，家計便只剩她一人獨撐，阿姝無奈地說，她原本在工廠擔任品管員，每月約有2萬元收入，但1年多前工廠倒閉後，便再也找不到穩定工作，只能四處應徵臨時工活，每月掙不到1萬元養4口家。

陽光基金會介入關懷

目前阿姝一家領有低收入戶補助及阿鈞殘障津貼共1萬1400元，政府每年也提供336個小時的免費到府居家照顧服務，陽光社會福利基金會社工張雅琪說，現阿鈞燒傷傷勢仍未穩定，近半年將會不斷地結痂、緊繃又出血，對他是非常痛苦的折磨。「為避免傷口沾黏，必須使用價格較高的石蠟紗布，每天約要300多元，經濟負擔很大。」阿姝說，除了阿鈞的醫療耗材開銷沉重外，氣爆當時造成住家2樓毀損，修繕費用約10萬元，且波及鄰居，需理賠約5千元，雖經政府「馬上關懷」急難金及慈濟協助部分費用，但仍欠約4萬元無力償還，蘋果基金會訪視後，已先撥款協助償還部分修屋費用。
阿姝疼惜看著身旁丈夫說，因居家服務員每天都會到家裡幫忙照顧阿鈞，她特別利用空檔找了一份兼差工作，每天工作4小時，每月約有8千元收入來貼補家用，「現在家裡開銷大，無論如何還是要想辦法靠自己的力量賺錢，能賺多少是多少。」
