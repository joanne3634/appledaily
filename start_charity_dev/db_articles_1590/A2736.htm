癌魔難纏 父嘆拖累孝兒

「從沒請過病假 現在被迫放長假」

2012年03月30日

55歲的高慶龍今年1月罹患上喉癌第四期，因氣切無法說話，只能用筆寫下病後心情，他表示，自己住院化療都靠30歲長子阿緯照顧，人家說養兒防老，自己不是老到不能動，卻得這種難纏病兩人都沒法工作！報導．攝影╱仲芝蓉

高慶龍的長子阿緯說，爸爸從去年底就咳嗽咳不停，要他去看醫生，他只說：「這點小毛病，去藥房買消炎藥吃就好。」又繼續咳了2個月，才注意到他右耳下方脖子腫起一塊雞蛋大小的包，沒想到是腫瘤。醫生說他腫瘤太大幾乎無法呼吸，緊急氣切手術，接著做化療和放射線療，等腫瘤變小後再開刀，「爸爸做木工從沒請過病假，沒想到被迫放長假。」

目睹同事意外返鄉

阿緯繼續說，自己跟著爸爸做木工，記者見阿緯身體健壯，問他怎沒想到外地蓋大樓，或許薪水高一些，阿緯回說，自己曾到台北做過建築工，「蓋高樓大廈工作危險性也高，自己在台北工作一年多，目睹同事不幸遭作業車輾斃，想起家鄉的爸爸老後還得靠自己，決定還是回鄉。」

家境苦弟車禍往生

阿緯回述，爸爸的工作收入不穩，爸媽常為經濟問題起口角，10幾年前離婚後就沒來往了。高慶龍接著寫到過去家況，表示早年生活辛苦，阿緯只有國中畢業就急著跟自己工作，幫忙分擔家計、供他弟弟讀完高中，但和他差1歲的弟弟去年車禍往生了，阿緯沒交女朋友，願意留在身邊照顧自己。

低收入補助難請領

高慶龍的同事潘先生說：「我們做木工的清晨5時30分就得到工地，只有中午吃便當時休息一下，接著做到天色昏暗才收工，現在很多年輕人吃不了苦，大家都羨慕阿龍教出好兒子，沒想到他突然罹癌，盼能早日康復。」當地公所急難救助承辦員說，高慶龍灌食費用每月約9000元，公所已撥「馬上關懷」急難金1萬元協助，並轉介蘋果慈善基金會，基金會訪視後已先撥款暫紓困境。
當地家婦中心社工說，高慶龍未滿65歲、阿緯也已成年，兩人都列計工作人口，政府低收入戶及相關補助都無法請領，阿緯照顧爸爸也難有穩定收入，盼民間資源介入愛心協力。
