兒罹癌 貧嬤拾荒掙家用

忍腰痛工作「還能做的一天就做」

2008年04月14日

台南市區的一處死巷子裡，44歲的陳德勝與67歲的母親秀蘭，忙著整理撿拾的資源回收物，「無法度，沒靠撿這個，就都沒生活費！」秀蘭嬤駝背埋首，臉上分不清是汗還是淚。報導．攝影╱孫敏聿

走進秀蘭嬤月租7千元的透天厝，電視、櫃子與椅子全是撿來的回收物，久為坐骨神經痛所苦的秀蘭嬤，腰椎根本直不起來，加上雙膝患有退化性關節炎，每走一步，就從腳底痛到腰，「為了我兒子、我兩個孫子，我還能做的一天，就得做。」

長子突死丈夫驟逝

秀蘭嬤說，家中這6年來很不平靜，先是赴大陸娶妻的長子莫名客死異鄉，辦完後事還未滿百日，深受喪子打擊的丈夫就接著因氣喘驟逝，而長女遠嫁宜蘭得育3子，么子患有僵直性脊椎炎，日益嚴重，卻為了3子仍勉力至噴漆工廠工作，「他們顧自己都顧不好了。」
一旁的次子陳德勝說，他兩次離婚後，帶著就讀高二和國一的兒子共同生活，曾跟父親討海捕魚，父逝後就幫忙養殖牡蠣，勉強度三餐，不料今年1月診斷出罹患肝癌第二期。「以前我一天抽三包菸，朋友找就去喝，現在真的很後悔。」

「幸好女友不離棄」

陳德勝罹癌後，住院1個月進行手術，切除部分肝臟，沒了工作收入，生活頓時陷入困境，「幸好我女朋友沒有離棄我，還願意當我免費看護，大兒子也懂事去打工賺零用錢，是他們，我才有活下去的勇氣。」陳德勝說，相識3年多的女友靠著擺攤賣青草茶及幫忙市場刷地等清潔工，賺取微薄收入，「但我病了，不想拖累她，也鼓勵她有好人家快點去嫁。」
台南市社會局3月已撥1萬1800元急難救助金，低收入戶仍在申請中，縣府社工表示，「若低收通過，陳家每月可領5800元補助，若無法通過，將轉申請為期半年、每月3千元的弱勢兒少補助。」蘋果基金會已先行撥款暫紓生活困境。

無法計算的人生

陳德勝：「以前和朋友喝酒跑攤絕不腳軟，而今病了，看到兩個兒子痛哭、母親撿回收維生、女友願意照顧我，我想，如果還有來生，我發願當個顧家的好男人。」
