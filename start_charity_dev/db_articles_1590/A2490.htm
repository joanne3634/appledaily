「為了我們太累 媽才中風」

兩子失依 零工掙錢

2011年06月14日

「命不好啦。」阿錦說，丈夫10年前病逝後，留下1對分別16歲及15歲子女，她靠著打掃、洗碗零工等勞力粗活每月掙得1萬多元讓孩子吃飯、上學，怎知會中風倒下，「我才國小畢業，只能靠出力流汗賺錢，手沒力，就等於斷了謀生能力。」報導?攝影╱韓旭爾

目前就讀高一的長女小珍說，爸爸過世時她才6歲，對爸爸沒什麼記憶，印象中就是媽媽每天幸苦工作，「若不是為了我們，媽媽也不用這麼苦，每天工作到晚上9時、10時才回家，說不定因為太累，媽媽才會中風！」

難忘母昏迷情景

「當時我還以為媽媽會丟下我們離去！」小珍憶起媽媽中風仍心有餘悸，她說，多年來，1家3口都是同寢而睡，每日早上6時媽媽會準時叫人起床，「比鬧鐘還要準時！」但當日已6時多，卻見媽媽仍昏睡床上，她用手輕拍幾下，媽媽全無反應，「我直覺反應媽媽生病了，馬上叫救護車送她到醫院。」小珍說，看到媽媽現在還能有說有笑，感覺一家人能擁有健康就是最大的幸福了，「媽媽中風昏迷不醒情景一輩子會印在我的心中。」

陪母工作才知苦

阿錦就讀國二的兒子小全說：「國小6年級時，有天陪著媽媽到餐廳洗碗、端菜，我才發現她有多辛苦。」小全說，從此他每天放學後，就到餐廳分擔媽媽工作，直到1年前他因騎腳踏車跌倒左手骨折，痊癒後仍會感覺痠麻，只能中斷到餐廳幫忙，但現在他每天放學後還是會到相框工廠打工補貼家用。工廠吳姓老闆說，跟一般國中孩子相比，小全不會沉迷漫畫、網路遊戲，每天騎2、30分鐘腳踏車到工廠打工，「實在很可取。」

術後無力難工作

阿錦的大伯說，阿錦一家雖多年來與中風的媽媽同住，但親戚們也清楚阿錦婆媳間關係不好，長期經濟都是阿錦他們自己想辦法籌措。中風的媽媽現由自家3個兄妹共同出錢聘請外勞看顧起居，原本阿錦還能工作時，她也會分擔部分費用，「只是現在阿錦病倒，我雖偶爾能幫忙接送就醫，但自己以零工為生，在金錢上實在出不了力。」
醫院社工說，阿錦中風術後，短期內仍無法勝任粗重工作，一家3口靠低收入戶、家扶補助每月共10600元為生，因此轉介蘋果基金會協助，而基金會訪視後，已撥款暫紓阿錦一家生活、醫療窘迫。
