男車禍 「撞掉平順生活」

高一長子打工月賺萬元 撐4口家計

2012年07月27日

37歲的阿袖（簡小袖）一面幫車禍傷腦的丈夫阿龍按摩手腳，一面嘆氣地說，5月底時，阿龍酒駕騎車撞路旁卡車，幸好沒傷到他人，「阿龍喝酒闖禍，家裡少一個賺錢的人，我要顧他也無法工作了！」報導．攝影╱韓旭爾

阿袖說，夫妻倆育有2個分別就讀高一、國一的兒子，多年來，阿龍開車送貨每月約有3萬多元收入，她則在工廠擔任時薪零工每月掙得約1萬元，夫妻雙薪維持家計，一家4口還能安穩過日子。5月時，46歲的阿龍下班後去找朋友喝酒談天，返家途中發生車禍，造成顱內出血，現開刀取出部分顱骨尚未放回，雖已清醒，但記憶、認知能力受損，未來還有漫長復健路要走，生活起居都要有人照顧，「他這一撞，把家裡平順的生活撞掉了，心裡難免會有埋怨，但是事情遇到了，我只能硬撐下去。」阿袖無奈地說。

躁鬱又罹糖尿病

回溯過往，阿袖說，阿龍因長期沒日沒夜南北奔波送貨討生活，工作壓力大，約4、5年前出現失眠、長期頭痛和情緒低落情況，「時常開了一整天的車子，嘴巴唸著好累，卻都睡不著。」她說，有時阿龍變得脾氣暴躁，甚至出手打人，到身心科就醫，醫生診斷罹患躁鬰症，需定期服藥控制。阿袖接著說，這次腦部受到重創後，阿龍的脾氣變得更壞了，「我感覺他躁鬰症症狀加重，醫生又發現阿龍有嚴重糖尿病，每日需施打胰島素控制血糖。」

至少要復健1年

記者問阿龍記不記得車禍情況？阿龍回答說：「我哪有發生車禍！不知發生什麼事？我的腦袋竟然變成軟的。」接著用手指輕按沒有顱骨的頭皮，阿袖見狀急忙拉住阿龍，要他不要亂按。醫院社工說，阿龍至少需要復健1年，記憶及認知能力才能逐漸復原，「能恢復到什麼程度還不知道？也要待腦部水腫消除後才能將顱骨裝回。」社工說，雖阿龍有勞保可請領傷病給付，但因之前住院時間不長，理賠僅約3000元，醫生也正進行殘障手冊的評估中。
阿袖二姊說，自家共有4個兄弟姊妹，阿龍車禍後，她不定期拿個2、3千元資助，但自己是個單親媽媽，能幫有限。阿袖就讀高職建教班的長子小哲說，6月到9月可打工，扣除生活開銷，每月約有1萬元可幫忙家計，「希望能減輕媽媽經濟負擔。」蘋果基金會訪視後，已先撥款暫紓阿袖一家生活、醫病窘迫。
