婦顧一家老病 棲陋屋苦撐

夫癱子幼 公婆殘疾 5口借貸度日

2010年06月14日

「阿立，幫你倒水，要吃藥了。」43歲的賴貮立因糖尿病須洗腎、左眼失明、手腳神經病變癱坐輪椅，生活起居得靠44歲的太太阿玲照料，但家裡孩子才讀國二，同住爸媽也因中風和骨刺手術失敗致行走不便，他愁煩生計難熬。報導 攝影╱向高彬

賴家居住的鐵皮屋裡潮霉殘舊，房裡沒幾件家具，有台至少超過20年歷史的古董級14吋電視機，賴貮立用他僅存右眼視力說，家裡3個病人，有點聲音，比較熱鬧，不然雙手連推輪椅力氣都沒有，躺床上悶得不知還能做什麼。
他說：「每周一三五出門，太太帶我去洗腎，現在已不敢想身體還能變好，只希望別再惡化，至少還留隻眼睛，看著孩子長大。」

照顧家人無暇上工

賴貮立回述，和太太阿玲僅生1個現讀國二的兒子，他原本開鐵工廠，鄉下地方雖工活不多，但總算勉強可過，只是3年前他糖尿病發洗腎後，無力再做，只得歇業關廠，沒想到病變持續惡化，前年手腳末梢神經萎縮，無力行走，吃飯都得人幫忙餵。去年左眼失明、剩右眼也視茫茫，這段期間生活全靠勞保理賠和微薄積蓄撐持，最後還跟親友借了16萬元。
阿玲原還偶做麵攤洗碗零工貼補家用，但賴貮立說，同住的63歲母親中風4年，年初漸有失智情況，73歲父親也在去年中因頸椎骨刺手術失敗，行走不便，「阿玲也沒法出去做事了。」

「只能求神多保佑」

採訪時，見賴貮立的母親蜷躺長椅上，眼神渙散的她喃喃自語表示手腳癢，用紅藥水不停塗抹手腳，一旁她老伴阿文伯念著，要她別一直擦，滴得滿地都是。阿文伯感嘆近年家運歹，家人紛病，小兒子也因刑案重罪入獄，「也只能求神多保佑，看家裡能不能早日平順點。」
賴貮立的大哥阿一在外租屋做雜工育子，他說，現在搬磚工少，常一個月不到10天，自家都難過活，「幫不了家裡忙，也很難過。」當地李姓村幹事說，阿玲列記工作人口，目前公部門僅列賴貮立的兒子具三款低收，家中老弱現每月共領補助1萬3200元，游姓村長說，若有米油救濟物資時，會多發一份給賴家。蘋果基金會訪視後，也從「視障專戶」撥款暫紓生活窘迫。
