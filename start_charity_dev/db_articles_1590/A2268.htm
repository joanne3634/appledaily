病婦穿鐵衣 忍痛育精障兒

夫女棄世 婦無力謀職 「不敢想以後」

2010年09月16日

靈堂前，61歲的阿英嬤一如往常地幫丈夫阿俊準備菜飯、毛巾，但卻再也聽不見他安慰的言語，她不禁悲從中來地說，大女兒、丈夫相繼自殺身亡，家裡還有精神分裂的兒子要照顧，「我不敢多想以後的生活。」報導 攝影╱游竹君

阿英嬤說，與丈夫阿俊育有3名子女，長女7年前燒炭身亡、精障的長子5年前心肌梗塞突然病逝，32歲的次子阿賢10多年前車禍腦傷導致精神分裂，不料，2年前丈夫阿俊清潔貨櫃時又不慎摔落導致腳後跟粉碎性骨折，只能拄拐杖緩步行走。阿英嬤難過地說，阿俊生前連走路都吃力，卻還體貼地說，「我會趕快好起來、可以幫忙煮飯，你就不用這麼辛苦，但沒想到7月初他卻想不開喝鹽酸自殺，不想再拖累我們母子。」

長年吃藥撐病體

阿英嬤說，阿俊自殺後送往加護病房，口腔食道嚴重灼傷，無法說話，只有眼淚不斷從眼角溢出，精神分裂的次子阿賢焦慮地在加護病房裡進進出出，嘴裡喃喃說著：「爸爸，我會乖乖吃藥、聽媽媽的話。」但阿俊仍在8月8日病逝，喪葬事宜已由當地慈善會協助完成。
記者訪視時，領有中度精障手冊的阿賢，連扣襯衫扣子的簡單動作也得花兩、三分鐘才能完成。阿英嬤說，阿賢情緒起伏不定，也需分神照顧，自己早年也有腰椎滑脫、退化性關節炎的毛病，但忙著照顧丈夫、兒子，長年只吃止痛藥、穿鐵衣硬撐，短時間恐也無法再工作。當地精神康復之友協會主任表示，像阿英嬤接連經歷家庭重大變故，更需有人持續關懷、讓悲傷的情緒有出口，協會社工也將協助阿賢穩定就醫。

鄰居也心疼救濟

熟識阿英嬤的鄰居阿姍說，鄰居很心疼阿英嬤情況，鄰里間已主動募捐約6萬元讓阿英嬤繳納醫療費用，當地里幹事說，已協辦「馬上關懷」急難金3萬元，阿俊病逝後，每月僅賴7000元殘障津貼實在難敷病弱母子兩人的家計開銷。蘋果基金會訪視後，也已撥款協助生活及醫療開銷。
