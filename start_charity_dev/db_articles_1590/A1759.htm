高三女心繫遺言 含悲撐家

精障母出院團圓 病父竟不忍截肢自殺

2008年12月17日

「妳要代替爸爸好好照顧弟妹和精障的媽媽。」看著爸爸阿立的遺言，讓就讀高三的小蘋在喪父劇變中，強忍哀痛、擦乾眼淚面對未來。她難以置信地說：「沒想到爸爸因將截肢而想不開，留下我們就這樣離開。」報導 攝影╱游竹君

享年45歲的阿立，5、6年前因工作從二樓鷹架上摔下來，導致右腳嚴重骨折受傷，無法工作。小蘋說，這幾年來，爸爸的右腳有著巴掌大的傷口一直無法痊癒，10月中旬右小腿甚至嚴重發黑，「醫生診斷是骨髓炎，擔心細菌感染全身，必須要截肢保命，也許這個打擊對爸爸來說太大，3天後他就選擇跳樓自殺。」

長子擬復學要父安心

由於家中經濟一直都不好，17歲的長子小祐今年8月辦理高二休學，爸爸走後，他在心裡對爸爸說：「爸爸，對不起，對不起，我到最後一刻都還為了沒找到穩定的工作跟你吵架，我答應你，我一定會照顧好這個家，明年也會復學、完成學業，你安心離開吧！」
小蘋說，爸爸生前擔任鐵工，常四處做工才能養活全家，「小學、國中期間，我和弟弟、妹妹都在親戚家搬來搬去生活，有精神疾病的媽媽則在高雄的養護中心養病，最近3年來媽媽病況比較穩定，終於回家同住，我們也才能一家團圓。」
導師余佳珍說，小蘋家經濟長期不佳，每學期3、4千元的美髮材料費都由班費先墊繳，「小蘋很有責任感，都從日常開銷一百、兩百省下來慢慢還。」老師說，學校得知小蘋家中變故，董事長已承諾全額補助小蘋不足的學費，也已申請教育部「學產基金」應急，讓她安心完成僅剩半年的高中學業。余老師說：「班上同學發現，小蘋為了省錢，中午常不吃飯，同學們擔心她的健康，便會主動找她一同分吃便當，跟她聊心事，希望陪著她度過人生難捱的時刻。」

親戚嘆「難長期協助」

里長胥樹文表示，由於小蘋父母在濱海老家尚有利用價值不高的農地、難以處理，導致低收無法通過，喪葬費也靠著民間慈善單位幫忙才完成，之後每月殘障及單親兒少、家扶補助共1萬200元。姑姑阿雪表示，以前兄弟姊妹知道阿立生活難過，偶爾湊幾千元貼補家用，「但是現在景氣差，自身難保，也難長期協助小蘋姊弟的生活。」蘋果基金會訪視後，已撥款暫紓小蘋家的困境。

後記
夢見爸沒拄柺杖 面帶微笑

採訪時，就讀小六、寡言內向的小婷，一直安靜地在旁聆聽，直到最後記者問小婷有沒有夢見爸爸時，她突然眼睛一亮地說：「我有夢見爸爸站在廚房裡做飯給我們吃，他不用再靠柺杖就能走路，而且面帶微笑，好像不再痛苦了。」話說完，小婷頓了頓，又悵然若失地補上一句：「可是我才一睜開眼，爸爸就不見了。」霎時間，家中又陷入一陣哀傷靜默，從頭到尾獨立、旁人難以察覺情緒的姊姊小蘋，此刻正暗自低頭拭淚。
