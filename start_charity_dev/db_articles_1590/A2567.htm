母助癱兒復健 盼坐起來

「求上帝憐憫 讓我有臉見先人」

2011年09月14日

27歲的阿諺因車禍癱瘓臥床而手腳關節僵硬，57歲的母親吳秀英幫忙活動手腳做復健，阿諺卻一會兒擠眉咧嘴笑、一會兒又痛得脹紅雙頰，張大嘴卻只能吐出氣音。報導．攝影╱仲芝蓉

母親吳秀英心疼地說，阿諺今年2月酒後騎車自摔，左腿粉碎性骨折、左肩神經叢損傷，雙腳癱瘓無力，「阿諺當初住了兩個月加護病房後才脫離險境，但仍無法說話，全身上下只剩右手能動。」吳秀英接著說，阿諺雖然不能說話，但和他說話他都知道，「我每天祈求上帝憐憫阿諺，至少能坐、能照顧自己，以後我和丈夫才有臉面對天上的老人家。」

兒精障發病10年

講起阿諺的事，吳秀英難掩心酸，阿諺原本就受精神疾病所苦，17歲那年原因不明發病，常說有人在他耳邊講話，要他去報仇，「家裡也沒人得精神障礙這種病，我偷偷帶阿諺去廟裡請道士做法，不敢讓基督徒的丈夫知道，但求神問卜都沒效，阿諺不斷進出醫院，10年就這樣過了。」
吳秀英繼續說，今年初阿諺精神狀況穩定，才出院回家休養，卻發生車禍。
吳秀英回述，和64歲的丈夫原生有2子2女，阿諺是次子排行老么，長子5歲那年在溪邊玩水溺斃，「阿諺是我們家唯一的命根，他發病後，我辭掉工廠工作，家裡靠丈夫做挖隧道工程，我在空地種菜、種辣椒賣，生活勉強還過得去。」吳秀英的丈夫阿財伯說，自己年紀大了，沒力氣繼續做粗工，2年前阿諺病況穩定時，他還去科技公司上班寄薪水回家，但他的工作都沒法超過半年，發起病來同事都怕他，只好返鄉休養。

月剩5千元生活

阿諺大姊說，自己和妹妹都嫁到外地，只能常打電話回家關心弟弟狀況。長照中心潘先生說，阿諺還年輕，建議積極復健，但吳秀英一家目前每月補助僅1萬元，扣除尿布和回診復健交通，就剩5000元生活，公所和醫院已申請「馬上關懷」及民間急難金共4萬5000元協助，蘋果基金會訪視後，也已撥款暫紓困。

我有話要說

吳秀英說：「阿諺發病時打人、也鬧過警察局，這麼多年來我們都靠信仰力量撐過來；只是萬萬沒想到，我和丈夫都老了，阿諺卻癱了，這個擔子實在太重，快背不動了呀！」
