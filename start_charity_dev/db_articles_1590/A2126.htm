妻罹癌 兒無依 癱男痛哭

每月剩7千元養4口 「恨不得馬上去工作」

2010年03月24日

今年大年初二，48歲于德珠因癌症放射性治療造成嚴重血尿，送急診住院治療，就在她煩惱自己久病難癒，4口生活陷困之際，雪上加霜的是，年初六晚上10時多，于德珠接到護士通知，52歲的丈夫阿駿（朱為駿）在工作時中風，也緊急送到醫院急救。報導 攝影╱韓旭爾

所幸阿駿病況已受控制，並無生命危險，但右側手腳癱瘓無力，後續須長期復健。

國一兒奔波忙照料

看著丈夫氣虛體弱地躺在病床上，于德珠不捨地說，阿駿原本身子就不好，曾有心肌梗塞病史，這2、3個月來常念著自己手腳無力，講話也像含著滷蛋一樣不清楚，「大年初六下午，當時他來醫院探望我時，就叫他順便掛號檢查，但阿駿只隨口回應一句：『工作要緊，改天再去吧。』」沒想到才隔5個多小時，阿駿就倒下了。
于德珠說，她與阿駿共育有1女1子，分別為16與14歲，一家原本靠阿駿從事保全工作每月約1萬5000元的收入維持生計，自己則是從事清潔零工，每月掙得7、8千元貼補開銷，兩人靠著微薄收入勉強度日。但半年多前，于德珠發現罹患子宮頸癌3期，家計重擔便落在阿駿一人身上。
面對夫妻倆雙雙住院治療，阿駿心急又自責地說：「如果身體健康，只要肯吃苦，生活沒問題，只是我和妻子輪番生病，現在孩子的生活被我們拖累。」所幸2個孩子懂事，夫妻倆同時住院，就讀國一的兒子小邑請假在醫院照顧父母，兩間病房來回跑，幫忙餵飯、擦拭手腳，小邑說：「爸媽平時為了撫養我們姊弟兩人辛苦工作，現在病倒了，我一定要好好照顧他們，讓爸媽趕快好起來。」

擔憂日後復健費用

于德珠說，現一家收入僅有政府低收入戶及慈濟補助每月共1萬2200元，扣除5000元的房租，就剩下7000多元。醫院社工說，于德珠夫妻因有低收福保身分，住院醫療開銷不多，但阿駿還有嚴重糖尿病，每月約需1000多元購買血糖試紙，未來還有來回醫院的復健交通費問題，經濟負擔不輕。
阿駿任職的保全公司主管說，阿駿工作認真，將來病況穩定後，願意再給他工作機會。阿駿說：「我恨不得馬上站起來去工作，好讓妻子能安心抗癌，孩子正常求學。」蘋果基金會訪視後，已撥款暫紓醫病困境。
