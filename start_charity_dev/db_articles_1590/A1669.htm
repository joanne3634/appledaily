癌女乳潰爛 借住漏水屋

沒錢延誤治療「痛起來像刀割」

2008年08月26日

「痛起來像刀子一直在割、在刺，整個人、整個頭鬧哄哄，根本沒辦法睡，我也只能求菩薩，求祂讓我減輕痛苦。」52歲的賴信諺患乳癌三期已近四期，獨身的她如今無力工作謀生，也擔憂往後抗癌路如何走下去。報導 攝影╱向高彬

賴信諺如今借住姊姊阿玉家，但阿玉家也不好過，兩層樓老屋破漏，7月底鳳凰颱風來襲時，二樓用了4、5個水桶接雨。賴信諺感嘆病弱，屋漏又逢連夜雨，「有時想，如果能好，就早些好，我有力氣幫姊姊做事；若不能好，就早點讓我離開，結束一切。」

姊還得顧躁鬱女兒

賴信諺指著左胸說，目前約10公分腫瘤部位潰爛嚴重，正進行化療與電療，「醫師看到訝異說，妳真能忍，這麼嚴重才來治療，我心裡想，是真沒辦法了，一來怕面對，二來也因為沒錢。」
賴信諺說，前年底就發現左側約3公分腫塊，但當時不太痛也不以為意，直到去年底潰爛化膿，左腋下又另冒出一顆腫瘤，她心裡有數大事不妙，趕緊借錢繳健保欠費，回花蓮老家投靠姊姊阿玉，開始接受檢查治療，如今她三餐得靠姊姊張羅，喝不起營養品，「因她要顧一個嚴重躁鬱症的女兒，已經很不好過，我卻連買100、200元紗布都要跟她拿錢，很拖累她。」

「可能要賣房子了」

賴信諺回述，從小家窮，沒讀什麼書，年輕時做遊覽車小姐、擺攤賣過女裝，也到北部做餐飲廚工，10年前在南部開小咖啡店，因經營不善虧光積蓄，「兩手空空回台北找工作，過了40歲，麵攤洗碗工都不好找，只能做點雜工租屋餬口。」
同樣未婚的姊姊阿玉說，年輕時收養一女，但現年30歲的養女卻在5年前躁鬱症發作，為顧她已夠煩心，現還得支應妹妹醫病開銷，而她擺攤賣早點飯糰，一天賺不到500元，「可能要賣房子了，租屋、醫她們的病，走一步算一步吧。」
當地公所社會課表示，因賴信諺現無殘障手冊，無法辦理殘津補助，當地里長劉秀香說將先協辦急難救助。蘋果基金會訪視後，也先撥款暫紓賴信諺醫病急窘。

後記
念佛消業「好受些」

採訪時，賴信諺跪在寶相莊嚴的觀音菩薩像前說：「以前不懂，很困惑，後來想通凡事因果輪迴，該還的、受苦的，是我的業，這樣想，心裡好受些。」遇到難題當成是「業」，念念佛，跟過往一筆勾消，或許是目前獲得安定的最好方法。
