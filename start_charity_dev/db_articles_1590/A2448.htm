妻洗腎夫罹癌 「怎養孩子」

住家偏遠 津貼4000元 只夠就醫一趟

2011年04月22日

39歲的阿珍洗完腎拖著疲累的身軀，看著罹癌正接受化療的丈夫阿成，無奈地說：「阿成2年前脖子上長出囊腫，今年2月才檢查出扁桃腺癌，我們倆都病了，該拿什麼養孩子。」
報導．攝影╱仲芝蓉

今年42歲的阿成說，脖子上的囊腫不痛不癢，直到去年底腫得像拳頭那麼大，才覺得不對勁，原以為是腮腺炎打針消腫，不到一個月又繼續腫大，才發現是癌症。當地家婦中心高社工說，阿成以重大傷病身分可減免醫療開銷，但他們家位處偏遠鄉間交通不便，就醫包車來回要4000元，阿成夫妻還育有2女1子分別就讀國三、國二和小六，阿珍照料丈夫也無法工作，目前一家僅領有阿珍重器障津貼每月4000元，光付車資就不夠生活了，將協助申請「馬上關懷」急難金，蘋果基金會訪視後，已撥款暫紓困境。

為多賺千元上夜班

阿珍說，原本一家5口在都市生活，丈夫在摩托車零件廠幫忙送貨、自己先後在電鍍廠、塑膠廠工作，兩人收入合計約5萬元，貸了款買房子，「我為了多賺2、3千塊上大夜班，卻沒想到3年前急性腎衰竭開始洗腎。」阿珍嘆口氣接著說，後來房子也保不住，只好搬回鄉下娘家跟母親住。

夫考取看護卻生病

阿珍的母親說，鄉下地方難找工作，阿成只有靠養雞、種西瓜等零工收入不穩，他一直說很不好意思帶一家5口回來白吃白住，但天下哪有父母捨得自己女兒在外吃苦，去年4月阿成很爭氣考到看護執照，「我和女兒開他玩笑，說他一個大男人粗手粗腳當看護，沒想到生活才剛開始平穩，阿成卻病了。」

盼兒就學不當貧戶

阿珍的母親心疼回述，阿珍有4個弟弟、2個妹妹，她生父在她4歲時病逝，小學才畢業繼父也病逝，因家貧阿珍未升學到外地工廠工作，「阿珍那麼認真努力，卻把身體拼壞」。阿珍接著說，自己小時候苦過來不怕再吃苦，無論如何想讓3個孩子至少高中畢業，不要再當貧戶。 阿珍次女小儒的導師姜亭安說，「小儒是個心思細膩的孩子，去年教師節還發動全班同學寫賀卡送我，讓我很感動，沒想到父母卻相繼病倒，學校將協助申請急難救助。」

暖流見證 母做清潔工 女兒也幫忙

洗腎中心護理長羅月梅說，阿珍3年前開始洗腎，知道她生活辛苦，於是讓她來洗腎室做清潔工，「阿珍很珍惜工作機會，她的女兒放學後也會來幫忙，真的很讓人感動。」
