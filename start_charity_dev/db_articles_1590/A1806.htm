婦顧病夫 長子零工撐6口

月入不到萬元 好心鄰居接濟青菜

2009年02月23日

44歲的尤美珍在簡陋平房裡，忙擦洗丈夫便溺床上的大小便，「先生肝硬化嚴重，造成肝腦病變，意識變得不清楚。」她還煩惱的是，有3個孩子在讀書，能做事的大兒子卻零工極少，一時家用成愁。報導 攝影╱向高彬

尤美珍撐著51歲丈夫阿輝起身，一口口餵他吃稀飯，阿輝臉色蠟黃，有時慢慢咀嚼還會不小心嗆到，尤美珍趕緊替丈夫拍背，她說，丈夫肝硬化造成代謝異常，神經發生肝腦病變，精神狀況不佳，最近已連房門在哪兒都搞不太清楚，大小便也無法控制，「想拉就拉，我忙著洗褲子、床單和擦地板。」

每月醫費至少6千

睡整天，意識稍好時又扶牆到處亂走，「有時亂轉瓦斯點火、有時到門外亂脫衣褲，不然就突然喊痛，要快去醫院急診。」見他們家廚房燒黑一小角，就是前幾個月阿輝燒的，由於丈夫不時出狀況，讓尤美珍只得隨時照料，難外出做事。
尤美珍回述，夫妻育有分別23、19、14及9歲的3子1女，丈夫原從事碼頭工， 7年前患肝硬化體弱，改做板模零工，勉強餬口，但這兩年先生病況更差，已無力工作，原本還靠她及大兒子阿華做零工維持家計，「景氣很差，阿華去工廠一直應徵不上，3個月前經朋友介紹做民宿房務清潔工，洗一間100元，但房客少、工就少，一個月常連1萬元都賺不到。」她還說，幸好阿華懂家裡苦，微薄工資幾乎全給家裡，自己在外僅花吃飯和機車油錢，但如今阿輝病況重，每月自費藥物約6000元，又不時進出醫院，積累約6萬元債務。
阿華說，媽媽有高血壓，爸爸又病重，家裡如今僅靠他做事，「1年前做過綁鐵零工，一天工錢1000多元，那時每月還能賺近2萬元，但現在工程都停了，想做也沒得做。」

低收資格仍待審核

鄰居麗花姊說，美珍家孩子都乖，有時衣服破了，也沒錢換新衣，所以她常拿些親戚送的二手衣或青菜幫忙他們，「前些時候發消費券，他們也都給媽媽還欠村裡雜貨店的賒帳，再買些家裡的日用品。」當地俞姓村幹事說，尤美珍與大兒子列計工作人口，所以低收未過，但鄉間工少，現正協辦低收資格審核中。蘋果基金會也撥款暫紓阿輝家急窘。
