父亡母病 10歲孝女憂家計

安慰媽媽「要和我一樣乖爸才走得安心」

2008年05月13日

對10歲的小筠來說，剛過的母親節很悲傷，因為才送走了長期父代母職的爸爸，更讓她憂心的是，媽媽精神狀況不好，阿公已86歲了，未來如何照顧得了她和妹妹與媽媽。
報導˙攝影╱向高彬

讀小四的小筠說，爸爸在5月4日因頸部淋巴腺癌病逝，將在5月17日（本周六）出殯。去年母親節，她曾寫卡片給生病的媽媽祝她早日康復，今年，她則寫紙條留下一段話：「媽媽：您之前因為佳佳半夜的哭鬧而睡不著時，我也很ㄉㄢ（擔）心您會睡不著，想盡半（辦）法讓您睡著也沒用，您要吃安ㄇㄧㄢˊ（眠）藥睡ㄐㄧㄠˋ（覺）時，我不讓您吃是未（為）您好，所以您現在要和我一樣乖每天都洗澡，爸爸才不會ㄉㄢ（擔）心，可以走的（得）安心一點。」小小年紀必須提早面對生活的挑戰，令人心疼。

老父嘆獨子太早辭世

小筠的阿公李金龍看著精神病況嚴重的37歲媳婦阿瑟和年幼兩孫，哽咽地表示，「該走的應該是我，不是我兒子，他才47歲，他還有這兩個孩子要養。」他說，獨子李寶鹿3個月前發現罹患頸部淋巴腺癌，原本想等待奇蹟，未料5月初醫院傳來噩耗，「我活一把年紀也夠了，本來還望他養我到百年，我有子送終，誰知卻變成這樣，他比我還先走。」
李金龍說，兒子生前做工又內向，35歲娶了原就輕度智障的媳婦阿瑟，但她當時僅稍憨鈍，沉默乖巧、能料理家務，一家也算和樂，不過一年多年前生了小孫女後，突然性情大變，不時嚷嚷大吵，「兒子認為媳婦中邪，花了不少錢請神明辦事化解，但都沒用，送去醫院治療，沒幾天她又吵著要出來，兒子常請假顧她、忙家務，因此欠了點錢，也把身體累壞了。」

鄰里義助湊齊喪葬費

雖然老人家身體尚硬朗，一時仍難忍白髮送黑髮悲慟，家裡大小事務暫由嫁出去的二女兒阿珠打理，阿珠說，哥哥喪葬費約18萬元，經鄰里好心協助能讓哥哥圓滿出殯，「哥哥2歲的女兒我會先帶在身邊照顧一陣子，小筠讓她陪著阿公、媽媽生活，只是往後一家老弱殘，我也很煩惱。」阿珠說自己是九二一受災戶，與先生還靠在夜市擺攤賣飲料償房貸，她說，另兩姊妹也能力有限，難幫娘家太多。
記者採訪時見阿瑟眼神渙散，身上散發出微微異味，她喃喃自語說每晚都睡不著，還表示自己已一個月沒睡了，對丈夫的去世毫無回應。小筠說，媽媽以前會做飯菜、愛乾淨，天冷了會叫她多穿件衣服，問她吃飽了沒……，但自從生了小妹後，整個人就變了，「都不洗澡，晚上我跟她睡，她翻來翻去都不睡。」小筠還說，有時媽媽亂發脾氣大叫，她心裡覺得害怕，會抱著小被躲衣櫥裡或書桌下。

列高風險家庭須關懷

目前南投當地學校及社會處社工都介入輔導，社會局社工說，老人家曾有想不開的念頭，現已將李家列為高風險家庭長期關懷，評估阿瑟若有必要時會強制安置就醫，甫通過三款低收戶資格，每月可領16600元，蘋果基金會也先撥款紓困。

後記 憶嚴父 盼保佑全家

記者初次探訪時，見已氣切的李寶鹿在加護病房病床上，雖已完全不省人事，但喘息聲似乎仍在為家人做最用力的掙扎，緊皺的表情像是還在使力與死神拼鬥，但如今他一切病痛已隨人逝遠去。跟小筠談起父親，她說，爸爸很嚴肅，當她考不好時會罵她，要她考好些，並說，只要電腦很強，就能賺很多錢，要她以後念資料處理科。小筠說，以前的學校功課、聯絡簿都是爸爸簽名，如今爸爸的話語只留給小筠無盡思念，「他去很遠的地方了，希望他保佑媽媽、阿公身體健康。」

專家意見 躲桌下避衝突「很聰明」

從事親子輔導教育的友緣基金會主任林淑棻認為，小筠拿被子躲在衣櫥或書桌下，顯示她很聰明善良，懂得保護自己，她知道在媽媽發病時，若跟媽媽再起衝突可能會受到斥責傷害，因此可暫時躲到自己的堡壘，等到暴風雨過去。

鼓勵朝正向思考

以親子相處及長期精神治療來說，若阿瑟無攻擊性，林淑棻建議能讓小筠與媽媽暫時先以此方式相處、不要分開母子較好，但鄰近親友、社工及學校師長應對小筠持續關懷並告知媽媽情況，「媽媽像感冒一樣生了病，只是需要長一點時間吃藥治療。」而據研究，若孩子適應良好，孩子會與母親找出相處之道，往後個性發展也會較成熟；但若孩子出現尿床、上課打瞌睡、成績退步或無故亂發脾氣情況，師長就應注意加強輔導，「學校及親友應讓她多正向思考，爸爸是去了一個更美好的地方，也鼓勵孩子有其他舞蹈或表演活動可多參加，讓她用成就感轉移悲傷。」
