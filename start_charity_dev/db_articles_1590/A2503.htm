師大男撞癱 復健拼復學

單親媽花光積蓄救兒 「看他這樣 心都碎了」

2011年06月30日

「曾想當導遊，到世界旅行，但現在要實現夢想，怕有點難了。」師大地理系大二學生曾華偉正展開飛揚青春，沒料到今年1月23日一場意外車禍，胸椎脊損折斷了他的夢想羽翼。 報導 攝影╱向高彬

車禍至今半年，華偉以手拉推復健器材，力氣仍不夠，沒一會兒額頭冒汗，20歲的他說，胸椎4至6節脊椎神經完全斷裂，現剩雙手有點力氣，下半身完全癱瘓，「只記得那晚騎機車經過泥沙路面，車子打滑眼前一黑，醒來已在加護病房，醫師告訴我脊椎神經斷裂，心裡就覺得慘了。」

克服想輕生念頭

華偉嘆說，傷後不時發燒感染，神經嚴重抽痛，曾想輕生，但經親友及醫師、社工鼓勵，心情漸平靜，「至少手能動，腦袋好好的，只要身體好些，日後仍能有貢獻。」
華偉表示，爸媽10多年前離婚，之後靠49歲單親母幫傭打掃，掙錢租屋撫養他和現年24歲的哥哥長大，但哥哥這兩年出外工作不順，勉強養活自己，「我考上師大，媽媽很高興，說我是他的驕傲。」但這一摔，摔碎了他的青春藍圖。
目前華偉長期住院復健，白天僱請看護，晚上是哥哥下工後照顧，「多年沒見的爸爸曾來幫顧一個多月，但他說已有家庭，後來沒來了，可能他看我這樣也難過吧。」華偉說，目前已辦休學，過一陣子要進行自費神經再生手術，或許有機會再站起來，「今年年底將到脊損潛能中心訓練生活照顧，計劃明年秋天能復學。」
母親張家華說，至今用光十多萬元積蓄，公所已撥急難金4萬元，目前每月領重殘津貼4000元，「看他這樣，心都快碎了。」醫院社工說，已評估補助部分看護醫費，「華偉傷勢嚴重，後續復健1年以上，長期照護經濟壓力重。」

盼上天出現奇蹟

記者兩度至醫院探訪，都遇到華偉的一群同學探訪，見識他的好人緣，大學同學阿安說，「期待華偉再回來上課。」高中同學小儀每兩周來醫院探望，「希望上天出現奇蹟，讓他再站起來。」

暖流見證
系上辦樂捐 幫度難關

師大地理系主任丘逸民表示，學校已協辦急難救助2萬元，目前系上還發起樂捐，也轉請學生輔導中心安排心理輔導和之後生涯規劃。我們知道孩子單親家境不好，往後手術和復健照護開銷沉重，也再轉請蘋果基金會關懷，感謝基金會訪視後，也撥款協力華偉醫護窘迫。
