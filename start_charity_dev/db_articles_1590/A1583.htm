8旬嬤智障孫 拾荒維生

不捨送社福安置 「我得給他更多愛」

2008年05月15日

83歲吳秋蘭的長孫三月初過世後，她和重智障次孫阿民每月僅賴相關補助一萬元及零星的資源回收過活，雖然心急日後無依的生活，但她無助的眼神，更難掩內心失去至親的悲痛。報導．攝影／張嘉恬

五年前，吳秋蘭的長孫阿華為就近照顧年事已高的秋蘭嬤，回到花蓮靠模板臨工收入，維持祖孫三人生活，但因阿華多年前的車禍宿疾，導致身體多處內傷，加上長年飲酒傷肝，不幸於今年三月初往生，秋蘭嬤和重智障次孫阿民，頓時少了生活照應與依靠。

每周回收不到百元

當地楊姓村長表示，現在家家都主動作回收，秋蘭嬤每周資源回收不超過一百元，且因她年紀大需經常至醫院就診，「他們祖孫倆每月扣掉日常用品費、醫病及交通等開銷後，常有不足，只好常向雜貨店賒帳。」
在鄰人翻譯下，秋蘭嬤用阿美族語說道，早年先夫常在外地打臨工，她則靠剪檳榔、插秧把一子一女帶大，「後來養孫子時，兩個我都背去田裡工作，插秧時就把孫子放在田埂上，回來再幫他們洗澡、餵飯。」秋蘭嬤表情堅毅地訴說著往事，如今生計陷困，感到不知所措，她甚至泛紅眼眶，難掩無法再與阿華相依為命的傷痛。
秋蘭嬤望著重智障的次孫阿民，語帶不捨說道，有人建議她把阿民送到社福機構安置，「他小時候父親就過世，母親又不知去向，我得給他更多愛才行。」
重智障的阿民雖然沒有工作能力，但長年與秋蘭嬤相依，不僅分攤秋蘭嬤的資源回收工作，有時也在生活上表達貼心關

「晚上會幫我蓋被」

採訪時秋蘭嬤才從外頭完成資源回收工作，阿民一邊幫忙推著回收車，一邊替秋蘭嬤綁好回收物，可說是秋蘭嬤的好幫手。「晚上要睡覺的時候，他也會幫我蓋被喔！」秋蘭嬤微笑地說著。
楊姓村長表示，秋蘭嬤雖然尚有一女，但因家計重擔，無餘力協助。楊村長並說，秋蘭嬤長孫阿華的喪葬費用，已由地方慈善會及鄰里協助完成，「但祖孫倆一個年紀那麼大、一個重度智障，都沒有工作能力，未來令人擔憂。」蘋果基金會
