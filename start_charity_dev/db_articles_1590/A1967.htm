兒顧癱母「把屎尿不辛苦」

6口生計窘 父：醫費不知哪來

2009年09月01日

40歲的泰雅族人阿芬（陳敏芬）6月底因腦中風導致癱瘓臥床，無法言語的她坐在輪椅上，望著不聽使喚的右手右腳嘆氣，19歲兒子阿堯沮喪地說：「把屎把尿沒什麼辛苦，最難過是的無法聽到媽媽開口說話。」報導．攝影╱游竹君

阿芬中風後，必須靠著丈夫阿仁（周廣仁）和兒子阿堯兩人費力攙扶，才能起身坐上輪椅，但才稍微挪動身體及手腳，阿芬便痛得皺起眉頭，嘴裡發出「嗯……嗯……」的聲音。記者行駛蜿蜒山徑近1小時才來到阿芬偏遠的老家，這趟漫長顛簸的路途，對需要頻繁就醫回診的阿芬來說，實在折磨。

只靠砍竹零工持家

阿芬與49歲丈夫阿仁育有5名子女，阿仁和22歲長子在山上以砍竹子零工餬口、長女已出嫁，另有3名子女分別是升大二的阿堯及就讀小五及小二弟弟妹妹，一家6口同住。阿堯無奈地說，家人也想安排媽媽下山接受復健，減少奔波，「無奈現無復健病床，我趁暑假在家先幫媽媽伸展手腳，別讓媽媽退化太快，爸爸也才能照顧年幼的弟妹。」
阿堯說，媽媽是個意志堅強的人，8年前懷小弟時引發肺積水、險些送命，但她還是撐過難關把弟弟平安生下；這次媽媽中風出院後，看到年幼的弟弟妹妹，她才安定下來，眼睛總隨著他們的一舉一動而移動，「媽媽雖然無法言語，但我知道她心裡放心不下弟妹，她一定會為年紀還小的弟妹勇敢活下去。」

月僅領補助5100元

寡言的阿仁談到未來，臉上滿是憂愁，他說，自己本來和大兒子只靠砍竹子的零工維持家計，「山間工活不定，這2、3年陸續向老闆、朋友借貸7、8萬元應急，這次阿芬中風，醫療費還是家扶中心與世界展望會社工幫忙付的，我得照顧阿芬，沒法工作，日後阿芬還要開刀放回頭蓋骨的醫療費，還有孩子生活開銷，都不知道哪裡來。」
阿仁稚子幼女的小學校長王富美表示，阿仁工作認真，一心努力存錢讓阿堯讀大學，只是沒想到突遭變故，連就醫及生活開銷都成問題，阿仁家每月領有家扶補助5100元，校方及「馬上關懷」急難金共已撥3萬元紓困，也協助申請教育部學產基金。
村長錢光輝說，待阿芬殘障手冊核發後，將協助申辦低收入戶補助，蘋果基金會訪視後，已先撥款暫紓困境。
