手殘癌男 求老天給時間

災厄連連 簽器捐盼遺愛人間

2010年11月11日

採訪時，患膀胱癌的陳貴享不到半小時頻喊內急衝廁所，「血尿尿得便盆都紅的，不尿忍不住、尿時又灼痛，好難受。」52歲的他舉起前年車禍撞殘左手使勁也握不攏，「手殘人病，快不知還有什麼是能把握的。」報導 攝影╱向高彬

與陳貴享訪談時，他氣色憔悴，「醫師說我膀胱癌擴散，還要再手術切除膀胱，後續做放射治療。醫師要我專心抗癌，如今我接受上天安排，也簽了器官捐贈，但仍求老天多給我時間陪孩子。」

窮得只剩200元

陳貴享說，5月時發現血尿灼痛，但手殘的他靠打零工為生，當時生活都成問題，拖至9月才就醫，當時醫師要求立即住院，但他窮得只剩200元，「醫師說再不治療，等膀胱血水爆開，就醫不活，才緊急接受治療。」
醫院社工表示，醫院針對醫費評估補助，「但他往後抗癌期間生活交通開銷是難題。」當地公所社會課表示，目前陳貴享每月領殘津3000元，已撥「馬上關懷」急難金1萬元。蘋果基金會訪視後，也撥款暫紓陳貴享醫病急窘。
回溯往昔，陳貴享說，早年曾從事航空地勤工作，做過房屋仲介，婚後生有一雙現就讀大二、國二子女，那時生活美滿，沒料到改投資古董生意後，13年前借友人票據卻遭其跳票跑路，致虧盡千萬元家產，前妻氣他太輕信友人不聽勸，雙方9年前離婚，「我落魄後慚愧再提當年，低頭彎腰做搭鐵屋雜工，租屋養2個孩子，沒想到前年車禍撞殘左手，養不起孩子，只好讓孩子跟媽媽，孩子和我很好，常來看我。」只是災厄又襲，陳貴享如今病得沒力氣做工。
陳貴享目前暫與單親做工的大弟租屋同住，但大弟零工不穩，無力幫忙，陳貴享的妹妹阿敏說，大哥落魄病殘，「現在連看病錢都沒有，我只能偶爾匯點錢。」相識近20年的徐姓友人說，陳貴享很重義氣，曾接濟幾回，沒想到人生無常，他竟又罹癌。

女兒心疼重病父

陳貴享的前妻說，20多年前她到蘭嶼旅遊時，與在當地航空站工作的前夫相識，他人很好、愛孩子，只是當年太衝事業、卻踩空，「緣盡了，現在我養孩子，沒力多顧他。」陳貴享的大女兒說，沒想到家道中落、爸又罹癌，在外讀書常打電話安慰爸爸，她知道爸爸病痛卻硬撐，「很捨不得他病了，希望快好起來。」
