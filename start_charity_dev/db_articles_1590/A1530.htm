離婚罹癌 厄運重擊壯男

女友不離棄舉債醫病 欠友10萬也陷困

2008年03月12日

44歲汪正道一年半前罹患下咽癌，採訪時他掩面痛哭、聲音沙啞說：「現在只能靠醫院的嗎啡才能止痛。」他因與歸前妻監護的孩子們在路上相遇形同陌路深感心痛外，更因面臨營養品、往來醫院治療的交通費等開銷無著落而心急如焚。報導．攝影／張嘉恬

汪正道原以承包貼磁磚的工作，支付妻子與一子兩女的生活開銷，兩年半前離婚時經協調後，名下的房子及三子監護權均歸前妻，當時他不僅孑然一身，還背負一身工程材料的貨款債務。阿道目前同住的友人阿蘭說：「當初我跟他說，我可是比你大五、六歲，女人比較快老，不要到時候我老了你把我踢走啊！」

女友照顧收入也中斷

阿蘭表示，她與阿道原是鄰居，三年前她喪夫時，阿道曾伸援手，協助喪葬及阿蘭和五個孩子的短暫生活，阿蘭因心中念記這份恩情，在阿道離婚又背債的低潮期，持續勸說並鼓勵日日買醉的阿道，「後來阿道說要正式交往，他是在我孩子們的同意下才住進來分攤房租。」
未料厄運接踵而至，阿道債務還未清，癌症又找上他，生計因醫病開銷而雪上加霜。阿蘭語帶無奈說道，她原本靠國內遊覽組團的出車業務維持生活，而她的長女靠工廠作業員單親養子，長子離家年餘，在美容院工作的次女每月薪水六千元僅夠自己開銷，「本來孩子們就沒辦法幫我，跑車業務不固定，一趟三、四千元，我還會把握工地做工機會，現在要照顧阿道、要負擔念高三小女兒的生活，無法工作開銷又增加……」而阿蘭自阿道罹癌後，為了醫病及生活開銷已積欠友人十餘萬元的債務。

光營養品每月耗2萬

阿蘭表示，阿道目前正進行放射線治療，無法像正常人吞嚥食物，更需要營養品補充體力，「一瓶約150元他一天要喝四罐，光營養品一個月就快兩萬元，從苗栗到台中醫院一趟車資也要一千元，沒收入實在負擔不起。」阿蘭看著縮著身子、低頭閉目沉思的阿道，語帶不忍提及，阿道最近在夜半時分經常疼痛加劇，「他還有骨刺，但身體狀況不能再承受手術，雙倍的疼痛讓他晚上常像三歲小孩一樣抱著頭哭。」
徐姓里幹事表示：「汪正道有四兄弟，也都靠臨工維持家計，無餘力協助他。」蘋果基金會訪視後，已提撥急難救助金協助。

無法計算的人生

阿道說：「我以前有65公斤，生病後只剩40公斤，沒想到人生變調，可以比體重降得更快，自己生的孩子在路上看到我像陌生人，我只能看開一點了。」
