病嬤憂2孫 求神助癌兒

家計斷炊 盼度難關

2011年06月28日

徐家客廳裡，76歲阿好嬤雙手合十向菩薩默禱，讓今年2月突患鼻咽癌的二兒子阿泉安度難關，「3個兒子走掉2個，就剩阿泉，不知醫得好嗎？他2個男孩還在讀書，我怎能不煩惱。」
報導．攝影╱向高彬

阿好嬤有青光眼，她茫茫望著正以鼻胃管灌藥的43歲兒子阿泉說，兒子出生時就嚴重唇顎裂，小時很自卑，出門一定把嘴摀著，怕別人嘲笑，小學時動過8次修補手術，才總算講話清楚些。
阿好嬤說：「沒想到他現在又得鼻咽癌，如今整天病懨懨，有時咳得痛到整晚睡不著，我心疼他病關過不完。」阿好嬤也說，阿泉離婚單親15年，多年來阿泉靠餐廳廚工，辛苦撐養現就讀高三及國三的2子，如今阿泉病倒，家裡生計快撐不住。

持續化療病況未卜

阿泉話聲含混地說，去年起常感頸部疼痛、右眼模糊，但一直不以為意，到今年2月劇痛難忍，一檢查竟已鼻咽癌轉移入骨為四期，醫師研判與他抽菸及工作長期接觸油煙有關，「最近常覺頭痛，醫師說可能癌細胞侵噬到頭，正持續做化療，我問醫師情況，他只說好好治療，別想太多。」儘管病況未卜，阿泉說，2兒很懂事，老母年邁，「無論如何要拼著好起來。」
阿泉回述，20歲時經介紹學廚工，專長在刀工巧快，都在大型餐廳做事，婚後生2子。

外地工作有苦難言

但15年前離異，之後由他做工養家，母親幫忙帶大孩子，「這幾年在外地工作，有時餐廳收掉又換一家，有苦不敢跟家裡講，硬撐籌錢寄回讓老母和孩子讀書生活。」
採訪時，19歲大兒子小青說，爸爸病後學校協辦急難救助，自己剛考完統測，還未確定能上哪個學校，「想選擇離家近的學校，如果爸的病有狀況，可趕回家幫忙。」小兒子的北北基聯測則成績優異，考上成功高中。
阿泉的妹妹說，娘家男丁剩大弟阿泉，大姊出家為尼，自家2個孩子正就學，「常回去探望媽媽，貼補點家用，但沒辦法太多。」

月領補助僅7750元

當地公所表示已撥「馬上關懷」急難金1.5萬元，目前阿泉家每月共領家扶、老金及3款低收補助7750元。蘋果基金會訪視後，也撥款暫紓阿泉家醫病生活窘迫。
