壯男突癱 無力顧精障母

開銷全靠姊籌措「不想一輩子坐輪椅」

2011年04月08日

阿松（許富松）一臉無助的躺在醫院病床上，雖拚命使力抬起右腿，但還是只能舉起10公分高，阿松說，越想用力卻總覺得使不上力，「感覺好像汽車快沒油了，用力踩油門，車子卻越來越沒力。」報導．攝影╱韓旭爾

今年2月初的中風讓阿松身體右半邊癱瘓無力，語言功能受影響，阿松口齒不清地感嘆：「自己才34歲，怎麼也想不到會突然中風！」阿松接著說，爸爸已過世10多年，他原本與58歲的媽媽相依為命，4年多前媽媽精神病發，又因車禍造成行動不便，現安置於養護中心。

煩心母親養護費

阿松說，這幾年來他靠著水產送貨司機零工掙錢維持家計，只是現在中風，雖幸運保住性命，但後續還有漫長復健路要走，也得煩心媽媽每月2萬元的養護費無著，他說：「媽媽雖有精神病，但還認得我，2、3個月沒法去看她了，不知她過得如何？」
阿松已出嫁的姊姊阿萍回述說，爸爸生前有賭博惡習，家裡常有人來討債，多年來媽媽承受沉重經濟壓力，4年多前媽媽開始出現異常行為，她說：「3年多前媽媽半夜外出拾荒，不慎發生車禍，造成骨盆碎裂，變得沒法走路。」阿松說，當時他需要工作，實在分不了身，只能把媽媽送到養護中心由專人照顧。

「感覺沒有尊嚴」

記者到養護中心探視阿松媽媽58歲的玉女婆時，問她：「會不會想兒子？」她點頭說會，之後便一直憨笑，養護中心主任說，了解玉女婆家裡困難，養護費已減免2成。
阿松說，中風後吃飯、洗澡，甚至大小便都要人幫忙，「感覺做人的尊嚴都沒有了，要努力復健，不想在輪椅上度過下半輩子。」阿松零工老闆說，阿松個性憨直，只要有工作，他一定會來上班，工作很認真。阿松說，除了姊姊阿萍外，他還有一兄一弟，但彼此甚少聯絡，自己中風這個2、3個月來，生活、醫療開銷全靠姊姊籌措。阿萍說，自從媽媽入住養護中心後，她每月已盡力分擔數千元養護費。

月津貼僅4千元

社工指出，阿松一家社福補助僅有媽媽重殘津貼每月4000元，正協助阿松申辦殘障手冊中。阿萍說，阿松中風不久，是復健的黃金時期，他還年輕，重新站起來的機會很大，蘋果基金會訪視後，已先行撥款協助阿松生活及媽媽養護費用。
