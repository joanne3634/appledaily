壯漢中風半癱 不忍妻吃苦

2憨兒待養「想快好起來顧家」

2009年10月13日

「拔拔，我幫你拿藥藥和水水。」5歲小峰踮起腳尖，幫33歲中風爸爸阿鴻（洪瑞鴻）拿藥袋，但拿不到，他靈機一動拿起抓癢棒高舉終於拿到。其實小峰患中度智障，阿鴻說：「孩子上早療，最近才較會說話，但我生病了，很煩惱生活。」 報導 攝影╱向高彬

阿鴻垂著無力左手坐床上，他雖正當壯年，但黝黑憔悴模樣讓他外表看來比實際年齡蒼老，阿鴻吃力地開口說，3年來中風3次，前兩次輕微中風後，經努力復健勉強能到工地做清掃零工，但去年底三度中風，讓他左半身無力，如今得硬撐四腳拐杖才勉強行走，「太太每周4天帶我去醫院復健，5歲和4歲兒子也要做早療。我想快好起來，有力氣再照顧家人。」

小兄弟日托早療

一旁32歲的阿鴻太太林亜靜說，老大原不懂說話，生氣只會跺腳大哭，去年中檢查為口語遲緩，社工協助為兩兄弟進行日托早療，孩子好許多，但未料先生卻癱了，還好先生都要求復健要勤練到流汗，手腳漸有起色撐起走動，只是如今一家四口每月靠三款低收補助18400元，難敷房租家用及長期復健開銷，「積蓄早用完了，親友都知道我家窮，現在已很難再借。」由於兩小孩較吵鬧，他們與鄰居關係緊張，林亜靜說想搬到偏郊平房，但苦愁搬家費和長期往返醫病交通，蘋果基金會訪視後，先撥款暫紓生活窘困。
林亜靜回述，阿鴻是獨生子、雙親已逝，阿鴻算是輕度智障者，但僅是識字、計算差，能生活自理，茹素的他曾是素食餐廳師傅。

持續復健會好轉

採訪時，見林亜靜在約8坪租住套房裡，忙追喊兩個好動孩子「坐好、不要吵」，但體型胖壯的孩子，仍不聽指揮爬上爬下。她說，出門帶孩子都得緊緊抓牢，否則就是沿路追小孩。只是孩子雖皮，但記者問及爸爸病況，卻能稍安片刻認真回說：「我有乖，幫拔拔拿藥喔。」當地早療中心黃姓社工說，1年來看到孩子表達越來越進步，爸爸也努力復健，父子都在治療黃金期，未來需持續復健，都有轉好機會。
