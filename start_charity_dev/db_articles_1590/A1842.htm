男抱病咬牙掙錢養5口

「我不能倒」 想聽遲緩么兒叫聲爸

2009年04月03日

41歲的阿賢（蔡寶賢）罹患肌無力症，常因全身無力、雙眼複視而發生意外，右腳小腿留下一道道意外後的疤痕，過年後疾病復發的他，仍咬牙做臨時工，「我不斷告訴自己不能倒，如果我病了，這個家就垮了。」報導．攝影╱游竹君

阿賢與31歲妻子阿君育有3名子女，分別就讀小2、大班，年僅4歲的么子阿呈則因語言發展遲緩至今還沒叫過他一聲「爸爸」。阿賢年輕時擔任車床學徒，原本全家在北部租屋生活，去年10月因不景氣加上阿賢因疾病無法穩定工作，生活總是捉襟見肘，最後決定舉家南遷租屋，節省開銷。

肌無力曾害孩子摔地

談起疾病，阿賢說6年多前因胸線瘤就醫時發現罹患肌無力症，醫生叮囑不能過度勞動，但原本就以粗活維生的他，沒有其他技能，只能硬著頭皮工作養家。
阿賢說，3年多前曾嚴重發作，住院一個多月，又休養大半年才逐漸恢復體力，醫生叮囑要穩定就醫服藥，病況才不易惡化，但因沒錢，無法固定就醫。記者發現，與他兩三小時訪談下來，阿賢已顯得氣喘吁吁，他語帶懊惱地說：「那時連走路都會暈厥，甚至還曾因為手部肌肉無力，害懷中孩子摔在地上。」
位於鄉間的低矮房舍裡，家具簡單陳設卻整潔，訪視時，語言發展遲緩的阿呈，興沖沖地拿著書，嘴裡發出「咿咿啞啞」的聲音，滿臉笑意想叫人念書給他聽，媽媽阿君無奈地說：「雖然知道要讓孩子接受治療，但是阿賢身體不好、收入有限，連基本生活填飽肚子都成問題，健保也欠費，以家中現況，真的沒餘力帶阿呈接受早療。」

工作不穩定難敷支出

北部社福中心曹社工表示，阿賢之前因病，工作一直不穩，有協助申辦弱勢兒少及民間慈善會補助，村幹事表示將協助申辦「馬上關懷」急難金。老闆陳先生說，阿賢年輕時曾向自己父親學習車床技術，知道他生活負擔重，「他很認真，有叫工他都會來，只是大環境不景氣，一個月頂多只能給他做半個月工，收入約1萬多元，扣除房租5000元，要養活一家5口實在困難，阿賢總是預支薪水、東籌西借的過日子。」蘋果基金會訪視後，除撥款協助阿賢一家生活，並轉介當地早療中心協助阿賢兒子及早進行早療。
