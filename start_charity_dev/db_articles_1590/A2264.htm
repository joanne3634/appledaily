一家之主中風 7口斷炊

妻子要顧5子女6 歲老三罹病待手術

2010年09月10日

44歲阿河今年3月中風，左半邊不良於行，妻子阿慧與5名年幼子女圍繞在他病榻旁，有的人餵飯、有的人幫他按摩，有的替他作復健，他心感愧疚說，中度多重障礙的老三小君本來要進行置換引流管手術，但「都因為我中風耽誤了。」
報導．攝影／張嘉恬

阿河說，他本仰賴駕駛砂石車維持一家7口生計，但自金融海嘯後工時減少，家用收支日漸緊絀，「今年過年前瓦斯爐、熱水器都壞掉，為讓孩子過個好年才借了10萬元小額貸款，沒想到過年後就因沒控制好高血壓而中風。」而10萬元也早用盡於生活和醫病開銷。

難分身顧女兒病情

採訪時屋內充滿著孩子們的童語笑聲，唯獨老三小君靜默無言。阿河說，與妻子阿慧育有現年最大10歲、最小2歲的2子3女，而6歲的老三小君因出生時缺氧導致腦水腫，手術多次外，為排除腦內積水，至今體內從腦部至腹部仍裝有引流管，卻間接影響小君下顎發育及語言學習，右手也無力。她除了因身軀日漸發育已屆置換引流管的年紀，也必須進行下顎手術，以利嘴部張開，順利進食、學習語言。阿慧說，她目前每周三天得陪阿河至醫院作復健，醫囑阿河至少得復健一年，「我們兩個都無法工作，我也分不開身處理小君的事情，很擔心日後她的醫療、交通費用。」

親姊肢障無力協助

當地簡姓里長說，阿河雖有兄姊，但多無往來，僅有四姊住鄰近，「但她四姊本身為單親肢障人士，尚需煩惱國中、小學兩子的生活，也無力協助。」阿河全家雖領有低收及慈濟基金會補助每月共2萬8000元，但扣除房貸1萬3000元後，7口就靠1萬5000元生活，蘋果基金會訪視後，已撥款暫紓困，當地鎮公所也撥急難金2萬元協力，當地社福中心表示，有不定期物資協助阿河家。
