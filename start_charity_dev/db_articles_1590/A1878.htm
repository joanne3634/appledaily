「愛他心甘情願」 妻舉債醫夫

盲夫癱床「記得我生日」 越妻感心不離不棄

2009年05月21日

41歲雙眼全盲簡明輝（阿輝），去年1月腦梗塞住院，不到2個月又跌倒導致頸椎及脊髓損傷，仍在復健時卻又於今年4月中在浴室滑倒，右大腿嚴重骨折。他躺在床上，一旁越籍妻阿鳳擔憂丈夫病況，更為斷炊生計發愁。報導．攝影／張嘉恬

阿輝與36歲阿鳳結褵9年未育子女，阿輝以視障按摩維生，阿鳳擔任工廠作業員，兩人收入每月共約3萬元，扣除房租1萬3000元後生活尚可。但自阿輝去年腦梗塞又接連跌倒後，1年多來已不能從事按摩，生活只靠北縣低收補助及阿鳳收入清貧度日。

向同鄉借錢付房租

但因去年下半年景氣轉壞，阿鳳面臨無工可做，積蓄也隨阿輝復健、針灸的醫病交通費耗盡。阿鳳無奈說：「這幾個月我都向同鄉借錢付房租和生活費，等補助下來再還。」
記者初訪阿輝時，他尚未進行手術，和阿鳳正在做復健練習，阿鳳邊搭著阿輝雙手引導他緩步前進，一邊回憶道，9年前阿輝初到越南娶老婆，「當時只覺得他眼睛怪怪的，來台後看他拿柺杖才知他看不到。」經過多年相處，兩人培養出濃密感情。阿鳳說：「每年他都記得我生日，一定會買蛋糕，還帶我去逛夜市買禮物。他常說要帶我四處走走，但現在他傷很重，以後可能沒機會了。」

顧夫半年無法工作

阿輝自去年初因腦梗塞及跌傷住院後，表達及思考能力大不如前，多數時間舉止都像孩子，雖智能減退，但夫妻倆對彼此由衷的關愛卻沒有改變。阿鳳說，兩人有次口角，她一氣下想回越南，整理行李時卻發現阿輝偷偷在哭。阿鳳說，朋友不捨她嫁來台灣吃苦，得辛苦照顧另一半，但她說：「阿輝對我那麼好，我愛他是心甘情願，不是可憐他。」
但阿鳳想到接下來得在阿輝身旁看顧，至少半年無法找工作，她不由得蹙起眉頭，「只靠每月低收補助約1萬元，連房租都付不起，更別說阿輝尿布、來回醫院復健交通費和基本開銷了。」
當地陳姓里長說，公所已核發「馬上關懷」急難金1萬元，里長說：「會再評估阿輝近況盡力幫忙。」蘋果基金會訪視後，已先自「歐舒丹．蘋果視障專戶」撥款暫紓阿鳳夫妻生活。
