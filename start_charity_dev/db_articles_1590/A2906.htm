男傷頸 陋屋4口難溫飽

摸黑省電 床舖總被雨淋濕

2012年10月26日

記者到家訪問時，58歲的川仔（楊清川）脖子仍圈著護頸，一臉憔悴、表情盡是無奈，他說，6月初騎車外出購物時，疾速行駛的貨車迫近呼嘯而過，他被嚇到摔車，造成第4至5頸椎椎間盤突出，醫師交代養傷1年。報導．攝影╱韓旭爾

現他搬不動磚塊、水泥，沒法再做建築零工賺錢養家，老婆頭腦憨傻難工作，2個兒子才16及15歲，都還在讀書，一家4口溫飽成愁，川仔說：「現在我不只脖子痛、手麻，想到沒錢生活，頭也跟著痛了起來。」川仔說，有聯絡的親人大家都窮，也幫不了多少。

憨妻零工補家用

川仔說，他有過2段婚姻，20多年離婚後，18年前再娶現年42歲、領有重度智障手冊的妻子阿華，「我年輕時開計程車，阿華媽媽生前常叫車看病才認識阿華，自己離過婚，年紀又大她很多，阿華媽媽不嫌棄願意將女兒嫁給我，就要好好照顧她。」
川仔說，阿華雖頭腦不好，但還能與人正常說話互動，也懂得做家事、照顧孩子，只是除了自己名字外，國字全不認識，情緒控制力差，常發脾氣，覓職不易，只能偶做清潔、農務零工掙錢貼補家用。採訪時，阿華對記者的詢問總是憨笑回答說：「我不知道，要問川仔。」

長子盼工讀掙錢

提起過往，川仔說，他長年工作不穩定，只能租住偏僻鄉間老厝，雖逢雨漏水連連，但因房租才3000元，只能勉強住下，「下雨時，兒子們的床舖總是被雨淋濕，只能打地舖，我這個當爸爸的看了也難過，卻也無能為力。」川仔接著又說，平時用錢總是能省則省，「能不開燈就不開燈，上2個月電費雖只有500多元，但還是繳得很心痛。」此時，記者才了解為何2次晚上到訪時，家裡都是漆黑一片，還以為沒人在家，原來一家人總是關著電燈看電視。
川仔就讀高一的長子小崇說：「我會努力找工讀賺錢，分擔爸爸的壓力，讓他好好養病。」小崇國中導師說，小崇自國中時就懂得體諒家裡經濟困境，假日就會跟著爸爸到工地打工，「是個貼心的孩子。」
當地公所社會課表示，已協辦急難金2.5萬元，之前因川仔前段婚姻所生1女已成年，被列工作人口，加上他名下有2輛老舊車輛因欠稅還未報廢，仍列計財產，一家無法獲得政府低收入戶補助，目前每月只領阿華重殘津貼4700元，公所正協助申請低收覆核，蘋果基金會訪視後，也撥款暫紓生活、復健窘迫。
