父驟逝 來不及等6子長大

兒女乖巧「少吃點」 寡母：不捨他們挨餓

2009年01月09日

「爸爸很注重家庭和樂，記得去年我們6個小孩和爸媽吃年夜飯圍滿一桌，還抽小紅包，很熱鬧，但今年少了他，感覺會不一樣了。」讀高二的小玲與弟妹在歲末送走心肌梗塞猝逝的爸爸，眼看農曆新年將到，心裡有濃濃哀傷。報導．攝影╱向高彬

小玲的爸爸蘇秋龍走得突然，連臨終交代都未留，小玲說，原本一家八口擠在20坪木造平房，平常爸爸獨自睡客廳椅上，爸爸走的那天清早，自己照常起床到客廳跟爸爸道早，卻發現他臉色發黑，「我叫醒媽媽，叫救護車，用學校教的CPR心肺復甦術幫他做人工呼吸，但他已斷氣，送醫急救也沒效。」

喪葬費靠籌措完成

蘇秋龍結束匆匆36年人生，由於家境窘迫，10萬元喪葬費還經學校、親友和公部門、慈善會等籌措完成。採訪時，小玲的媽媽陳美惠說，丈夫很愛小孩，「他常說，大人可以吃苦，但不能讓孩子吃苦，做工再累、再晚，只要回家看到孩子吃飽飽、睡到打呼的臉，再累都值得。」
陳美惠說，17歲大女兒小玲是丈夫年少第一段婚姻所生，他與丈夫婚後再生下現分別14、13、11、9、8歲的5名子女，丈夫生前從事模板工作，每月加上三款低收補助，維持一家八口家用，她說，丈夫會教孩子寫功課，期望他們做對社會有用的人，「他很樂觀，都說凡事要往正面想，有一天，會愈來愈好，但沒想到他來不及等孩子長大，就先走了。」

「需要時間調適悲傷」

陳美惠往後得獨力扛起照養孩子重擔，但苦於現今景氣差，鄉間零工難覓，陳美惠說，孩子正值發育期，學校午餐有補助，但早晚兩餐得吃掉500元，「孩子會說省錢少吃點，但我捨不得讓他們餓肚子。」陳美惠每月偶有不到5天的洗荖葉零工，仍難解困窘家計，更遑論償還先前因生活積欠約30萬元債務，「除了這6個乖孩子，丈夫什麼也沒留下，只能走一步算一步。」
孩子小學校長梅媛媛說，孩子乖巧懂事，學校正申辦學產基金補助，「孩子需要時間調適悲傷，老師已在輔導關懷。」當地社會處社工也介入協助陳美惠穩定一時生活與輔導就業，蘋果基金會訪視後也先撥款協力。面對未來，陳美惠盼丈夫在天之靈，佑護她與子女在新的一年更有力量去面對生活挑戰。

後記 「爸爸 要快樂去當神仙」

採訪時，午後下起大雨，瞬間積滿庭院，苦雨沿牆柱滴灑在簡陋靈堂，看到孩子表情恭敬為父親擦去靈前水漬、拜飯、燒紙錢、念誦佛號，還說：「爸爸，要快樂去當神仙。」眼神流露對亡父千萬不捨，大女兒小玲說：「他躺在冰櫃裡，守靈這幾天我早晚去跟爸爸問好，習慣每天一定要看到他。」濃濃親子情，是他們在貧困生活中最富足的享受。
