癌夫傷見骨 妻愁扛5口家

嬤中風 2女年幼 「沒人能賺錢」

2010年01月08日

33歲楊梅（阿梅）的丈夫阿海（張海光）去年10月罹患口腔癌，術後取小腿皮膚修補傷口，腿部卻因糖尿病反覆感染。她對丈夫小腿流出血水、膿水，已不覺怵目驚心，因為一家5口的生計更令她煩憂。報導 攝影╱張嘉恬

採訪時正逢護士小姐替阿海換藥，繃帶拆解後，傷口深可見骨，並不斷流出紅色、粉紅色、綠色的血膿。「不能吃東西還好，現在連靠自己下床站立的力氣都沒有，後續的化療或放射線療程也因此暫緩。」阿海無奈說。

背負30萬元卡債

41歲的阿海和陸籍妻阿梅育有7歲、5歲2幼女，並與60歲母親同住，阿海早年開水泥預拌車，但因2年前營造業不景氣轉行當公車司機，加上阿梅電子工廠薪水，雖然每月共有約6萬元收入，「但因換過兩部水泥預拌車，兩部車貸就要約5萬元，根本無法儲蓄。」多年來車貸雖已償畢，但為支付基本生活開銷，至今還有約30萬元卡債。
而阿梅為了照顧阿海也只好辭去工作，熟悉阿海家況的當地劉姓里長表示：「阿海的母親罹患高血壓，早年也中風過，走路不太穩，僅能在家幫忙照顧2孫女，目前沒有人能賺錢。」

同事合捐3萬元

採訪當天，阿梅說，因為女兒很久沒見到阿海，她便抽空回家帶女兒到醫院探父，一見到阿海，2個女兒隨即上前握住阿海的雙手，長女更上前輕撫阿海術後臉頰說：「爸爸你有沒有好一點？」
阿梅說，女兒很久沒有看到爸爸了，只能在家打電話問候，「每次問到爸爸什麼時候才可以回家，我也不知道怎麼回答。」
阿海父女們相聚沒幾分鐘，阿梅又得急忙帶著女兒趕搭最後一班車將女兒送回家睡覺。「奔波往返辛苦都沒關係，只希望阿海趕快恢復體力，和我一起撐這個家。」阿梅說。
阿海的黃姓主管說，與阿海共事近3年，「他工作認真負責，尤其對老人家很孝順，以前就知道他家計負擔較重，同事聚餐時，他都說寧可把時間拿來上班多賺錢。」阿海同事們已合力捐助約3萬元，蘋果基金會訪視後，也撥款暫助阿海一家生活。

後記
「再苦也要撐住這個家」

阿梅在訪談間，提及無法預知的未來，數度落淚，尤其想起獨自在手術室外等丈夫手術時，更哭著說：「別人都是一家人手牽手在禱告，那時我最感到孤單。」但她仍堅強地說：「現在只有我能撐這個家，再怎麼辛苦也得走下去。」
