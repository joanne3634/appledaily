癌婦寧吃苦 盼老小溫飽

「只怕無法看小兒子長大」

2011年01月31日

50歲單親媽陳月卿2年多前罹患肝癌，切除三分之二肝臟後，雖穩定控制病況，最近卻發現癌細胞有擴散跡象，體重不到40公斤的她，眼神明亮、語氣堅定地說：「艱苦的日子我自己能咬著牙過，但家裡的老小不能餓著。」報導 攝影╱張嘉恬

「她為了家裡開銷，3個多月前還去做洗碗工，才做2天身體撐不住又跑醫院。」陳月卿的78歲老母敘述女兒苦撐家計，還提及自己也為補貼家用到檳榔攤求職，「但人家看我這年紀，沒有錄用我。」

「兒給我抗癌力量」

陳月卿和老母及現年21歲、13歲的2子租屋同住，家計本仰賴她做裁縫收入維持，2年多前罹癌後無法工作，今積蓄也用盡，「大兒子今年元月去當兵，我想工作又力不從心，只怕自己時日不多，無法看小兒子長大。」
陳月卿說，自小家境貧困，她小學3年級便每天凌晨3點就起床和父母一同撿磚頭補貼家用，看著雙親辛苦持家，「從小就立志日後要讓家人過好日子，我小學畢業就到電子工廠做童工。」而長大成年後，她的兄弟也長年因案服刑未分擔家計，「我習慣了過苦日子，也把養家視為自己的責任，卻沒把握能通過病魔的考驗。」她說，小兒子小勝見她因擔憂未來，躺在床上流淚時，總會替她拭淚，見她作家務過了用餐時間，「也會提醒我要吃飯，或煮麵給我吃，還好他體貼懂事，讓我多了些與癌病戰鬥的力量。」一旁的小勝不捨地說：「媽媽常常忘記吃飯，她身體已經很差了，我很擔心她這樣會讓身體更糟。」

兒子想快長大養家

小勝的楊姓導師說，雖然小勝家境陷困，但他樂觀開朗，「他曾在作文中寫道希望自己趕快長大，才有能力賺錢養活全家，是個獨立堅強的孩子。」楊老師說，小勝家目前「一家3口老、病、幼，小勝母親癌病在身無法勝任工作，很辛苦。」楊老師表示，學校已協助申請學產基金急難金5萬元，也讓小勝每天帶回學校的愛心飯菜以節省餐費。
目前陳月卿全家生活每月僅賴低收等相關補助共約1萬2000元，扣除房租9000元後供3口生活，陳月卿自罹癌後至今已獲「馬上關懷」急難金共3萬元，林姓社工說，針對陳月卿困境，已協助申請民間慈善單位資源，蘋果基金會訪視後也撥款暫紓困境。
