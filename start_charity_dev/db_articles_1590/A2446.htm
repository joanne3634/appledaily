殘父久病 孝女工讀養家

心疼母顧父還得打工 「爸媽只能靠我了」

2011年04月20日

54歲張銘豐不僅自2年多前開始得終身洗腎，難負重工作，去年中更因糖尿病併發症截去左小腿，他坐在輪椅上，怔怔地望向窗外，心中因無力持家、無法給妻兒穩當的依靠而發愁。
報導．攝影╱張嘉恬

張銘豐本賴卡車司機工作維持家計，他因家族遺傳患有高血壓、糖尿病，至今因急性心肌梗塞進行了6次心臟手術，他說，2年多前開始洗腎後因無力工作，將自宅貸款投資生意卻慘賠500多萬元，於是將辛苦攢錢購置的房、地售出償債，未料去年中又因糖尿病併發左足潰爛而截肢。

腳潰爛常需就醫

張銘豐說，截肢後請領勞保殘廢給付約24萬元，但扣除7萬元義肢自費後，餘款已盡用於10個月來醫病與生活，「因蜂窩性組織炎引起的傷口難癒合，今年1月才裝義肢，現在還在復健期，醫生說以我狀況來看，需用助行器練習約一年才能獨立使用義肢。」而近一個月張銘豐右腳足部也出現多處潰爛，常需就醫，一家3口每月僅賴女兒工讀、妻子打零工及他的殘津共約1萬4000元。
張銘豐的51歲妻子說，自先生截肢後，「他的生活、三餐還需我打理，我僅能利用空檔作摘菜等零工，讓女兒打工養家，心裡很無奈、很不捨。」

仍想找第二份工

採訪時小嘉才從早餐店收工回家，近午時分，她先將父親旁的便桶拿至廁所清理，又匆忙地趕著外出尋覓第二份工作機會，她告訴記者說：「這些事都是我分內該作的，只希望爸爸的病況都能控制穩定，他不用再因病痛難受，媽照顧起來也較不辛苦。」
張銘豐夫妻倆養育的18歲女兒小嘉，現就讀高職夜間部二年級，她說：「知道爸爸必須截肢時，我嚇了一大跳，但心中也做好準備，了解到從那時候開始，爸媽只能靠我了。」小嘉說，目前在早餐店工作每月收入約7000多元，小嘉的賴姓導師說，小嘉生性樂觀積極，「雖然家境陷困，卻不會把肩負家計的責任當成壓力或負擔，反而不斷地想盡更多的力量。」
當地里長說，張銘豐兄弟各自有家計重擔，也因債務牽連鮮少往來，而目前張家居住太太的弟弟名下房屋，張銘豐自截肢後陸續已獲「馬上關懷」等急難金共2萬8000元，蘋果基金會訪視已撥款紓困。
