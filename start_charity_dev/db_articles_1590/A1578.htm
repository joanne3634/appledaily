嬤罹癌 兒孫照護生計窘

半個月得化療3天 工作被迫停擺無收入

2008年05月05日

59歲的莊方蕊抱著14歲的孫女小雅，眼淚奪眶而出：「我這病再拖也沒多久，妳要好好照顧自己和爸爸，知道嗎？」罹患肺癌末期的莊方蕊說，人生已過半百，生死看得很開，最放心不下的就是唯一的孫女。報導．攝影╱孫敏聿

「阿嬤，妳不是胸口很痛，不要說話了啦，躺下來休息。」就讀國一的小雅說，自幼父母離異，爸爸長年在外打零工，阿公和阿嬤就像她的父母，下課一起去買菜、假日還會跟阿公去採草藥，「阿公得肝癌十多年來，都是喝自己煎的草藥，沒有化療，不過2年前還是過世了。」

兒女經濟狀況都不好

小雅說，阿公生病後，沒法再和阿嬤一起做建築零工，過世時的喪葬處理還靠親戚幫忙，之後爸爸跟著工地南來北往做零工，「阿嬤也去幫忙代班包水餃，一個月大概4、5千元收入，直到今年初發現得癌症，才沒了收入。」
「現在我每兩個禮拜要來住院化療3天，小雅下了課，就騎半小時的腳踏車來看我。」躺在病床上的莊方蕊接著說，她與丈夫育有一子一女，女兒嫁人經濟狀況不好，兒子莊秋龍國中沒畢業就從建築學徒做起，「這種工作收入不穩，像他3月份跟老闆到桃園工地，每天吃住還得扣1百元，下雨又停工，一個月才賺2萬多元。」莊方蕊說，現在兒子得照顧她，已沒了收入。

外甥女送飯菜助生活

莊方蕊的外甥女黃瓊美表示：「我舅媽很苦，年輕時跟著舅舅挑磚頭、跑工地，兒子離了婚，舅媽就幫忙照顧孫女。」黃瓊美說，經營麵店的她三不五時就拿些米、菜接濟，「但醫療期間的房租和生活仍是問題。」
里幹事呂湘河指出，莊家經濟狀況不佳，卻因兒子莊秋龍仍為工作人口，無法通過低收申請，3月底已申請每月3千元的殘障津貼，蘋果基金會獲悉也先行撥款紓困。

無法計算的人生

莊秋龍說：「我父親肝癌、姑姑肺癌、表叔公也癌症過世，現在不抽菸的母親竟是肺癌末期，就怕哪天癌症找上我，那女兒怎麼辦？」
