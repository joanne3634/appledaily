抱殘父坐輪椅 高三女無怨

3姊妹無依 「不知爸爸何時再站起」

2012年03月05日

「爸，我推你去散散步。」讀高三的小雯知道爸爸王仁祥在床上躺大半天，一定覺得悶，她熟練將截肢的43歲爸爸抱上輪椅，穿件防寒外套，推爸爸在家附近兜一圈，雖然初春山間微寒，這就是她的父女溫暖時光。
報導．攝影╱向高彬

來到深山泰雅原民小部落，人煙稀少沒幾戶住家，記者進到小雯家，簡陋平房布滿滲水霉痕，小雯正與36歲媽媽趙惠玲幫王仁祥換穿尿布衣褲。

母女須輪流照顧

小雯說，爸爸12年前糖尿病及蜂窩性組織炎發作，切掉右腿，原本裝義肢，之前能緩步行走，做點以工代賑臨時工貼補家用，但左腿長久出力支撐變形，去年中左髖關節遭細菌感染壞死，不時進出醫院打抗生素，平常臥床休養，「爸醫院一住就一個月，媽媽、我和大妹輪流請假到醫院顧爸爸。醫師說，爸爸髖關節壞了要換掉。」
小雯說，爸2月中手術換人工髖關節後需休養，加上他右腿難施力，之後得重新練平衡學走路，「爸病了，我照顧是應該的，只是不知他還要多久，才能再站起來走路。」
趙惠玲回述，先生沒截肢前，以山上砍竹零工為生，夫妻育有現各就讀高三、國二及小四的3個女兒，「先生很肯做事，只是年輕就截斷腿，他常說，女兒乖，是以後的希望。山上生活苦，要盡力栽培女兒把書讀好，別只能做苦工。」

「以後想當社工」

小雯高職讀幼保科，每天長途通勤一小時到山下上課，正準備5月統測，想讀社工系，她說，曾受社工及師長鼓勵幫助，「喜歡照顧老人，以後想當社工關懷弱勢家庭。」 王仁祥姊姊遠嫁外地，趙惠玲剩一兄照顧老母親，親人無力多幫。
轉介小雯家的公所社會課表示，目前小雯家每月領低收及家扶補助共約2萬餘元，山上就醫交通費是個大負擔，蘋果基金會訪視後，已撥款暫紓山間貧家窘迫。

無法計算的人生

王仁祥說：「去年整理家族祖墳，我身體差、不知剩多久，就預留位置給自己。原本我還照顧一位單身叔叔，他比我健康，我認為他會活得比我久，結果今年初他突然心肌梗塞過世，那位置先給了他，人生無常就是這樣。」
