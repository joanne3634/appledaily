子喪夫跛 嬤種椒養2孫

社工不捨 「她總替這家解決問題」

2012年11月12日

53歲黃貴妹今年9月經歷喪子之痛，心情尚未平復，先生卻又因雙腳痛風嚴重發作無力行走，兒子還留下了就讀小五與小六的2名子女，她頂著烈日在山坡上收成青椒，即使揮汗如雨，仍說：「只要有工可作，辛苦一點沒關係。」報導．攝影╱張嘉恬

採訪時黃貴妹的2名孫子女也在旁幫忙收成，就讀小五的孫子說，爸爸因蜂窩性組織炎病逝後，「我經過爸爸房間還是會想到他，阿公也生病了，現在我們只能靠阿嬤，很想趕快長大賺錢。」

夫痛風先截腳趾

黃貴妹平時和58歲先生陳福永、過世的兒子向親戚借地種青椒為生，黃貴妹說，近期先生常因痛風發作不能幫忙農務，兒子過世後，先生接著在10月初也因蜂窩性組織炎就醫，「醫生判斷截掉左腳掌避免感染擴散，但考量先生意願目前先截了腳趾。」而陳福永本就有糖尿病、高血壓宿疾，現又因痛風、腳傷感染需妻子黃貴妹照顧，她說：「青椒可連續收成4至5個月，平均下來每月收入約1萬多元，加上兩孫補助7200元但仍不夠用，其餘時間我只好作砍草、採茶零工補貼開銷，「就算1天能多個100元收入也能多買包米」。但目前須陪先生往返醫院減少了工作時間，收入更微薄。
黃貴妹說，她和先生都有前段婚姻，過世的兒子是她前段婚姻所生，兩人再婚後共育了現年18歲兒子。「我前段婚姻還有個女兒但很早出嫁，和肢障夫婿要照顧4名子女自顧不暇，先生前段婚姻的獨子也在今年6月過世。」
而現年18歲兒子今夏畢業後，在外地從事時薪工作，他說：「看媽這麼辛苦心很疼，我盡量在回家時幫忙她務農，讓她覺得還有我、還有希望。」而他每月扣除在外房租生活開銷後已盡力協助3000元，仍難敷家中4口生活，蘋果基金會訪視後已撥款紓困。

師讚嬤知足感恩

長期關懷的縣府社工說，黃貴妹過世的兒子是單親爸，黃貴妹對孫子女而言取代了母親的角色，「因先生病弱，家裡出狀況，她就得扛起處理，她一直在為家庭付出，總是在替這個家解決問題」。
黃貴妹就讀小五的孫子班導朱老師說，孩子在校乖巧懂事擔任副班長，「孩子的阿嬤也很知足感恩，學校知悉家況提起協助申請相關補助時，她客氣說一樣一樣來，不好意思同時申請太多」。讓朱老師印象深刻。
