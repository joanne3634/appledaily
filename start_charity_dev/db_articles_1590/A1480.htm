跛男病纏身 無力顧癱母

開刀臥床 月靠7千津貼過活 難籌醫費

2008年01月02日

家住南投的全春德去年（2007年）10月才因左腳罹患蜂窩性組織炎，住院手術，還沒能出院，上個月又再因膽結石動刀一次，疾病纏身讓他多次進出加護病房，支持他與死神搏鬥的動力，就是放心不下53歲下肢癱瘓的母親。報導?攝影╱孫敏聿

病床上的全春德痛得不停抽搐，見到記者還不停問道：「我媽媽還好嗎？你們一定要幫幫她！」38歲的全春德說，自己年輕時曾經有過一段6年的婚姻，之後平靜分手，便一直與媽媽相依為命，「以前還會想沒有生孩子，將來沒人送終，現在想想幸好沒生，不然孩子真的很可憐。」

傷腳上班久站腫麻

全春德說，媽媽3年前腰椎病變，歷經4次手術造成腰部與下肢無力，平常得靠旁人抱到輪椅上才能行動，母子倆生活就靠他四處打零工維生。
不料去年全春德髖關節壞死，陸續置換左右髖關節後，只能跛行，半年前好不容易找到幫砂石車指揮交通的工作，但月薪僅1萬2千元，勉強餬口。
「我換了髖關節後，去上班還得站8小時，回到家整隻腳都是腫的、有麻麻的感覺。」全春德說，媽媽癱了之後，他每天上工前得花幾百元央請鄰近親友代為看顧，「請他們幫忙媽媽買菜、煮飯和換尿布，不然一回家媽媽又餓又臭，我看了都掉眼淚。」
全春德說，母親因沒有低收資格，他又無力負擔安養院每月2萬元，他只得把媽媽託給鄰近的阿姨照料，阿姨金賢慧說：「我看姊姊這樣，心裡也很難過，但我每天種菜工錢800元，載姊姊下山一趟油錢和看診費都不夠，真的很困難。」但一想到姊姊老來貧苦，還是咬著牙幫忙照料。

低收因哥哥未通過

當地教會松牧師表示，全春德還有一個哥哥，多年前結婚後搬至台東妻子家，逢年過節也很少回來，也沒資助家裡，全春德目前無法工作，母子倆只靠殘障津貼7千元和親友接濟，難支應醫療與生活開銷。
村長伍春財指出，全春德母子處境堪憐，2007年的低收因全春德尚有哥哥無法通過，今年（2008年）的低收入戶資格尚在申請中，預計今年2月可得知申請結果，如能通過每月補助增至1萬1千元，屆時全母亦享有公費安養資格。蘋果基金會得知已先撥款暫紓此階段困境。
