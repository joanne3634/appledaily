婦2天沒睡 工作兼顧癌夫

2兒年幼 積蓄用盡 家計陷困

2012年11月08日

44歲李建良今年5月底罹食道癌，術後只能靠胃造口灌食，虛弱的他癱躺病床，妻子正替他按摩減緩痛楚，6歲長子見一旁4歲次子大聲嬉鬧，懂事地對弟弟說：「噓……小聲點，ㄅㄚˇㄅㄧˊ（童語：爸爸）不舒服，不可以吵他。」
報導．攝影╱張嘉恬

採訪間李建良因癌病疼痛每每說不了幾句話，便痛到無法言語，甚至哭了出來：「不知道自己還能和妻兒相處多久……。」而33歲的妻子裴氏香神情憔悴，為兼顧工作與先生，她已連續2天2夜沒有好好入眠，她說，夫妻倆原都在養護機構擔任看護，先生5月罹癌至今積蓄用盡，所獲的「馬上關懷」急難金、教育部學產基金急難金及慈濟急難金共約4萬元也皆用於生活。

夫化療需長期照護

裴氏香說，任職的養護機構體諒她的困境，讓先生住養護機構以每日350元計價，她工作時間自晚間8時至白天8時，其餘時間則照顧先生，兩幼子則託母親從越南來台照顧，「先生病後除了體弱，情緒更不穩，我不能離開他太久。」
目前李家4口生活仰賴妻子月薪2萬8000元，扣除房租6000元、住養護機構平均1萬元的費用後，僅剩1萬2000元，不敷開銷，太太裴氏香說，目前先生開始進行化療，「我要長期照顧他，沒辦法工作，收入完全沒有了。」
林姓里長說，李建良罹癌無力撐家，「太太能力有限，短期急難金僅能應急，所以轉介蘋果基金會。」里長並將進一步協辦低收資格，蘋果基金會探視後也已先撥款紓困。

6歲長子乖巧懂事

夫妻倆工作的養護機構羅姓護理長說，裴氏香與先生同在此機構工作10多年，「兩人都認真盡責，太太還是組長，看她獨自兼顧工作、病夫、2幼子，體力無法負荷，同事這麼多年，我們能盡力的是減輕她先生費用負擔。」
採訪間，李建良說想回住處看看，記者陪同到家，6歲長子自動自發將帶去學校的餐盒拿出清洗，李建良夫妻在旁瞬間眼泛淚光。

【無法計算的人生】

裴氏香說：「從事看護工作，10多年都在照顧別人，沒想到有天自己的先生會需要被照顧，癌症難醫，我想不到長遠未來，只能珍惜先生和我與孩子相處的每一天。」
