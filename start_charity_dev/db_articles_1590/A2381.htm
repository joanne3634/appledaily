7旬嬤抗癌憂殘兒失依

16年前車禍傷腦智力退化如3歲童

2011年01月23日

午餐時刻，69歲的劉秋葉一匙一匙餵著48歲的兒子（劉醇洋）阿洋吃飯，16年前，未婚的阿洋因車禍傷腦，起居全靠劉秋葉照料，不料去年（2010）11月劉秋葉診斷罹患胃癌，面對漫長抗癌療程，她心裡惦掛的不是自己病情，而是擔心重度殘障的阿洋該怎麼辦？
劉秋葉（以下簡稱劉）與記者（以下簡稱記）訪談母子當前困難：
劉秋葉口述 記者韓旭爾採訪整理

月3千勉強度日

記：這些年來母子生活如何？
劉：16年前當警察的阿洋騎車買便當時，不幸發生車禍，頭部重創，肇事者逃了，雖撿回一命，卻造成智力退化，變得像3歲小孩，整天胡亂吼叫，只能勉強站著沒法走路。當時警局給了約100萬元提前退休金，我帶著阿洋四處求醫、拜神，錢都花光，阿洋還是沒好轉，現在吃飯、洗澡和大小便都要靠我幫忙，就好像在養一個永遠長不大的孩子。
為了看顧阿洋，我沒法外出工作，因為20年前去世的老伴是林管處技工，我們一直住在免費的員工宿舍，母子靠低收入戶補助和阿洋殘障津貼每月1萬3000元生活。但去年9月時，林管處收回宿舍，我們必須租房子，政府補助扣掉房租、阿洋尿布錢，我們母子每月只剩3千元，就煮些白粥，配肉鬆、醬菜，有鹹味，就能度三餐了。

肚痛未治竟成胃癌

記：癌症發病情況？
劉：我2、3年前肚子就開始悶痛，一直沒去看病。去年11月時，肚子痛到受不了，半夜到醫院掛急診，檢查結果是胃癌，醫生說我有嚴重貧血，不能手術只能先化療，等到體力好再開刀，我住院時，社會局社工說可讓阿洋接受公費安置，但每月要繳3、4千元的尿布、衛生紙這些耗材費用，這錢我付不起，只能叫大兒子阿海每天到家裡來餵阿洋吃三餐。

長子拖病無力協助

記：大兒子能再多幫些忙嗎？未來打算如何？
劉：我的大兒子阿海是單親爸爸，兩個孩子還在讀書，他肝、腎都有毛病，沒法工作，也是靠政府低收入戶過活，現在我罹胃癌，在經濟上他幫不上忙，只能幫我顧一下阿洋，我對阿洋說：「阿母生病了，你要乖一點！」阿洋好像聽得懂，會靜靜的看著我吃藥。對生死我看得很開，只是放心不下阿洋，謝謝蘋果基金會給我急難金幫忙。我活著的每天，都希望能好好照顧阿洋。
