嬤喚癱兒「要養女兒長大呀」

妻照護無力打工4口家計成愁

2011年05月19日

「紹華，行行好，不要讓媽媽這麼煩惱，你女兒還小，還要你養她長大呀。」67歲的張徐桑妹喚著近植物人狀態的兒子紹華，但38歲的紹華雖睜大眼，但雙眼無神，沒絲毫回應。
報導．攝影╱向高彬

災禍起因都怪貪杯誤事，張徐桑妹說，兒子之前做搭鐵屋零工，「酒品是不太好，醉了亂罵人，我們都跟他說少碰酒。不料他元旦時休假，中午到小吃店喝茫了，跟別桌酒客起衝突，被棍子敲頭。」張徐桑妹說，事發後媳婦找到他時，他躺地上已沒呼吸心跳，也因休克缺氧造成腦病變、全身癱瘓，醫師評估他可能餘生如此。

愛女見父淚流不止

困境突來，張徐桑妹說，若早知紹華就剩一口氣、半條命，那時也不跪求醫師救活，「不如一次走，我一次哭完，不要我看一次，掉一次淚。」此案正以殺人未遂案開庭偵查，張徐桑妹說，對方表示無力賠償「就去關吧。」
張徐桑妹感傷回述，紹華傷癱前脾氣雖大男人，但他顧家疼愛女兒，「每天大清早，都是他幫10歲獨生女小珊準備書包，笑嘻嘻騎車載她上學，結果他這家庭支柱卻倒了。」女兒小珊在旁看著爸爸，眼淚撲簌簌流。

接回家「沒放棄他」

採訪時，紹華33歲的太太阿蓮說，從先生傷後，她伴著先生按摩手腳，跟護士學管灌、抽痰和拍背，但無力負擔安養費，且覺自家人照顧較周到，近日將紹華接返家中照顧，「回家裡住，他會感覺我們沒有放棄他。」只是阿蓮一時無力做工，張徐桑妹每月領老人年金3000元，得幫還紹華先前積欠分期健保費4000元，加上紹華尿布管灌醫療耗材費，往後家計成愁。
張徐桑妹的大女兒說，常探望小弟，但自家有就學子女，能幫不多。當地公所協辦急難救助金2.8萬元，有犯罪被害保護協會關心後續，邱姓里長說，後續待醫師評估張紹華殘障鑑定，協辦低收，「對方以自衛過失辯護，案件審理不知何時有結果。」蘋果基金會訪視後，也撥款暫紓生活急窘。

報你知 因犯罪而傷 可請補償

法務部於1999年籌設財團法人犯罪被害人保護協會，於各縣市設立分會提供相關協力。若有因他人犯罪而被害死亡、重傷，受害家屬可於知有犯罪日起2年內提出申請，由地檢署被害補償審議委員會依案情佐證及傷殘程度審核，死亡最高補償210萬元，重傷最高180萬元，審案時間長短及補償金額依各案有所不同。相關諮詢可洽全國保護專線：0800-005850。
