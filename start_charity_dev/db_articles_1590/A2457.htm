妻癱子病 失業漢苦撐家

生活開銷沒著落 「只能勇敢面對」

2011年05月03日

51歲阿珍罹患腦癌，癱躺在病榻上，對於先生阿富的呼喚，她沒有回應，只是怔怔望了先生再望向遠方。阿富說，太太完全無法自理生活，需要他在旁照顧，兒子才讀國一，憂心未來生活。
報導．攝影╱張嘉恬

現年59歲的阿富說，妻子本在5年前發現良性腦瘤，手術取出後生活、工作皆正常，但今年2月妻子卻出現走路不穩、口齒不清及昏睡情形，經檢查後確定為惡性腫瘤。阿富說，3年前他因金融海嘯失業致家計困頓而欠卡債，累積至今約欠百萬，經協商每月分期須繳1萬8000元，「舊債未清現又沒收入，妻子營養品、每月房貸1萬元和生活開銷都沒著落。」

國一養子情緒障礙

阿珍的好友侯小姐說，阿珍一家3口過去生計仰賴先生在家具工廠作外務工作維持，阿珍自幼自學電子琴，婚後為補貼家用，在琴行教琴賺取時薪，她說，與阿珍相識20多年，「阿珍父親早逝，間接造就她自立自強的性格，前幾年她先生失業家裡經濟出狀況，她教琴之外，還瞞著先生接設計圖案的外快工作，很多事情自己扛下來。」侯小姐也說，阿珍夫妻婚後因久未生育，13年前領養了現年就讀國一的兒子，養子雖在3年前經鑑定為情緒障礙，但「在工作之餘，她仍全心投入照顧兒子，沒想到她說倒就倒。」

「過一關又來一關」

阿珍在琴行的同事楊小姐說：「她為了生計，5年前得良性腦瘤時，開完刀沒休養就再工作，但她病後再出發，學生人數已銳減，原本每月尚約有3萬元可和先生一起養家，後來收入僅剩1萬多元。」而阿珍兒子的學校唐姓輔導老師說，學校除了為孩子安排心理諮商，也正協助申請學產基金急難金。
阿富看著妻子說，多年前他失業時，夫妻倆因生計問題口角不斷，「當時吵到要離婚，但我堅持不肯，同心協力過了一關，現在又來一個難關，我只能勇敢面對。」阿珍雖尚有兄弟與妹妹，但除皆有家計重擔，尚需分攤照顧罹患痴症母親的責任，難有餘力協助，當地里長正協助申請「馬上關懷」急難金，蘋果基金會訪視後已撥款暫紓困境。
