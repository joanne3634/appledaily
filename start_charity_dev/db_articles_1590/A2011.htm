夫吐血猝逝 妻含淚產子

「孩子出生就沒爸」 外配抱憾育2娃

2009年10月23日

「每次餵孩子喝奶，看著他的小臉，我就會想起過世的先生。」23歲的印尼籍外配彭莉莉，9月中眼見癌重先生吐血猝逝，隔兩天遺腹子在哀戚中誕生，如今她得撫養兩年幼子女，未來育子路長，讓她傷神。

以下為彭莉莉（以下簡稱彭）與記者（以下簡稱記）的訪談：彭莉莉口述╱記者向高彬採訪整理

癌夫病發 送醫中斷氣

記：先生發病到猝逝的情況是如何發生？
彭：我先生阿鈞42歲，今年2月咳不停，到藥房配成藥吃，但一直沒好，3月時，因3歲大女兒感冒，他帶去看醫師，自己順便看，居然被告知是大病，是下咽癌4期，我當時已懷小兒子，挺肚陪他化療、電療大半年，醫師認為狀況算穩定，我心想他一定也期待看到小兒子出生，心情好有助病好。卻沒想到那天半夜他突喊醒我，用毛巾摀住冒血鼻嘴，救護車來不及載他到醫院急救，半路就斷氣了，隔兩天，我小兒子出生，阿鈞終究來不及與孩子見一面，孩子一出生就沒有爸爸了。

公婆罹病 照料更費神

記：先生猝逝後，妳又在坐月子，這段時間的生活怎麼過？
彭：這段時間跟親友借了快8萬元醫病生活，先生喪葬用農保理賠、奠儀和馬上關懷救助金辦好，他半句話都沒交代，只留這兩個小娃娃給我。先生之前做裝潢工，原本他不想結婚，後來經71歲的公公催促要傳後代，4年前才娶我，原本我們在外地租屋勉強可過。他對我和女兒很好，有時逛夜市他點碗愛吃的土虱頭湯，一家合吃得很高興。
他病後，我們搬回先生鄉下老家的老三合院住到現在。只是公公今年4月腦膿瘍開刀，67歲婆婆2年前曾中風稍失智，都行走不便，兩老靠大伯照料，但他也有家庭，但至少讓我們有地方住，生活難題得想辦法解決。

女兒童言：爸爸睡著了

記：妳會和小孩長期在台灣居留？
彭：娘家在印尼加里曼丹偏遠的鄉下，工作很少、生活非常困苦，這兩年才有電。先生病逝，我不想跟娘家的人說、也沒打算回娘家，怕他們擔心。
台灣的社會福利、醫療和教育好、人情味濃，我把居留證辦好之後，決定要在這裡做工，好好養大我的孩子，吃苦沒關係，我以前也做過苦工。
女兒還小，什麼都不懂，像他爸爸過世時，她只知爸爸身體冰冰、說他睡著了。等兩個孩子大一點，我會跟孩子說，他們的爸爸到天上做神仙了，他是好爸爸，正在天上保護照顧著我們。
