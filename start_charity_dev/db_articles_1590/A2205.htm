父母接連去世 3兄妹孤苦無依

19歲長子忍悲拼指考 盼父「保佑考試順利」

2010年07月01日

剛高中畢業的小賢19歲新接戶長，前年底剛送走媽媽，又在5月底送走心肌梗塞猝逝爸爸，母墳土才乾，就要新造父墳，往後留下他與仍讀國二大妹小琪及小五小妹小柔孤苦相依，但他必須抹去憂傷，全力為明天（2日）的大學指考衝刺。報導．攝影╱向高彬

蘋果基金會當時得知此一不幸消息，即於金宗賢的爸爸金輝南出殯前一天，前往深山部落關懷貧弱家庭，並與3兄妹的學校師長、家扶及展望會社工、村長及公所社會課人員商討，協辦金輝南後事。基金會曾於2008年8月時撥款2萬元救助小賢母親醫病急難，並同時以「克蘭詩．蘋果助學專戶」助學3兄妹至今，基金會現獲悉金家再逢困厄，往後仍漫長，也撥款濟助3孤生活。當地社工表示，金家現每月有三項低收、家扶補助共12800元，生活暫有鄰近年邁大伯和打零工的舅舅照應，社工會持續關懷金家3兄妹。

【棺前守靈】

爸爸金輝南的棺木剛送到，小賢說，家裡簡陋僅1房，爸爸生前打地舖睡客廳，做農雜工的爸爸5點半會叫他們起床上學，但那天6點半大妹醒來，發覺爸爸已沒了呼吸，往後再聽不到他叫喚大家起床。

【冰櫃禱告】

功課不錯的小賢2月時原本已推甄上淡江大學，但爸爸苦家計緊絀、只得讓他放棄，小賢說，爸爸原本答應要再陪他參加大學指考，但爸爸卻猝逝失約、冷冰冰躺冰櫃裡，「和妹妹禱告盼他安息，也希望他保佑我考試順利。」

【撫墳思親】

小賢說：「媽媽前年底過世時40歲，她肝病嚴重拖3個月病逝；爸爸生前雖好酒，但對我們兄妹功課仍很關心，也說有天若他先走，就近埋了陪媽媽。爸爸走時51歲，那天清早沒留一句交代，但我們知道，把他葬在媽媽旁邊，比較不孤單。」

【雙親遺照】

小賢將爸爸的後事都辦好，回到家他把爸爸的遺照掛在客廳牆上，擺在媽媽的旁邊，小賢說，和妹妹一進家門，就會看到爸媽，儘管他們已不在人世，但想像他們將一直在天上守護著，「妹妹還小，以後我也要想辦法照顧她們長大。」

國中導師 來回600公里陪考

5月底小賢爸爸猝逝，讓他慌亂手腳，而且出殯後隔兩天就是警專考試，當時小賢得收拾心情，求助來參加爸爸追思禮拜的國中班導張致瑋，張老師立即答應幫忙，讓小賢很感謝。張致瑋現任大妹小琪班導，教授自然科學，常協辦獎助學金，他說，小賢功課操行一直很好，也考上第一志願公立高中，大妹也是全班第一名。

上網找赴考資訊

張致瑋犧牲六、日假期，自掏腰包車資食宿約6000元，來回奔波超過600公里，很少上台北的他上網找資料，帶小賢輾轉客運、高鐵、捷運赴考。周六下午到台北時，他讓小賢專心看書，自己趕搭公車幫小賢看考場。雖然，最終，小賢未考上警專，但他相信，小賢大學指考會有好成績。「山上部落和都市真是兩個世界，也疼惜有困難的學生，我只是做分內的事。」
小賢的小妹小柔學校教導主任伍曉玲也說，小柔功課在班上前二名，曾代表學校參加演講比賽。不過下學期初她感冒造成顏面神經失調，變得口歪有點口齒不清，本來活潑變得內向，老師常關心輔導，也帶她下山做復健，已慢慢好轉。爸爸猝逝，我們會持續輔導，讓孩子盡快走出悲傷。
