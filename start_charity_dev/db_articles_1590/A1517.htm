夫強忍痛風 為盲妻打針

「像千百支針往關節扎」無力維生

2008年02月29日

剛過的「年」，除貼上春聯，余國興家中毫無過年的熱鬧氣氛。一如往常，吃完早餐，余國興忍著手指關節長滿痛風石的疼痛，拿起胰島素針劑，為視力幾乎失明的妻子美貴打針，他感慨地說：「現在家裡只有我們夫妻倆相依為命，就把命運交給老天安排吧！」報導．攝影╱韓旭爾

「痛起來好像有千百支針一直往關節裡扎，真的生不如死！」53歲的余國興十多年來飽受痛風所苦，他原靠當建築臨時工，每月掙二萬多元維持家計，但這幾年痛風病況加重，早已無法勝任粗重工活。53歲妻子美貴有多年糖尿病史，以往還能擔任農產品加工零工，但二年前視力惡化，左眼失明，右眼只剩0.01，幾乎什麼都看不見，只能約略感受光影。余國興說：「美貴突然失明，她自己很難適應，原本簡單生活起居，現在凡事都要我協助。」

養女遠嫁難協助

余國興表示，家裡有痛風家族史，他的母親就是因進行痛風石切除手術遭感染去世，他一直很注意自己病情，除定時服用藥物治療，還很重視飲食控制，但病情還是每下愈況，余國興無奈地說：「不管如何治療，痛風還是愈來愈嚴重，現在雙手雙腳長滿大大小小的痛風石，痛到無法入睡。」雪上加霜的是，去年六月余國興突感覺胸部持續疼痛就醫，結果診斷出心肌梗塞，住院進行冠狀動脈繞道手術，才救回一命，現在仍需每月回診追蹤。
美貴感慨地說：「自從我失明後，我們倆好像連體嬰，我實在離不開國興，就連去年他住院進行心臟手術時，自己在家，因看不清胰島素劑量，無法自己打針，只好跟著到醫院住，讓護士幫我打針。」
余國興夫妻結婚多年未生育子女，領養的女兒已遠嫁，無力提供協助，余國興說：「女兒沒工作，女婿每月工資才二萬多元，要租房子，又要養三個孩子，就算想幫我們，心有餘而力不足呀！今年過年，因年假不長也未能帶孫子回來看我們，心裡感覺特別寂寞。」

賴殘津接濟餬口

三年多來，住附近的鄉鎮調解委員會主席張事民偶爾拿些白米、沙拉油與泡麵接濟余家。余國興說，現在兩人只能靠殘障津貼每月七千元度日，但每次就醫車資就需一千元，平均每月兩人就醫車費就要二、三千元，還要付房租，「平時左鄰右舍會贈送些自家種的蔬菜，里長也會拿東西來，他們的好意讓我們至少不用餓肚子。」蘋果基金會訪視後，已由「歐舒丹．蘋果視障專戶」撥款紓困。張事民說，現正申請更改鑑定殘障程度，需費時三、四個月，更改殘障程度後，案主夫妻便能申請低收入戶補助，減輕經濟負擔。

無法計算的人生

余國興說：「不知自己這輩子走什麼樣的霉運，年輕時生不出子嗣，老了夫妻倆還同時生病，這都是命中注定！」
