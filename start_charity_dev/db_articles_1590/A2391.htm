3稚女陪母縫鞋 醫舌癌父

一隻鞋賺35元 「每一塊錢都是救命錢」

2011年02月11日

走進32歲阿萍（鄭雅萍）家裡，只見她在客廳埋首做著手工皮鞋，阿萍說，從丈夫阿榮去年10月發現罹患3期舌癌後，5口之家就靠這份代工過活，「花半小時縫一隻鞋才賺35元，雖不多，但每一塊錢都是我們一家的救命錢！」
報導．攝影╱韓旭爾

阿萍與36歲的丈夫阿榮共育有3女，分別為13歲、9歲及6歲，家計原本依靠阿榮擔任吊車司機維持，阿萍則在家照顧孩子，利用空檔時間從事家庭代工每月掙得數千元貼補家用。
阿萍說，2年多前受金融海嘯衝擊，阿榮失業近1年，當時不只每月1萬5千多元的房貸欠繳，房子被查封法拍，「甚至連一家三餐都成問題，只能1、2千元跟親友借來應急，去年初好不容易阿榮找到新工作，沒想到工作才7個多月，卻發現罹患舌癌。」

婆嘆兩子接連罹癌

阿萍說起阿榮罹癌，眼泛淚光，「收入少，吃差點、吃少點就能過日子，但現在最煩惱就是擔心阿榮病況。」阿榮住在老家祖厝的媽媽阿馨婆說，每年阿榮都會帶妻兒返鄉過年，去年因長子阿泰癌末讓家裡愁雲慘霧，今年春節阿泰病情更加惡化、阿榮也罹癌，她說，「有時覺得老天對我好不公平，兩個兒子都得了難治的歹症頭，萬一白髮人送黑髮人，還有誰能依靠？」
回憶阿榮發病情況，阿萍說，阿榮嚼了5年多檳榔，1年多前，因大伯發現罹患口腔癌末期，阿榮開始戒除吃檳榔習慣，沒想到去年8月阿榮感覺喉嚨疼痛，原以為只是感冒，到診所就醫數次不見好轉，過了1個多月，頸部左側出現明顯腫脹才到大醫院檢查，「醫生一見到阿榮，就說情況不太妙，切片化驗後確診為舌癌，並擴散到淋巴及氣管。」阿萍說，現在阿榮需先化療，待腫瘤縮小再進行手術。

苦惱管灌食物開銷

當地公所承辦說，阿榮因無殘障手冊，仍被計入工作人口，一家暫難獲低收入戶核可，目前正協助申請特殊境遇家庭的相關補助中。
阿萍說，家庭代工每月最多只有8000、9000元收入，但房租就要7000元，「現在阿榮吞嚥困難，醫生正評估要插上鼻胃管，又多了管灌營養罐頭開銷，真的很傷腦筋。」蘋果基金會訪視後，已先撥款暫紓阿萍一家生活、醫病困窘。
