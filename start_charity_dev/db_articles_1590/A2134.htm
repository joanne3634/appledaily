夫癱女病 婦泣嘆度日

女出院「媽媽頭髮花白 快認不得」

2010年04月04日

去年底，54歲的阿達因腦中風導致左側手腳無力，21歲女兒小如也因見爸爸病倒而躁鬱症發病，一度強制安置就醫，家中接連變故，讓妻子阿馨不禁泣嘆。報導 攝影╱游竹君

「這幾個月來，媽媽突然多了很多白頭髮，我都快認不得。」過年前出院的女兒小如，情緒已較和緩，這些小如脫口而出的話，讓53歲滿頭花白頭髮、滿臉疲倦的媽媽阿馨，聽了不禁眼眶泛淚。

無法請領殘障補助

阿馨說，自己曾有一段婚姻，早年離異後，與丈夫阿達育有一女小如；多年來，兩人靠鄉間零工養家，小如前年高中畢業後則在便利商店、飲料店打工，家計尚可維持。
阿達說，自己有遺傳性糖尿病，近年來體力日虛，去年11月初騎車時因暈眩而撞上冷凍車後門，當時沒有外傷也就沒去檢查，直到一星期後突然昏倒，就醫才發現疑似車禍後的腦出血導致腦中風，左邊手腳無力，左眼視力也模糊一片，「沒想到女兒也因擔心家裡經濟加上感情受挫，1月中旬又吞服大量安眠藥自殺，強制就醫。」
阿馨坦言，夫妻間常口角衝突不斷，「原想夫妻情分已盡，沒想到去年11月阿達中風後根本沒處可去，念在夫妻一場，還是接回家照顧。」阿達說：「我的手腳還是不時僵硬麻木，眼看阿馨快撐不住了，我也想拚命復健，快點好起來再找工作。」阿達雖領有殘障手冊，但因母親名下尚有積蓄，導致無法請領殘障補助。
阿達弟弟陳健全說，無論颳風下雨，母親每天上街撿回收變賣，「都是我們兄弟不爭氣，接連中風、體弱，她為了存老本，我們兄弟實在沒臉開口要母親多幫忙。」

每月需繳9千房貸

村長羅偉菘也說，鄰里間也清楚阿馨婆媳間關係不好，長期經濟都是阿馨他們自己想辦法籌措。
當地身心障礙個管中心沈姓社工表示，阿馨每月還需繳納房貸9000多元，也出現不自主哭泣、失眠等症狀，雖曾建議她與女兒同時就醫，但總因擔心掛號等就醫費用而遲未行動。
小如目前仍處於急性發作期，必須靜養一段時間後才能再外出工作，醫院社工已協助減免小如部分醫療費用。蘋果基金會訪視後，也已撥款協助醫療及生活開銷。
