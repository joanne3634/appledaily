洗腎男撐病體 想伴兒長大

連撿破爛都沒力氣 4口生計無著

2010年03月26日

時序入春，天氣回暖，62歲的紀榮華病厭厭坐在自家庭前曬太陽，一臉愁容，直到看見就讀小學3年級的兒子小恩回家，才露出笑容問說：「今天在學校過得好不好？」報導 攝影╱韓旭爾

紀榮華說，自己多病纏身，現在只希望能多撐幾年，「讓我好好陪伴年幼的兒子長大！」他除每周洗腎3次外，去年底還因嚴重肺炎住進加護病房2個多月，農曆年前雖出院返家休養，但病況仍不穩。

妻唇顎裂覓職困難

記者訪談過程中，紀榮華沒講幾句話，便咳嗽不止，他無奈地說：「白天還算好，一到晚上要睡覺時，整個肺都好像要被咳出來一樣。」說著說著紀榮華又開始咳嗽，一旁的42歲的妻子阿英（張瑞英）見狀，忙用手拍紀榮華背部。
紀榮華說，他與阿英育有2子，分別為21歲及9歲，阿英先天性唇顎裂，不只語言能力受損，加上反應遲鈍，找不到合適工作，多年來一家生計全靠他做綁鐵工，每月掙得3萬多元維持開銷。4年多前，紀榮華因腎臟病變，須定期洗腎，再也無氣力從事綁鐵粗活，改以資源回收工作每月掙得4、5千元勉強餬口。在去年底紀榮華肺炎病重，連外出撿破爛也無法勝任。
阿英見丈夫為了生計發愁，她口齒不清地說，自己也很想找份工作，但「老闆們看到我連話都講不清楚，都不想用我。」
提及紀榮華21歲的長子，他搖頭無奈地說，長子遺傳了阿英的唇顎裂，智力也不好，學歷僅國中畢業，覓職困難又愛玩。9年前阿英意外懷上小恩，紀榮華老來得子，現在最掛心的就是年幼的么兒小恩，「小恩功課不錯，體育也很好，去年曾獲全國比賽冠軍，將來一定出人頭地，他是我這輩子的驕傲。」

就醫往返開銷沉重

當地村長余國憲說，現紀榮華一家四口全靠政府低收補助共1萬6千多元度日，但紀榮華現在跑醫療交通開銷大，每次包車就1千多元，負擔沉重。蘋果基金會訪視後，已先撥款暫紓紀榮華一家困窘。

後記
「拿好成績給爸爸看」

採訪時，才9歲的小恩表現很成熟，他說：「我了解爸爸的辛苦，我會努力贏得好成績給爸爸看。」小恩的導師王維說，他很懂事，同學們偶爾會向他開玩笑說：「你爸爸的年紀好大，看起來像爺爺！」但小恩都不以為意，每天放學便幫爸爸按摩手腳，是個貼心小孩。
