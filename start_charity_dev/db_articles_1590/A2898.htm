父子罹癌 「病痛如吞針」

清潔婦難撐家嘆 「營養品可買2便當」

2012年10月17日

9月初，記者初至張家關懷時，36歲未婚的阿銘（張志銘）身體蜷縮床上，頸部右側紅腫嚴重，不停呻吟哀號，37歲姊姊張雅慧安慰說：「忍耐一下，過幾天開刀後就會好一些！」她說，弟弟舌癌復發又轉至頸部淋巴，60歲的爸爸罹口腔癌術後也1年了。 報導．攝影╱韓旭爾

阿銘10月初術後出院，目前尚要進行電療，並僅能以營養品灌食。阿銘爸爸阿南伯說，阿銘的痛苦他也經歷過，「感覺就像把整把針往嘴裡吞一樣，比死還難受，我都撐過來了，阿銘一定也可以！」阿南伯邊講話，邊用衛生紙擦拭嘴角溢流的口水，他在1年多前發現罹患2期口腔癌，經歷2次手術後及放、化療後才控制病情，卻留下無法吞嚥的後遺症，「吃東西只能用倒的」，目前仍需每月回診追蹤。

切除腫瘤又復發

張雅慧說，她5年前離婚後搬回娘家，2個兒子歸前夫監護，原本做清潔零工維持自己生活開銷，也分擔部分兒子教育費用。但近2年來，先是爸爸罹患口腔癌，沒過3個月媽媽中風去世，現在弟弟又舌癌病發，「這陣子忙著跑醫院，工作也做不了，生活只能靠親友小額資助，只是大家幫久，也沒法再幫了。」
張雅慧的前夫說，念著昔日夫妻情分，他手頭寬鬆時，會拿幾千元援助，「只是現在景氣差，修車工作也不好賺錢，也幫不了多少。」
阿銘說，他原為貨車司機，今年初時出現舌根潰瘍，但他不以為意，拖到3月時，右耳出現聽力障礙，就醫檢查發現有腫瘤壓迫內耳，診斷為3期舌癌，「雖開刀切除腫瘤也做了放、化療，但8月發現復發。」
阿銘說，他以前每天需開車近15小時，只能靠嚼檳榔提神，「每天都吃2、30顆檳榔，舌頭、牙齦破皮習慣了，對於癌症前兆完全沒警覺。」阿銘又說：「為了開車顧三餐，不得已才吃檳榔提神，但吃檳榔造成舌癌，是我自己『討皮痛』（台語，意謂自討苦吃），也怨不得別人。」

7千元津貼度日

當地社會課葉姓承辦說，目前張家靠阿銘、阿南伯兩人殘障津貼共7000元度日，3月時已協助申請政府「馬上關懷」及中華基督教救助協會急難金共3萬5000元，因阿銘術後營養品開銷大，轉介蘋果慈善基金會協助，張雅慧說，阿銘腎臟不好，特別配方每罐就要120元，都可買兩個便當了，實在喝不起。基金會訪視後，已先撥款暫紓張家生活、醫病困窘。
