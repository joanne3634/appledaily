女友遣返 病男憂難再聚

「命是她救活的」 老弱3口失依

2012年05月02日

49歲蕭文弘因肺疾復發身不離氧氣機，他拿下氧氣面罩虛弱地說，女友與1歲兒子四月中被遣返柬埔寨，病重的他和80歲老母及14歲女兒無人照顧，而且「這次分開不知到什麼時候才能再相聚」。他惆悵地說。報導 攝影／張嘉恬

「蕭文弘有肺部疾病，為養家常抱病上工，他工作很認真，但1月底染上流感後，一搬重物就滿臉通紅喘到快斷氣，當時還硬撐。」蕭文弘老闆黃先生說，蕭文弘家有老小，僅靠他1人撐養，平時生計即清苦，病重後「我能作的是協助個幾千元，日後他身體恢復，體力負荷若只能工作半天，公司可以配合，見人有難，體諒幫忙是應該的」。

依賴氧氣機過活

蕭文弘7年前不慎將門窗緊閉噴灑殺蟲劑致肺部重傷，3年前又切除一半右肺，持有重度的重器障手冊，平時即因病進出醫院收入不穩，1月底因流感使得原已病弱的肺部引發肺氣腫及慢性氣道阻塞，無法自行呼吸，得仰賴氧氣機，醫囑得休養至少半年。
偌大客廳裡，蕭文弘想起昔日1歲兒子往他身上靠近，小手小腳不停向他揮舞，他感嘆肺病纏身、生計貧困，但有愛相伴，「醫生說我要有人在旁邊照顧，現在女友和兒子回去了，我也照顧不來老母和女兒，我的病能不能恢復不知道，現在情感依靠也沒了。」他的女友在台原有段婚姻，離異後經人介紹相識，「她不嫌我窮又病，每次住院都寸步不離照顧我，也照料我老母及與前妻生的14歲女兒，沒有她我早活不了，我的命是她救的。」

移民署同仁協助

移民署鄭姓科員說，蕭文弘女友在台婚姻離異後應返柬埔寨，留下照顧蕭文弘7年屬非法居留，「因她主動來報到，且家境貧困屬實，易科罰金5萬多元已找尋民間團體協助，同仁也協助5000多元，因柬埔寨禁止人口外移，女友遣返後，沒法用結婚方式再來。」蕭文弘女友遣返後，他和老母及女兒3口僅賴補助約1萬元，不敷房租8000元及生活，蘋果基金會訪視後已撥款紓困。
