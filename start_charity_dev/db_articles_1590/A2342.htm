阿公癱 父罹癌 全賴孫看顧

醫院家中兩邊跑「恨不得有分身」

2010年12月10日

阿嘉（李嘉益）55歲的爸爸阿仰伯去年中發現罹患3期喉癌，病情惡化，經常進出醫院，阿嘉說：「我不只要到醫院照顧爸爸，家中還有一個癱瘓臥床的阿公。」近半年來，阿嘉來回奔波，身心俱疲。報導．攝影╱韓旭爾

未婚的阿嘉說，去年7月時，55歲的爸爸開始出現吞嚥困難情況，就醫檢查發現罹患3期喉癌，醫生評估先進行化療及放射線治療，待腫瘤縮小後再進行手術。阿嘉說：「自己長年在外地工作，知道爸爸罹癌後，原本想馬上辭掉工作陪同抗癌，但爸爸很堅強，總是說，他自己可以照顧自己，要我以工作為重。」但未料爸爸今年2月開刀摘除腫瘤後，4月時病情開始惡化，癌細胞擴散頸部淋巴組織，他只能辭去工作返鄉照顧爸爸。
阿嘉回溯說，9歲時媽媽病逝，他國中畢業後便到外地擔任作業員，爸爸則在家中從事農務零工，家中還有分別83歲及82歲的老邁祖父母。

阿嬤雙腿不良於行

阿嘉說，阿公原本就有糖尿病，今年初時，因泌尿系統感染導致敗血症，住院約1個月才撿回一命，出院後，阿公卻出現癲癇、老人痴呆情況，三餐、大小便都要人幫忙，「若申請政府居家照顧服務，每天2小時每月就要3千多元，也是一筆負擔。」阿嘉說，阿嬤有高血壓、退化性關節炎毛病，雙腿略有不良於行，「現在我守在醫院看顧爸爸，又擔心家裡阿公、阿嬤情況，恨不得自己能有分身術。」阿嘉無奈地說。
記者訪視時，只見阿嘉的爸爸靜靜地躺在床上，眼睛不時開閉，口中塞著紗布，阿嘉說，爸爸傷口出血不止，每次幫他塞上紗布止血時總是特別難過，「我希望爸爸能堅持打敗癌症。」

妹妹擺攤難伸援手

阿嘉的阿嬤阿蜜婆說：「現在我擔心到吃不下，也睡不好，萬一兒子比我先走一步，我也撐不下去了！」阿蜜婆說，老伴現在變得憨傻反而好，「什麼事都不知道，也就不會傷心。」
當地公所急難承辦說，目前阿嘉一家領有祖父母老農年金及父親殘障津貼共15000元，正協助申請「馬上關懷」急難金中。阿嘉說，每月光是阿公、爸爸尿布開銷就需7、8000元，「政府補助就去掉一半了。」阿嘉說，他雖還有一個妹妹，但她出嫁多年，「靠著跟丈夫在夜市擺攤賣玩具，在經濟上實在幫不上忙。」蘋果基金會訪視後，已撥款暫紓阿嘉一家生活、醫病急難。
