孤苦腦麻男截肢 欠13萬醫費

日賺300元維生 裝義肢努力復健

2008年08月28日

「我還想用腳走路，靠自己面對未來。」患腦性麻痺的劉秋良本就行動不便，從小父母雙亡，由爺爺撫養長大，平常在市場賣抹布餬口，沒想到三個月前從火車上失足摔下致截肢，剛裝上義肢，正練習再出發。報導．攝影╱向高彬

午後陽光灑入室內復健步道，29歲劉秋良撐雙手扶握護欄，一步步奮力向前，汗珠從額頭滲下，他左小腿截肢面傷口剛癒合，正進行義肢適應訓練，「每天都在練習，因為我本來腳就不好，復健老師說我適應和練習時間會比一般人更長。」劉秋良說，他對未來約一年期間復健有信心。

父母兄長都已過世

劉秋良說，平常在桃園獨自生活，雖然雙親、哥哥與撫養他長大的爺爺均逝，但他仍不時回中部鄉下老家探望年邁阿姨、姑姑，「5月12日那天我要回桃園，不小心下錯站，火車已開動我又想追上，一慌張重心不穩就摔下去，腳被輾碎痛昏了，醒來時，我在醫院病床，身上有大小擦傷，也已截肢了，我痛哭著大喊，『我的腳呢？』」來探望他的姑丈江茂雄說：「這孩子有夠苦命，爸媽、唯一哥哥都早年意外或病逝，撫養他長大的爺爺10年前過世後，他出外討生活，賣一天抹布賺三、四百元，剛好飽三餐租房子。」
劉秋良住院期間看護醫療費近20萬元，由低收補助加上親友東籌西措幫忙，鐵路局運務處營業科長林竣曜說，局裡也曾多次派員慰問，因過失為劉秋良本人造成，他的情況依交通部「鐵路行車及其他事故損害賠償暨補助費發給辦法」，有上限七萬元慰助金，目前已上簽呈申請中。

低收補助不夠開銷

只是劉秋良現一邊積極復健，一邊煩惱義肢費及往返回診復健生活開銷，目前扣除義肢公司折扣及公部門補助，仍差約13萬元，「光靠每月三款低收補助7000元，連生活費都不夠，我也還要一段時間才能工作，一時不知怎麼辦。」蘋果基金會訪視後，已先撥款暫紓劉秋良醫病生活急窘。
義肢公司負責人鮑慶生說：「健保給付等級的義肢不太適合他，我們為他裝配的義肢經過練習，絕對可以再站起來，甚至走得更遠。」

無法計算的人生

劉秋良說：「我知道賣抹布是大家同情才來跟我買，本以為已夠慘了，沒想到又摔斷腳，而且還是我較有力氣的那隻腳，但我就是很固執，要練習靠自己再站起來，不想就這樣跟命運低頭。」
