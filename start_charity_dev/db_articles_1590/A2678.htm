癌男咬牙上工 腹脹如孕

小六女扛顧家重擔 「好想爸爸」

2012年01月15日

55歲的單親爸爸阿彬（郭增彬），因肝癌、肝硬化挺著大如懷胎十月的腹水，獨自住院接受放射線治療，心卻懸在百公里之外，擔憂著家中3名年幼子女和80歲高齡、行動不便的母親，他說：「家裡全是老幼，我病了，往後他們依靠什麼？」報導．攝影╱游竹君

阿彬與前妻育有3名子女，分別就讀國二、小六、小一，其中2子領有中度智障手冊，記者前往訪視時，孩子正乖巧地收拾餐後碗筷，就讀小一領有智障手冊的么子小豐，口齒略含糊地說：「爸爸生病了，在好遠的地方，我好想他，但不知道怎麼去看他。」阿彬的母親阿妹婆脊椎手術後，得靠著助行器緩步行走，她感傷說，「孫子還小，也不知道能照顧他們多久。」

上山採筍勉撐家計

因阿彬家住偏遠山區，得顛簸一個半小時的車程到大醫院就診，去年年中時，他因工作腳受傷就醫時，便發現肝臟有陰影，B型肝炎也惡化為肝硬化，世展社工說，「阿彬很盡心照顧孩子，怕孩子沒人顧，自己也就拖著沒就醫，直到腹水嚴重、疼痛難耐時才住院，確診為肝癌第四期，實在讓人不忍。」社工表示除現有每學期註冊｝ 費用協助之外，後續也將協辦醫療補助，減輕家中經濟負擔。
阿彬原本在工廠工作，3年多前帶著孩子返回老家居住，靠著山間零工、務農勉強維持一家生計，就讀小六的女兒小婷說，「爸爸生病後，還挺著大肚子去山上採竹筍，連彎腰都很辛苦，我很擔心，但不知怎麼勸他去看醫生。」
學校老師說，小婷的哥哥和弟弟，因缺乏刺激、有學習障礙，都領有中度智障手冊，爸爸住院後，「小婷負擔起照顧年幼弟弟的責任，帶著上下課、照顧生活起居，很乖巧，但小小年紀要承擔照顧家人的工作，也很讓人心疼。」學校教育儲蓄專戶也已撥急難金一萬元應急。

醫療開銷壓垮經濟

阿彬家中共有4手足，他排行老大，小弟4年多前也因心肌梗塞過世，其他人各自家計重擔。村長潘啟元說，阿彬家有三款低收資格，每月低收、殘津及老農補助共2萬5800元，但阿彬體弱包車就醫一趟就得1500元，後續還需持續抽腹水、往返化療，開銷沉重。蘋果基金會訪視後，也已撥款暫紓生活及醫療開銷。
