憨兒摔癱 老父母苦顧

無收入難撐 「求菩薩保佑好起來」

2011年10月23日

記者至病房訪視時，潘金英和丈夫阿福伯正幫兒子阿昇（張元昇）活動手腳，只見阿昇手腳癱軟，雙眼直視天花板，潘金英說，阿昇原本就頭腦不好、憨傻內向，「原以為平安養大他，我的責任就盡了，沒想到發生意外。」報導．攝影╱仲芝蓉

現年31歲的阿昇今年7月9日騎車意外摔癱，55歲的母親潘金英一邊幫忙按摩復健，一邊說：「阿昇意識清醒、會說話、脖子痛還會叫我幫他揉，求菩薩保佑他要好起來啊！」潘金英繼續說，阿昇車禍進加護病房2個月才脫離險境，「醫生說阿昇還年輕，把握黃金期復健說不定有機會進步。」

原靠兒微薄薪過活

醫院社工說，阿昇傷及高位頸椎，影響呼吸中樞暫時無法脫離呼吸器，雖然阿昇已通過低收入戶，可公費安置養護機構，但租借氧氣機及其他耗材，每月仍需自付約1萬5000元差額，張家每月僅領有阿福伯老人津貼3000元，阿昇的3姊妹也已出嫁各育幼子，恐一時難籌安養開銷。當地公所社會課長說，已申請「馬上關懷」和公所急難金共1萬5000元協助，再轉介蘋果基金會，訪視後已先撥款暫紓安養復健開銷。
潘金英回述，阿昇國中時鑑定為智能障礙，但他畢業後還去印刷廠工作，「阿昇真的很乖，工作6、7年，每月薪水2萬元都寄回家。」潘金英嘆口氣繼續說，可惜後來被不良損友利用，差點染毒犯刑，只好帶回鄉下和損友斷絕往來，「不知是否上輩子沒燒好香，怎麼又遇上車禍劫難。」車禍對方李先生說，對發生事故感到遺憾，後續和解待肇事責任釐清再談。

「不知要顧到何時」

潘金英的丈夫阿福伯說，鄉下工作機會少，妻子去年做了半年擴大就業砍草工，下次不知何時才能排到，原本阿昇還會釣魚，拿去市場也能賣個100、200元買菜錢，「阿昇雖然頭腦不靈光，好在還學會釣魚，本想以後靠阿昇照顧，現在卻不知還要顧他到何時！」

我有話要說

潘金英說：「我每天去幫阿昇做復健，要搭1小時公車、再坐2小時火車才到醫院，「丈夫跛腳不方便，剛開始我也怨嘆，為什麼接二連三發生困難，但想想阿昇不抽菸、不喝酒、也不吃檳榔，他就像天使，所以再苦再累也沒關係，只要能進步一點也好。」
