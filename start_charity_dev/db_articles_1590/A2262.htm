養癌兒幼孫 跛翁「拚命做」

老夫婦揮汗種芒果 「工錢都不夠」

2010年09月08日

夏日午後，暑氣仍燠熱難耐，領有肢障手冊的71歲阿雄伯（童吉雄）以鋤頭支撐跛行左腳，在芒果樹叢間揮汗整理雜草，阿雄伯說：「只能拚命做，罹癌的兒子和2個孫子才能生活。」報導 攝影╱游竹君

罹患下咽癌及甲狀腺癌的46歲耀仔（童榮耀），看著跛行的父親阿雄伯和69歲的母親秀花嬤頭頂烈日農地作活，難掩自責地表示，自己沒辦法讓老人家享福，「一把年紀了，還整天讓他們為我和2個孩子操心，心裡很難受。」阿雄伯10年前工作從三樓高跌落地面，導致脊椎斷裂及左大腿骨折，只能撐柺杖緩步跛行，秀花嬤不僅車禍跛行、2年多前也因心肌梗塞心臟裝4支支架，無法太過操勞，但仍盡力務農貼補家用。

一年收成4、5萬

耀仔說，自己與前妻育有就讀國二及國一的一對子女，平時以泥水零工養家，與父母同住祖厝，去年11月發現罹癌直到今年4月結束化療及電療，因左腳髖關節也壞死，等體力恢復時必須再開刀置換人工髖關節。就讀國二的女兒小羚說，因媽媽早年離家，爸爸失意時常藉酒澆愁，「爸爸罹患癌症後，就已經戒酒，我反而撿回一個不喝酒、好脾氣的爸爸，只是他之後腳還要動刀，我還是很擔心。」
阿雄伯說，家中原種蓮霧需費工施肥、噴藥，但自己與妻子年邁、行動不便，兩年前改種芒果，現因果樹矮小、扣除成本一年只能收成約4、5萬元，「有時候連工錢都不夠，但還是得硬著頭皮做。」當地村幹事表示，阿雄伯因農地略超過標準，低收無法通過，目前每月領有老農、殘津及單親兒少共14800元5口生活，已請地方慈善會提供急難濟助。蘋果基金會訪視後，也已撥款協助生活及往返醫療開銷。

後記
遇風雨 什麼都沒了

記者跟在跛行、以鋤頭為杖的阿雄伯夫妻身後，穿過芒果樹間，微弓的背，這是爺嬤努力生存的背影，早該是含飴弄孫的年紀、卻得拖著行動不便的雙腳，在烈日下揮汗工作養孫。就像阿雄伯說的：「務農是看天吃飯，一個颱風、大雨可能就什麼都沒了，但我只能往前看、拚命做，生活才能繼續。」這便是長年赤腳踩踏土地、務農人家對生命磨難的堅毅。
