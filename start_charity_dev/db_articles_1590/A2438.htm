癌漢顧殘妻 「要撐下去」

雙兒早逝 積蓄用完 憂房租沒著落

2011年04月11日

63歲段廷裁今年2月罹患下咽癌，因腫瘤壓迫氣管而作氣切的他，雖不時咳吐帶血痰液，仍使勁地扯開嗓門說：「還有老婆要顧，我要撐下去啊。」報導 攝影╱張嘉恬

病房裡，段廷裁的妻子阿淑姨（林惠淑）因腰腿痠痛坐立難安，正等著來接班照顧的女兒，57歲的她說，自己16年前因重大車禍傷及腰、腿，雙邊大腿至小腿皆裝鐵片，久站久走或坐立太久便感不適。阿淑姨說，她和先生育有2子1女，但長子、次子在年幼時分別因車禍及重病往生。

連累妻女「過意不去」

段廷裁說，原靠在鐵工廠做工養家，但5年前因年歲體力走下坡遭辭退，無力繳付房貸，夫妻倆便靠他的退休金及賣屋餘款共約100多萬元，以及資源回收零星收入補貼房租與家用。「沒想到積蓄用完不久，今年過完農曆年後還發現罹癌，房租已繳不出來了。」
阿淑姨說，先生總把自己放在最後，「自己生重病不擔心，只想著房租沒著落，怕我沒地方住。」先生病後痛到難忍時，「看他用手搥著被舖發洩，我心也很痛。」採訪時段廷裁出嫁的女兒阿瑩匆忙趕來接替母親，在傍晚至翌日早晨照顧父親，段廷裁見到因長期睡眠不足、雙眼布滿血絲的女兒，愧疚說：「我一人不舒服，連累兩個人，良心過意不去啊。」

女兒沒錢「只好出力」

「爸，你不要這樣講，這跟良心沒關係，好好跟醫生配合治療。」阿瑩說，自己有3名就學的子女，家計沉重，「我沒能力出錢幫爸媽，也只能出力了。」

清苦又罹病雪上加霜

在段廷裁罹癌前便已介入關懷的慈善會社工說，因兩人皆未滿65歲，妻子的車禍後遺症也未達肢障鑑定標準，無法獲公部門資源，「未料近來關懷訪視時才知段廷裁罹癌，本會正評估提高原本每月4000元的生活補助至8000元。」
轉介段家困境予蘋果基金會的當地游里長說：「段廷裁很努力，平時他都利用晚間至清晨時段推著手推車沿街撿拾店家打烊後的回收物，生活清苦，罹癌真是雪上加霜。」
目前段廷裁夫妻僅賴地方慈善會補助過日，不敷房租與生活醫療費用，蘋果基金會訪視後已先撥款紓困。
