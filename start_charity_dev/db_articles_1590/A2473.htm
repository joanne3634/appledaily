夫喪子逝 婦「再苦都得撐」

月內連辦兩喪事「孫女是唯一希望」

2011年05月23日

52歲的胡梅芳今年3月中經歷喪夫之痛，不到一個月，單親長子病逝，留下讀高一的孫女小琴，胡梅芳嘆：「女兒精神障礙，次子嚴重糖尿病，孫女是家中唯一希望，再苦再累都得撐。」
報導．攝影╱仲芝蓉

記者採訪時，來到胡梅芳與孫女相依棲身的鐵皮工寮，三面鐵皮牆面，以木板隔出一小房間，門上沒有掛鎖，外出時僅用鐵絲鉤住門扣，老伴和兒子的遺照，就簡單擺在一旁桌上。胡梅芳說，74歲的老伴金福伯（蔡金福）6年前撞癱，氣切仰賴呼吸器維生，並以鼻胃管灌食，沒錢請看護，只好和46歲的女兒阿華輪流照顧，「今年3月老伴因呼吸道感染住院，我好不容易有公所臨時僱員的工作，才讓女兒晚上睡醫院照顧老伴，沒想到那天晚上，老伴一口痰卡住無法呼吸，就這樣走了。」想起驟逝的金福伯，胡梅芳難過自責地掉下淚來。

女精障兒生病

胡梅芳說，長子阿國和媳婦個性不合，婚後生下孫女小琴，在她小學時便分開，「可能阿國單親帶孩子壓力大藉酒澆愁，喝多了肝病都跑出來，10年前開始常肝昏迷住院，走的時候才45歲。」小琴說，爸爸常生病住院，自己就跟阿公住，阿公癱瘓後家計全靠梅芳阿嬤，「雖然梅芳阿嬤不是親阿嬤，但她照顧我就像自己的孫子一樣。」
金福伯妹妹回述，其實阿國三姊弟是哥哥和元配所生，20多年前阿華不慎摔傷腦袋，變成智能精神多重障礙，接著嫂嫂中風過世，10年前哥哥教職退休靠種水蜜桃養老，並再娶胡梅芳，「兒女都身體不好，哥哥只好做到老，開車上山種水蜜桃卻不小心翻落山溝，脊椎損傷癱瘓。」

還欠50萬債務

胡梅芳接著說，6年前癱瘓時金福伯的保險解約理賠和存款共70萬元早已用盡，房地也變賣，只能借住友人鐵皮工寮，為湊管灌尿布耗材開銷，還積欠50多萬銀行債務。
村長蔡文豪說，月內連辦兩場喪事，村人幫忙辦喪葬，安置金福伯的精障女兒，辦理孫女小琴監護權轉移，才能申請低收等補助，恐還須耗半年，蘋果基金會訪視後，已先撥款暫紓困境。

暖流見證 「小琴早熟得令人心疼」

小琴的導師施翠瑩說，小琴懂事時家境就遇變故，所以她比一般孩子早熟獨立，在校成績都保持中上，學校安排她住校，並減免學雜住宿開銷，「這次至親接連過世，小琴為了不讓大人擔心，仍表現堅強，早熟得令人心疼。」
