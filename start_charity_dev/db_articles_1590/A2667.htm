殘嬤住陋屋 「怕家會垮」

木牆腐爛樑蛀空 逢雨就漏水

2012年01月11日

66歲的馮明珠去年（2011年）6月開刀置換人工髖骨後，因心臟衰弱，勉強能扶四腳拐行走，但木造老屋房門狹小，得將四腳拐放直才能通過，馮明珠說，「木頭蓋的牆壁、屋頂都被蟲蛀蝕，好擔心大風把家吹垮。」報導 攝影╱仲芝蓉

記者沿著蜿蜒山路，來到位於山間部落一處矮小木屋，這裡是馮明珠和71歲丈夫林榮信及39歲次子住了快半個世紀的家，部落裡左右鄰居都已改建為水泥或鐵皮平房，她家是全村僅存木造老屋。

「求天保佑兒子」

泰雅族的馮明珠以不甚流利的國語說著，家裡天花板和樑柱木頭都被白蟻蛀空，每次下雨都會漏水，屋舍地勢又比柏油路面低，「每年颱風大雨，後面山坡沖刷土石，家裡就會有小溪流，牆腳木頭都腐爛了，好害怕家突然會倒。」
當地公所課員說：「馮明珠去年開刀後，村長協助申請急難救助，才知她家殘破亟盼修屋，故轉介蘋果慈善基金會。」馮明珠接著說，丈夫和兩個兒子每年都自行修屋頂，但長子前年底病逝，丈夫年紀也大了，次子零工收入不穩，根本存不夠錢修房子，「公所說要幫忙轉介修房子，我們都好期待。」
馮明珠回述，這房子是已逝前夫民國57年蓋的，前夫20多年前因高血壓疾病猝逝，自己靠著醫院清潔工薪資撫養和亡夫生的2子和1養女。講起兩個兒子，馮明珠無奈說，前夫早逝，兩個兒子酒癮重管不動，次子阿明現已戒酒，「女兒嫁了家計負擔也重，只求老天保佑阿明身體。」

寶島義工團協助

公所課員說，馮明珠與第二位丈夫兩人都是虔誠教徒，公所已申請「馬上關懷」及民間急難金共2萬元，但離重建材料經費約40萬元仍有差距，蘋果基金會訪視後，也拋磚引玉撥款盡棉薄之力，並聯繫專幫弱勢家庭修屋的寶島義工團一起協力修屋，盼早日讓馮明珠兩老安心養病。
夫妻和善愛助人
同村的鄰居阿蘭說，馮明珠夫妻倆身體一直很硬朗，直到2年前長子過世，馮明珠又因糖尿病又動心導管手術才身體虛弱，「前幾年馮家隔壁李奶奶中風，馮明珠常去探望，還幫忙煮三餐照料，夫妻倆都很和善，颱風天教會和鄰居都接濟讓她們借住，但村民大多以零工為生，也難協力修屋。」
