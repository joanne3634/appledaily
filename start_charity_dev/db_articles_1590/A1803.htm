婦伴夫抗癌7年 家產耗盡

「上天還是帶走他」 稚女常哭喊要爸爸

2009年02月13日

看著得年51歲丈夫阿永掛在牆上的遺照，42歲的盧君梅深深嘆息，她說，結婚8年時間，有7年在陪丈夫抗癌，「我懷孕7個月時，阿永便發現罹癌，抗癌7年多以來，我從來沒有放棄希望，但上天還是把他從我們母女身邊帶走了！」報導．攝影╱韓旭爾

盧君梅止不住淚水說，阿永是1月23日去世，就是要過年的前兩天，「過年期間，我每天就跟女兒到殯儀館幫阿永誦經、摺紙蓮花，上周五已辦完後事，希望阿永一路好走！」
8年多前從大陸嫁來台灣的盧君梅說，丈夫阿永原為模板工人，7年多前因胸痛、呼吸不順到醫院檢查，發現罹患支氣管癌初期，雖經手術及化療，病情卻持續惡化，並擴散至脊椎及腦部。

新婚罹病妻從未離棄

回述過往，盧君梅說：「阿永是個很老實、愛家的男人，7年前首次癌症手術後，才一出院，就硬撐著身子到工地上工，他說，『女兒要出生了，要多賺點奶粉錢。』但才工作2、3個月，回診時就發現癌細胞轉移了。」之後阿永便不斷進出醫院，家裡生計開始出現困難。盧君梅雖還沒有工作權，但仍要一肩挑起生活重擔，「我都利用照顧丈夫的空檔到蓮霧園做小工、到山裡撿蝸牛販賣，只要不做壞事，什麼工作我都願意去做。」盧君梅回憶，當時新婚不久，有朋友勸我乾脆帶著女兒回大陸算了，「但我覺得嫁雞隨雞，阿永也沒什麼親人可以依靠，要是我走了，就等於是害了他。」
「女兒可以說是在醫院長大的。」盧君梅感嘆表示，對於爸爸時常進出醫院，8歲的小綾習以為常，但對於生死卻還一知半解，盧君梅說，有一次阿永住院期間，隔壁床的病人病危，經醫生電擊急救還是回天乏術，小綾被當時的景象嚇到了，常常會問：「爸爸以後是不是也會變成那樣子，看起來好痛喔！」盧君梅說，現在阿永走了，小綾常常哭泣，晚上睡覺作夢時，也會喊著要找爸爸。

喪葬費得靠補助支應

盧君梅說，自己身心俱疲，3年多前開始出現失眠、莫名地流淚、時常激動生氣，到醫院就診後，才知得了憂鬱症與腦下垂體病變，已不太能工作。她說，阿永去世後，低收入戶補助已被取消，「親友能借都借了，一時間也無法償還，另外，自家住宅也因4年前曾辦理貸款30萬元來支應生活、醫院開銷，每月房貸為4000元，目前也無法按時繳交。」小綾學校的班導黃瓊淑老師說：「小綾很乖巧，學校會輔導，讓她早日度過喪父之痛，並幫忙小綾的餐費、學雜費。」
記者是在１月中旬時初次探訪盧君梅、阿永夫婦，在醫院病房裡，見阿永眼睛不時開閉，但對旁人的呼喚全無回應，１星期後，阿永終究敵不過癌末折磨去世。
盧君梅說，因家裡貧困，雖有慈善會提供棺木、火化補助，但仍不足約4萬元，蘋果基金會訪視後，已先撥款協力。
