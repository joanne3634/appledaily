盲男喪妻 愁稚女成長路

背債48萬 慈母臨終仍念子女未來

2009年06月29日

13歲的小雲原盼媽媽來得及參加她小學畢業典禮，擁抱告別童年，卻沒料到34歲媽媽阿惠跟子宮頸癌魔搏鬥近半年，仍在5月底去世，留給她與視障的爸爸無限感傷。報導．攝影╱向高彬

樣貌秀麗的小雲有雙大眼，黑亮眼神透著早熟憂鬱，她說，農曆年後媽媽查出患子宮頸癌，病得很厲害，49歲的爸爸廖致榮為醫治媽媽，專程奔波帶到外地大醫院住院求醫，沒料到5月底媽媽說想回家，回來不到3天，就病逝了。

摸索顧妻已成追憶

小雲爸爸廖致榮右眼全盲、左眼僅餘眼前約2公尺霧濛濛視力，天一暗幾乎看不見，他說，與太太感情好，當時在醫院摸索陪太太服藥、推輪椅做治療，還談往後如何栽培小雲，如今都成追憶，「以前我們擺攤從早忙到晚上7點收攤，一個月才一萬多元，但阿惠和我，總想只要老實過日，平安把孩子養大也是成就。」廖致榮又說，阿惠臨終一再交待，「一定要好好把小雲撫養長大。」
而這段期間生活醫療開銷及後事，蘋果基金會於6月初訪後，已以「歐舒丹．蘋果視障專戶」撥款協力部分喪葬，廖致榮說，有鄰里幫助及公部門救助但仍不足，尚欠8萬元。

女兒懂事助扛家計

小雲的班導師黃瑋琦說，孩子相當乖巧懂事，自知家境不好，之前課後假日都會去幫媽媽賣黑輪，學校也正辦教育部「學產基金」協助。
廖致榮回述，他做工因意外截去左手3指，16年前車禍致失明，他靠泥水零工撫養第一段離異婚姻的1女1子，及至認識阿惠，兩人合力擺攤賣黑輪養小雲，家計長年辛苦，還有約40萬元的卡債。
廖致榮的小哥也正和食道癌搏鬥，阿惠的媽媽患糖尿病。廖致榮和前妻生的獨子阿寶說，自己剛入伍服志願役，姊夫租屋做工、姊姊懷孕待產，能幫有限。當地王姓里長說，廖致榮視力差，每月領4000元殘津，撫養小女兒長大還有漫長一段路，會再幫忙找零工和申辦相關補助。

後記
母愛不死 佑女平安

人世無常，記者猶記5月底在醫院見到當時病重的阿惠，雖氣弱游絲，但仍堅強撐起身體，「孩子需要我，我一定要好起來。」但如今天人永隔，希望往後能如廖致榮所願，阿惠天上有知，母愛不死，保佑小雲平安長大。
