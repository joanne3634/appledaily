19歲女抗癌魔 「我不認輸」

母獨力顧1老4小 自責不及格

2009年06月10日

19歲，應該是女孩們恣意享受青春的大好時光，但小秀（黃佳秀）卻沒如此幸運，年紀輕輕便腎癌上身，現在少了右側腎臟，後續還有漫長的抗癌療程，她堅強地說：「我認命，但不認輸。」報導．攝影╱韓旭爾

小秀說，父母在7年前離婚後，她便跟著媽媽生活，因家境困窘，3年前國中畢業後，便開始四處打工協助家計，她說：「當年考上私立高職後，接到3萬多元的註冊單，把媽媽嚇壞了，看到媽媽的反應，就知道沒法繼續讀書了。」小秀說，因學歷低，只能到茶飲攤打工，每月賺3、4000元貼補家用。社會的現實，讓小秀體認到讀書的重要，她說：「連到加油站應徵，都要求高中畢業，感覺好挫折。」小秀原本打算明年報考夜間部，沒想到身體卻出了問題。

「人生還有希望」

回想發病經過，小秀說，今年初就覺得腹部不適，「吃了胃藥都沒效，到4月時更加嚴重，常痛到發抖、冒冷汗。」小秀說，到大醫院檢查才發現罹患腎臟癌，開刀摘除右腎，「幸好醫生說只剩一個腎臟的腎功能和一般人沒有差別，只要癌症能治癒，將來還是能正常生活，讓我覺得人生還有希望。」
小秀41歲的母親阿淑說：「我覺得自己是不及格的母親，不只無法給孩子完整家庭溫暖，連女兒的健康都沒照顧好。」
阿淑與前夫共生育7個子女，她說，7年前兩人離婚後，她獲得小秀及現年分別14歲、12歲共3子女的監護權，靠著餐廳雜工維持家計，3年前，她結識一男友，兩人生下1子後男友便失聯，「小兒子還小，我根本無法工作，最後連房租都付不出。」阿淑說， 2年多前，她帶著4個孩子返回部落娘家，並照顧中風臥床的母親，一家開銷依靠父母老農年金每月1萬2000元度日。

機車送醫省荷包

目前小秀才剛完成手術，正進行化療，阿淑說，每次到醫院的包車費用就要1000元，實在負擔不起，她只能騎著機車載小秀到醫院治療，來回近4小時車程，「小秀不但要忍受化療不適，還要承受乘坐機車的路程顛簸，實在捨不得。」小秀說：「沒法幫忙家計，反而增加家裡負擔，覺得對不起家人。」
原住民家婦中心社工陳慧敏說，因目前申請低收，需併計前夫財產，因此無法通過核可，現僅領有展望會每學期補助兩個就讀子女每人3000元學費，正協助申請單親兒少補助，蘋果基金會訪視後，已撥款暫紓小秀醫病急難。
