單親父抗癌 放不下4兒

「每天都怕是最後一天」

2011年07月27日

突來病變，讓39歲的單親爸爸張土文慌亂了手腳，他說，病痛與生計讓他煩愁，「躺手術台上，怕若萬一突然掛了，在手術室外等候的女兒，知道怎麼通知家裡嗎？每一天都怕是最後一天。」
報導．攝影╱向高彬

張土文離異單親8年，現育有4個分別16、14、11、9歲的2子2女，他說，年初突然發覺右腹持續悶痛，檢查竟是肝癌二期，進行栓塞治療，以為控制趨穩，但未料6月兩側腋下有腫瘤，回診確認有移轉擴散，「本想若好些，就再去做工，沒想到醫師說目前狀況較棘手，要有長期拼戰準備。」

欠親友30萬債務

由於張土文母親名下有祖傳農地略超過低收規定，未符低收，當地公所已撥「馬上關懷」急難金3萬元，醫院社工說，目前張家共領殘障、單親兒少及家扶補助共1萬3700元，醫院已辦醫病補助，也轉予蘋果基金會協力。蘋果基金會訪視後，撥款暫紓生活窘迫。
張土文回述，8年前離婚後，將年幼孩子託往鄰近現58歲的母親協力照料。他坦言，曾有段時間意志消沉借酒澆愁，後經親友勸誡，擺脫酒瓶再振作，至工廠擔任倉管，雖然養4子擔子重，積累約30多萬元親友債務，但生活總算勉強可過，只是未料癌魔突襲，如今一家遭逢重大危機。

摺金紙貼補家用

採訪時，將讀高職汽修科的大兒子小男說，等高職畢業，想服志願役幫家裡，「爸這陣子常說，若他不在，要我顧弟妹，我跟他說，你會好的。」他又說，媽媽失聯多年，但曾在去年突然簡短來電，沒隔兩天他再回撥，卻已號碼停用，「不知她怎麼了，希望她平安。」國一的小萍和小五的小慈都說擔心爸爸，平常會幫阿嬤摺金紙代工貼補家用，讀小三的小兒子小斌常幫燒香求神明，保佑家人平安。
小男的林姓班導師說，正辦學產基金救助，張土文的小妹阿鳳哽泣說，自有家庭能力有限，這幾年娘家事多，「大哥很顧孩子，但他現在病這樣，很擔心孩子以後怎麼辦。」

無法計算的人生

張土文說：「爸媽辛苦做工養大5弟妹，結果爸沒享清福就病逝，大妹8年前車禍癱瘓，小弟前年心臟病發死了，我又得了肝癌，怕媽媽承受不了，現在也忍著不敢多說。」
