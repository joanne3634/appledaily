男癱床 5歲兒哭找爸爸

不捨4子女寄養 盼早日團圓

2012年05月28日

「爸爸加油！我和弟弟妹妹都很想你，希望你趕快接我們回家。」8歲的長子小翰透過電話為爸爸阿國打氣，阿國的女友阿美（杜美貴）在旁幫忙今年1月中風的阿國拿著電話，讓阿國一解思親之愁。報導．攝影╱仲芝蓉

現年31歲的阿國（宋建國）癱躺病床，吃力咬字說：「我想…趕快好…小翰沒有爸爸…睡不著…。」阿美嘆了口氣告訴記者，小翰和2妹1弟4人，是阿國和亡妻的孩子，長子小翰8歲，最小的是對龍鳳胎才5歲，「不料今年1月初阿國因先天腦動脈畸形，引發中風右半身癱瘓，4子女只好暫讓他二姊寄養。」

現須積極復健

阿美繼續說，阿國原本靠水泥等零工為生，他的妻子4年前車禍往生，當時龍鳳胎才1歲，只好託阿國母親幫忙帶，但2年前母親病逝，才帶著4子女在外租屋。阿美說，自己也有過婚姻，因個性不合離異，之後結識阿國，「我幫人種山蘇、摘菜打零工，平淡過日」。
當地公所主任高文山說，針對阿國家庭所遇急難，政府依兒童及少年寄養辦法，提供協助4子寄養的親屬（也就是阿國二姊）每月3萬元補助，而阿國年紀輕，醫師建議把握黃金復健期積極復健，將來若接回子女，每月可領低收補助1萬9600元。
阿國的二姊說，自己3姊妹都已出嫁，么弟失聯已久，只能盡力幫忙代為照料4子，「5歲的龍鳳胎不懂爸爸生病，常半夜起來哭著找爸爸，看了實在很心疼」。她說，自家也靠零工為生，領的補助費僅打平生活。

回診開銷沉重

當地社工說，目前阿國領有殘障津貼，扣除房租剩6千多元為生，當地交通不便，就醫回診每月開銷負擔重，公所正申請「馬上關懷」急難金，並轉介蘋果基金會，基金會訪視後已先撥款紓困。家扶中心主任陳清輝說，阿國因重症暫無法照料子女，才會進入寄養，但不論在機構或親屬寄養，最終目標還是讓子女回歸原生家庭，鼓勵阿國努力復健。
