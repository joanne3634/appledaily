男遭停薪又斷腿「很歹運」

妻月掙3千元養家 水電費得借錢繳

2010年03月19日

44歲的陳文平去年12月中騎腳踏車自摔，右腿粉碎性骨折以鋼釘固定，家裡行動不便的84歲老母和7歲的兒子，全靠48歲妻子照料，陳文平說：「家裡接二連三出事，現在只希望傷快點好，可工作養家。」報導．攝影╱仲芝蓉

陳文平說，自己原是大客車司機，去年6月因為一場車禍，遭到留職停薪處分。妻子江秀蘭說，原本靠丈夫3萬元薪水養家，婆婆骨質疏鬆、心臟又不好，自己就留在家當家庭主婦，照顧婆婆和兒子。
江秀蘭繼續說，家裡住的是遠房親戚借住的祖厝，2年前修補屋頂漏水破損、裝修門窗，前後累積卡債40萬元，每月還債後剩2萬元，一家4口生活只能剛好打平。

資源回收湊買菜錢

江秀蘭說，丈夫被留職停薪後，原本想再找其他工作，但客運司機臨時缺少，只好做擦油漆、拔雞毛零工，自己也開始賣水煮玉米，兩人每月合賺1萬多元，吃飯都成問題了，債就更難還了，「這半年來生活雜費帳單也都拖欠，收到斷水斷電通知才趕緊向親戚借錢來繳。」
陳文平自責的說，生活已經夠辛苦了，沒想到自己又因天黑騎腳踏車視線不良，跌到大排水溝裡摔斷腿，現在只能靠四腳柺杖行動，擦澡、上廁所都要妻子幫忙，「她還在床邊放了水桶，讓我夜裡尿急時可用，現在就剩她一個人賺錢養家，但擺攤賣玉米每月也頂多賺3千元，有時還去撿瓶罐資源回收，就為湊幾百塊買菜錢，看得真的很心疼。」陳文平歎。

傷勢還須休養半年

隔壁30幾年的老鄰居說：「陳文平的大哥二哥都早逝，他結婚10幾年一直沒生孩子，7年前才收養1子，現在孩子還小、母親年紀又大，都得靠他工作，卻遇上留職停薪，接著又摔斷腿，實在很歹運。」
醫院社工說，陳文平的傷勢狀況至少須休養半年無法工作，且因他有糖尿病舊疾，須觀察復元情況再評估是否開刀取出鋼釘，目前已有「馬上關懷」急難金和當地村長協助募款繳納9萬元醫療費，但尚有2萬8000元欠費，蘋果基金會訪視後，已撥款暫紓困境。

後記
判決確定才能復職

陳文平說，法院已裁定車禍的責任自己無過失，但因對方往生，家屬仍繼續上訴。陳文平服務的客運公司莊姓站長說：「陳文平服務16年來表現優良，這次因為對方已往生，不得不慎重處理，一切得等到法院判決確定，才能再復職。」
