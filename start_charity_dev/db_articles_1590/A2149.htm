家中老病殘 孝子辭職顧家

獨力照顧4人 「只能求神保佑」

2010年04月23日

南部偏遠鄉間一處老舊平房，晨曦從老舊破裂的石綿屋頂照入屋內，34歲的阿順一起床，先為父親、爺爺與叔叔、嬸嬸準備早餐，之後便騎著機車載著叔叔阿慶到醫院回診。5口家中4人老病殘，他只能求神明保佑，助他早日度難關。報導．攝影╱韓旭爾

阿順說，自從1年多前52歲的叔叔因糖尿病後遺症、左小腿受傷潰爛截肢後，住院的時間比在家裡長，阿順說：「叔叔2月時才因肺積水、心衰竭住進加護病房急救，治療1個多星期才撿回一命，現在病況還不穩定。」
阿順無奈地說：「叔叔原本還有約40萬積蓄，但住院時看護費用每月就需6萬元，加上家裡三餐、水電開銷，這筆錢在半年前早就花費殆盡。」當地村長李進福說，由於阿順的叔叔之前有存款，雖都用完了，礙於法令還是到明年才能重新送件申請低收補助。
阿順回溯說，多年來，因63歲的父親罹患嚴重糖尿病，無力工作，家中還有90歲高齡的爺爺，他自退伍後，便在外地的科技工廠擔任作業員，每月掙得3萬多元維持家計，家裡同住還有52歲的叔叔阿慶夫妻兩人，阿順說，因嬸嬸是中度精障者，無法工作，叔叔則做零工貼補家計。
自從叔叔病倒後，阿順只能暫停工作，返家張羅叔叔住院就醫事宜，「爺爺年紀大了，走路不方便，吃飯、洗澡都是爸爸幫忙，如今叔叔又生病，嬸嬸整天窩在房間內喃喃自語，爸爸一人忙不來！」

盼康復別誤姪前途

阿順說，原本以為只要叔叔病況穩定後他便能重返職場，沒想到叔叔一病不起，「請假時間太久了，公司也無法通融，只能自行辭職。」
叔叔阿慶看到姪子對自己盡心盡力照顧，他哽咽地表示，夫妻倆膝下沒有一子半女，幸好阿順這個孩子乖巧，把自己當成爸爸一樣地孝順，只希望自己能早日康復，「不要再拖累阿順，耽誤他的前途。」
阿順說，現在家計收入僅靠爺爺的老農年金、叔叔夫妻兩人殘障津貼共1萬3千元度日，「叔叔到地區醫院回診一次掛號費就要3百多元，負擔很大。」蘋果基金會訪視後已先撥款紓困。阿順說，只要叔叔病況穩定，他便能外出工作，靠自己的力量扛起一家重擔。
