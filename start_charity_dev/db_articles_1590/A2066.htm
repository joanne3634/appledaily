視障男截肢 妻摘菜養老小

借錢裝義肢 「至少還活著陪孩子長大」

2010年01月05日

「爸，我幫你穿套筒。」讀國二的小婷幫剛截右小腿的爸爸王福霖裝義肢，陪他一步步練習走路。小婷說，爸爸患糖尿病洗腎又失明，現再截肢，「鼓勵他有信心，陪我們走下去。」報導 攝影╱向高彬

王福霖的左眼全盲，僅右眼隱約看到1公尺前的模糊黑白光影，49歲的他撫著冰冷沒溫度的義肢嘆說：「本來人生只剩黑白，沒想到現在右腳又沒了，路越走越坎坷。」
王福霖還在費力適應新義肢，視力不便也沒力氣久站，來回走約20公尺就氣喘吁吁，他說，年輕時做過守衛和送貨工養家，但遺傳自爸媽的糖尿病、高血壓，已發病21年，11年前併發視網膜病變致失明後無力做事，8年前洗腎，原還有點手腳力氣照顧自己，不拖累家人，沒想到去年（2009）8月時突然高燒不退，右腳感染潰瘍，在醫師建議下只得截肢保命。他哽嘆，還讓太太跟人借10萬元來過活和裝義肢，「一個大男人變這樣沒力氣養家，一想到就很難過。」
王福霖的同齡太太王瑞端要他別想太多，夫妻一場就是緣分，相互照顧是應該，「至少還活著陪孩子長大。」王瑞端因小時生病致顏面神經失調略臉歪嘴斜，她說，夫妻當年考量家境和先生有遺傳糖尿病，僅養育一女小婷，在先生無力工作後，由她至工廠做女工，平日還照顧洗腎失智的69歲婆婆。

每月僅靠萬元過活

王瑞端說，沒想到前年底她遭裁員，中高齡失業又得照顧病殘老小，難覓合適工作，只能到鄰近菜園做採菜零工，每月收入加三款低收補助共約1萬9200元，但得償還3年前修繕漏水屋的貸款、每月還8000元，未料如今先生截肢須人顧，讓她一時難做工，生活更顯困難。
目前王福霖的媽媽已暫由弟弟阿沛接去照顧，阿沛說，自己還有8旬養父及2幼兒須照養，「但看哥哥家更是辛苦，我接媽媽來顧幫些忙。」當地公所已協辦王福霖1萬3000元急難救助金，里長古雲騰也說，王福霖截肢後，曾多次前訪關心，會再撥冬令救濟米油協助。蘋果基金會訪視後，也撥款暫紓一時生活復健急窘。

無法計算的人生

王福霖說：「年輕時曾做過乩童，那時爸媽和我常看病吃藥，身體不好，問神才知有機緣要做神明事；但這些年經歷很多，現在我認為神佛非萬能，不可能人所求全答應，一切有命數，或說像我，該受的就須受，是人生在世的修行，信神只是求心安。」
