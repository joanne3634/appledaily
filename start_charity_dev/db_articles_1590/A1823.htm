弟死妻重殘 病漢力竭

葬弟掏空積蓄 「手頭已沒錢了」

2009年03月12日

56歲的鄭聰敏蜷臥床舖，想起身喝水，但老半天卻頭昏無力爬起，他半年前發作腦血管硬化無法工作，家裡還有洗腎又中風重殘的46歲妻子阿霞，他說：「家裡都靠我，現在我也病倒，很頭痛往後生活。」報導 攝影╱向高彬

採訪時，見鄭聰敏臉色發白，話聲有氣無力，得坐近才能聽見他說話，他說，早有糖尿病問題，又發作這病，剛開始時頭痛和手腳無力，以為季節交替加上過累所致，但後來頭暈得路都走不動，奔波醫院檢查才知是腦血管硬化，「再惡化可能腦中風，醫師說要多休息，按時服藥。」他又說，長期暈痛讓記憶力大受影響，「最近常忘東忘西，想不起吃過藥沒？」

妻染敗血症腳植皮

只是鄭聰敏煩惱自身病況，也擔憂健康更差的妻子阿霞，才剛醫病出院返家養病，他感嘆妻子受病痛多折磨，已洗腎21年，每周須洗腎3天，7年前曾出車禍遭撞斷腿，3年前又兩度輕微中風，目前雖能行走，但有時神志不清，平常由他陪伴照顧，「結果她過年前不小心左腳被刺傷，因本來抵抗力就差，結果細菌感染敗血症，住院植皮，住了半個多月才出院。」

夫妻病弱生活陷窘

鄭聰敏回述，妻子的第一段婚姻因前夫家暴離異，生有1女阿惠，他與阿霞兩人10年前相識結婚，未再生子女，「認識妻子時，她已在洗腎，但身體還好，當時看她照顧病重老丈人很孝順，後來在一起，只是沒想到她身體越來越差。」鄭聰敏以開堆高機養家，之前還要負擔照顧癱瘓的小弟安養費，生活僅剛好可過，「半年前我開始生病，弟弟也在今年初過世，這段期間醫病生活和弟弟後事，用光微薄10多萬元積蓄，手頭已經沒錢了。」
目前鄭聰敏就醫時，就由已嫁出的25歲女兒阿惠來回娘家照料，病弱夫妻完全沒有人可以工作，沒有經濟來源。女兒阿惠說，丈夫做模板零工，自家有一名兩歲幼女，「景氣很差，丈夫從過年後休息到現在，還在等工頭的電話，但娘家媽媽又病了，我也只能家裡、娘家兩邊跑，經濟上幫不了家裡忙。」
目前鄭家每月領阿霞重殘津貼4000元，當地鄉公所已辦「馬上關懷」救助金2萬元協助醫費，張姓社會課長說，後續協辦相關社福低收資格，以減輕醫病開銷，但夫妻兩人同時病弱，生活復健是辛苦，蘋果基金會訪視後，已先撥款暫紓生活困窘。
