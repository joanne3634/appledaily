2幼子畫圖 為癌父打氣

4口支柱倒 「計劃全亂了」

2012年09月19日

46歲的阿來因罹患鼻咽癌第四期，渾身不適疼痛難耐，27歲的妻子阿碧在旁幫忙塗抹藥膏止痛，阿來無奈嘆了口氣說，自己晚婚，兩個孩子才分別6、7歲，「我檳榔戒掉20年，沒想到還是逃不過罹癌命運」。報導．攝影╱仲芝蓉

阿來7歲的女兒小佳和6歲兒子小皓在紙上畫圖，幫爸爸加油打氣，希望爸爸早日康復，小皓今年才上小學，不會寫國字，以注音寫著加油，還畫出爸爸皺眉忍痛的表情，小佳則懂事地畫爸爸正說著謝謝，感謝大家替他加油。妻子阿碧在旁搖頭嘆道，小佳個性早熟又敏感，雖才7歲，卻已知爸爸病得嚴重，「小佳常躲在門邊偷聽我們講話，然後去學校跟老師講，實在是很不好意思，但我知道小佳比誰都關心爸爸，希望爸爸的病趕快好」。
阿碧繼續說，其實丈夫去年9月就感覺脖子腫大，當時以為只是感冒，喉嚨發炎疼痛，拖到今年初，痛得無法睡覺才看醫生，「醫生說丈夫得的是癌症時，我眼淚就掉下來，完全不知道該怎麼辦」。阿來接著說，20年前親眼看過自己的二舅得口腔癌，開刀動手術清瘡，臉上大小傷疤不但嚇人，最後還沒辦法張口吃東西，連說話都困難，「我們做鐵工靠勞力工作，累了就嚼檳榔，像吃口香糖一樣，真的是親眼看過才會怕，自己也就戒了檳榔」。

拼做工讓孩子讀書

阿碧繼續說，家計靠丈夫做鐵工，剛好去年底房貸還完，小皓今年也上了小學，自己還找到紙廠作業員工作，想說以後只要夫妻倆多努力，就能多存點錢讓孩子讀書、過好一點的生活，「怎料丈夫突然罹癌，原本的計劃全亂掉了」。
長期關懷阿碧一家的當地社福中心社工說，阿碧家因去年度所得超過，暫不符低收入戶申請標準，公所已協助申請「馬上關懷」及民間急難金共3萬3000元，學校也申請學產基金2萬元協助，縣府另申請緊急兒少補助每月6000元，但阿來因吞嚥困難僅能喝流質營養品補充體力，每月開銷至少9000元，故提供蘋果慈善基金會電話讓阿碧求助，基金會訪視後，也已撥款暫紓困境。

親友負擔重難伸援

阿來的弟弟回述，10年前中風5年多的母親病逝，當時自己和2個妹妹都已成家，哥哥為了照料母親才遲遲沒有成家，如今自己和妹妹們的孩子都才國中、高中，家計負擔沉重，實在無餘力幫忙。
