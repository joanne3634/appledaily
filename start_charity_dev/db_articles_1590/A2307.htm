母忍膝痛做粗工 醫罕病女

體內長瘤像地雷 「不論多難治 跟它拼了」

2010年11月01日

32歲的寂姝8年前發現罹患神經纖維瘤後，便成為醫院常客，54歲的媽媽王陳寶貴說，女兒外表看來正常，但體內各處都可能長瘤，至今已開刀10多次，「這些瘤就好像女兒人生的地雷，不知何時會爆炸？」報導．攝影╱韓旭爾

寶貴說，3年多前，寂姝的大腦也檢查出長了好幾顆纖維瘤，造成女兒不時癲癇，「一發作就全身痙攣抽搐、口吐白沫，一天10多次，看了好心疼。」寶貴說，醫生評估開腦手術風險大，建議自費進行放射線迦馬刀治療，每次費用需1萬6千元，前後已進行2次，原有改善，但7月時腦內又發現腫瘤。

平衡感差常跌倒

寶貴說，她與62歲的丈夫秋龍育有1子1女，她從事餐廳雜工，輕度肢障的秋龍擔任作業員，夫妻倆每月掙得約4萬元維持家計、照顧寂姝。去年初，丈夫放起無薪假，寂姝又不時住院，丈夫便辭工顧女兒。未料近半年來，她自己雙膝退化性關節炎日益嚴重，「實在沒法勝任廚房粗活，也辭去工作。」寶貴說，醫生建議置換人工關節，但她不敢，「家裡有一個病人就夠累了，要是連我也去住院，丈夫怎麼顧得來？」目前寶貴為家計，忍著腿傷到火鍋店擔任打雜零工，日薪約600元，寶貴同事說，看她動作緩慢搬菜、洗菜，知道她撐得很辛苦。
寂姝說，24歲前，都不知道自己得什麼病，只覺平衡感差，常跌倒，腳踝關節還常莫名腫大，當時看醫生也不知為何，爸媽還罵她走路不小心。「直到24歲時，喉嚨很痛，吃飯講話都痛，到大醫院檢查，才發現自己罹患神經纖維瘤症候群。」

目前無有效療法

寂姝無奈地說，神經纖維瘤嚴重影響生活，「我現在會耳鳴、聽力不佳，常常手指痠麻、肩頸痠痛，都是腫瘤壓迫神經造成的，腫瘤手術後又會再長出來，有時我也會想，這樣的人生有什麼意義？」一旁的寶貴聽了，不捨地安慰女兒說：「不論這個病怎麼難治，我們一起跟它拼了！」
寂姝的主治醫師說，寂姝罹患第2型神經纖維瘤，為政府公告罕見疾病，腦及脊椎等中樞神經系統易生長神經瘤，此病目前尚無有效療法，需定期以核磁共振追蹤腫瘤的大小。
當地里長說，寶貴一家僅有殘津補助6000元，會送些白米、罐頭等物資接濟。寶貴的長子說，他在外地工作每月收入僅2萬多元，扣除租屋、生活花費，每月能資助家裡只有4、5千元，「幫助不多，我也很難過。」蘋果基金會訪視後，已撥款暫紓寶貴家困境。
