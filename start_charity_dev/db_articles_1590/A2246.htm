單親爸 把屎把尿顧癱女

「盼在她有限生命裡 能多陪她」

2010年08月20日

29歲的阿蓮因罹患罕病多發性硬化症，意識雖尚清楚，但卻無法自理排泄、癱躺在床上，他56歲的單親爸阿杉，正替她清潔身體，他說：「沒什麼好尷尬的了，只希望在她有限的生命裡，多爭取些陪伴她的時間。」 報導 攝影／張嘉恬

「我爸爸照顧我很累、很辛苦，我要好好作復健，賺大錢孝順他！」阿蓮像孩子般大聲地說著。阿杉說，阿蓮4年前罹患多發性硬化症，是種中樞神經發炎的疾病，從四肢麻木無力、步伐不穩漸進到目前近乎癱瘓，智能也逐漸減退。為了兼顧照料與生計，阿杉利用凌晨至清晨在市場做夜工。但阿蓮今年4月下背部褥瘡潰爛嚴重、傷口深可見骨，再加上泌尿道反覆感染、發燒而頻繁住院，阿杉因得在旁看顧請假太多無法繼續工作。目前父女每月僅賴相關補助1萬2000元，但房租就要5000元、阿蓮尿布、紗布等耗材費用也要3000元。

復健至少要花一年

阿杉說，曾讓阿蓮試住養護中心兩天，但「護理人員說她每天一直要爸爸來照顧她，更吵著要回家，因她意識清楚，醫生說要尊重她個人意願。」阿杉低頭停頓了半晌說：「自己的女兒，又不是昏迷不醒，我也捨不得這樣做，讓她有被遺棄的感受，醫生建議我專心陪伴阿蓮往返醫院進行復健至少一年，配合藥物控制病況惡化，讓她坐起來，在家用助行器行走，我才能分身工作。」

前妻打零工難援助

阿杉說，早年做小生意持家，負債後，6年前與妻子離異，小女兒由前妻監護，長女阿蓮高職畢業後獨自在外生活，他說，當年阿蓮有一穩定交往男友，「曾論及婚嫁，但阿蓮發病後兩人沒再往來，她沒把難過情緒表現出來，只對我說：『阿爸要不是你，我可能流落街頭了。』」
阿杉前妻說，自己背負銀行債務約50萬元，「現在靠洗碗零工薄薪，沒辦法幫忙。」謝姓里長說，阿蓮已獲「馬上關懷」2萬元，蘋果基金會訪視後，已撥款協助阿蓮復健開銷。

後記
「總算有機會彌補親情」

長期關懷阿杉父女的慈濟楊姓社工說，阿蓮發病前長年在外，與父親情感並不密切，阿杉也提及阿蓮與男友在外生活時，多年未返家探訪他，再次聯繫上卻是「女兒男朋友通知我到醫院接她，雖然她生病前不算有孝順到我，我老了也不可能期待她照顧，但早年我忙於工作、沒好好照顧女兒，現在也算讓我有機會彌補了。」
