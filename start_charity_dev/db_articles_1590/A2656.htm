妻中風獨子腿殘 翁揪心

3口擠陋屋 「不知未來怎麼走」

2011年12月29日

冬夜霧濃，在秋錦華的深山小屋外四周靜悄，霧濃到幾步路遠就看不清，57歲的他剛幫中風太太換尿布，走出門外透口氣，他發愁著，老伴中風，獨子阿文腿傷殘了，生活也像被濃霧籠罩著，不知未來怎麼走。報導．攝影╱向高彬

上山到秋錦華家，是探訪一大挑戰，從最近的深山泰雅族小部落出發，得再進入深山半小時車程，狹小林道崎嶇，僅容一車通行，他家簡陋鐵皮屋位於幾無外人到來的山坳處，秋錦華說，最近一戶鄰居在1公里遠，到山下醫院，得花1個多小時。

深夜載下山急診

進房裡，他的53歲太太阿珠癱坐輪椅，嗚啊聽不出說什麼，秋錦華幫她拉好毛帽，阿珠給先生一個開心笑容。秋錦華說，太太年輕時就有高血壓、糖尿病，7年前輕微中風，本來能勉強照料自己，他可忙農作，但前年太太二度中風癱瘓，「幫她包尿布、餵三餐，有時突然發燒，深夜趕著載下山看急診，我很心疼她病了。」
他又說，34歲大兒子阿文3年前離婚後，回老家幫忙做農雜工，但去年中不慎被農用車撞斷大腿，今年又髖關節病變，最近剛動手術撐拐杖行走，「醫師說他可能沒法再做搬重粗活，要再復健一段時間。」當地村長說，秋錦華一家3口生活靠每月殘障及老人補助共7000元，復健交通開銷吃力。蘋果基金會訪視後，也撥款暫紓病殘急窘。
秋錦華和太太生有1子1女，獨子阿文愧疚地說，沒讀什麼書、也沒一技專長，「現在只能多休養復健，在家會撐拐杖煮飯菜給爸媽吃，希望腿快康復，再去做工養家。」

小女兒無力接濟

秋錦華的小女兒阿妹已嫁在外，最近剛生小女娃，她說，自家經濟壓力重，「先前微薄接濟家裡，現在手頭緊，有時買點魚肉上山，幾乎天天打電話回家，聽到爸媽好，心裡才安心。」

我有話要說......

秋錦華說：「山上是我長大的家，和太太種青椒、番茄，一輩子做山上的活，有什麼吃什麼，從沒想要請人家救助，但太太、兒子病了，鄉代表知道我家困難跟蘋果基金會求助，很不好意思麻煩大家了。」
