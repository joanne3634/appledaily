喪偶無子女 婦重病沒錢醫

恐需仰賴洗腎 大伯罹病無力援助

2008年04月22日

48歲的李秋銀去年底才送走因病過世的丈夫，今年2月自己又因肺炎併發呼吸衰竭、急性腎衰竭等病症住進加護病房，由於夫婦倆無生育子女，遠在澎湖的娘家早已失聯，身體欠佳的大伯嚴福田相當煩惱弟媳的醫療費無著。
報導．攝影╱孫敏聿

現年51歲的嚴福田說，家中四兄弟國小畢業後外出打拼，原本自己擔任砂石車司機協助家計，不料12年前先是賀伯颱風毀了家園，之後兄弟們合資修屋，還未完成，3年來大弟與二弟竟因心臟衰竭與敗血症陸續過世。

出院後照料成問題

「8年前我發現走路不穩，手腳無力，下肢麻痺，才知是神經病變。」嚴福田說，他容易跌倒、腳底經常發麻疼痛，每周得跑復健診所兩次，無力再當司機開車，只能返家鄉打零工，與弟弟、弟媳同住遇雨則漏的祖厝。嚴福田說，年近80歲的父親患嚴重氣喘，常需仰賴氧氣製造機呼吸，現由么弟照顧，么弟要負擔父親醫療與照顧，也無力再經濟援助。
奇美醫院加護病房部沈醫師表示，李秋銀目前意識逐漸清醒，4月初轉至呼吸照護中心，「我們預計最長訓練呼吸功能6周，目前病人有百分之五十的機率未來需仰賴洗腎，6周觀察期後若無併發感染，就會請家屬帶回照顧，或建議轉至安養機構。」3月底縣府社會局與鄉公所已提撥1萬4千元急難金，蘋果基金會獲悉亦撥款繳清部分醫療欠費，暫紓急難。

無法計算的人生

嚴福田：「我生病8年來，以為自己身體最差，沒想到大弟和二弟比我還短命，喪事都讓慈善會幫忙，現在弟媳又住進加護病房，想到要大家幫忙，真的很不好意思。」
