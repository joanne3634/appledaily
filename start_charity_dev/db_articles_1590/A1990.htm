男治痛風成半癱 4口失依

小手術變大毛病 努力復健盼再養家

2009年10月02日

49歲的阿宗（詹慶宗）原本只是想摘除腳掌妨礙走路的痛風石，沒想到卻因術後感染，併發敗血症，經住院治療2個多月才撿回一命，他難掩憂煩地說：「小手術變大毛病，現在半身不遂，真不知要怎麼辦？」報導．攝影╱韓旭爾

南部一處溪邊堤防旁的低矮平房內，午後炙烈的陽光透進屋內，四處充斥霉悶味，記者甫入門，便見到阿宗躺臥於房間木板床上，訪談時他不時變換姿勢，紓減腰部疼痛。阿宗說，現在大小便無法自己控制，「雙腿沒知覺，但腰部卻痠痛的要命！」阿宗苦笑說：「以前常抱怨腿上痛風讓我痛到睡不了覺，現在半身癱瘓唯一的好處大概就是，就算痛風石傷口爛到化膿了也不會覺得痛。」

原靠打零工拾荒餬口

阿宗回述病發經過，他無奈表示，自己有20多年的痛風病史，手腳布滿大小不一痛風石，今年3月因左腳掌痛風石不時發炎，「走路好像針在扎！」醫生建議，開刀摘除痛風石，沒想到卻遭感染，引發敗血性休克。39歲、輕度智障的妻子阿廷在一旁心有餘悸說，當時阿宗不省人事，「我一直叫，都叫不醒，醫生只說他很嚴重了，我不知該怎辦，只能打電話叫姊姊幫忙。」在阿廷姊姊安排下，將阿宗轉到城裡大醫院才保住性命。
阿宗說，他與妻子阿廷生育兩子，分別就讀國二及國一，其中次子小承經鑑定為中度智障，一家原本靠他擔任建築零工、資源回收勉強餬口，「現在我只能躺在床上，三餐、洗澡都需要有人照顧，一家4口真不知要怎麼辦？」

里長協助申請急難金

阿宗說，現每月雖領有補助共1萬5400元，但自己的尿布導尿管等開銷就要5000元，包車至城內醫院每趟就要2000多元，蘋果基金會訪視後，已先撥款紓困，當地里長也協助申請「馬上關懷」急難金2萬5000元應急。阿宗說，現在他只想努力復健，只要能走路，就能再掙錢扛起養家重責。

後記
長子顧智障么兒「很欣慰」

阿宗談到智障的次子，眉頭深鎖，他說，小承2歲都還不會叫爸、媽，鄉下人也不在意，一直以為是大雞晚啼，直到6歲還是只會講「我要吃飯、尿尿」，一檢查才知道頭腦有問題。「他今年升國一了，還不會看時鐘，想到就傷腦筋。」但一說到長子，阿宗欣慰的說：「小吉很懂事，課後常幫忙處理家務」。採訪時，小吉在一旁對記者說：「我了解自己的責任，我會好好讀書，將來養活爸媽、照顧弟弟。」14歲的國二少年肩上，已有千斤重擔，讓人疼惜。
