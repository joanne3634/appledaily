夫打零工 供妻洗腎育4子

6人擠10坪陋屋 小兄弟輪做家務分憂

2009年12月30日

「媽媽，今天我洗碗，水好冰喔！」讀小二的小香蹲在狹窄的廚房地上洗碗，她和兩個哥哥為讓洗腎的媽媽休養，兄妹每天排做家務。小香媽媽徐嘉雯愁煩自身病弱，才32歲就洗腎，拖累家計，讓孩子跟著吃苦。報導．攝影╱向高彬

徐嘉雯說，因高血壓引發腎臟病變，9月時醫師判定終生洗腎，「剛聽到時覺得天昏地暗，這輩子不就沒望了。」她說，一度想尋短，但想到孩子還小，得陪他們長大，只是當她告知孩子須洗腎時，他們還天真回說：「是不是去醫院把腎拿出來，像洗碗一樣就好了。」

「米袋空了也很急」

未病前，徐嘉雯還可做麵攤工作洗碗貼補家用，但如今只靠丈夫做噴農藥零工養六口，「冬天工很少，上個月不到10天，賺不到1萬元，房租也沒付。」採訪時，見小香一家租住約10坪大簡陋鐵皮屋舍，6人擠睡一間房，小浴廁不到半坪。鄉公所社會課表示剛協辦鄭家三款低收，每月補助8800元，蘋果基金會訪視後，也先撥款暫紓小香家生活困窘。
徐嘉雯回述，和35歲丈夫鄭朝仁婚後生有現分別11、10、8和5歲的3子1女，丈夫以模板工、搭鷹架等零工餬口，但常遇僱主落跑拖欠工錢，她前年因高血壓病弱，近3年向親友借款和卡債近60萬元，「之前我們夫妻總想還年輕，孩子生了，只要肯打拼，一定可還債、合力養大孩子，但沒想到景氣變糟，這時他沒工，我又病，還好鄰居阿婆常燉肉和拿菜來幫忙。」
小香爸爸鄭朝仁愧疚沒能力多掙錢養家，他說曾到處問工、到就業服務站登記，但總是「等候通知」，「一堆比我年輕、學歷高的也在找工作，我只能一直等。看米袋空了，我也很急。」

房東讚孩子有禮貌

對於妻子病況，鄭朝仁說，想捐腎給她，但「我有腎結石，也不行，現在登記移植腎臟，又怕排到了，也不夠錢醫她。」鄭朝仁還盼景氣快好轉，努力多做些工，好撐起家。
小香家的房東楊先生說，底層生活無奈，他心疼孩子也大半年沒收租，還借小錢救急，「雖然環境困頓，但看到他們夫妻認真教養孩子，孩子有禮貌，每回都是靜靜擠在小桌寫功課。」
鄭朝仁僅有一妹，單親育一女的妹妹阿玲說，媽媽10年前子宮頸癌病逝，當年爸爸為醫母病和遭倒會背債，一直家計窘迫，「卻沒想到大嫂跟媽媽一樣，年輕就患重病，但我單親租屋養小孩，想幫他也不知從何幫起。」
