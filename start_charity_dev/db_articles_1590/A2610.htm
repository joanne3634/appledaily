男苦撐抗癌 搏命伴女兒

手術毀容 妻不捨「他永遠最帥」

2011年11月06日

「來，不要怕痛，一下就好了。」46歲的阿姿（阮麗姿）細心地幫罹患口腔癌的丈夫阿富（鄒新富）撕下紗布準備換藥，她說：「雖然手術讓他的臉看起來有點嚇人，但他永遠是我最帥的老公。」
報導．攝影╱郭恩新

訪視時，50歲阿富正在家裡休養，今年7月因放射性骨壞死接受腐骨切除手術，他指著自己的臉口齒含糊地說：「現在臉上少了這麼一大塊肉，我都不敢出門了。」阿富表示，自己過去忙於工作，錯過女兒許多成長階段，一場大病讓他發現生命中最重要的就是家人，「我現在最大心願是兩個女兒能找到屬於她們的幸福」。

去年復發病情惡化

一旁的太太阿姿表示，先生早年靠鐵工收入維持家計，自己則在蘭花園打零工貼補家用，夫妻倆育有2女小育與小詩。5年前先生罹患口腔癌3期，經手術與化療後情況穩定，沒想到才回工作崗位沒多久，去年5月又發現復發為4期，「口腔癌治療對先生來說非常折騰，看在我們眼裡也是於心不忍，廓清、重建、化療等他都一一撐過來，就是想要換取多一點時間來陪女兒。」

次女想當志工助人

阿姿說，自己蘭花園零工一個月約收入1萬4千元，26歲長女小育獨自在外地租屋工作，每月拿4千元幫忙家計，加上先生中度殘津4千元，每月約有2萬3千元收入，但扣除房貸9千元、先生管灌營養品8千元，一家3口只剩5千元維持生計。
阿富就讀大三的次女小詩說，爸爸生病後，教會與各社福團體都曾介入關懷，「不管是欠人的人情或是債務，將來都必須要還，因此選擇就讀社工系，希望將來可以用社工的專業，幫助更多人。」看著阿富長大的村長陸旨成表示，阿富無法工作後，太太一人難獨扛家計。蘋果基金會訪視後，已先撥款暫紓阮麗姿一家醫病、生活困境。

無法計算的人生

阿姿說，先生過去幾個從部落一起長大的好友，這幾年因車禍與意外一個個不幸離世，少了這些好友的陪伴，這幾年他也顯得較鬱鬱寡歡，「沒想到又換他得癌症，真是人世無常，我也學到要好好珍惜當下，珍惜跟先生在一起的每一天。」
