孕妻顧癌男 累到早產

兒漸胖 夫卻瘦 「再痛都要撐過」

2012年06月25日

5月下旬那天雨落不停，陳麗敏的先生黃志強剛做完第二次口腔癌化療出院，才到家不久，懷胎9月的她開始陣痛，緊急入院待產，凌晨時她倆的第一個小娃迫不及待早到人世，這聲啼哭帶給虛弱的黃志強一股大力量。
報導．攝影╱向高彬

陳麗敏懷抱小男娃說，可能這段時間獨力照顧罹癌先生太勞累，孩子提早一個月出世，「孩子急著想跟爸爸見面，要他加油把病醫好」。
39歲的陳麗敏說，與42歲的先生交往同居兩年，去年中懷孕，兩人打算今年結婚，未料先生2月突然右頰腫脹化膿，檢查竟口腔癌4期，「先生一度想放棄，要我別理他，但他真心對我好，我怎麼能在他最需要我的時候離開，5月7日我們辦理了結婚登記，要一起努力養孩子長大」。

喜獲兒「相信奇蹟」

陳麗敏回溯，兩人都二度婚姻，她第一段婚姻生3子女歸前夫，現任先生未有子女；先生之前開計程車和遊覽車，有嚼檳榔、抽菸習慣，原本租屋生活可過，未料得子、罹癌竟「好壞事一起發生」，但陳麗敏又說，老天送來幸運兒，相信有奇蹟。採訪時，黃志強說，嘴破潰爛副作用很痛苦，「但看到兒子可愛的臉，我想無論如何為了他，再痛都要撐過去」。

尚須治療半年以上

目前陳麗敏產後顧幼娃，陪先生抗癌療程，她說，兒子喝母奶、一天胖一點，很欣慰，但看先生忍痛喝營養品，卻一天瘦一點，已瘦快20公斤，讓她擔心，「醫師說他還得治療半年以上，要有耐心，但家裡已困難」，她打算等先生好些，再去做事撐家。陳麗敏的娘家爸媽常探望，但能幫忙的有限。
黃志強的母親說，志強是大兒子，自己靠單親育兩女的小兒子打零工養，「我關節退化，已不太走得動樓梯；但看志強夫妻苦，我想幫媳婦帶小孩，讓她先顧志強。」當地公所及社工正協辦相關社福救助，蘋果基金會訪視後，也撥款暫紓急窘。

我有話要說

黃志強說：「以前我少生病，現在住院才知道有這麼多人為癌所苦，甚至手術切得面目全非，生病像活在另一個世界，體會很多，只要我熬過難關，一定要為家人再多做好事積德。」
