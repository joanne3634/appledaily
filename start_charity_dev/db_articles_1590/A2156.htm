妻勞累罹癌 扛債夫愧疚

忍痛硬撐小吃攤「癌症怎會找上門」

2010年05月05日

41歲的阿春（葉秀春）因罹患鼻咽癌4期住院治療，她躺在病床上彎著身軀咬牙忍痛，額頭上冒出一顆顆汗珠，丈夫阿福（宋福生）在一旁輕聲安慰：「護士馬上就來幫妳打止痛針，再忍耐一下。」報導．攝影╱韓旭爾

現年45歲的阿福提起妻子心中多有不捨，阿福說，夫妻倆育2子1女，分別就讀國三、小六與小四，他本擔任工頭，承包小型擋土牆、公路等工程，阿春則在住家前院擺攤賣小吃，但在2年多前上游包商倒閉，共倒了100多萬元工程款，所有積蓄都用來償還積欠的建材費與工人薪資，連賴以為生的怪手都賣了，之後從事收入不穩的零工維持家計，每月收入僅約2萬元。

子乖巧慰病母

「家裡經濟困難阿春都看在眼裡，雖然她1年前就發現頸部左側有硬塊，還是硬撐著做小吃攤，直到今年3月痛到受不了才去就診，一檢查就是鼻咽癌4期。」講到這裡，阿福一時哽咽，心疼妻子的辛苦，也愧疚自己無能。
阿春說當時就診，醫生曾問為什麼這麼嚴重才來做檢查？她只能沉默不語，阿春紅著眼眶對記者說：｢家裡沒有任何存款，真的不是不想儘早檢查，何況自己不嚼檳榔、不喝酒，從來沒想到癌症會找上門。｣一旁就讀國三的大兒子小偉安慰媽媽說：「媽媽不要難過，只要好好治療，妳的病一定會好。」
目前阿春因腫瘤範圍太大，無法進行手術切除，只能先進行化療控制病情，而阿福除擔心阿春病況外，長年與大姐同住的母親也因口腔癌住在同一家醫院治療中，妻子、母親同時罹癌，讓阿福感到心力交瘁。

親戚幫忙看顧

阿福的大姊宋淑珠說，家裡兄姊們收入都不高，，「只能偶爾幫忙看顧阿春，讓阿福有更多時間去打零工，大家都盡力了。」
當地村幹事羅鳳萍說，因阿春尚無法申請殘障鑑定，一家難以申請低收入戶補助，已協助申請政府「馬上關懷」急難金，蘋果基金會訪視後，先撥款協助阿福一家的醫療和生活急難。

暖流見證
管區警：懷念阿春的菜

當地管區警員左富雄說，阿春小吃攤已營業13年，警局很多人是老顧客，她賣的東西很多，早上賣饅頭、包子、豆漿，中午之後就賣麵食、快炒，「很勤快，一碗意麵才30元，不管地點再遠，她都願意外送，希望阿春早日康復，讓我們能再吃到她做的菜。」
