夫死伯逝 婦顧癱婆養5口

採西瓜獨扛生計 「有責任把家撐起來」

2008年08月11日

「爸爸呢？」4歲的小仲問媽媽江惠麗，「爸爸去山上採地瓜。」「那大伯呢？」，「大伯去山上找爸爸了。」先生與大伯相繼猝逝，江惠麗面對孩子童言問題，總覺心酸，如今還得吃力扛起家中一老三小的重擔。報導．攝影╱向高彬

江惠麗才23歲，但結婚生子早，現在已是兩個分別7歲、4歲孩子的媽，3年前先生心臟病發猝逝，「本來大伯還在外地做工，寄錢讓我幫忙顧中風婆婆，但他竟也在5月底和先生同樣心臟病發走了，還留下一個剛國中畢業的兒子讓我照顧，我每天都覺得壓力大的不得了。」

「記不得爸爸的樣子」

由於江惠麗的74歲婆婆賴雪花中風已5年，平常僅能癱坐，一家五口住在花蓮縣山間部落，這陣子鄰近西瓜園陸續熟成，江惠麗就靠每周兩三天採西瓜臨時工養家，算算這月約近8千元收入，讓她東支西借的緊絀家計稍喘口氣。
江惠麗說：「帶著兩個小的一起去田裡採西瓜，做工的朋友會幫忙看著，但家裡剩婆婆一個人，我去做工，很放心不下，一下工要載小孩騎機車飆回家看她和做晚飯。」
採訪時，兩個孩子正拿蠟筆塗鴉，小仲天真繪著五顏六色的星際大戰，將讀小一的大女兒小慈原本想畫爸爸，但後來她畫了自己與小狗，有點苦惱地說：「我越來越不記得爸爸的樣子了。」

照顧婆婆不回娘家住

江惠麗說，與先生結婚後原本在台北從事模板工，先生過世後，母子只得遷回老家蜷住，「娘家曾要我帶孩子回去住，但如果我們搬回去，由誰來照顧婆婆和大伯的孩子，我有責任要把這個家撐起來。」所幸大伯的兒子小雄聽話肯吃苦，明白家計沉重，打算半工半讀念高職，也趁暑假期間隨村人做載西瓜跟車小工賺生活費。
雪花婆以太魯閣族語說，她很難過孫子還小就沒了爸爸，更疼惜二媳婦獨力持家勞苦，「我只能求上帝賜給我們家力量，度過難關。」蘋果基金會已提撥急難金暫紓雪花婆一家老弱困境，並將以「克蘭詩．蘋果助學專戶」協助小慈完成國小階段學業。

這家情況是村子裡最困難的，但公部門每月低收補助6600元很有限，二媳婦打零工養一家老小很吃力，我們正協辦小雄的低收資格，有米油物資救助時，總先送給她們家。
