「爸爸生病了」 4口陷困

妻做臨時工扛家計 自責：害孩子跟著吃苦

2009年05月08日

33歲罹患肝硬化的阿揚（戴景揚），去年底開始因腹水、肺積水反覆入院就醫，就讀小二的小豪摸著爸爸腫脹的肚子說：「爸爸生病了，所以肚子像吹氣球一樣一直變大，裡面都是水。」報導 攝影╱游竹君

阿揚與妻子阿欣育有就讀小二及大班的2子，一家4口租屋生活，4月初阿揚又因細菌性腹膜炎、肺積水緊急入院，加上貧血、血小板凝血功能不佳，每天都得輸2袋血保命，阿欣則家裡、醫院兩頭跑，「一天來回3、4趟，有時騎上摩托車，都快搞不清楚自己要去哪裡。」

負債百萬借酒澆愁

阿欣說，丈夫阿揚年輕時做不鏽鋼學徒，3、4年前靠著信用貸款及公公資助開了不鏽鋼工廠，不料2年前因經濟不景氣、生意直直落，「收掉工廠後，還負債100多萬元，阿揚因心情不好，常借酒澆愁，導致罹患酒精性肝炎，去年10月因肚子痛就醫，發現惡化為肝硬化。」眼見健康逐漸惡化，阿揚說自己已經一段時間不敢碰酒，「如果不戒酒，就等不到孩子長大了。」
記者初訪時，阿揚因肺積水、腹膜炎，肚子已經鼓脹得不成比例，講話也氣喘吁吁，但他還是勉強打起精神去做工，阿揚說：「好不容易遇到有工作，就得咬牙做，但現在不景氣，零工少，3月份才做3天。」阿欣也在椰子乳工廠擔任臨時工，「一個月做不到10天，連房租、孩子的學費都得東籌西湊，害孩子跟著吃苦。」

無法請領殘障津貼

記者二度訪視時，阿揚已入院將體內6公斤積水排出。
學校陳姓導師說，阿揚夫妻很關心孩子學習狀況，學校已為孩子協助申請減免營養午餐費。阿揚的姊姊說，弟弟高中畢業就離家，「之前父親也曾拿20到30萬元幫忙開工廠，加上年邁父親中風、還在復健，家人實在無力再多協助。」
里長朱家成說，阿揚工作也很認真，阿揚現在雖領有中度殘障手冊，但因居住地與戶籍地不同，無法請領殘障津貼，將協助處理戶口遷移問題，也申請了「馬上關懷」急難金。醫院王姓社工於去年底阿揚發病時，曾協助繳納第一期健保分期費用，協助就醫，蘋果基金會訪視後也撥款協助阿揚家生活及醫療開銷。
