單親爸抗癌 為兒活下去

17歲子煮粥顧家 憂父「營養不夠」

2010年08月04日

洲仔（江明洲）剛從醫院做完化療回到家，坐在客廳椅子上，抗癌痛苦湧現腦海，「口腔癌已治療1年，原本6公分的腫瘤僅縮小1公分。」洲仔心裡出現不祥念頭，他急忙拈香向供奉在客廳的神明祈求：「求菩薩保佑我堅強活下去！」報導．攝影╱韓旭爾

現年39歲的單親爸爸洲仔苦笑地說，醫生說存活率高，「但癌症治療起來很麻煩，我撐得好辛苦。」洲仔說，目前正進行化療，醫生評估須等腫瘤縮小後才能進行開刀切除，術後還要持續進行電療及化療，「可能還要再折磨一年半載，但只要能好，再痛苦我也能忍。」洲仔說，現在口腔內膜不時破皮潰瘍，痛得難受，但想到獨子小羿還在讀書，他就有勇氣堅持下去 。他說，至少要撐到孩子能工作賺錢，自己照顧自己才能放下心。一旁17歲的兒子小羿說，爸爸還年輕，也都按醫生指示治療吃藥，「我相信爸爸一定會好的！」

補助難支付房租

回溯過往，洲仔說，7年前與妻子離婚後，靠著油漆零工掙錢撫養獨子，雖工作不穩定，但父子倆省吃儉用，還能平穩度日，未料癌症突襲，讓他手足無措。洲仔說，自己嚼檳榔已有20年，去年8月，口腔出現難以痊癒的潰瘍，9月切片確定為口腔癌，之後開始進行化療至今，目前洲仔醫病期間，實在無力工作，家計僅靠洲仔輕度殘障補助及單親兒少津貼共4400元，「這些補助光是付每月5000元的房租都不夠用。」
洲仔說，今年暑假過後升高二的兒子小羿十分乖巧懂事，平時放學回來，就會淘米煮粥，也會幫忙洗衣做家事，自己癌症病發後，小羿為協助家計，曾利用周六、日幫人洗車，每天掙得4、5百元協助家計，但因影響功課已停止。
小羿說，爸爸現在嘴巴咬合困難，只能吃流質食物，「沒錢買營養品罐頭，只能煮些粥，配肉鬆、魚鬆，父子倆就能解決一餐，只是爸爸這樣吃，營養不夠，身體虛弱，我比較擔心。」學校導師說，小羿這學期常請假到醫院照顧爸爸，成績有些退步，但他的程度還不錯，已順利考上工業配線丙級證照，「對於小羿的家庭困境，學校正協助申請教育部急難金。」

親友協助仍有限

當地村幹事說，因洲仔仍列工作人口，在低收申請上暫難核准，2月時已幫忙申請「馬上關懷」急難金應急。洲仔說，他還有4個兄姊，因各自都以零工為生，只能偶爾協助。蘋果基金會訪視後，已撥款暫紓洲仔父子困境。
