洗腎漢寒流擺攤撐家計

2天僅賺250元 還需顧幼子精障妻

2008年02月27日

新春期間，寒流來襲，陣陣冷風吹來，但罹患嚴重肝硬化的凃世仁依舊外出擺攤賣茶壺，他無奈地說：「雖然街上人潮來來去去，但生意卻很差，擺了兩天，才賣了250元，一家四口日子真不知道要怎麼過？」報導．攝影╱韓旭爾

目前43歲的凃世仁說，自退伍後，一直在路旁或夜市擺攤賣茶壺維持生計，「十多年前，台灣景氣好，民眾也覺得新奇，大陸進口的茶壺很好賣，每個月都有5、6萬元收入，但這幾年來，生意真的不好做，常整天沒開市。」凃世仁肝硬化、尿毒症等病纏身，無力另覓他職，只能將就擺攤，「現在一個月賺不到3千元，也不知道該怎麼辦？」而凃世仁42歲的印尼籍妻子阿珊因罹患精神分裂症，也無法外出工作分擔家計。

肝硬化纏身逾八年

凃世仁說，12年前花了30多萬元積蓄至印尼將阿珊娶過門，沒想到，阿珊生了兩個現年10歲與9歲的子女後，便罹患精神分裂症，這幾年來，他要看顧阿珊，同時還要擔起照顧子女的責任。凃世仁說：「阿珊自生完么兒，坐完月子不久，有一天她突然不理人，有時坐在客廳裡自言自語，有時甚至脫了衣服到處亂跑，到醫院檢查，結果發現得了精神疾病。」
凃世仁表示，他已有八年多肝硬化病史，為了治療肝病，除西醫治療外，還服用草藥治療，沒想到肝硬化未治癒，反而造成尿毒病，現每周必須洗腎三次，目前他皮膚及眼睛泛黃，腹部腫脹，像顆充滿空氣的氣球，他說：「現在肚子都是積水，每月都要抽一次腹水，三不五時還會發生肝昏迷，經常送醫院急救。」雖然身體不適，但在街角總還是時常看見凃世仁的攤子，「我要是能夠休息，早就休息了，出來工作，就有機會把茶壺賣出去，能賣多少是多少！」凃世仁現在最擔心的是，萬一他倒下，妻子和孩子該怎麼辦？

拒子女住寄養家庭

當地里幹事羅銘溪表示，凃世仁擺攤掙不了多少錢，一家四口靠著殘障津貼、低收入戶與家扶補助每月21000元過日子。凃世仁說：「看著自己的身體一天比一天差，總不能坐著等死，現在每星期定時前往雲林診所治療，每次藥費加上車資就約需900元，而且不時因肝昏迷而住院，住院時偶爾也需聘請看護。醫療花費負擔大。」蘋果基金會訪視後，已撥款紓困。
羅銘溪說，看到凃世仁病況不穩，曾與他溝通過，希望能安排阿珊至機構安置，而子女讓寄養家庭照顧，放下負擔好讓自己安心養病，但凃世仁堅持不肯，「會與社福單位密切關切凃世仁病況，若有發生緊急情況，將協助處理阿珊與孩子的安置事宜。」

無法計算的人生

凃世仁說：「我也想不透，為何會淪落到現在這種悽慘情況，只能說一切都是命，但是小孩還小，我一定要拼下去。」
