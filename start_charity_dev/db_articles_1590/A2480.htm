單親癌父為3子 忍痛硬撐

親戚接濟生活 「再苦也要熬」

2011年05月30日

「做化療除了想吐外，全身發癢，好像很多隻螞蟻在身上爬。」39歲單親父親阿親（廖志親）剛結束大腸癌住院化療返家，一進客廳，臉色發白地癱坐沙發，「一次次入院治療，真的好痛苦，但3個孩子還在讀書，再苦也得把病治好，把家撐著。」報導．攝影╱韓旭爾

阿親說，今年1月發現罹患2期大腸癌併腸套疊，「醫生說，我還年輕，治癒的機會很高，要我不要擔心。」阿親說，但因常感到疲倦無力，連想起身給放學的孩子做晚飯也爬不起來。

3兄妹分擔家務

阿親與前妻共育有1子2女，分別為16、14及10歲，3年前離婚後，靠著塑膠工廠作業員工作每月掙得約2萬元持家，還可勉強度日。記者採訪時，阿親較年幼的2個女兒剛放學回到家，一進門放下書包後，她們就幫忙收衣服、做飯，就讀小學四年級的小女兒小吟說：「媽媽在爸爸手術時曾來看我們，但沒法長期留在家裡照顧我們，現在我們3個兄妹要學著長大，大家分工合作來做家事，讓爸爸能專心治病。」
回想起發病之初，阿親說，去年5月時，他便常感到腹部劇痛，雖有數度到診所就醫，但總診斷是腸胃炎，吃藥控制，但病情未獲好轉，「我擔心請假到大醫院看病，少了工錢，孩子們也沒人照顧，一直拖，直到痛到受不了，到大醫院檢查，醫生說是癌症，我也嚇一跳。」阿親就讀國二的大女兒小紋說：「爸爸跟我們說患了癌症，一開始我不敢相信，以為是醫生診斷錯了，爸爸還這麼年輕，相信爸爸一定能治好。」

盼能不拖累家人

阿親無奈說道，病重後生活多仰賴親戚協助，爸媽雖住在附近，但爸爸又因中風行動不便，生活也困苦，「兩個哥哥以零工粗活養家，都有家庭重擔，不能再拖累人家。」阿親的媽媽春江婆說，阿親從小身體就不好，國中時便因腦瘤開刀治療，但後來平衡感不佳、左眼也斜視，沒想到現在又罹患癌症，「當媽媽沒有生給兒子健康身體，實在對不起他呀！」
當地李姓村長表示，阿親由於父親名下祖傳土地超過規定，導致低收無法通過，每月僅有單親兒少補助4200元，他生病後，已協助申請公所及地方慈善會的急難金共4萬5千元應急。蘋果基金會訪視後，已撥款暫紓醫病及生活困境。
