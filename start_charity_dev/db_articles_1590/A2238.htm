母癱又失智 孝女棄學撐家

犧牲青春 只為讓弟妹繼續學業

2010年08月11日

43歲的田英枝今年5月中風，右半身麻痺癱瘓，躺在床上像小孩嚷著：「哈哈嘿、哈哈嘿。」20歲長女小凡看了搖頭嘆：「媽媽中風後智能減退，連話也不會說，只會發出簡單的氣音，也不知道是要上廁所還是肚子餓？」
報導．攝影╱仲芝蓉

小凡說，原本父母都在山上打零工，幫人摘菜、拔草，養育自己和現年17與16歲的弟弟妹妹。但2年前爸爸外遇離家，自己當時讀高二，為了幫媽媽分擔家計，只好暫停學業，到外地打工。和小凡一起工作的表姊說：「小凡年紀輕就扛家計，一個女孩子做鐵器包裝、綑工都不嫌重，也不喊累，自己省吃儉用，每月賺1萬多元寄回家，供弟弟妹妹讀書。」

辭職在家顧病母

隔壁鄰居說，小凡媽媽田英枝每天都會一起做農雜工，那天村人發現她沒來，到她家中才發現她倒臥在地，緊急送醫卻已半身癱瘓。醫院林姓社工說，田英枝左側腦中風，不但失語、智能也受影響，小凡只得辭職回來照顧。小凡說，媽媽連上廁所都不會講，必須包尿布，且因心臟病常身體不舒服，卻不會表達，「有一次才吃過早餐，媽媽就開始叫『哈哈嘿』，換了尿布還是叫不停，給她吃藥、陪她玩，都沒辦法，一直到媽媽臉色發白，送急診原來是心臟病發，緊急送醫撿回一命。」
小凡的姑姑說，曾試著聯絡小凡的爸爸，但他在外已有家庭，「小凡的父母之前就有些個性不合，雖分居但沒辦離婚，原先嫂嫂還能打零工維持家計，現全靠小凡。而且嫂嫂行動不便，上下床要人扶，比照顧嬰兒還累，小凡已經失學，現又無法工作，她為讓弟妹繼續學業，犧牲自己的青春。」

「努力讀書回報」

小凡弟弟小廷學校的導師說，家庭雖不健全，但小廷認真負責，功課都在前5名，「已申請原住民學雜費減免60％，但仍需自負3萬元，幾乎都到學期末才繳清，已另申請慈善會助學金一學期2500元。」今年要升高三的小廷說：「姊姊高中美髮科沒畢業，就代父職撐家，我和妹妹只有努力讀書回報。」
公所社會課蕭姓課員說，已幫田英枝辦殘障手冊，等離婚手續辦妥，再申請相關福利，先以「馬上關懷」急難金1萬元協助，蘋果基金會也提撥急難金協助。
