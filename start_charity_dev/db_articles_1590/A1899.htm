單親母打零工 顧罹癌唐氏兒

「她是我心頭的寶 希望上帝別帶走她」

2009年06月14日

53歲單親媽媽阿玉守在醫院，神情憔悴，腳踝因骨刺疼痛，走起路來步履蹣跚，她說，女兒桑妮1月發現患血癌，「桑妮是個唐氏兒，出生就比一般孩子不幸，現在又得了癌症，上天為什麼會這麼殘忍？」報導．攝影╱韓旭爾

記者與阿玉來到病房，14歲的桑妮孱弱地躺在病床上，見到護士過來，不像一般孩子恐懼哭鬧，乖乖地伸手讓人打針，阿玉感慨地說：「桑妮頭腦不好，根本無法了解自己的病多嚴重，所有擔憂都讓我這個當媽媽的一人承擔，我撐得好辛苦。」

丈夫5年前即猝逝

阿玉說，丈夫在5年前過世，之前是建築工人，兩人除育有桑妮外，還有一個現年9歲的女兒，她回憶，產檢並未檢出桑妮是唐氏兒，桑妮出生後，夫妻倆雖傷心，「但她是我懷胎10月的孩子，不管怎樣都是我心頭的寶。」阿玉說，為養育桑妮，丈夫每天頂著大太陽在鷹架爬上爬下，每月能有5萬多元收入，一家過得不錯。
不料丈夫在5年前心肌梗塞猝逝，這些年來，阿玉靠林務零工勉強維持家計。阿玉說：「唯一願望就是平安撫養兩個女兒長大，沒想到這麼不順。」
阿玉十分感嘆地說，唐氏兒易有先天性疾病，桑妮9個月大時便因心室中膈缺損開刀，現又罹患癌症，「希望上帝不要把她從我的身邊帶走。」唐氏症關愛協會社工江秀敏表示，唐氏兒罹患血癌的機會，的確比一般的孩子高一點。
阿玉說，桑妮對化療的反應特別敏感，併發症也多，出院時程一延再延，阿玉望著桑妮強忍哽咽，桑妮似乎無法了解媽媽的擔憂，只是傻笑。阿玉轉述醫生意見說，只要化療療程順利，血癌病患能有6成以上治癒率，「一定不要放棄希望。」

兒癌化療恐須3年

當地村長說，目前阿玉一家領有低收戶補助及桑妮重度多重殘障津貼共1萬1400元，但山區部落離醫院須3小時車程，每次包車要花3000元，交通開銷很大。醫院社工表示，目前桑妮的學生平安保險提供1年上限5萬元醫療補助，兒癌基金會也提供4萬元生活補助，但兒癌化療可能長達3年，阿玉為照顧桑妮很難安心工作。蘋果基金會訪視後，已撥款暫紓阿玉急難。

師生祝福

自從桑妮住院後，學校導師和老師不時前往探望，送來特教班同學製作的祝福卡片（圖），寫著「早日康復」。老師說，桑妮不識字，但仍專注地看著卡片，「相信她一定能感受到家人、師長與同學的關心。」
