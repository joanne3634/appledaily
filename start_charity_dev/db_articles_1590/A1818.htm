視茫夫冒險騎車 載癌妻看病

為省車錢不得已 「窮人沒生病的本錢」

2009年03月06日

走進住家附近的廟宇，59歲的潘秀梅望著神明，一臉虔誠禱望，她跟記者說：「去年11月發現得了乳癌後，我除聽醫生指示按時到醫院努力做治療外，其他能做的，就只剩聽天由命，請老天爺多多幫忙了。」報導．攝影╱韓旭爾

潘秀梅與54歲的男友阿賢共同生活近30年，雖未登記結婚，但彼此早以夫妻相稱，「我們認為若有心要一起生活，不用在意那張結婚證書，所以就遲遲沒到戶政事務所登記。」潘秀梅回述說，30多年前丈夫因病去世，數年後，與男友阿賢交往，之後她便帶著與亡夫所生兩個年幼兒子與阿賢同居，並再生一女。

嘆長子孝順卻早逝

回想自己罹癌經過，潘秀梅感嘆說：「窮人沒有生病的本錢。」她說，前年就發現乳房有硬塊，但是忙著工作顧三餐並不在意，等到發現不對勁、到醫院檢查時，已是乳癌二期，「當初要是能早期發現、早期治療，痊癒機率會更高，現已動了切除手術，正持續化療中。」
阿賢受訪時則靜坐在一旁，滿臉落寞表示，自己身體情況也不佳，長年飽受糖尿病所苦，工作時，手腳若不小心受傷，傷口總是痊癒特別慢，現在眼睛因糖尿病造成視網膜病變，視力變得模糊不清，也無法到工地上班，阿賢無奈地說：「秀梅回診治療時，她不會騎機車，只好由我騎車，秀梅在後座幫我注意路況，雖知道這樣子危險，但為了省車錢，這也是不得已。」
現潘秀梅罹癌，本應有子女承歡膝下，但她提起與前夫所生的2個兒子，深嘆一口氣，露出無奈，她說，長子孝順，但17年前因車禍逝世，33歲的次子不學好，偷竊、吸毒樣樣來，「二兒子在監獄的時間比在家還多，去年剛假釋出獄，現在又因偷竊被警察通緝。」至於她與阿賢所生的女兒，潘秀梅說，10年前，女兒才17歲，便因未婚懷孕而出嫁，4年前離婚後已不知去向，「一年多來，從不曾打電話回來，我們也很擔心。」

掛號150元都付不出

因此，近10多年來，潘秀梅夫妻全靠自己打零工維持家計，奉養高齡89歲的公公。她說：「我知道像我們這樣靠勞力掙錢的辛苦人很多，只要身體健康，就有本錢去打拼，但只要生病，問題就一堆！」
與潘秀梅一家相識約30年的老鄰居阿甘婆說，潘秀梅與阿賢兩人都生病，長子乖巧，卻英年早逝，次子三不五時鬧事，看了叫人感嘆。潘秀梅說，已約4個月沒有收入，有時連回診150元掛號費都拿不出，只能向朋友借錢應急，蘋果基金會訪視後，已撥款暫紓生活、醫療困窘。面對後續漫長的抗癌路，潘秀梅說，我一直跟自己說，事情遇到了，就只能堅強面對，「只能求上天保佑要趕快好起來，快點去賺錢。」
