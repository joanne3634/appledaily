男攜憨兒拾荒 養癌妻癱母

6口家4病弱「我老了他們怎活」

2011年06月24日

傍晚，52歲的阿生（陳福生）與22歲中度智障長子阿男在庭前空地整理拾獲鋁鐵瓶、寶特瓶，他說，與妻育有2子1女，但兩子皆有智能障礙，家中還有74歲老母，6口生活皆靠他一人，無奈近年來老母中風，妻罹胃癌，頻繁進出醫院，生活雪上加霜。報導．攝影╱韓旭爾

阿生說，他與47歲妻子阿媛是經媒人介紹結婚，「相親時，阿媛看起來正常，笑起來特別親切。但婚後才發現阿媛反應遲鈍、理解能力也差，數字1到100也不會數，才知阿媛有智能障礙，根本沒法工作。」

妻一度拒化療

阿生居於祖厝，靠著他打零工與拾荒清苦度日。4年多前阿生同住的母親因中風癱瘓、2年前阿媛又發現罹患胃癌，現兩人常進出醫院，「原本日子就不好過，家中又多了兩個病人，真的好頭痛。」
一旁的妻子阿媛聽到阿生無奈感嘆，略帶生氣地說：「生病又不是我願意的，肚子痛當然要看醫生，不看怎麼會好。」阿生見狀輕聲回應說：「沒說妳不好，只是我們家運氣不好。」阿生接著說，對阿媛來講，她沒法了解癌症是會要人命，「當時抗癌化療很苦，她嚷著說不要看；但肚子一痛又說要看醫生，想法很簡單。」阿生說，阿媛雖腦筋不好，但能洗衣、做飯，處理家務，「有她在家，我才能安心外出打拼。」
回憶兩子出生之初，阿生說，兩個兒子大概到3、4歲才會叫爸媽，當時還以為只是大雞晚啼，直到阿男上國小時，老師建議到醫院檢查，才發現他是中度智障，接著把阿宏也送鑑定，結果是輕度智障。一想到兩子皆智障，阿生無奈的表示：「老天很會開玩笑，兩個兒子外表長得像老婆，都是帥哥，可是頭腦也像老婆一樣憨憨，現在我還能看顧他們，要是將來我老了，顧不動了，真不知他們要怎麼活下去？幸好女兒正常，將來還有她可以依靠。」

小女懂事扛責

阿生就讀大一的女兒小雯說，兩個哥哥雖然憨傻，但兄妹間感情很好，「家裡有好吃的，他們會留一份等我回家吃。」阿雯說，從她懂事以來，爸爸就不停叮囑一定要把照顧哥哥們當作一輩子責任，「我會好好讀書，將來找份好工作，努力賺錢好好照顧哥哥、媽媽。」

謀生賺錢路遙

當地村長說，目前阿生家領有政府社福補助共2萬2000元供6口生活。身心障礙者職業重建協會老師說，阿男、阿宏目前雖受職訓，但「要能謀生賺錢還有一段路要走。」蘋果基金會訪視後，已先撥款暫紓阿生一家生活、醫療困境。
