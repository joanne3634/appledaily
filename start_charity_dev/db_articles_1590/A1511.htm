子罹癌收入斷 嬤愁醫費

「要不是年紀太大 再苦工作也做」

2008年02月20日

71歲王楊彩霞對著佛經、雙手合十，「老伴肺癌過世，兒子得結腸癌，不敢想未來，只能誦經求平靜。」長年承擔家計的次子阿銘罹癌無力工作，「大兒子智能障礙不能幫忙，低收補助全付房租，阿銘醫療耗材費根本付不出來。」王楊彩霞無奈地說。報導．攝影╱張嘉恬

王楊彩霞說：「阿銘從來不生病，沒想到一病就是這種要人命的。」彩霞嬤回憶道，她和老伴共育兩子一女，長子因早產導致智能障礙，精神障礙的次女獨自在外租屋、靠低收補助生活，「老伴12年前往生後，家裡只有阿銘能賺錢，他這個支柱倒了，我們連水電、瓦斯都付不出來，要不是我年紀太大，再苦的工作我也願意做。」

須長期化療至少三年

彩霞嬤說，阿銘一年半前罹癌後，她找過多家小吃店洗碗工作，「大家都說我年紀大應該在家享福，怎麼還出來工作？真是有苦難言啊。」而最近阿銘的癌細胞更已擴散至肝及肺，得長期化療至少達三年。
採訪時阿銘剛從洗手間走出，「我清洗了一下人工肛門的便袋，不然排泄物過久沒清會有異味。」現年46歲的阿銘說道，往常他以每月的司機薪水約3萬元支付家中開銷，扣除每月1萬4000元房租後，剩餘約1萬5000元已難敷全家三人生活，而他罹癌後全家雖通過北市低收資格，但老津及殘津補助恰巧夠付房租，一年多來生活開銷，則仰賴親友在他當時住院時所贈的10多萬元紅包錢。阿銘補充說：「一年多前切除癌細胞後，心想病情若穩定仍可上班，沒想到便袋得全天帶著，排氣、排瀉都有聲音，還要不時清洗，工作起來不方便也很尷尬。」
而阿銘現在更因癌細胞擴散，體力極差，連步行太久都會氣喘吁吁，著實無力勝任工作。彩霞嬤說，阿銘得住院做化療，但常排不到健保床位，因阿銘付不起每天1900元的自費，只好每天來回醫院，彩霞嬤說：「雖然把化學藥劑帶著走還要另外的裝置袋，算一算每月袋子花費會比住院便宜，沒錢什麼都要算得細一點。」

長子智障無工作能力

阿銘在去年10月申請勞保失業給付每月約一萬元，但仍不敷耗材費及全家三人的三餐、水電、瓦斯及阿銘營養品開銷，且給付也為期半年僅至今年三月止。里長許世傳表示，阿銘家裡老母年事已高，哥哥又因智能障礙無工作能力，「唯一能賺錢的人得癌症，也不可能工作養家了。」蘋果基金會訪視後，已撥款暫紓阿銘醫療耗材費、全家生活困境。

無法計算的人生

阿銘說：「我從來不生病，沒料到癌細胞找上我，信心是和癌細胞打仗的最佳武器，我一定會好起來、參加職訓，還有一片好光景等著我！」
