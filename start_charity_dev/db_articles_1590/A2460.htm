癱父頻住院 少女醫院為家

「小時你幫我把屎把尿 換我照顧你」

2011年05月06日

「爸爸，麵包就在你旁邊，餓了就吃！」早上一起床，17歲的聖子幫脊損癱瘓的爸爸洗臉、拍背，在臀部褥瘡傷口擦藥，把昨晚放學途中購買的麵包放在床邊矮櫃上，接著到學校上課，如此父女相依生活，數年如一日。報導．攝影╱韓旭爾

今年讀高一的聖子說，13年前爸爸從事模板工作時，不慎自4樓跌落，造成頸椎及腰椎損傷，當時原本還能跛行，但8年多前病況惡化，只能以輪椅代步。近1年來，更因長期臥床造成臀、背部產生褥瘡，不時住院治療，只要爸爸一住院，醫院就變成她第2個家，白天上課，晚上就睡在醫院躺椅上。

三餐靠慈善會接濟

聖子57歲的爸爸阿清說，意外發生後，以祖產土地貸款100萬元支應生活及治療脊損花費，但約8、9年前就已花用殆盡，且無力還貸款，土地已被查封。這些年來，父女兩人每月依靠政府殘障、兒少及家扶補助共7100元度日，還好三餐有地方慈善會送米、罐頭及泡麵幫忙，聖子學費由多個慈善會協助支持，如果不是因為近來經常就醫，無法負擔醫療開銷，父女兩人再苦都還是會撐著過。
阿清回述說，父母均已去世，一兄一弟甚少往來，自己有兩段婚姻，聖子是第二段婚姻所生，但妻子在１2年前離家出走，至今失聯未辦離婚。阿清說，當年聖子才只有4歲，他父兼母職，幫聖子換尿布、洗澡，「沒想到，現在聖子正青春，本來只要專心讀書，放假跟同學外出玩樂，但現在卻還要幫我清理大小便。」

冬天忍耐洗冷水澡

阿清哽咽說：「家裡曾經因沒錢叫瓦斯，聖子曾在冬天裡連洗了一個多星期冷水澡，女兒這麼乖巧，卻要陪我吃苦，實在捨不得。」聖子見狀急忙安慰說：「我小時候你還不是幫我把屎把尿，現在換我照顧你，這是應該的呀！」
聖子更心疼爸爸也因省錢強忍病痛，有次褥瘡潰爛深及血管，造成爸爸大出血，「我放學回家，看見地上都是血，把我嚇死了。」阿清說：「住院一天，餐費就要150元，有時還須聘請半天看護照顧起居，每天就要1000元。」
聖子高中導師說，為了家計聖子曾想放棄學業，「我常鼓勵她一定要好好讀書，將來才有能力找份好工作，讓爸爸得到更好的照顧。」

曾獲家扶自強兒童

已協助聖子一家4年多的家扶社工說，去年度聖子曾獲選家扶自強兒童。目前阿清因未辦妥離婚，難獲低收核可，現已經由法津訴訟中。蘋果基金會訪視後，已先撥款協助聖子父女生活、醫療困難。
