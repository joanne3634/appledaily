夫肢殘 獨育癌妻2幼子

緊守愛妻「怕再組的家垮掉」

2008年11月03日

44歲的陳來發小心翼翼地扶著妻子董鳳琴，平均每20分鐘就得跑廁所一次，「她今年6月罹患膀胱癌，手術後頻尿，每月還得化療三次。」陳來發深怕久病精神不穩的妻子想不開，緊守在旁，就怕再組的家庭垮掉。報導 攝影╱孫敏聿

憶起10年前，前妻自殺身亡，現在的妻子又罹患癌症，陳來發感嘆萬分，「不知道是不是我命硬，會剋到另一半。」他說前妻因無法為他生育，愧對身為獨子的他才自殺，當時是「年頭才結婚，年尾就辦喪事。」

尚欠5萬元手術費

陳來發說，妻子帶著與前夫所育一子再嫁，婚後再育一子現年6歲，靠著他做電焊工，月入3萬多元持家，不料4年前他工作時誤觸高壓電，左手險些截肢，領有輕度殘障手冊，再難找穩定工作。
「當時僱主理賠25萬元，因我工作不穩定，2年前用畢了理賠費。」陳來發左手負傷後，僅以打零工維生，近2年來的臨時僱主謝先生表示，「陳再發工作還算認真，不過因為手不方便，只能算臨時工，平均一個月做10天左右，一天薪水雖有1700元，但沒做就沒錢，常跟我預支工錢買菜、生活費，自從太太生病，幾乎沒法來做。」
陳來發指出，妻子6月份罹癌時積欠3萬多元健保費，是在慈濟基金會協助下辦理分期才就醫，8月份妻子又因頸部椎間盤突出，手術治療，至今尚欠5萬多元醫療費用。

長子開朗做事認真

陳來發說，「我收入不穩，現在要照顧妻子，又走不開去工作，親友能借的都借了，真是煩惱。」
青山國小王老師表示，陳來發兩子分別就讀國小5年級與1年級，營養午餐與學費都由校方減免。「5年級的小山個性開朗，做事認真，在班上擔任衛生股長，校方得知家境後，亦協助申請2萬元學產基金急難救助金。」
國泰醫院社工員陳小姐說，6月份董鳳琴因膀胱惡性腫瘤入院手術，8月份她動頸椎手術時需自費3萬8000元使用椎間融合系統，「兩次住院期間尚欠5萬2000元，院方正申請愛心基金協力部分。」
七星社福中心社工員洪小姐亦表示，由於陳來發一家被妹妹申報為撫養人口，今年無法申辦低收入戶，9月份起每月可領有弱勢兒少補助及家扶中心共9400元。蘋果基金會獲悉，已先行撥款紓困。
