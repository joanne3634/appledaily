無子女積蓄 兩老病纏身

翁盲嬤腰傷 「天一變 好像千萬螞蟻在咬」

2011年02月17日

窗外吹過冬季凜冽的寒風，只見秀卿婆（趙秀卿）坐在客廳嚷著腰椎、髖關節痠痛不堪，連吃了止痛藥都沒有效，老伴仙做伯（謝仙做）聽了在一旁安慰說，過陣子天氣回暖就會舒服些了，「妳再忍耐一下啦！」報導．攝影╱韓旭爾

仙做伯今年85歲了、老伴秀卿婆79歲，兩人未生下一男半女，多年來，兩老相依為命。仙做伯無奈感嘆，人生臨到老來，求的不多，只求將來百年之時，能一路好走就是最大的福氣了，「可是現在夫妻兩人全身都是病，不只身子痛苦難受，看病的花費更是傷透腦筋。」仙做伯說，自己在9年前因車禍右眼失明，3年前左眼又因為白內障視力不良，現只能隱約感受光影，而老伴秀卿婆半年多前，騎機車不慎發生自摔車禍，造成右側髖關節骨折及腰椎損傷，雖已手術更換人工關節，但還是不能久站，腰椎不時痠痛難耐。秀卿婆接著說，「冬天濕冷，腰、腿關節變得像氣象台，只要天氣一變，疼痛就跟著來，好像千萬隻螞蟻在咬。」

姪兒幫忙保棲身處

回憶過往，仙做伯說，他與秀卿婆兩人都是二度婚姻，30多年前兩人結婚後，夫妻以宰殺羊隻販賣為生，生活平穩，雖無生育子女，本以為只要有間房子，再存點積蓄，兩老便能安度晚年，但約11年前，為借款給友人應急，以自宅貸款200萬元，「但朋友竟然跑路，加上賣羊肉的生意越來越差，每月1萬多元的貸款付不出來，房子因而被查封拍賣。」仙做伯無奈地說，幸好姪子將房子標下，兩老才不至於無處棲身。仙做伯的姪子說，自己有子女要撫養，每月約1萬3千元房貸付得很辛苦，「也不知還能付多久？」

遲未獲低收入核可

目前仙做伯兩老只能依靠每月共6000元的老人年金度日，當地李姓老鄰居說，每次看到仙做伯兩老三天二頭就到醫院看病，卻沒晚輩扶持照顧，都會幫忙叫車接送，常聽見仙做伯感嘆，老來沒子女孝順，真的很悲哀，「聽了令人心酸！」蘋果基金會訪視後，已先撥款暫紓仙做伯兩老生活、醫療困境。
當地的里長說，仙做伯年輕時曾任附近廟宇管理委員，廟宇建地登記在他的名下，因此兩老一直未獲低收入核可。廟宇現任管理委員說，當年建廟時，並未向政府進行社團法人的登記，現正補辦登記程序，屆時便能變更土地產權為法人名下。仙做伯說，人老了總有一些事要面對，對他們夫妻倆而言，還能吃多少年的飯呀？但只希望剩下歲月不要有太多病痛折磨就好了。
