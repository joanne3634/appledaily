憨兒腿傷 老母抱病撐家

種菜養雞 「不知能做多久」

2012年03月13日

78歲阿梅嬤（李玉梅）無奈嘆道，56歲的三子阿裕（潘豊裕）去年12月不慎鋸傷自己右小腿，又因糖尿病感染，傷口反覆發炎，「我的三個兒子和二個女兒都離家打拼去，就剩我和阿裕相依為命，只求他平平安安，我這一生才算功德圓滿。」報導．攝影╱仲芝蓉

阿裕皺著眉頭忍痛換藥，口中念念有詞地說：「求佛祖菩薩保佑，讓我的腳趕快好起來，一定是祖先生氣了，我才會受傷……。」阿梅嬤見狀嘆了一口氣說，阿裕28歲那年曾出車禍，命是撿回來，但撞壞了腦袋，之後講話就顛三倒四，有時講什麼沒人聽得懂，不過，還可以跟村人去做砍樹、噴農藥等零工，「這兒子注定討不到老婆，10多年前我老伴走了後，就剩我和他相依，沒想到他鋸木材時不小心受傷，還發現他有糖尿病，傷口一直發炎不好，希望老天爺多保佑阿裕。」

腿傷一直好不了

阿梅嬤指著兒子右腳踝潰爛發炎的傷口說著，只見阿裕時而躁動、時而靜默，對阿梅嬤的憂心似乎充耳不聞，阿梅嬤繼續說，阿裕的傷一直好不了，連走路都跛腳，「以前生活辛苦，我們靠種田養孩子長大，好不容易讓阿裕小學畢業，只能做勞力工，但現在他腳受傷，根本沒辦法工作。」
「老伴留下這間祖厝給我和阿裕，我自己種菜、養雞，鄉下生活簡單，但近幾年我高血壓、痛風老毛病常發作，腳也不方便，不知還能做多久！」阿梅嬤擔心地說。

月靠7千元過活

公所急難承辦員說，阿裕雖因車禍腦傷反應憨傻，但未領有殘障手冊，無法申請殘障津貼，當地家婦中心社工已協助阿裕鑑定，目前阿梅嬤母子倆靠老農津貼7000元為生，但每月就醫光車資就需3000元，對他們來說是一筆沉重的負擔，公所已申請「馬上關懷」急難金1萬元協助，並轉介蘋果慈善基金會，基金會訪視後已先撥款暫紓困境。
阿梅嬤的長女阿美說，鄉下工作真的不好找，自己和兄弟都在外地謀生，但在外租屋生活開銷都高，「阿裕意外受傷，我們幾個已盡力湊幾千元幫他繳醫藥費，希望阿裕的傷能趕快好。」
