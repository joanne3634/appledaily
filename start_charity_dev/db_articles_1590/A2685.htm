壯漢罹肛門癌 日日難眠

生計陷困擠陋屋 殘母打氣：媽還要靠你

2012年02月06日

47歲陳藝文得少見肛門癌，術後傷口疼痛讓他坐立不安，他想到剩他和68歲肢障母親江娥相依，從知得病那天起，沒一天好睡過。報導．攝影╱向高彬

陳藝文拉開上衣，軟黃便液緩瀉腹部便袋，他又拉下褲子，露出右腹股溝一顆鵝蛋大腫瘤，他說，肛門癌4期，去年底化療及放療後，肛門腫瘤縮小，但現麻煩的是移轉腫瘤，「我告訴自己要想開，去面對。」
陳藝文單身未婚，罹癌後無力做馬路清掃工，只能靠曾車禍右手輕殘的母親照顧，當地社工說，陳家母子現為二款低收戶，月領補助14000元。
慈濟社工說，送米油物資關懷，「生活仍吃力，再協助營養品。」蘋果基金會訪視後，也撥款暫紓醫病急窘。

伴癌兒「老來坎坷」

陳家在狹小巷弄，5坪屋為承租國有地，江娥亡夫生前改建3層小閣樓，她嘆，曾遭鄰居倒會百萬，畢生積蓄一空，豈料老來更坎坷，怕老天連個兒子也不留，「我陪他回診，要他加油，媽媽還要靠你啊。」

頻換便袋「好難受」

之前陳藝文做過廚師，他說，曾患胰臟炎、體力差，改做保全、清掃工，前年肛門長瘤，以為是痔瘡，切除兩次，但仍痛到不行，去年轉診大醫院，醫師查出是少見肛門癌。陳藝文去年底裝人工肛門，每兩三天換便袋，每月醫療耗材和回診就約6000元，他說：「得這病好難受。」
陳藝文回述，爸生前做泥水工，與媽媽生他和一弟，爸爸10多年前肝癌逝，單身弟弟5年前心臟病猝逝，「我沒感情緣，本想和媽媽平安過日也好，沒想到卻病倒。」媽媽江娥說，兒子得癌後，「半夜我想到就哭。」
