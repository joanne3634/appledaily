火燒屋 7童：何時有新家

兩戶11口挨年關 亟盼40萬重建家園

2011年01月30日

74歲的任金旺面對去年（2010）底祝融肆虐後，仍是一片焦黑的殘破屋子，心裡感傷，幾個小孫子問他：「爺爺，我們還有多久才有新家？」他兩眼落寞，不知怎麼回答。報導．攝影╱向高彬

任金旺是太魯閣族原住民，他以不甚流利的國語說著，與乳癌病逝已7年的老伴辛苦流汗大半輩子，在家鄉部落一磚一瓦搭造起來，與各自成家的3個兒子生活居住的房子，竟在去年12月8日疑似房屋老舊電線走火而突然發生火災，幸消防隊及鄰人協力搶救得快，雖人員無傷，「但3間相連的小屋子一口氣兩間燒光光，只剩一間，但一家住小兒子阿勇夫妻一家6口，一家住四兒子阿貴夫妻一家5口，這兩家11口，現在都沒地方好睡，還得擠住附近親友家，過了一個沒家可以安住的春節。」

子做粗活僅供溫飽

任金旺家火警新聞曾在《蘋果》去年12月9日地方新聞披露「火燒屋 童泣 天冷要睡哪」，當地村長表示，當時慈善會紛送暖，濟助衣物棉被食物，暫先讓任家一大家子三餐生活稍安定，並獲救助金共約10萬元，但離重建材料經費約40萬元仍有差距，蘋果基金會訪視後，也拋轉引玉撥款盼盡棉薄之力，專幫弱勢家庭修屋的寶島行善義工團表示，屆時可協助出力修繕，盼早日幫鄉間苦弱家庭重建家園。
任金旺回述，他與亡妻生有5子2女，女兒已嫁，平常他與3個兒子共同生活，兒子沒讀什麼書，平常只靠做粗活雜工勉強養家，孫子女又都還小在讀書，兒子做工掙沒幾個錢，前兩年金融風暴時，還失業待工大半年，靠貸款借錢過活。

家人平安萬中之幸

好不容易這1年多來景氣慢慢復甦，兒子經介紹到外地做道路橋樑噴漿工程臨時工，每月有近3萬元收入，漸漸生活可過，但還得還債，僅能勉強溫飽日常三餐，「沒想到一把火燒壞房子，剩空空四面牆，實在沒多半塊錢重修屋子。」
任金旺的四兒子阿貴說，平日在外做工，家裡剩孩子和太太，失火時他人在外地，接到家人電話告知失火屋毀時，「人都傻了，幸好孩子沒事，不然可能活不下去了。」只是起火點始於他家廚房，延燒波及小弟阿勇家也全毀，讓他非常過意不去，「自己也還有債要還，也借不到錢，孩子都很難過，家燒掉了。」阿勇說，都是一家人，只盼有機會快把家修好，平安就好。
