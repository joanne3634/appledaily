單親母罹癌 「孩子怎麼辦」

借錢醫病 為兒拚命活下去

2009年01月22日

「最近失業問題嚴重，我擔心就算身體好也工作難保，更何況乳癌已移到腦和肺，有時很想乾脆農藥喝一喝，但想到兒子才高一，我不拼著活下去，他怎麼辦？」新春將到，但單親媽媽金怡伶望著病房窗外灰濛天空，卻心中茫然。報導．攝影╱向高彬

採訪時，在病房見到40歲的金怡伶，她光禿頭頂有約5公分大紗布包覆處。她說，頭頂也有移轉的瘤，癌症真恐怖，為腫瘤做了9次化療和電療，「本來還想我以前很健康，從不生病，就算生這種病，開個刀、吃吃藥也應該很快好。但和它對抗都快一年了，控制卻很有限，不知道接下來會怎樣？」

母子抱頭痛哭

金怡伶因離異，單親撫養一子，她說，去年（2008）初發現左乳有腫塊疼痛，經檢查竟是乳癌，她顫抖告訴兒子「媽媽得了乳癌」，現年16歲的兒子聽後，兩人就抱頭痛哭。之後她持續進行治療，但目前病況仍移轉惡化為三期，「醫師很盡力，也叫我別想太多專心養病，我現在就是該做的治療、開刀、吃藥都按時努力配合，抱著正面樂觀心情面對。」其實金怡伶話聲爽朗，若沒見她病樣，還真難相信她患重症。

擺攤虧了60萬

金怡伶回述，12年前因前夫外遇及經濟問題離婚，前夫工作不穩定，由她擺檳榔攤撫養獨生子小鴻，原本兩母子相依可度日，但5年前生意每下愈況，虧錢結束經營，還欠貸款卡債約60萬元。她改做旅館房務清潔工，月薪勉強支應租套房及孩子求學等家用。
未料她突患重病，至今她已跟親友借了7萬元醫病生活。更糟的是，去年5月時她因缺錢賣詐騙人頭帳戶遭查獲，審理中帳戶被凍結，母子生活雪上加霜。金怡伶之前的同事阿佩說，她很疼孩子，做完12小時夜班，清早6點下工，還幫孩子準備早點，「她病了，最擔心的還是以後孩子怎麼辦」。

前夫暫付房租

採訪時，小鴻說，很擔心媽媽的健康，「但我能做的不多，只能多鼓勵和陪她，提醒她吃藥，問今天好不好」。小鴻又說，與生父少聯絡，但還好去年10月起他知道媽媽病重又家計窘迫，目前每月先寄5000元暫付房租。小鴻的爸爸阿松現於外地做工，他說，不知工作能做多久，自己有租屋債務負擔，「我46歲了，有工作就不容易，只能盡力，先想辦法讓他們母子有地方住」。
金怡伶從小父逝母改嫁，僅有一兄阿海，但哥哥家為養育6名子女的三款低收戶，生活也難捱，「看到妹妹更苦，也把補助的錢擠一點幫忙」。目前金怡伶家月領兒少補助1800元，家扶及學產基金救助申辦中，當地林姓里幹事說，已發「馬上關懷」救助金1萬元，正協辦低收資格。蘋果基金會訪視後，也撥款暫紓病弱母子醫病生活急窘。
