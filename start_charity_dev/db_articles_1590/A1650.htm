夫肺疾重症 婦做女工苦熬

「他住院不敢去顧 怕丟工作一家怎活」

2008年07月30日

「現在別說出力做工，我連爬上3樓都要停好幾次，喘5分鐘才上得來。」呼吸原本是件容易事，但55歲的潘金治去年初發作塵肺症，如今喘得無力做工、常住院，太太微薄工資難撐一家四口家用，已需借支過活。報導．攝影╱向高彬

潘金治的55歲太太鄭焦說，醫師說先生肺功能僅剩不到常人一半，如今只能多休養避免惡化；她最怕先生患感冒或天氣變化，常掛急診住院好幾天，但家裡現只靠她在木材工廠女工收入過活，還要照顧兩個國中年紀兒子，「他住院我也不敢請假去顧他，只讓他自己住院，我一把年紀了，萬一常請假丟掉工作，一家以後怎麼活？」

無奈房子恐遭法拍

鄭焦說，工廠體諒她家苦境，已讓她先預支3萬多元薪水，但每月扣薪還款後，僅剩1萬元出頭，鄭焦說：「房貸都欠了5個月，銀行不時打電話來，再不繳就法拍，我說要賣就賣吧，再逼也沒錢。」
鄭焦回述夫妻過往情路坎坷，各有一段婚姻，潘金治早年跑遠洋漁船，入贅後因前妻外遇離異，而鄭焦則因前夫家暴而逃家，之後兩人相識生下現分別15、14歲的兩子，幾經波折兩人於9年前正式結婚，「先生之前一直照顧罹癌婆婆到她病逝，8年前我們才存點錢貸款買現住房子，本想兩人做工，可以安心扶養孩子長大，沒想到他卻病了。」

2國中兒盼助家計

採訪時，大兒子小勇說知家裡辛苦，但自己對讀書很不行，所以打算明年國中畢業去學技術做事，好幫家計；讀國一的小兒子小華說，她覺得媽媽好辛苦，一人做工難養家，「為了想幫家裡一點忙，同學的媽媽幫我找了在輪胎工廠鎖螺絲打小工，8月有1個月工期，能賺1萬多元。」
當地陳姓里幹事表示，已協辦潘家急難救助，但因兩夫妻有前段婚生成年子女，且潘金治尚無殘障手冊，仍計工作人力，因此未過低收，但邊緣弱勢家庭一時因病弱，生活辛苦。蘋果基金會訪視後，也先撥款暫紓潘家醫病生活急窘。

塵肺症小檔案

●高危險群：礦工、耐火磚製造業、噴砂業、砂炒花生人員等。
●治療：至多穩定病情，嚴重者需靠氧氣筒與藥劑度餘生。
●預防：工作佩戴防塵呼吸防護具；工作場所加強排氣通風設備。
●致病原因：長期吸入礦物粉塵（如二氧化矽、石綿、煤）沉積於肺，導致肺部非腫瘤反應的所有症狀總稱。
●症狀：呼吸急促，由乾咳變濕漱，血痰，末期呼吸困難。
資料來源：台灣礦工醫院胸腔內科主治醫師楊傳音
