殘妻顧癱夫 泣無醫費

「他倒了很無助 只能在旁守候」

2011年03月08日

49歲陳芳蓮蜷縮著中風過的左手，看著去年底因腦中風癱躺病床的55歲先生，她不禁淚眼婆娑，不停地喚著沉睡中的先生，只希望他能恢復往日健康，繼續一起攜手走完人生。
報導．攝影╱張嘉恬

陳芳蓮說：「我中風前後樣貌差很大，手腳不靈活、也老了很多，但他對我還是一樣好，到哪裡都帶著我，不怕麻煩也不怕人笑；現在他倒了，我只能無助地在旁守候。」
陳芳蓮說，她和先生皆各自有過一段婚姻，當年她的兩子歸前夫撫養，先生則帶著獨子阿豪與她再婚，兩人婚後未再生育子女。陳芳蓮曾於3年多前中風、持有中肢障手冊，先生則以擴大就業零工維持生計。

兒苦撐家難援手

陳芳蓮說，兒子阿豪與媳婦育有現年10、3及1歲共3子女，5口人仰賴兒子種菜等務農零工維持，因收入微薄不穩定，「他自身難保，之前還靠先生協助他們水電等部分開銷。」

夫哽咽盼病快好

陳芳蓮說，自己肢障無法實際負擔照顧責任，正因家中無人能看顧先生，為著看護費用、醫病開銷發愁。
採訪時，陳芳蓮先生才剛從睡眠中清醒，在太太說明後他不停地對前來訪視的記者說感謝，因病表達無法順暢，他語帶哽咽、一字一句緩慢說，自己一定會勤作復健，「趕快好起來，才不會拖累家人。」
陳芳蓮兒子阿豪說，自己生母早逝，父親再娶，母親陳芳蓮對他視如己出他無奈說：「爸爸住院到現在我都在這裡看顧，但是妻兒生活沒辦法不顧，太太需要在部落照顧幼子，我更得回山上工作才行。」

把握半年復健期

醫院蔡姓社工表示，陳芳蓮先生須把握至少半年的黃金復健期，「半年後視復健效果才能進行身心障礙鑑定。」
蔡社工表示，待陳芳蓮的先生通過身心障礙鑑定後，才能進一步辦理低收資格。當地林姓村長說，陳芳蓮先生已獲「馬上關懷」急難金1萬元，蘋果基金會訪視後也先撥款協助。
