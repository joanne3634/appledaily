寡婦養癱母幼子 「快垮了」

擺攤賣雜物 「只求三餐白粥配肉鬆」

2009年12月29日

35歲的阿麗（吳美麗）蹲坐在自己二手雜物攤位旁，上午8時開市後，已快到中午時分，還沒賣出半樣物品。她想到中風的媽媽躺在醫院治療，還有3個年幼孩子，「生意不上門，日子要怎麼過？」報導．攝影╱韓旭爾

記者跟著阿麗來到醫院病房內，阿麗55歲的媽媽阿閃婆（吳陳阿閃）身上插著鼻胃管和醫療針管，意識時而清醒、時而迷糊。阿麗說，11月中旬，媽媽早上起床時便直嚷著頭痛、手腳無力，「醫生診斷為腦中風，現右半身癱瘓，講話也有困難。」

擔心孩子教養問題

阿麗說，多年來全靠媽媽在家幫忙看顧孩子，她才能安心擺攤賣些二手雜貨，原本每月還能掙得約1萬元養活5口之家，「媽媽病倒後，我要接送孩子，又要做生意，醫院、家裡往返奔波，自己感覺也要垮了！」
丈夫6年前病逝後，阿麗便帶著3個分別13、11與9歲的子女與媽媽同住，其中老大、老三是輕度智障，老二的智力也不好，「自己不到30歲就當了寡婦，3個孩子又都要特別費心照顧，日子原本就過得吃力，現在媽媽又病倒。常覺得自己只求三餐能有碗白粥配肉鬆，能度日就好，但在現實生活中，卻也大不易。」
現阿麗除了擔心媽媽病況外，更為3子女的教養發愁，阿麗就讀小四的么兒小霖的導師陳秋慧說，小霖最近到校時衣服顯得骯髒，有時還會散發出異味，同學常會排斥他，下課常看到他自己一個人玩耍。老師接著說，學校了解小霖外婆生病，家長較辛苦，孩子難照顧周全，她有時會在學校幫小霖洗澡，讓他保持身體清潔，「在小霖就學方面，低收入戶未補助的費用，學校也幫忙尋找資源支應，以減輕家長負擔。」
阿麗說，自己原本還有一個哥哥，但已在6年前病逝，現媽媽只剩她一人能依靠。阿閃婆住院治療期間，因阿麗無法全天守在醫院照料，需聘請看護幫忙，每天就要1900元，阿麗說：「公所撥了3萬元的急難救助，但付半個月的看護費就沒了。」

月靠萬元撐5口家

當地里長林其瑩說，阿麗一家領有低收入戶補助、子女殘障津貼共1萬4600元，加上她每月數千元的微薄收入，但扣除房租與攤租，僅剩近1萬元讓5口家餬口。蘋果基金會已撥款暫紓阿麗一家生活、醫病困窘。
