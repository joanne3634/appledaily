少女工讀養癌嬤癱翁 苦相依

「慶幸有外公外婆疼 他們只有我」

2009年08月21日

19歲的小帆（顏逸帆）將升專五，暑假期間每天下午3時到深夜12時都在便利商店打工，以維持家計，小帆說：「外公、外婆年紀大了，身體又都有病，只有我能依靠。」報導．攝影╱韓旭爾

小帆說，母親未婚生下她後，便到外地工作，從小與外公、外婆相依為命，58歲的外婆2年多前發現罹患舌癌，當時經過手術及電、化療，原以為病況已控制，沒想到今年初又復發，癌細胞擴散至淋巴，小帆的外婆說：「家裡本來就一直為錢發愁，現在自己癌症又復發，真是雪上加霜。」說著說著她嘆氣連連，並看著在一旁的65歲的丈夫金霸伯說，老伴身體情況比自己好不了多少，有心臟病、高血壓，4年前便曾發生中風，現仍行動不便，不時發生癲癇，「6月中旬，他才又因腦梗塞送到醫院急救，在加護病房住了2天才撿回一命。」

兩老擔心孫女累倒

小帆外婆回述，她與17年前去世的丈夫育有1女2子，3個孩子成年後相繼離家生活，12年前她改嫁金霸伯，並未再生子，夫妻兩人原以建築綁鐵零工為業，但近4、5年來夫妻接連罹病，無力再做工，「我們都是靠勞力生活，頂著大太陽，在工地綁鐵、挑磚，工作再苦也能忍，但現在身體病了，真不知道如何是好。」她無奈地說，現在家中只剩外孫女小帆來維持家計，「她要顧學業，又要擔心我們兩個老人的身體，真擔心她也跟著病倒了。」

為籌醫費不怕辛苦

提起小帆的生母，她感嘆，女兒的姻緣路不順，讓小帆跟著我們這兩個老人受苦，她說：「女兒現已另組家庭，生了2個孩子，但日子也不好過。」小帆的生母阿君接受記者採訪說，丈夫有嚴重痛風，又有肝病，根本無法工作，「現在只靠自己當綁鐵零工，每月才賺得約1萬元，要應付一家4口吃穿，雖擔心娘家爸媽和小帆，卻又幫不上忙。」
小帆看著外婆的頸部皮膚因電療而紅腫、焦黑，嘴唇也潰爛、發炎，她憂心忡忡地說，雖然從小家裡苦，但有外婆、外公細心呵護，「跟別人相比，我從來不覺得自己少了什麼，很慶幸能有外婆、外公的疼愛。」小帆說，為了讓他們能好好治病，無論邊上學邊打工有多累，她都會堅持到底。小帆打工的便利商店店長說，小帆在店裡工作已快滿1年，是個工作積極、認真的女孩。
目前小帆祖孫3人領有殘津補助共6000元，光用在兩人看病的交通花費及住院的生活開銷都不夠用，而小帆打工的薪資，每月才8、9千元，蘋果基金會訪視後已先撥款紓困，村長也正協助申請鄉公所急救金。
