壯男罹癌 憾一家緣分短

妻照料斷收入 難養幼子病雙親

2012年12月05日

「爸爸！什麼時候才能回家？」5歲的小甯向躺在病床的爸爸阿義（黃明義）撒嬌，天真地要爸爸趕快出院帶她到海邊玩，37歲的阿義聽了心酸，哽咽地哄騙說：「只要妳乖乖的，上帝就會讓爸爸早點回家。」
報導．攝影╱韓旭爾

阿義說，近1年多常乾咳，但因有抽菸並沒放在心上，但到9月「感覺肺都要咳出來了，受不了才掛急診。」沒想到竟檢查出4期肺腺癌並擴散腦、骨，他說：「想再活幾年，不是怕死，是遺憾一家人的緣分太短。」一旁32歲的妻子阿菱（潘紫菱）安慰說：「醫生說還不到最壞的地步，還能治療，你無論如何要撐著。」

「孩子笑勝止痛針」

阿菱接著說：「阿義一下說喘不過氣，一下說頭痛，吃藥、打針好像都沒效，心裡很急。」夫妻育有7歲、5歲的一對子女小群與小甯，原本阿菱做超商收銀員，阿義做空調安裝工人，雙薪共3萬多元養家。阿菱說，如今她得照料癌弱丈夫，家裡全無收入，「由醫院社工得知蘋果基金會能幫忙，實在不知怎麼辦？才請社工代為打電話求助。」基金會訪視後，已先撥款暫紓窘迫。
9月底阿義發病至今，阿菱在醫院照顧，2個孩子只能先交給公婆照顧。阿義說：「現身體隨時都感到疼痛，但看到孩子的笑比打止痛針還有效，每星期都期待孩子來看我。」阿菱說，阿義看孩子的眼神充滿不捨與不安，「我相信，為了孩子，他會努力活下去。」

母：怕孫沒爸爸陪

阿義的媽媽玉華婆說，她跟老伴都有糖尿病、心臟病，10月中老伴還因蜂窩性組織炎住院1周，兩老原靠阿義養「如今阿義住院要花錢，我們幫不上忙，只能幫忙帶孩子。」玉華婆說，最近小群常作噩夢嚇醒，說爸爸被壞人抓走，哭著要找爸爸，「我看了也難過，很擔心孫子沒爸爸陪伴長大。」阿義的姊姊說，她單親撫養殘障兒子，「生活壓力也大，能出力的部分少。」
小群的導師說，孩子在校乖巧，「學校知道小群爸爸生病，除減免午餐費外，也盡力幫孩子申請獎助金補助。」當地村長說，正協助申辦「馬上關懷」急難金與低收入戶補助，核可後，可減輕部分生活、醫療開銷壓力。
