單親父視茫體衰 獨養2子

患糖尿病後妻離家 懂事兒煮飯

2008年03月21日

屏東縣鄉間一處矮房裡，51歲的鄭進賢吃力起身，卻險些跌個四腳朝天，「以前我一個人扛10斤米都不是問題，現在不行了，可憐兩個孩子以後不知道怎麼辦。」報導．攝影╱孫敏聿

鄭家牆上還掛著鄭進賢與菲籍妻子恩愛的結婚照，不過提起妻子，鄭進賢越顯沉默。「她不想待在家裡，每天都往外跑，3年前只好離婚。」原來鄭進賢與菲籍妻子結褵15年，陸續產下小志、小成兩兄弟，一家四口幸福和樂，生活靠著鄭進賢做水泥工、田裡粗工維持，不料4年前鄭進賢罹患糖尿病導致視力惡化、體力衰弱，「貧賤夫妻百事哀，我也不怪她，只是很對不起孩子，沒給他們完整的家庭。」

「有時氣自己無能」

鄭進賢說，妻子離家2年，獨自在外租屋，僅偶爾回來探望孩子，直到去年他走路開始不穩，無力工作，妻子在外僅靠種花零工日薪700元，難以支付房租與生活，才勉強搬回同住，「但有時一通電話來，她又馬上騎車走了，我和孩子生病，她也不管。」鄭進賢說，有時很氣自己無能，但見到孩子有媽媽在旁照料，快樂許多，也只好忍讓遷就，「不過我前妻除了偶爾買點菜，賺的錢也不會拿回來。」
對鄭進賢而言，眼前的世界一片模糊，因糖尿病影響視力，讓他3年來無力工作，生活就靠低收補助每月3600元。他也想栽培孩子讀書，給孩子最好的生活，無奈年過半百就眼茫、語障、行動不便，沒法賺錢，生活真是無以為繼。

無法工作賺生活費

就讀國一的小志導師表示，小志課業成績中等，但從不遲到早退，平常和國小5年級的弟弟一同負責家裡的打掃清潔和洗衣煮飯，「小志和弟弟乖巧懂事，現面對父親病況嚴重，生活著實令人擔心。」
村長張綿絨說，鄭進賢的兩兄弟在外工作，鮮少往來，無法提供經濟協助，而他平常不菸不酒，生病前勤奮工作，「離婚後鄭進賢很努力，就為了照顧兩個兒子，前妻儘管會回家，也無助於家庭經濟。」村幹事黃建棟指出，鄭進賢2年來已列冊三款低收，就醫免繳健保部分負擔、孩子學費也有減免，但就是無法工作，生活費無著。蘋果基金會獲悉，已先撥款暫紓醫病與生活急難。

無法計算的人生

鄭進賢：「我不辦什麼信用卡、現金卡，也沒有欠人家錢，賺得少就得省，我自己不要緊，問題是孩子可憐，以後怕連累他們得反過來照顧我。」
