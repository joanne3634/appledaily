獨養老小4口 「若我倒了這家怎辦」

2012年11月05日

「想問老天為何是我？！」42歲周淑敏直腸癌復發後，很震驚感傷，單親的她有更大無奈，看著身邊年邁爸媽和仍讀高二的獨生子，深深擔憂湧上心頭。報導．攝影╱向高彬

周淑敏年初動直腸癌手術後，腹部裝人工肛門，每兩天換便袋，她說，有時流汗多或動作大，黏貼便袋鬆脫溢出，很不方便，「8月癌症復發，化療後10月又再次動手術，經濟壓力和心裡煩惱更多。」

沒休養忙工作病發

周淑敏原在鍋貼店工作，她說，去年持續血便快半年，本以為痔瘡，但越拉越嚴重，檢查才知直腸癌三期，做化療及手術，原本尚穩，由於她獨扛一家老小4口房貸生計，顧不得好好休養，她趁體力好時再回店做事，卻在回診發現復發，醫師交代二次術後真要靜養抗癌了，「老爸爸87歲、我家只我一個女兒，我兒子才17歲，如果我倒了，這家怎麼辦？」
周淑敏述往，自己是養女，她66歲養母許金連當年因病切子宮不孕，與曾是老兵的養父周貴福領養她，兩老先前租間2坪大的小攤，賣水餃養大她。

自責用罄老父積蓄

之後淑敏嫁人，生獨子小良，但7年前與前夫離異，她帶小良回娘家，做事照應一家老小，原本生活單純，直到她罹癌病弱，老父在今年中因體力差也收攤，如今老父棺材本已用罄，「很對不起老人家，不能孝順，還讓他們操煩。」
採訪時，兩老談到女兒很擔憂，母親許金連嘆：「很無奈，求老天保佑女兒度過難關」，周貴福憂自己成負擔，現至榮民之家安住，假日回來團聚。他說，就這麼個女兒，從半歲小胖娃養大她，「女兒說下輩子要做我親生女兒，我苦笑，別了，欠妳的，這輩子還就好。」但老人家嘴巴說，眼裡仍是捨不得的心疼。
兒子小良知家境差，為了省800元上學車費，早晚走路1小時到校，走半個月，「後來媽媽發現我出門早、回家晚才知。」

獨子想考軍警貼補

小良又說：「剛好她手機到期，附近手機店辦續約活動，她擔心我這樣走花太多時間沒精神讀書，就續約換贈品單車。」他高中畢業要考軍校或警專，「想早點做事幫家裡。」
周淑敏之前工作的店家李先生說，淑敏兒子乖巧、暑假時也在店裡打工，「我們能幫的都盡量幫，跟她說，好好養病，康復再回來。」目前周家月領榮民就養、老人及單親兒少補助共約1萬9千元，公所已撥急難救助2萬1千元，蘋果基金會訪視後，也撥款暫紓醫病急窘。
