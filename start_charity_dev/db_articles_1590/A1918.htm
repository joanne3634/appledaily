癌末父求神「給多些時間」

盼為愛女跆拳助威 女兒：為父打場精采比賽

2009年07月06日

37歲吳保璋強撐癌末瘦弱的身軀在佛堂前誠心跪求，以道士為業的他，過往總為往生者祝禱一路好走誦經祈福，但他這次卻含淚為自己祈求著：「盼天上的神佛保佑，給我多些時間，好安頓妻女往後的生活。」
報導．攝影╱游竹君

罹患食道癌的吳保璋，4月中住院接受化療、電療1個多月，由於吞嚥困難，體重由55公斤瘦到剩40公斤，連腳步都踉蹌蹣跚，就讀小一的女兒小瑩擔心地陪在爸爸身邊，她說：「爸爸變好瘦，連下樓梯手都要扶牆壁慢慢走，我好擔心。」

苦撐養家延誤就醫

6月初吳保璋出院當天，小六的女兒小儀剛好獲得全縣跆拳道比賽冠軍，一進家門看到女兒遞上的冠軍獎牌，他臉上才顯出安慰驕傲的笑容，彷彿這獎牌能讓他忘卻身體的疼痛，但他卻又憂心能否參與女兒8月中參加全國少年跆拳錦標賽。
吳保璋說，1年半前因嘴巴痛就醫便發現口腔有腫瘤，「那時不敢面對，怕就醫家裡就沒收入，沒想到拖到今年4月，連喝水都會吐，檢查才知道食道全被腫瘤塞住，很後悔沒有早點治療，一切都是命。」
吳保璋與27歲妻子阿娟育有就讀小一及4歲的兩名女兒，並扶養前段婚姻的長女小儀，小儀4年前開始練跆拳道，她說：「有比賽的時候，爸爸總會抽空到現場幫我加油，雖然8月份的全國比賽也希望爸爸能到台北幫我加油，但路途太遠，不知道爸爸的體力夠不夠，我希望為爸爸打一場精采的比賽，別讓他失望。」
謝教練說，小儀爸爸一直全力支持小儀練跆拳道，爸爸罹癌多少也影響小儀練習的心情，但大家都鼓勵小儀把內心擔憂化為努力的動力，「贏得名次，當作送爸爸的禮物。」

全家只靠親戚接濟

阿娟說，丈夫一直努力工作養家，近1年來道士工作量少，丈夫便利用沒工作的日子擺攤賣炭烤，家裡還有年邁的公公和曾患腦瘤的婆婆同住，丈夫從3月住院後，家計只能靠親戚接濟，還得清償遭法院查封車輛後的車貸差額每月5000多元。急難救助承辦黃小姐說，公所已核發「馬上關懷」急難金2.5萬元，並協助申請弱勢兒少補助，村長黃鴻澄也已協助向地方慈善會申請急難救助，蘋果基金會訪視後，已撥款協助暫紓困境。
