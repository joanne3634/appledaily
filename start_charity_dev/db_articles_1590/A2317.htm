全身器官壞一半 漢難扛家

妻腦傷 兩兒仍稚齡 夫感嘆「未來怎辦」

2010年11月12日

9月中，阿賢（彭厚賢）因罹患兩側髖關節壞死，住院接受左側人工關節置換手術，出院回家不到1天，又遇大量吐血急忙返院急救，診斷因肝硬化引發食道靜脈瘤出血，阿賢說：「自己才41歲，怎知全身器官已壞了一半！」報導．攝影╱韓旭爾

阿賢31歲的妻子阿蓉（張苗蓉）一邊幫丈夫擦澡，一邊說，阿賢完成髖關節手術出院時，一到家便開始冒冷汗、臉色蒼白，還不時抱怨肚子脹氣什麼東西也吃不下，「我以為剛開過刀，身體比較虛弱，沒想到晚上時阿賢開始吐血。」阿蓉心有餘悸地說，阿賢連嘔了10多分鐘的紅色鮮血、黑色血塊，「客廳地上都是血，有些血塊就像雞蛋一樣大，我還以為阿賢把肝臟都吐出來，嚇死人了！」一旁的阿賢接著說：「阿蓉當時完全不知所措，連救護車都是我自己打電話叫的。」命撿回來了，「但醫生檢查我有嚴重肝硬化，人工關節手術的傷口恢復情況也不好，將來恐怕三不五時就要來醫院報到了。」

「連上廁所都無力」

阿蓉說，夫妻倆共育有分別10歲、7歲的兩個兒子，她因多年前車禍傷腦，留下右側手腳無力、視力不清等後遺症，還會不時發作偏頭痛，無法工作，家計全靠阿賢不穩定的建築、農務零工維持，一家還能勉強度日。不料去年中開始，阿賢開始出現髖部疼痛，醫生診斷需兩側輪流更換人工髖關節才能恢復行動，「醫生說，阿賢是從事粗重工作，建議更換自費的關節比較耐磨，需要6、7萬，我們負擔不起，只能先用健保治療，能撐多久是多久！」阿賢聽到妻子語氣盡是無奈，他感嘆地說，他很擔心這次自己的病好不了，孩子還這麼小，未來不知該怎辦？「現在自己上廁所都無能為力，真的很沒用。」

孩子撿柴燒水洗澡

訪視當天，阿賢的哥哥正來探訪，他看著阿蓉正準備幫姪子洗澡，心疼地說：「他們住的環境很不好，只能撿柴火燒水洗澡，現在阿賢又生病，我打零工討生活，在金錢也沒能力幫上忙，只能常常來看看、關心一下。」當地里長說，目前阿賢家4口就靠低收、家扶補助共19800元生活。
阿蓉因年輕時車禍造成頭痛後遺症也時常就醫，阿蓉說：「就醫包車一趟就要5、600元，真是吃不消！」蘋果基金會訪視後，已先撥款暫紓窘，醫院社工也提供免費尿布襄助。
