顧癱夫癌兒 嬤：淚流乾了

「萬一兒拼不過這關 怎養大2孫」

2012年11月20日

採訪時，一進病房就見躺在床上的阿忠側身拿著塑膠袋嘔吐，61歲媽媽阿玉婆（陳阿玉）替他拍背舒緩痛苦，「8月阿忠發現罹患多發性骨髓瘤，他才38歲，求菩薩保佑給阿忠一個活命機會。」
報導．攝影╱韓旭爾

阿玉婆說，7月初時阿忠右腿感覺痠麻抽痛，醫院診斷是脊椎骨刺，復健1個多月卻連左腿也出不了力，轉診醫學中心檢查竟是癌症。阿玉婆擔心感嘆，阿忠2年多前離婚，2個孩子才11歲、8歲，一家6口靠他在清潔用品公司當業務送貨賺錢維持，「萬一他拼不過這關，以後我怎麼養孫女、擔起這個家？」
阿玉婆回述，她與丈夫育有2女2子，女兒1個已出嫁、另1個自幼出養，兒子除老三阿忠外，老二阿勝反應較遲鈍又口吃，「做零工賺的錢只夠自己生活。」

「盼病好再撐起家」

阿玉婆說，夫妻年輕時在種香蕉，7年前66歲的老伴龍山伯中風，現四肢癱瘓、略有失智，這些年來全靠阿忠一人，「如今老伴、兒子都病了，好像年紀越大越不順利，我眼淚都流乾了。」
當地里幹事說，目前一家社福補助僅有龍山伯中低老人年金每月7200元，9月時阿忠已領有「馬上關懷」救助金2.5萬元，後續待他殘障證明核發後，再協辦低收補助。醫院呂姓社工說，醫院已協助約3萬元看護費，蘋果基金會訪視後，也撥款紓困。
採訪時，阿忠虛弱地向記者說，被宣判罹患癌症時，這輩子未曾有如此的無力感，「但後來想想，我還年輕，有本錢跟癌症對抗，我這年紀應該是撐起整個家的時候，等穩定些，想回公司上班。」阿忠公司劉姓經理說，阿忠是位值得肯定的好員工，「只要阿忠病況好轉，歡迎他回來工作，會安排體力負擔較輕的內勤職務，他一定要加油。」

孫女貼心照顧阿公

探視阿忠後，記者隨阿玉婆返家，龍山伯癱臥客廳床上，記者問他知不知道阿忠生病，他支吾半分鐘才說：「兒子住院，好久沒看見他。」之後便眼眶跟著紅了。阿忠讀小六的大女兒小庭說，最近阿嬤家裡、醫院兩地跑，看她也快累壞，「我會很乖，自己準時起床上課，也幫忙照顧阿公，不會讓阿嬤再煩心。」
