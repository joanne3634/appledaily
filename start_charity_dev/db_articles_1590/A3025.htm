撐家爸摔斷腿 「怕成累贅」

白飯讓孩子吃 自己日食一碗泡麵

2013年03月25日

45歲的阿藤費力撐起全身，扭動身軀從輪椅移到沙發上，一旁49歲的太太阿真趕忙攙扶。阿藤嘆道，去年底不慎從3樓高摔下，這一摔，除了將他的雙腿摔斷之外，也斷了一家5口的生計。
報導．攝影╱江品璁

阿藤無奈的說，自己國中沒畢業就去學做焊接鐵工，至今已有30年經歷，沒想到去年12月16日，在搭建鐵皮屋時，承載重量的樑柱還沒焊好，自己一時疏忽爬上去卻斷掉，從3樓高摔下。阿真難過的說：「阿藤傷後到現在3個月了，雙腳還是不能走須坐輪椅，生活起居都要我來照顧，醫生說還要再復健3個月，但往好處想，至少沒摔到脊椎或頭部，人還在，就怕雙腳好不了。」

借錢付工人薪水

阿藤回述，10幾年前自己包小工程，生活本還可過，後來上游廠商惡性倒閉，收不到工程款項，只好借錢去付工人的薪水，導致負債百萬元，勞保也停繳多年，我跟太太一起打拼還錢，但這些年3個孩子陸續出生，最大讀小6，最小才5歲，「我沒讀什麼書，孩子生了就要養，但小孩多請不起保母，太太辭去工作在家帶小孩，自己去打零工賺錢，再苦也要撐，但現在跌斷腳，反而成了家中的累贅。」阿藤的姑姑阿惠姨說，阿藤沒有任何保險，這次受傷無法工作，工頭包6000元慰問金也已用於生活了。

「不能苦到孩子」

訪視時，讀小六的長女小湄天真的問：「爸爸什麼時候會好？」阿藤撐起笑容說：「快好了，時間到就會好了。」小湄又問：「那會不會餓呢？」阿藤回應：「我不餓，爸爸喝水就會胖。」父女間一問一答，感情融洽，太太阿真偷偷告訴記者說：「其實阿藤一整天只吃了碗泡麵，家裡的白飯都留給孩子配肉鬆吃。」阿藤嘆道：「只能走一步算一步，但孩子還小，再苦也不能苦到孩子。」
小湄的郭姓班導師說，學校也已幫忙申請校內基金助小湄一家。阿藤大哥阿斌說，弟弟摔傷後有到醫院探訪，也幫忙處理手術、住院事宜，但自己還有2個孩子要養，經濟負擔大，其他2個弟弟也都打零工過活，幫忙有限。當地公所說，已協辦「馬上關懷」急難金2萬5000元，低收補助也正申請中。蘋果基金會訪視後，也撥款暫紓困窘。
