精障妻伴癱夫 5口家難扛

兒女工讀掙錢 「想對爸說不用操心」

2010年04月06日

35歲阿芳倚著先生阿輝（羅慶輝）臉頰，「輝，幹嘛不理我，昨天沒來看你生氣喔？對不起啦。眨眨眼告訴我你聽到了。」中度精障的阿芳，努力喚著中風癱瘓的阿輝，她雖表達有礙，但言語仍流露真切關愛情。
報導．攝影╱張嘉恬

阿芳應答略慢，逐字逐句對記者說，夫妻原本一起做油漆零工持家，但因收入不定，隨著2女1子日漸長大花費增加，自己在6年前因遺傳基因加上經濟壓力罹患嚴重躁鬱症，無法工作，此後家計僅靠阿輝收入及相關補助每月共約3萬元，而阿輝原即有高血壓病史，今年1月腦中風癱瘓，期間阿輝雖然出院，但因阿芳無法勝任照顧工作，使阿輝因背部大面積褥瘡引發敗血症，一度又進加護病房。

工作家事都難打理

阿芳說，17歲長女、16歲長子皆為助家計就讀建教合作職校，但「兩人最多每月協助共1萬元，要全家5口生活很辛苦。」走進阿芳家，一陣潮濕摻雜霉味的氣味撲鼻而來，客廳、廚房都顯凌亂，長子阿立說：「媽媽的病，讓她沒辦法工作之外，連家事也無力打理，我和妹妹只能在讀書之餘，盡力幫忙。」
阿立還說，因為「爸覺得男孩子要挑大樑，平時管我比姊姊和妹妹還嚴，所以常和他吵架，現在他病倒，看他躺在床上沒意識，才覺得自己很不該。」阿立低頭沉思半晌後，語帶盼望說：「想對爸說我不會再讓他操心了，只希望他有機會能聽到。」

安置仍須負擔2萬

與阿輝共事達20年的范姓老闆說，阿輝工作認真盡責，「常聽他說孩子多、家計重，希望多加班，同事收工要聚餐他也很少參加，說要省錢。沒想到他忽然病倒，妻子又有精神障礙不可能工作，生活一下子陷入困境。」
阿輝雖尚有姊妹與兄長，但哥哥靠清潔工收入養育就學中的2子，3姊妹也都各有沉重家計，無力協助。大姊阿珠說：「姊妹們能盡力的是提供些物資，長期的金錢支援真的沒辦法。」
當地賴姓里長表示，阿輝病後已協辦「馬上關懷」2萬元並核發殘障手冊，但因阿芳無法承擔照顧之責，阿輝日後得安置在養護中心，但扣除政府補助之外，阿輝每月仍得自付約2萬元，蘋果基金會訪視後已撥款暫紓困境。
