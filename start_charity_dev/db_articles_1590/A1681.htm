嫗顧腦傷兒 盼甦醒喚「媽媽」

每月1.8萬照護費 拖累兩老淚洗臉

2008年09月05日

午後突來雷陣雨，69歲林伍貴枝騎車在路上，雨水把她淋得濕透，但顧不了濕衣鞋褲便進呼吸照護病房，探望昏迷兒子阿民時仍先為他擦額頭細汗，「我每天騎車來看他，禱告求上帝讓他醒過來，再叫我『媽媽』。」報導．攝影╱向高彬

林伍貴枝指著阿民的隔壁床病友都是70歲老人家，而阿民不過49歲，病況卻比他們嚴重，接上呼吸器，雙眼微睜，卻像失了魂魄毫無反應，林伍貴枝說，阿民四月中酒後不慎跌倒撞腦送醫急救，還經6次開腦手術，雖挽回一條命，卻成了如今模樣，往後終生靠呼吸器維生，每月需18000元照護費。

每月4千房租付不出

林伍貴枝無奈看著阿民，她說，想接阿民回家照顧，但病況太嚴重，「我照顧不來，現在照護費要到處想辦法。」她也坦言，「是愛喝酒害了他，怎麼勸也不聽，把家搞得破碎，拖累我和老伴，真是快要用眼淚洗臉。」
林伍貴枝說，長輩沒留田產，她和現年76歲老伴早年忙幫人砍竹才能攢錢溫飽三餐，她感嘆家境差，沒能給3子1女好好栽培讀書，以致孩子沒什麼專長，長大後只用勞力謀生，二兒子也因年輕時曾摔傷手殘，平常工作不定，小兒子單親育女，女兒遠嫁屏東，都僅能養活自己，無力幫助兩老太多。
原做砍草零工的大兒子阿民因愛喝酒及個性問題，與媳婦阿滿已分居7年，阿滿帶著三個子女在外靠揀荖葉零工租屋，「不常聯絡，但我知道媳婦很辛苦做工，才能養活孩子。」
只是現讓林伍貴枝頭疼的是，為籌措阿民照護費，家裡每月4千元房租也拖欠了好久。她回述，兩年多前為償還先前生活不足借債50萬元，便以老家土地請人代辦借貸，卻遭代辦人詐騙轉賣土地房子，之後不見人影，也沒了屋地，如今兩老租住鄉間便宜房屋，債務現由二兒子每月償還約三千元，「拼了一輩子，竟然老了卻連家都沒得住，還為兒子的病這樣煩惱，真的很無奈。」

鄉公所協辦殘障鑑定

目前醫院社工正介入關懷協助，而兩老每月領補助共9千元，當地鄉公所社會課急難低收承辦許小姐說，知道林伍貴枝窘況，也盡力協辦公部門與民間急難救助，阿民殘障鑑定申辦中，若通過每月可領4千元補助，蘋果基金會訪視後，也撥款暫紓林家生活苦窘。
