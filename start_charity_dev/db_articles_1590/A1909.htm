癌父臥病榻 掛心兩稚兒

斷電又欠房租 19歲女打工難扛家計

2009年06月25日

48歲江英南（阿南）罹患胃腺癌且已轉移肝臟及淋巴，因會吐血、身形瘦弱的他，為避免感染躺在隔離病房裡，他面容枯槁地與妻子阿素無言對望著，正為斷炊的生計及積欠已久的房租發愁。報導．攝影╱張嘉恬

傍晚時分，醫院單人隔離病房的大塊方窗透著和煦的陽光，病重的阿南卻因體力孱弱、身體發冷，蓋著厚重的毛毯，阿南43歲的妻子阿素帶著7歲的么兒小年，準備在病房過夜陪阿南，「兩個大女兒都嫁人，景氣不好她們自身都難保，家裡因積欠電費已被斷電，屋內沒光線，只好把小年帶在身邊自己看著。」

為省錢少換尿布

阿南和阿素共育有3女2男，全家生活仰賴阿南以綁鐵、模板零工收入支撐，阿南早年便有偶發性胃痛，但為龐大家計只好吞胃散忍痛上工，直至去年8月胃痛難耐才就醫診斷出胃腺癌，因已轉移至肝臟及淋巴，為避免又再擴大轉移不適合手術，目前以化療醫治。
阿南長女已遠嫁他鄉多年，已婚次女一家5口原本還租屋同住，但去年景氣轉壞後，也以模板維生的二女婿因無工可做，無力分攤房租，只好在3月底搬離。二女婿阿雄無奈說道，大環境景氣差，「做臨時工的都苦哈哈，我們連房租都付不出來，現在也只能常回去探望爸媽，希望景氣轉好上工日多一些，才能協助爸媽。」
採訪時，阿素邊替阿南換尿布邊說道，阿南因接受化療體力相當孱弱，得補充流質營養品，「也是喝下去就吐，現在光靠19歲三女兒餐飲店打工收入每月約1萬元，扣掉阿南每月尿布費3000元，根本不夠基本開銷，最近為省錢，沒有常換尿布，他還因為尿道感染白血球數值一度飆高。」

里長協辦急難金

阿素傷感地說：「最近他痛到不行時，一直問我還可以活多久，真教我不知道該怎麼回答他。」阿南躺在病榻上平靜地說：「我其實不在乎自己的病，看著家沒有人可以撐起來才著急，最掛心的是年紀還小的兩個兒子。」當地里長魏君龍表示，去年10月協辦「馬上關懷」急難金1萬元，目前也已向內政部申請急難 救助金，蘋果基金會訪視後已先撥款紓困。
