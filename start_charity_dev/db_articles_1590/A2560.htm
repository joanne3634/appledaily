勇婦不放棄 忍淚顧癱夫

長子貼心 力拼學業盼助家計

2011年09月06日

43歲的胡勇福癱臥病床，雙眼無神凝視天花板，38歲的妻子林靜玉在旁幫忙活動手腳、做復健，一邊說：「丈夫今年6月初突然癱瘓病變，不認得人、聽話也沒反應，病倒2個月來沒說超過10個字，但只要他能進步，我不會放棄。」報導．攝影╱仲芝蓉

林靜玉說，丈夫6年前得鼻咽癌，化療、放射線療半年多後病情穩定，每年定期回診也沒有復發跡象，怎知今年6月初卻發現罹患急性神經病變，這次反而讓他倒下去。

「我知道他想變好」

林靜玉說，丈夫四肢癱瘓、腦筋也不清楚了，「幸好復健2個多月後，右半身漸恢復知覺，雖然跟他講話好像沒反應，但有次我在床邊跟人聊天，他突然伸手搔我癢，好像想跟著說笑，我知道他努力想要變好。」林靜玉繼續說，剛發病時，丈夫只說手腳沒力，沒想到隔天早上就全身癱軟，醫生說這種病起因不明，只能努力復健。
林靜玉接著說，和丈夫育有現年分別18、12歲的1子1女，丈夫6年前罹癌時，子女才要上國中和小學，他為康復撐家，不再碰菸酒檳榔。
18歲的長子阿龍說，爸爸罹癌後常說，大難不死必有後福，他撐著工作，堅持要自己和妹妹念書，「這次，爸爸倒下，媽媽雖然每天有說有笑地逗著爸爸做復健，不在我和妹妹面前掉淚，但我知道她很擔心，我會努力完成學業，早日幫忙扛家計」。

低收取消卻遇困境

當地派出所副所長殷中原說，胡勇福的子女因被親戚報稅撫養，今年低收入戶補助取消，卻遇上罹病癱瘓，公所已申請「馬上關懷」急難金3萬元協助，「我們再幫忙轉介至蘋果基金會，他們訪視後已有撥款協助」。當地家婦中心社工余小姐說，胡勇福74歲的母親古玉花曾獲模範母親表揚，她丈夫30幾歲就意外過世，古玉花阿嬤靠著農雜零工，拉拔9個孩子長大，但子女相繼過世，僅剩3子在外地打零工。

暖流見證 兒女積極 畢業拿獎

林靜玉女兒小悅的導師說，小悅和哥哥兩人今年剛好要上國中和大學，沒想到父親再次病倒，雖然家境狀況不好，但兄妹倆學習態度積極，小悅今年還得縣長獎畢業，「雖低收暫被取消，他們仍可以原住民身分減免學雜開銷，學校也將申請學產基金協助」。
