小六女求援：給聾啞爸工作

窮到租不起房子 3口家困居無電空屋

2008年07月20日

「爸媽不會說話，爸爸又沒工作，房東嫌我們窮都不租房子，現在借住空屋沒電，晚上只好點蠟燭。」小蓉剛小學畢業，語氣早熟惹人憐，她代替爸媽對外發言，哽訴殘障弱勢家庭心酸悲歌，只盼外界給他聾啞爸爸一份工作，租到便宜又有電的房子。報導．攝影╱向高彬

傍晚記者來到小蓉一家三口借住郊區荒廢空屋，屋裡雖有電視、冰箱，卻因無電不能用，小蓉的41歲爸爸陳譁方趁天色將暗煮晚餐，小蓉以手勢與爸爸溝通，她爸媽都是瘖啞者，41歲媽媽白翠錦還有輕度智障，三人每月靠低收補助1萬9800元過活。

全家差點流落街頭

小蓉說，過沒電生活已2個月，「本來借住外婆家，但舅舅在外欠債搬回，我們只好另外找房子，可是市區房子太貴、租不起，郊區房子的房東又覺得我們會付不出房租而不租。」三人差點流落街頭，幸經友人協助暫住空屋棲身，但這房子因建照問題無電多年，晚上只能靠蠟燭點燃亮光。她難過爸媽聾啞：「爸爸本來有工作，可是老闆希望他做夜班，但他有糖尿病、心臟病，不能太累就被辭掉，從去年中失業到現在已一年了。」
小蓉去年底看到新聞報導社福單位，還鼓起勇氣打電話，「有次打去一個單位，卻是安置智能障礙的，沒法幫忙；也有社工說會幫忙，但仍沒好消息。」
小蓉說，家裡一天花150元買菜，煮午晚兩餐，晚餐剩下菜肉，隔天早餐混煮鹹稀飯，「吃得飽就好。」陳譁方咿啊表示，只盼有份清潔工能養家。蘋果基金訪視後，先撥款暫時紓困，並以「克蘭詩．蘋果助學專戶」協助小蓉完成國中學業。

身障就業屢遭排擠

小蓉的外婆碧霞婆說，她也苦愁無力幫忙。當地村長李明忠說，多次幫陳家找工作及住屋，但「景氣歹，正常人都難找工作，好不容易有工廠想用殘障朋友，但一聽到聾啞，就怕溝通麻煩而不用。」東勢愛心家園就業輔導組長陳美芸說，景氣差，身障者就業被排擠情況愈嚴重，去年10家公司還有3家願提供身障者工作，但今年只剩1家，「給一個機會，他們能靠自己力量活得更好。」

後記
女童盼撐起這個家

採訪時，早熟的小蓉侃侃而談，她說，是因為從小先跟著奶奶住才順利發展語言。也因為她可以正常說話、表達，小小年紀便要擔任爸媽對外的發言人，但過程卻也比一般人來得辛苦。
像是最近要找房子常打電話，但對方只要聽到是小孩子，就以為是惡作劇不理會，即便見了面發現是一家殘弱也不想租，讓小蓉期待快快長大，好真正撐起這個家。
