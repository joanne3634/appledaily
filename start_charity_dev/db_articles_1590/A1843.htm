癌夫不敵死神 婦苦顧憨子

居陋屋又扛債 盼趕快有工可做

2009年04月05日

「爸爸，你要加油！跟你說，我們兒子今天也有乖乖做治療喔！」數日前，56歲柯清蘭這樣喚著病床上因口腔癌末期昏迷的先生阿時，不料，3月31日早晨，阿時還是不敵病魔去世，預訂4月7日出殯。報導 攝影╱向高彬

像顆陀螺打轉的柯清蘭這陣子以來其實已快崩潰，因為她在照顧病夫的同時，仍要憂心精障獨子阿文。柯清蘭說，29歲的阿文精神分裂情緒極不穩，「從療養院請假回來給他父親守靈，但他一直大聲咆哮，想到唯一的孩子，也不能好好送他爸爸一程，實在很心酸！」

撐重擔心力交瘁

柯清蘭哭腫了雙眼，她說，與先生婚後僅生一子阿文，但阿文13年前不明原因發病，出現妄想幻聽病況，而先生自從20年前經營小五金批發失敗後，變得意志消沉愛貪杯，僅偶做油漆零工，多年來家計重擔全落她肩上，先前她曾做打掃清潔工及保母勉撐家用，但先生去年6月發現口腔潰瘍疼痛，一檢查已口腔癌三期，且很快惡化成末期，同時阿文病況加劇，數度有暴力傾向，父子同時住院，讓柯清蘭心力交瘁。
記者初次到醫院探訪柯清蘭時，看到病床上的阿時已氣切無法言語，右頰一塊約10公分面積補皮手術痕，偶爾睜眼但眼神渙散，柯清蘭感嘆心腸太軟也苦了自己，「他都說這輩子運途歹，以前我沒離開他，是想夫妻一場，我不放棄他，如今又看他生病痛苦，我想，應該陪他走完最後一段路。」
之後，記者也與柯清蘭來到阿文療養的院所，他在醫護人員戒護下與媽媽相見，阿文坐在椅上焦躁難安，語無倫次嚷著說想出去，「我出去就會賺大錢，但是我脾氣壞，會跟老闆吵架。」讓柯清蘭只能順著阿文的話點頭。

菜湯配飯過一餐

目前柯清蘭一家租住約15坪大簡陋木造老屋，一塊門板玻璃在兒子住院前才被他發怒敲破，她難過地表示，家中有精障病患，不僅常被房東趕，平常也不敢跟鄰居有接觸。
夫兒皆重病，柯清蘭至今已跟親友借約27萬元過活，採訪時，見她廚房爐上僅一鍋白菜湯，她說是黃昏市場撿便宜一顆10元買的，滷一滷有鹹菜湯就能配白飯吃，辦妥喪事後，希望能趕快有工可做。
目前醫院社工已介入關懷，當地吳姓里幹事也說，去年底已協辦急難金3萬6000元，目前每月領殘津7000元，後續再協辦低收資格。蘋果基金會訪視後，也先撥款暫紓柯清蘭母子生活急窘。
