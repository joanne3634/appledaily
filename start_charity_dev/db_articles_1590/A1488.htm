夫成漸凍人 妻不捨苦撐

吃飯要人餵 嘆：不是一般人能體會

2008年01月13日

「先是手腳不能動，現在說話變得困難，感覺像漸漸枯萎的樹木，真的很可怕！」41歲的許明旭吃力地述說著，罹患肌萎縮性側索硬化症候群（俗稱漸凍人）的恐懼，發病前的他，是個靠勞力掙錢養家的汽車修護技工，體型壯碩，如今卻連幾句話都說得吞吐。報導．攝影╱韓旭爾

「四年前，突然覺得雙手無力，原以為只是工作太操勞傷了筋骨，才會連汽車輪胎都搬不動，跑遍屏東地區醫院都找不出病因。」許明旭說，一開始診斷認為是中風或是脊椎問題，三年前才確定罹患俗稱「漸凍人」這種罕見、無法醫治的運動神經元疾病。

「我會撐到撐不下去」

許明旭說：「發病前，在修車廠有份穩定工作，收入雖不多，但老婆和三個孩子吃、穿沒有問題，沒想到現在成了漸凍人，吃飯都要人餵，很悲哀。」目前41歲的妻子阿麗，不但要照顧三個分別年僅九歲、八歲與四歲的稚子，還需全天照料許明旭生活起居。
阿麗說：「一年多前他還可以緩步行走，現在他需要輪椅幫助行動，再下去會終日臥床，不能說話和進食，呼吸衰竭時就要氣切，到時我不知是否應付得來，壓力好大。」阿麗表示，她與許明旭溝通過，屆時如果無法照顧他，就在住家附近找養護機構接受公費安置，「因具有低收入戶身分，費用不多，我也能找份工作，好好撫養三個孩子長大，可是一想到要讓他獨自面對死亡，實在捨不得，我會撐到撐不下去。」
許明旭說：「起初實在很難接受自己得了這種病，蚊子叮了，很想抓癢，但手卻不能動，這種感覺不是一般人能體會，現在我認命接受了，我只是放心不下家人，尤其三個孩子年紀還這麼小，阿麗將來的負擔一定很大。」許明旭想到自己可能無法撐到子女長大，不禁愁眉深鎖，心裡有說不出的牽掛。

「迫己接受離開事實」

許明旭無奈地說，最後就算我只剩下眼睛能動，還是希望能陪在家人身邊，「可是考量現實環境，如果有一天我必須離開他們，我會強迫自己接受。」
現在許明旭一家無人可工作，只靠每月共約二萬元的政府低收、殘障與家扶補助度日，扣除四千元房租，常需依靠地方慈善會協助，阿麗說：「娘家會拿一些米和蔬菜給我們，省下不少錢。」蘋果基金會訪視後，已撥款暫紓許明旭一家困窘。
漸凍人協會秘書長林金梅指出，國內約800人患有此疾。漸凍人雖全身無法動彈，但記憶及思考與感覺神經能力是不受影響，因此，病患心理關愛特別重要，「漸凍人接受氣切後，約有十多年的存活時間，這是一個長期抗戰。」

無法計算的人生

許明旭說：「知道自己變成漸凍人時，腦中只想著：為什麼是我？但是，人真的不能不認命，我不知道明天會變成什麼樣子，只想未來能安寧的走。」
