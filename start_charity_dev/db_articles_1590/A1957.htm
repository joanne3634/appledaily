癌夫瞞病養家 延誤就醫

「只要撐到兩兒畢業 一生無憾」

2009年08月20日

「丈夫就是為了家硬撐，口腔出現了硬塊也不講，一直拖，現在才會這麼嚴重。」41歲的陳春雲守在病床邊陪伴口腔癌3期的丈夫阿鴻，眼神充滿不捨，她說：「希望一切不會太遲。」報導．攝影╱韓旭爾

陳春雲說，她與45歲的阿鴻育有2子，今年暑假後分別升高三與高一，自18年前兩人結婚，靠著經營帆布行維持家計，「原本生意還可以，一家吃穿還能勉強維持。」

妻打零工貼補家用

陳春雲說，但近5、6年受不景氣影響，家裡帆布行生意一落千丈，阿鴻每天四處打探哪裡有工程可做，還得忙著籌錢，她說：「剛收到的工程款，馬上就用來清償帆布材料費，錢總是左手進、右手出，還得向親友周轉，家裡的經濟缺口愈來愈大。」為增加收入，近2年來，陳春雲到鋁工廠打零工，每天掙得500多元貼補家用。
陳春雲感嘆，阿鴻早在2年前口腔內出現癌症徵兆，但他始終瞞著家人，記者詢問阿鴻當時的想法，他因剛切除口腔內組織，喉頭插著氣切管，無法言語，只好拿起小白板寫下：「景氣不好，沒錢。」陳春雲說，今年6月，阿鴻嘴裡的腫瘤已長到3公分，嚴重影響咀嚼，經再三追問，阿鴻才吞吞吐吐地說自己可能罹癌。之後陳春雲要阿鴻就醫，阿鴻回應：「小兒子要上高中了，要是住院開刀，一拖就是一年半載，家裡沒人賺錢不行，只要撐到兩個兒子高中畢業，一生就沒有遺憾了。」經全家勸說，阿鴻才就醫。
陳春雲說：「家裡每天3餐最多2道菜，偶爾多花50元買條吳郭魚加菜，孩子的衣服也都是破了再補……」說著、說著陳春雲不禁哽咽，「就是日子苦，阿鴻才會不管自己身體病痛，咬牙苦熬。」

沒有任何社福補助

目前陳春雲一家無任何社福補助，面對醫療、房租與一家生活開銷，「這筆錢不知道要怎麼籌？」陳春雲說，兄弟姊妹偶爾拿2000、3000元幫忙可以，再多也沒辦法。長子小翔現於泡沫紅茶店打工，但每月只賺約3000元，么兒小新說：「將來我會跟哥哥一起去打工，希望爸爸安心治病。」蘋果基金會訪視後已先撥款紓困，當地村幹事也協助送件申請「馬上關懷」急難救助金。

後記
吵架離婚 妻自責失職

陳春雲自責地說自己是失職的妻子，因為近2年，帆布行生意無法維持一家開銷，她勸阿鴻改行，夫妻常為此吵架，去年2月一時氣憤說：「乾脆離婚算了。」阿鴻竟回：「反正人生也沒有什麼希望了，離婚也好。」結果辦了離婚。她帶大兒子北上租屋，離家半年多，因放不下小兒子，去年8月又回家同住。陳春雲說，現在才知當時阿鴻已無體力工作，更別說轉行了，一切都怪自己太不了解他！
