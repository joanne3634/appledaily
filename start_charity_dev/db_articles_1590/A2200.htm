單親媽拾荒兼擺攤 養11人

還要顧癱姊一家5口「借錢借到不好意思」

2010年06月27日

天色灰濛之際，阿梅起床後，淘米煮了一鍋稀飯，將幾個罐頭醬菜放在桌上，給孩子當早餐，便跑山間溪溝旁釣青蛙，阿梅說，趁著昨晚剛下過大雨，青蛙都會在水溝旁停棲，「釣些拿到市場賣，賺得2、3百元就算是上天給的獎金。」報導．攝影╱韓旭爾

現年39歲的阿梅是單親媽媽，3年前因不堪丈夫長期家暴經法院訴請離婚，所生4名子女中，大兒子去年退伍後因覓職困難，現準備復學就讀高職，最小的女兒今年才讀幼稚園大班，阿梅說，為了生活和孩子教育費，她每天上午騎車跑遍大街小巷，沿路撿資源回收，下午到漁港碼頭批些魚、蝦、貝類到黃昏市場擺攤販賣，直到晚上7、8點回家。

外甥心懷感恩

阿梅44歲的大姊阿春就住阿梅一家隔壁，同樣是家暴失婚的單親媽，育有現分別就讀大學、高中及國中的4個兒子，阿春說，自己在10多年前被丈夫毆打造成雙耳耳聾，10年前離婚後，家庭開銷便靠阿梅幫忙，1年多前，她又發生中風，右側肢體無力，經過復健，現在才能跛足行走，阿春說：「阿梅自己的日子已辛苦，但她對我們一家5口還是盡心盡力的照顧。」阿春的20歲長子阿杰說，阿姨從小就把他們4兄弟當成親生兒子一樣疼愛，「自己學的是食品營養，將來會努力當上營養師，回報阿姨的照顧。」
當地的村長說，阿梅家中還有一個63歲的老母親，雙腿飽受退化性關節炎所苦，也因擔憂兩個女兒婚姻失敗、往後生計，罹患憂鬱症，不時就醫。阿梅苦笑說，自己現在一無所有，還好身體還算健康，「我手腳健全，還能做事，可是現實很殘酷，每天從早忙到晚，每月能賺到的錢還不到2萬元，要養活11口人，每月總要籌錢應急，借到都不好意思了。」

女兒曾想棄學

在阿梅攤位附近擺攤賣菜的阿玉說，阿梅一直咬牙硬撐，「加上現在生意不好做，每隔幾個禮拜阿梅就會開口向我借個幾千元周轉，有借有還，我很願意幫她。」蘋果基金會訪視後已先撥款紓困。
阿梅說，排行老二的小月今年就要高中畢業，靠著舉重的體育專長獲得國立大學的入學資格，但她擔心就學開銷原打算放棄升學，後經師長勸說，小月仍將往求學路上精進，小月的舉重教練說，小月不只體育成績好，功課也一直維持在班上的前3名，是個文武全才的好學生，如果能順利升學，畢業後可朝學校體育老師的教職努力，就能改善家裡經濟。
