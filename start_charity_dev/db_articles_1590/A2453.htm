夫癌妻打氣「做工養你們」

「多活10年 和我陪3稚兒長大」

2011年04月28日

38歲的陳厚昇戴毛帽、口罩，採訪時，他一張口，記者見口內滿布白色肉芽組織，像月球表面小丘蔓延佔領上顎及舌後，「很痛，沒法吞嚥，1個月瘦了15公斤，只能慢慢仰躺喝營養品，一喝快，就從鼻孔嗆流出來。」報導．攝影╱向高彬

現年41歲的妻子阿蘭說，丈夫去年底發現上顎潰瘍，遲遲不癒，今年1月到大醫院檢查，竟罹軟顎癌四期，屬口腔癌，「我眼淚直掉，不知道怎麼跟在門外等報告的丈夫開口。」阿蘭又說，做鐵工的丈夫雖沾檳榔菸酒，但少量，醫師說體質問題也會致病。

每日營養品須300元

目前陳厚昇正進行化療及放射療，阿蘭說，丈夫每日營養品300元，當地公所已撥「馬上關懷」急難金1萬元，剛通過每月領三款低收補助共6600元，但她擔憂，原本一家5口僅賴丈夫做工養家，如今丈夫病重，又有之前生活不足卡債20萬元，她得照料病弱丈夫及3個年幼孩子，一時生活成難題。蘋果基金會訪視後，也撥款暫紓窘迫。
阿蘭回述，自己曾有兩段離異婚姻，與曾任廚師的丈夫在餐館工作時相識，婚後生有現各8、5、4歲的2女1子。為貼補家用，平日丈夫做工，假日替換由丈夫顧孩子，她出門做清潔零工，但如今生活全亂。

小孩懂事拿藥捶背

阿蘭哽咽述跟丈夫說，無論如何要拼，「就算以後你沒力做事，但多活5、10年，我做工養你們，只要你和我陪孩子長大。」採訪時，讀小一的大女兒小芷幫爸爸拿藥，小兒子在旁說：「我會幫爸爸捶背。」

兄長做工無力接濟

陳厚昇的大哥說，雙親自小離異，各有家庭，弟弟從小由奶奶撫養長大，在弟病後他常去探望，但本身做工無力多幫。
陳厚昇大女兒的黃姓班導師說，將協辦餐費補助和仁愛基金，「家長對孩子課業很關心，之前都是孩子爸爸接送上下課；知道孩子爸爸生病，我們也會持續關心孩子家裡的情況。」

無法計算的人生
「生命剩半年 像另類刑期」

陳厚昇說：「以前生活不完整，曾很叛逆，甚至當兵時，胡塗跟著朋友做錯事，用漫長刑期悔改。後來有太太、孩子，是他們改變我，做個好爸爸。現在醫師說我或許剩半年，像另一種刑期，就算時間剩不多，也會一直愛他們。」
