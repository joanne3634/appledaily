夫躁鬱么兒病 勇婦扛家

老幼病殘全靠她 「做到做不下去為止」

2010年12月15日

「她很拚命喔！」當地村長談41歲陸籍配偶肖俊蘭，讚她吃苦耐勞。肖俊蘭與丈夫共育3子女，丈夫飽受躁鬱症所苦，同住的公公失智、行動不便，3歲么兒感染肺外結核，不時住院醫治，6口家中5人老幼病殘，生計全靠她獨撐，「就做到做不下去為止吧！」肖俊蘭（以下簡稱肖）與記者（以下簡稱記）訪談當前困難：肖俊蘭口述 記者韓旭爾採訪整理

肖俊蘭（左1）說，每到花生收成季節，她會到田地撿拾農人採收剩下的花生，2個女兒也會幫忙整理。圖左後為肖俊蘭3歲病子。

公公罹病又失智

記：談一下公公和兒子的病況？
肖：公公去年6月中不小心跌倒，送醫後發現公公除了外傷還感染肺結核，隔離治療後出院，開始出現失智情況，需要人照料。更慘的是，今年5月，3歲小兒子小翰脖子出現多顆硬塊，醫生說是淋巴感染結核菌，已動過3次手術，頸部淋巴結核還是一直長，現只能先吃藥控制，要是脖子淋巴結核長得太大或化膿時就要再開刀摘除，而且小翰也有發展遲緩情況，3歲了，只會叫媽媽、爸爸，社工跟我說想幫忙安排他接受早療，但肺外結核還沒好，過一陣子再打算。

拚命上工掙薄薪

記：談談多年來生活情況？
肖：9年前透過媒人介紹嫁來台灣，相親時媒人說，現年58歲的老公阿金在工地當綁鐵工人，每天能掙得2000元，我當時在大陸工廠上班，每月才賺到約6000元新台幣，以為依阿金的收入，應能輕鬆過日子，沒想到嫁來台灣後，才發現阿金有嚴重躁鬱症，不是大吵大鬧，就是整天躲房裡睡覺，平時要吃藥控制，每月上工時間不到1周，當時想著，既然嫁了就認命，幫鴨寮清鴨糞、幫人噴農藥，每天就掙個600元養活3孩子，並照顧86歲公公，這幾年來日子雖苦，但至少三餐溫飽。沒想到公公和兒子出了狀況。

盼子康復有出息

記：未來日子有何打算？
肖：現在家裡每月有公公老人年金和家扶中心補助共8100元，村長鄰居很幫忙，農田缺人手時，都會叫我去，以前每月大概能賺1萬元，只是兒子常看病、住院，工作時間變少，丈夫其他兄弟姊妹都去世了，我們沒什麼可依靠。幸好，蘋果基金會對我們伸出援手，外配中心社工也幫忙申請補助，接下來安排我先生去做精神鑑定，以便證明無工作能力，好申請低收補助。只是社工說，這還需幾個月的時間。我吃苦沒關係，只要孩子能好、有出息，我做牛做馬都值得。
