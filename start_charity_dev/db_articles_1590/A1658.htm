老伴中風 兒精障 癌嬤累垮

月靠1.4萬苦撐 「不知我們誰先走」

2008年08月12日

醫院外暑氣炙熱得快把馬路柏油熱融，67歲簡玉的心情也被眼前難題蒸得翻滾，她患乳癌4年，如今80歲老伴第7度中風意識不清，精障的二兒子也置換髖關節，父子同時住院，她拖著病體奔波照料，怕要累垮。
報導．攝影╱向高彬

簡玉奔波醫院陪伴老伴已1個月，老伴年事高，多次中風病體虛弱，近月來持續發燒感染，未料她精障二兒子阿賢也髖關節壞死而進行置換手術，讓簡玉疲於奔命，還好這段時間正放暑假，阿賢的14歲兒子小邦貼心，常趁輔導課餘陪奶奶照料爸爸、爺爺，「不過我身體不好、孫子小，他們兩個病人住不同樓層、且神志不清，我們抱不動，還是得請看護幫忙。」

「已借到沒得借」

目前簡玉雖另有1子3女盡力籌措看護費，但她的長子阿中說，自己與妻子在工廠做工，收入微薄，這幾年為幫爸媽、弟弟醫病和自家過活，入不敷出。
簡玉說，二兒子阿賢8年前離婚失業後，終日鬱悶酗酒，喝到精神出狀況，常覺得有人要害他，「勸也勸不聽，身體弄壞了，肝昏迷住院好幾次，去年底休克來不及急救，腦缺氧造成手腳行動不便，智能退化像個小孩。」加上還要照料老伴和孫子，簡玉形容像在照顧3個孩子。
簡玉4年前接受乳癌二期治療，曾硬撐病體想找洗碗工做，卻沒老闆想僱用她，而眼前更讓簡玉頭疼的是，現今孫子雖有世展會補助學費，但每月家用僅靠老伴的榮民就養金和殘津共約2萬3千元過活，扣掉房貸後僅剩1萬4千元苦熬，「兩父子不時住院，已借到沒得借。」簡玉看著小邦說，只盼孫子順利成人，「一家有1個人生重病已很糟，沒想到我得了乳癌，老伴和兒子也病成這樣，還不知道我們3個誰先走，最擔心若我先走，他們3個誰來照顧，我很煩，卻又不能不想，老了還這樣苦，真悲哀。」

低收補助未核准

病床上阿賢神情呆滯，問他名字如何寫，也說忘了。一旁孩子小邦說，覺得奶奶好辛苦，「以前我叫爸爸別喝了，他還罵我；現在他變這樣，我很氣，但他還是我爸爸，希望醫生能醫好他。」
當地里長陳茂松說，因余水宣領就養金及另有子女，低收一時難核過，已申辦急難救助，「老媽媽得癌症，還要照顧一家病弱，看了真覺得可憐。」醫院社工正評估補助相關醫費，蘋果基金會獲悉後，也撥款暫紓余家醫病生活困窘。
