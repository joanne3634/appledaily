半年喪2子 翁拖老命養孫

每逢年初一是忌日 「一家怎麼歡喜過年」

2012年03月04日

靈堂前，71歲的銀達伯（郭銀達）看著才8歲的孫子小揚跪在地上、拈香向爸爸祭拜，他在一旁哽咽地要孫子向爸爸說，要爸爸在天之靈保佑他好好讀書、平安長大。銀達伯不禁老淚縱橫、放聲大哭，再也說不出一句話來。報導．攝影╱韓旭爾

「以後大年初一就是兒子阿和的忌日，他才46歲就走了，一家要怎麼歡喜過年？」望著阿和的棺木緩緩送進熊熊烈火焚化爐，銀達伯表情哀戚地說，阿和是單親爸爸，靠零工撫養獨子小揚，他有高血壓病史，除夕夜全家團圓到晚上10時多，年初一當天，以為阿和睡到中午還沒起床，不料他是因中風在半夜猝逝。

花光兩老棺材本

才讀小一的小揚說：「那天我叫爸爸半天都叫不起來，搖他也沒反應，用手摸摸爸爸的鼻子，發現他沒呼吸了。」銀達伯說，雖急忙叫了救護車，但救護人員一到家就說，阿和已往生，他硬要求將阿和送醫院，結果還是回天乏術。
自從阿和去世後，銀達伯62歲的老伴秀鳳婆每天哭紅雙眼，秀鳳婆說，她生了2子1女，半年前小兒子阿榮才因意外去世，沒想到大兒子也命薄。當地里長說，半年多前，阿榮跌倒傷腦去世，因檢察官對死因疑慮，解剖驗屍讓阿榮喪禮延宕，冰櫃費用大增，喪禮花費約30萬元，把銀達伯兩老的棺材本全用完了。如今又要處理阿和喪禮，兩老手足無措，他已協助向慈善會申請約3萬元補助，並轉介蘋果慈善基金會協力，基金會訪視後，已撥款襄助部分喪葬開銷。

年邁裝心臟支架

銀達伯兩老，除喪子之痛尚未撫平外，更為年幼孫子養育責任憂心，銀達伯說，老伴有嚴重糖尿病，兩年前中風後仍走路不便。他平時承租國有林地種植香蕉、龍眼等水果，趁周六、日遊客到村裡爬山時，在山徑路旁擺攤販賣，每次收入僅6、700元。「我年紀也大了，心臟裝過兩根支架，不知能再做多久？」一旁孫子小揚天真地說：「我可以幫阿公賣香蕉，讓阿公賣到一百歲。」
里長說，目前銀達伯一家3口政府補助僅有老農年金每月7000元。銀達伯說：「孫子小揚是阿和留下的唯一血脈，拖著老命我也會好好把他養大成人。」
