A3528 撐家男跌癱 童哭聲救父

2014年11月24日

「把拔掉水裡，我哭哭。」今年3歲的小家對記者來訪時，指著坐在一旁的爸爸輕聲說著。小家37歲的爸爸林俊廷垂著頭、左邊肢體無力癱坐椅上，他無奈說：「穿鞋也沒力套進去，不能工作很痛苦。」
報導•攝影╱張嘉恬

一旁林俊廷的25歲妻子阿娟說，先生本就罹患肝炎、肝硬化，今年9月初因發燒久難退，檢查發現血糖值飆到300才知罹患糖尿病，9月6日先生騎車載3歲兒外出買便當，因暈眩，停車後坐在水泥石墩休息時昏倒跌到大排溝裡，村落鄰居看到兒子小家號啕大哭，才發現有人落水報警送醫。

罹肝病仍熬夜掙錢

阿娟接著說，先生在加護病房躺了一個月，沒想到糖尿病導致神經病變受損，至少需復健半年。
阿娟也說，夫妻倆育有現年5歲女兒、3歲兒子，原本先生上日班，她白天照料子女，晚上先生下班後，她再做夜間時薪收銀員補貼家用，以能照顧子女，合計每月收入不到3萬元，「今年9月他改找大夜班工作，我上白天班，粗計薪水合計可多2萬元，沒想到他沒做幾天就出事」，目前阿娟需常帶先生往返醫院，也只能回到餐飲店打工，每月收入加補助約2萬元，不敷醫療費與4口生活開銷，蘋果基金會訪視後已撥款紓困。
林俊廷的好友阿瑋說，林俊廷有肝病忌勞累熬夜，「但他常說，物價漲不停，很難生活，錢愈來愈薄，眼看2個孩子一天天長大，他也顧不了那麼多了」。

走過鬼門關更珍惜

林俊廷說，太太告訴他，落水送醫已命危，院方建議可料理後事，「聽說那時已經在找葬儀社了。」有幸從鬼門關走回來，還可以看到太太和2個寶貝孩子，這種感覺，感恩老天爺，說罷他看著身旁一雙兒女哽咽眼紅，說自己才國中畢業，不會形容內心感受，但終於可體會什麼叫做珍惜擁有。
阿娟說，公婆皆逝，小姑收入僅能支應自家，小叔則服刑中，她也說，自己是獨生女，出生沒多久父母就分開，由祖母、伯父帶大。「我同學有的還在念書，有的還煩惱找工作，我現在雖被生活壓得喘不過氣，但就想，提早遇到困難，提早盡力克服。」
