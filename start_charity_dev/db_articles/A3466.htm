A3466 寡母罹癌 11歲女扛家務

化療難負荷留職停薪憂生計

2014年09月12日

晚飯後，11歲的小儀洗碗收拾餐桌，倒了杯水提醒媽媽吃藥，小儀44歲的媽媽范惠敏嘆氣說，丈夫死後母女相依，沒想到自己今年5月罹患乳癌。報導•攝影╱仲芝蓉

范惠敏腰背骨頭就因化療副作用神經抽痛，她忍痛皺眉說，其實去年4、5月就摸到乳房硬塊，但醫生說是乳腺炎，吃兩周消炎藥就不痛了，沒再追蹤，直到今年初又開始腫痛，才再檢查，沒想到是乳癌二期。小儀在旁整理曬乾收下的衣服，一邊問媽媽：「等下洗好澡要穿什麼呀？」小儀的個頭雖已長得比媽媽高，卻仍孩子氣地黏著媽媽。

夫急性胰臟炎猝逝

范惠敏應了幾句，心裡煩惱著不知生命何時將到盡頭，雙眼含淚對記者說，丈夫4年前急性胰臟炎猝逝，自己帶著當時才讀幼稚園大班的女兒回到娘家附近租屋，「娘家雖有父母在，但年邁多病，靠么弟照料，我只能自己找工作撫養女兒。」
范惠敏繼續說，回來後就在當地飯店做房務清潔工，「小儀懂事又貼心，看我白天工作回家很累，家事都是她在做。」范惠敏的大姊說，手足4姊妹都已嫁各有家計負擔，「惠敏住院時我幫忙照顧小儀，希望她度過難關。」
中華民國乳癌病友協會秘書長林葳婕說，乳癌患者術後抵抗力弱，多休息降低感染發生率，是避免復發、轉移的關鍵，而惠敏從事房務勞力工，體力恐無法負荷，建議休養期應延長半年至一年。

年初發病忍痛工作

當地社區發展協會理事長說，范惠敏母女倆生活及房租開銷無著，公部門已協助申請急難金1萬元，再轉介蘋果慈善基金會，基金會訪視後也已撥款暫紓困。
范惠敏的同事說，惠敏年初就開始疼痛不舒服，但因過年正是飯店最忙、缺人手的時候，她一直撐病工作到4月多才看醫生，現已先留職停薪，她還跟大家說抱歉，「5月手術切除部分乳房和淋巴後，還回來工作幾天，後來要化療不得已才請長假，聽到她說抱歉，我忍不住抱著她流眼淚，大家都很喜歡跟她一起工作，希望她早日康復。」
