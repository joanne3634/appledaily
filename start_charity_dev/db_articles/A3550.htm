老母臥病女遲緩「勇敢走下去」

2014年12月23日

晚上9時，36歲的阿珠（辛碧珠）用輪椅推著68歲媽媽阿芳婆到房屋前庭角落，拉上布簾，幫媽媽洗澡。只見阿珠快速抹上沐浴乳，5分鐘就幫媽媽洗好澡，「天冷，洗快點媽媽才不會感冒。」
報導•攝影╱韓旭爾

阿珠說，爸爸早逝，她7年前離婚後，帶女兒小芷投靠娘家，因住處只有山泉水，水壓不足輸送屋後浴廁，不得已才在院子燒柴洗澡。媽媽阿芳婆喘氣說，她年輕時就氣喘，近2、3年因慢性阻塞性肺病，沒法正常呼吸，再也離不開氧氣機，因整天臥床，雙腿肌肉萎縮沒法再站。「不知還有多少命能活，幸好有阿珠照顧，人生最後幾年才有依靠。」
說到女兒小芷，阿珠不捨地說，小芷3個月大就診斷出第1對染色體異常，如今小芷發育遲緩，都讀小六了身高才116公分，只會講「媽媽」、「穿衣服」等簡易話語，又過動，領有重度殘障證明。

醫院家裡兩頭跑

阿珠說，這些年來，她抽空做零工，還能勉強度日，但近半年來，小芷常莫名其妙發燒，「醫生只說血小板太少、白血球過多，前陣子住院1個月還找不出病因，幸好用藥後恢復正常，醫生交代，只要高燒不退，就要趕快掛急診。」阿珠說，那時她在醫院顧小芷，中午、傍晚又要抽空回家幫媽媽準備三餐、洗澡，忙到不知所措。阿珠姊姊說，手足4人，大哥失業多年，二哥失聯，自己才做完換腎手術，「只能常去探望媽媽。」

住處惡劣無力修

當地社會課承辦說，阿珠一家領有政府補助共1萬1170元，住處環境惡劣，一直沒能力改善，又離醫院遠，就醫包車每趟都要上千元，故轉介蘋果基金會，基金會訪視後已撥款暫紓窘迫。
採訪到9時許，阿珠安頓好媽媽後，接著幫女兒洗澡、穿衣服，小芷洗完澡，便好動地在屋裡屋外亂跑，阿珠喊著：「幫妳綁漂亮！」只見小芷便過來把頭埋在媽媽懷裡撒嬌，阿珠說，對未來，她一片茫然，但為了撫養小芷長大，她會勇敢走下去。
