A3525「吃粥攪鹽」 阿公顧兒孫

5200元養3口「撐到子病好」

2014年11月20日

傍晚，72歲阿吉伯（陳萬吉）打開沒什麼食物的冰箱，拿出青菜料理，再煎顆讀小三孫女愛吃的蛋；他跟記者說，家裡已2個月沒錢買肉，「80元幾把青菜夠吃2天，再差一點，我吃粥攪鹽也能飽一餐。」報導•攝影╱向高彬

阿吉伯老來不得閒，打理一家大小事，照顧同住45歲單親兒子阿承及8歲孫女小伶，「我不顧也不行，阿承患糖尿病10多年，去年中發病嚴重，走幾步路腳軟手軟，兩眼茫茫比我還看不見，腎臟病變、手腳水腫，已1年沒力做事。」

動手術無法打工

這1年來家計成愁，阿吉伯去找掃工地零工，1個月近萬元貼補家用，但他8月也動小手術，體力差些，醫師叮囑不宜搬重，「但家裡沒收入怎過活，我又去排工，工頭看看我說：阿伯先回去休息啦。」
阿吉伯無奈老本用罄，身體老來不中用，如今一家3口靠每月老人年金和家扶補助共5200元生活，「現在房貸繳不出來，最後也只好走到賣房子這步了。」當地公所正協辦急難救助；醫院社工協力部分醫費並轉介蘋果基金會，基金會訪視後已先撥款紓困。
阿吉伯回述，與前妻有3子1女，20年前離異，大兒子跟前妻生活；他早年跑計程車為業，在二兒子、小女兒各自成家後，這些年與三兒子阿承生活，先前阿承做汽車鈑金每月約2萬多工錢撐家，娶越配生1女，3年前離婚撫養小伶，本來阿承身體還好，怎料如今病況惡化，「我只能再撐撐看，顧他到身體好一些。」
一旁阿承氣色憔悴地說，走10幾分鐘路去醫院回診，得邊走邊喘停下休息5、6次。小伶說：「以前爸爸常帶我去公園玩，可是現在都沒有了。」

「自己也跟著哭」

阿吉伯二兒子阿銘說，和印尼老婆做舉看板、工地零工，「每月2萬多元付房租、養2小孩和還債，再擠出1、2千元給老爸，已盡力了。」小女兒阿如住外地、家有2幼子，「爸有時打電話給我，只是嘆氣，聽他難過，自己也忍不住哭，但也無能為力。」

基金會編號：A3525
