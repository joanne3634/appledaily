A3927 夫妻皆罹癌 8旬母苦撐家

2016年05月12日

53歲胡金樹今年1月罹肝癌，還在治療中，未料4年前中風的同齡妻子阿梅隔月也發現罹乳癌，夫妻倆除面對病魔外，更煩惱8旬老母和12歲兒子生計。
報導•攝影／張嘉恬

胡金樹說，以往夫妻都做綁鐵零工，妻子4年多前中風，左側癱軟無力無法再做工後，就靠他1人工作，4口清儉度日，無奈現在他也倒下。「太太中風走路不方便，沒辦法自己到醫院化療，我做完治療很累，也只好撐著帶她去，遇到了沒辦法」。

12歲子想去市場洗碗賺錢

他接著說，和妻子結婚20多年沒有子女，12年前領養車禍過世姪兒的兒子小寶，「小寶出生沒滿月我們就抱來領養了。生病後我反而想得開，都得癌症了，我已經不在乎生死，還好有小寶，才會想繼續活下去」。訪談間胡金樹看似堅強，但提到兒子小寶則不禁掩面哽咽。
胡金樹也說，夫妻病後，12歲就讀小六的小寶，主動提出要到菜市場洗碗賺錢，「我不可能讓他去，只是他有這個心，我真的很安慰、很感動，聽到時候好像病好了一半那樣」，他眼眶泛紅說著。小寶說：「爸爸沒在賺錢，我很想幫他，以前媽媽中風現在又病得更嚴重，每次她去住院，我都很怕她不會回家了。」一旁胡金樹的妻子阿梅因中風身障，現又因癌病虛弱，不擅表達的她說：「想好好治病…希望病快好…看孩子長大。」

8旬老母嘆歲數大心裡無奈

採訪時，胡金樹最小的單身弟弟阿豪來探望，他說，兄弟姊妹共8人，3個姊姊都出嫁，胡金樹是三哥，其他3個哥哥也都成家分散各地，他說，因爸爸早逝，哥哥們都小學畢業就做工，現都靠勞力養家，「老母是大家的該奉養，三哥三嫂病後，我們也共幫了10多萬，已盡力，我來探望時會帶些營養品」。
胡金樹全家目前每月僅領妻子殘津4872元，加上老母老農津貼7256元後共1萬2128元，扣除就醫交通及營養品開銷9000元不敷4口生活，蘋果慈善基金會獲悉訪視後已撥款紓困。胡金樹8旬老母說：「我很想替兒子媳婦做點什麼，只是歲數這麼大了，幫不了什麼心裡實在無奈。」
