A3691 單親癌母病倒 19歲女扛家

2015年06月25日

傍晚時分，剛高三畢業的小雲才下公車，急忙去市場買了些青菜，接著快步走回家裡，「媽媽得乳癌剛化療完出院，炒菜時鍋子都舉不穩，現在都是我在煮，今天回來晚了，媽媽可能餓了。」
報導•攝影╱楊光翔

小雲49歲的母親阿凰，4年前因遭家暴結束第二段婚姻，從事家庭美髮，月收約2萬元，辛苦扶養小雲和國三女兒小雅，但今年2月阿凰檢查罹患乳癌，阿凰表示：「離婚後帶著2個小孩在外租屋，分別就讀國、高中，正是最需用錢的年紀，好不容易撐過了，現在小雲要讀大學，眼看將有美好前程，我卻在這時候倒下，身體苦痛不打緊，就怕耽誤到她們未來。」

每逢開學求延繳費

阿凰說，生病停工，不得已向第一段婚姻所生的兒子阿柏求援，「阿柏知道後，瞞著家裡盡力幫忙，但他工廠薪水不多，最近也論及婚嫁需開銷，我實在不敢繼續開口。」記者聯繫上阿柏，他說：「媽媽只有1個，但我能力有限，婚期受影響也一再延後，不知道能再瞞多久、幫多久？真的很煩惱。」
19歲小雲說：「媽媽一人養我們，每逢開學她總拜託校方通融延繳費；媽媽現在罹癌，我打算先讀夜大，利用白天工作，只是擔心高中學歷薪水不高，沒辦法照顧媽媽跟妹妹。」暑假後升高一的妹妹小雅說：「我有學處理媽媽傷口的引流管，希望媽媽快點好起來。」

房東同情不忍催租

小雅班導黃老師表示：「前陣子她上課常紅著眼眶，關心後得知家中狀況，已通報校方並協助申請行天宮急難金。」阿凰房東太太說：「女人家獨自帶2小孩真的不容易，阿凰沒罹癌前手頭本就很緊，常積欠房租，但知道她狀況我也不忍催收，偶爾還幫忙代墊水電費，希望她能再度振作。」
當地社工說：「阿凰父親74歲，母親已歿，手足4人，大弟、二弟都做零工，小弟剛服義務役，協助能力都有限。」現阿凰一家僅獲公所馬上關懷急難金2萬元，故轉介蘋果基金會，基金會訪視後已先撥款紓困。

基金會編號：A3691
