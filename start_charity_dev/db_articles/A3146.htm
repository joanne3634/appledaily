A3146 姊扛家顧弟 守護昏迷母

父逝弟智障 「我只能哭」

2013年08月12日

「媽，快點醒來，弟弟還在家裡等你。」22歲的小如（陳瀅如）在50歲的母親陳素美旁輕聲呼喊，她說，今年5月底媽媽因呼吸衰竭昏迷，至今仍無意識。
報導．攝影╱江品璁

小如說，讀高一時，爸爸中風，家裡無法負擔升學開銷，「幸好城裡的阿姨讓我住她那，供我讀書、讓我在她開的小麵攤打工，每月賺1萬多，匯5千元回家，爸爸去年12月二度中風過世後，我心裡像缺了一塊，常想家，但今年6月大學畢業後，擔心回鄉下找工作不易，就邊在阿姨家打工邊投履歷。」
小如說：「我每3個月回家一次，5月25日回家，吵著要吃媽媽煮的焢肉飯，但家裡窮不常煮，她答應我下次回來一定煮給我吃，沒想到5天後，媽就因敗血症衰竭昏迷，要靠呼吸器維生。」

母每月耗材萬元

小如掉淚道：「19歲讀高二的弟弟是中度智障，母親癱瘓後，每月耗材開銷約1萬多，我負擔不起只能哭，後來想到平常看《蘋果日報》暖流版常報導困苦家庭，才求助基金會。」基金會訪視後已撥款紓困。

「媽媽都不看我」

小如的同學小豪說，之前小如早上8點幫阿姨賣麵，下午5點趕到學校，她媽媽昏迷不能參加她大學畢業典禮，「她一直哭，我在旁安慰她，但也幫不上忙。」
記者到小如家訪視，只見小如弟弟小耿身高僅157公分，看起來像國中生，小如說，弟弟2歲時發燒過度傷腦，致發育不良、智力受損，東西拿了就往嘴裡塞，常到處亂跑，「媽媽昏迷後沒人可顧他，只好請隔壁的大伯代為照料。」小耿跟記者說：「媽媽眼睛都張好開，但都不看我，我好怕，但又好想她。」
小如的大伯陳永川說，父母已逝，家裡3兄弟，大弟、二弟都過世，二弟媳昏迷後，自己幫忙照料小耿，「但各人有各人的家業，我無法負擔他一輩子。」當地里長說，小如家為中低收入戶，領有智障補助4700元，已協辦急難金1萬5000元。
