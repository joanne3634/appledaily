A3415 妻顧癌夫 8千補助養家

幼女求神「別讓爸爸這麼難過」

2014年07月11日

「妹！噓，小聲一點啦！爸爸要休息，妳不要讓爸爸趕快好是不是？」8歲小妘對4歲妹妹說著。一旁43歲媽媽阿梅則透過鼻胃管，一邊幫罹齒齦癌的先生李明國灌食營養品，一邊擦拭他不斷從嘴角流出的唾液。報導•攝影／張嘉恬

僅2坪大的客廳裡，李明國因化療後的疲憊，坐在客廳已睡著，8歲小妘用餐前，正禱告著：「主耶穌求祢讓爸爸下巴不痛，不要讓爸爸這麼難過……。」阿梅說，家中沒固定宗教信仰，孩子因同儕學會禱告求上帝，她則把精神寄託給觀世音菩薩，她說：「先生倒了，我的心慌掉，孩子求上帝我求觀音，只盼能穩住這個家，求安心。」

難負擔管灌營養品

阿梅說，她和46歲先生同做粗工養家，先生今年4月罹癌，手術後先生曾昏倒過，她得陪同看顧，雙雙無法工作，目前家計僅靠低收等補助每月共8600元，但房租就要4500元，無法負擔管灌營養品1萬元與生活開銷，而先生的3兄弟皆有自家生計重擔，已盡力協助10多萬元。

「長大當護士助人」

阿梅家附近的機車店老闆廖先生說：「李太太常因車子故障來店裡維修，聊起來才知道李先生得癌，李太太雖樂觀面對，但想到女兒還小，生活吃緊，她很苦惱，我能做到的是維修小錢不收，因有看《蘋果》暖流版，所以告訴她可以求助。」蘋果基金會獲悉訪視後已撥款紓困。
「阿姨，爸爸說護士可以救人做善事，以後我長大要做護士幫助人。」李明國長女小妘天真對記者說著，一旁因體虛疲倦半闔眼的李明國望著2幼女，使盡力氣說：「人破病（台語：生病）時，什麼雄心壯志都不重要了，只要平安順利，我能多活幾年看2個女兒健康長大就好。」
阿梅說，自己從小讓人做養女，養父過世後她幾無親人，後與李明國在工地工作相識，她接著紅了眼眶說，先生病後兩人曾因生計問題意見相左，有次先生因覺得自己拖累她，「要我包仔款款（台語：行李整理整理）帶著孩子離開家，不要管他，結果我們抱頭痛哭。這個家雖然垮一半了，但我還是會硬撐下去。」
