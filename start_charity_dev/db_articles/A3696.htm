A3696 怕拖累妻女 癱男勤復健

2015年07月01日

晚間43歲李淑儀把脊損先生阿德從輪椅移抱到床上，幫他換尿布、做腿部復健，之後，她揉了自己的腰，跟記者說，好幾回閃了腰疼好幾天，「我知道先生很努力想好，我怎麼樣都得盡力。」報導•攝影╱向高彬

李淑儀回述，夫妻育有14歲獨生女小瑄，48歲先生10多年前曾開工廠失敗負債，他先前做電梯維修，10年前曾摔傷腿，休養1年後改做臨時僱工，她做人力派遣工，夫妻撐付房貸家用，未料3年前先生遭意外脊損，雙手可動、但下半身癱瘓，每兩三小時要導尿排便需她照料，「先生仍抱著再站起來的希望，每天勤做復健。」
李淑儀又說，先生癱後，之前先生公司曾負擔當時大半年醫護費，「接著勞保理賠近70萬元，我們一直到今年初用盡，如今3口每月僅靠三款低收補助共1萬800元生活」。在一旁的阿德說，只好求助工傷協會諮詢，協會再轉介蘋果基金會，基金會訪視後已再撥款急難金。

判決廢時家計生困

工傷協會專員說，此案調解破裂，正由司法審理，職災判決曠日廢時，5年、8年很常見，協會持續關懷，「但一家支柱重殘，家計出現難題。」
採訪時，阿德難掩鬱悶地說，癱瘓後拖累成家裡負擔，有回曾坐輪椅出門，被路邊乞討者罵別擋路，「心情難受，想越多，就越鑽牛角尖，還好太太安慰我。」他又說，近期打算假日讓太太推輪椅去賣刮刮樂，也想到脊損潛能中心做生活訓練，「要想辦法重新過日子。」

「養爸媽是我責任」

讀國二的女兒小瑄文靜話少，她說，媽顧爸很辛苦，她會幫忙洗碗家務，「家裡只有我一個小孩，我知道長大後，撫養他們是我的責任。」
阿德的二姊阿芳說，兄妹各有孩子要養，她常帶水果探望，「弟弟毛病多，弟媳很認真照顧。」李淑儀的大姊阿玲說自家尚有2子女就學中，娘家4手足中淑儀是小妹，「以前她很愛笑，但現在每回看她都苦著臉說心情不好。我70多歲老媽常去看她，也只能安慰她堅強一點。」
