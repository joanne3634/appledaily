A3413：抱病養4口 嬤孫撿回收

2兒病癱嘆：天氣如何都要做

2014年07月08日

傍晚五點多，公園旁聚集來自各家各戶提著垃圾袋的居民，當中，還有10歲小延和他64歲阿嬤廖桂椿的身影，在垃圾車來之前，大家都將回收物紙類瓶罐留給他們。報導•攝影╱仲芝蓉

採訪當天下著小雨，廖桂椿讓孫子先回家換衣，自己淋雨整理，她嘆了口氣說：「不管天氣如何都要做，收1個月可賣4000、5000元。」
廖桂椿手上不停搬拿一袋一袋的瓶罐，收拾好推車後，才終於喘了口氣，揉著腰皺眉說：「我有糖尿病、關節炎，家中除了孫子小延，13歲孫女小如靠我照顧，大兒子中風癱瘓、二兒子骨髓炎傷重，我不勤勞回收怎麼過生活。」
記者再隨廖桂椿返家訪視，小延已換好衣褲，正幫爸爸復健舉腿。廖桂椿說，39歲的二兒子阿明是小延的爸爸，原家計靠他做水電工，但他今年2月右膝蓋小腿骨髓炎復發，開刀4次還不好，連下床走路都難，「醫生說只能靠高壓氧治療，一次1500元，20次3萬元就耗盡積蓄。」

骨髓炎無法工作

廖桂椿愁容滿面繼續說，自己共生3個兒子，孫女小如是老三阿達的，「媳婦在孫女4個月大就離婚，阿明也離婚，42歲大兒子阿松沒娶，卻在37歲那年中風。」阿明說，自己因國中時車禍被撞，右手右腿當時惡化成骨髓炎，「父親早逝，母親做清潔工撫養我們，那時治好骨髓炎，國中畢業我就去做水電工學徒。」
阿明繼續說，自己一直做到水電小包，原也存了點錢，10多年前跟朋友投資想做大事業，卻失敗賠本，當時有了小延卻經濟困頓離婚，「近幾年收入勉強維持生活，弟弟4月才找到幫人代班開遊覽車工作，怎料我骨髓炎復發，又讓母親擔憂我的醫療費。」

補助不夠醫療費

當地公所急難承辦員說，廖桂椿家現領有單親及阿松殘津等補助共1萬200元，加上回收及三子代班開遊覽車薪水約1萬5000元，收入合計共2萬多元，但阿明每月高壓氧治療就需3萬元。
公所正協助申請急難金，並轉介蘋果慈善基金會，基金會訪視後已撥款暫紓困境。

基金會編號：A3413
