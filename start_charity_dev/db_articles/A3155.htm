A3155 癡情夫顧癌妻 「苦也值得」

兒打工 女棄學 5口相扶

2013年08月23日

41歲的阿蘭2月初發現罹患4期乳癌，至今已做過6次化療，療程還要多久醫師仍未有答案，只見她虛弱癱躺病床嚷著：「要回家！」一旁丈夫阿董說，阿蘭有精神分裂症難照顧，「想到醫療、生活費，一個頭兩個大。」
報導．攝影╱韓旭爾

今年51歲的阿董（蔡夢董）說，夫妻經朋友介紹結婚，20年前新婚時就發現阿蘭多話、易怒，原以為只是脾氣大，不久診斷為嚴重躁鬱症，定期服藥，還能做清潔零工月賺約8、9000元貼補家用。

精神分裂更難照顧

阿蘭罹患乳癌後，「躁鬱症變成精神分裂症，出現幻覺，脾氣更壞。」阿董指著阿蘭頭上傷疤說，這傷就是她在家休養時幻想家裡有小偷，想爬樓梯到2樓抓賊卻跌倒造成，「一定要有人全天陪在阿蘭身邊。」
阿董夫妻共育20、18和16歲3名子女，18歲長女小瑄說，她今年高職畢業後原想升學，但為顧媽媽暫停學業，「有我陪伴，媽媽焦躁情緒有穩定些，比較不會在病房大叫大鬧，吵到別的病人，我們全家對媽媽的病有信心，一定可以度過難關。」升大二的長子小龍平日住校，他說，利用假日到餐廳打工，月賺3000多元，「至少讓爸爸少點憂心。」而升高職的么女小婕也就讀建教合作班，想減輕家裡負擔。
提起過往生活，阿董說，家住離島偏鄉，平時做建築或漁船零工為生，多年打拼去年總算付清房貸，未料阿蘭罹癌，光往返醫院的交通費、營養品及尿布，每月就要2萬多元。這半年多，除了張榮發基金會已協助4萬8000元外，又向親友借約20萬元應急。蘋果基金會訪視後也先撥款紓困。

住離島得跨海看病

當地村長說，一樣繳交健保費，但離島居民最大的悲哀是許多重症需要到台灣治療。
村人都知道阿蘭精神有狀況，但阿董從不抱怨，阿蘭在家休養時，女兒幫忙看顧，有人叫工阿董就去做，是個老實有責任的丈夫。阿董說：「結婚就是一家人，從沒想過放棄老婆。阿蘭身體若能好，日子再苦也值得。」
