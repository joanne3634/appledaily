A3518 稚兒探癱父「爸眨眼了」

昏迷月餘6口等奇蹟

2014年11月12日

「爸，有聽到，眨一下眼睛。」讀小五的小桂和讀小一弟弟到長期呼吸照護病房，探望腦幹中風的40歲爸爸張庭鉦，「弟，快看，爸爸眨眼睛了！」報導•攝影╱向高彬

接著弟弟小瑋喊：「爸爸會捏我的手了。」兩兄弟對於昏迷癱瘓1個多月的爸爸從原本毫無意識，到如今稍能回應，都非常高興。小瑋說，之前在加護病房時見爸接滿管子，「我嚇到了，怕他會死掉。」

「還須老天多幫忙」

39歲孩子媽郭嘉欣說，先生有高血壓，除此之外身體還好，8月中突然病倒，「醫師說腦幹出血太嚴重，就算救活也成植物人。公婆只有這個兒子，堅持救他。」張庭鉦經搶救活命，現靠呼吸器維生，郭嘉欣說，先生有機會經訓練脫離呼吸器，要清醒還須老天多一點幫忙。
只是如今郭嘉欣苦愁僅靠她一份工廠會計薄薪約3萬元，但負擔先生每月照護病房費1萬元，及家中公婆和兩子共6口生活房租開銷。當地公所和孩子學校正協辦相關救助。醫院社工協力部分醫費再轉介蘋果基金會，基金會訪視後也撥款暫紓急迫。
郭嘉欣回述，先生之前跟公公做木材生意，但工廠倒閉負債，因公公稍懂中藥，近年偶幫朋友抓藥掙1、2千元零用，婆婆曾中風略跛；「先生沒什麼專長、找不到好工作。就在市場賣甘蔗汁，一個月約2、3萬元付攤租家用。」未料如今一家生活全亂了。
探訪時，見張庭鉦的63歲父親阿益伯臉色很憔悴，他顫抖地說，兒子病後至今每晚失眠，就算吃安眠藥也僅睡2小時，快崩潰了，「老實說，我有後悔救活他，要好好不了，醒來也不完全，但父母心，不救又怎麼忍心。」他哽咽，兩孫仍年幼，「我只剩老命一條，願意老天換兒子快好起來。」

女兒更怕老父病倒

郭嘉欣有1姊1弟，姊姊說，之前已貸10萬元協助周轉，難再多幫。張庭鉦的4妹已嫁，大妹說，與3妹都有年幼子女，心有餘力不足，「我更怕爸爸病倒，他常打電話訴苦，我安撫他看開點。」
