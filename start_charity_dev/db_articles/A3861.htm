A3861 夫重病 婦洗豬寮養5口

不怕髒臭 心疼孩子為省錢餓肚

2016年02月17日

33歲的阿梅（周麗梅）看著因肝硬化致腦病變的45歲先生阿峰說：「做尪的認不出自己的某，真無奈！」說話同時邊用病床束具固定阿峰，邊解釋：「等等得去接5歲的小女兒小嘉下課，沒人顧怕他亂跑，只好像這樣綁著。」
報導•攝影╱楊光翔

接完小女兒回家，阿梅說，與夫生有就讀國二、小六、幼稚園中班3子女，「家計原靠夫妻務農維生，但1年多前先生手腳無力，不願就醫，不再下田，他整日躺在長椅上，脾氣也變古怪，孩子們都不敢親近。「我現在做零工撐家，但收入僅1萬多，小孩為了幫我省錢，早餐常不吃，人都瘦巴巴的，還安慰我說營養午餐有拚命吃補回來。」

拜託校方通融分期

阿梅接著說：「但2個月前先生虛弱跌倒在地，急診後才發現嚴重肝硬化跟肺結核，已需全天看護，家裡沒人賺錢，小嘉幼稚園學費還得拜託分期。」小嘉老師說：「園方統一每學期繳費，但小嘉媽媽經濟狀況有困難，已協調通融把1萬3千元的費用攤6期繳。」因阿梅家計陷困，蘋果基金會獲悉訪視後已於過年前先撥急難金協助。阿梅小六長女小方班導說：「若不是媽媽告知，小方本還不想說出家中困難，校方現已幫忙申請獎學金，但金額有限。」
阿峰舅媽說：「阿梅其實是印尼人，有人問過她家裡狀況這麼差，怎沒跑掉，她只說台灣才是她現在認定的家，家人有難，她怎能離開？很讓人欽佩。」阿峰大弟阿霖說：「爸媽都因癌症去世，大哥現又生病，但我和妹妹都有家庭要顧，經濟協助有限，只能過來幫照看姪子女。」

子女假日分擔農務

阿梅說：「小瑋和小方都很懂事，會跟我學煮菜分擔家務，也會幫我安撫小女兒，多虧有他們，我才能放心看護先生。」
當地村長表示，阿梅很勤奮，「為了賺錢她什麼零工都做，洗豬寮全身髒臭也不介意，小瑋、小方常會用假日去幫忙親友拔草等農務。知道他們家陷困後，就趕快幫打電話到蘋果基金會求助。這些孩子都很有希望，盼別因家境影響到未來發展。」
