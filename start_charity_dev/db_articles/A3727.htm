A3727 17歲女 做手工顧癱母

單親媽嘆：拖累她很對不起

2015年08月14日

「媽，我推妳到外面曬太陽，整天躺在床上也不好。」趁著早晨陽光未熾，17歲的小慈推著中風媽媽阿貞到老屋前庭透氣，50歲單親媽阿貞癱坐輸椅無奈說：「只剩小慈能照顧我，拖累她書都快沒法念下去了，很對不起她！」報導．攝影╱韓旭爾

阿貞說，她曾有前段婚姻，所生2子女早已離家，之後與男友同居生下小慈，孩子1歲多兩人就分手，這些年來，她因常氣喘發作，只能做農務零工、家庭代工，每月只能掙約1萬元，「生活上想辦法盡量節省，只求平安把小慈養大就好，怎料今年2月突發中風。」
目前阿貞大小便、吃飯和擦澡都需小慈照料，小慈說，現正值暑假，她能全天在家顧媽媽，「前陣子上課時，我上學前先幫媽媽穿好尿布、餵早餐，午休時，再請假1小時回家煮午餐、換尿布。」

「原本想休學顧家」

小慈表示，本來也想休學顧媽媽，但媽和老師都希望我不要放棄學業。她說，雖找時間做分裝金紙代工，但1天才賺100多元。
小慈的導師說：「小慈很懂事，她4月就填好休學申請書，但我不簽字同意，勸她無論如何要高中畢業，家裡經濟很不好，所以要靠讀書才有機會翻身。」導師說，已協助申請學產基金急難金，蘋果基金會獲悉訪視後也先撥款紓困。
採訪時，小慈雖表現堅強，但講起對媽媽的擔憂，忍不住內心害怕，她哽咽說：「復康巴士每趟來回要500多元，沒錢讓媽媽坐車去醫院復健，手腳的活動能力都沒進步，很怕媽媽永遠都無法再站起來了。」
住附近的阿貞堂妹說，阿貞其他3手足在經濟上都自身難保，與前夫所生子女，只有女兒阿惠不定時寄4、5000元供家用，「我偶爾也幫忙買些尿布、牛奶。」

房遭查封恐無居處

當地公所急難救助承辦說，阿貞9月可申請殘障鑑定，核可後每月除能有4700元補助外，每月也有50至90小時的免費居家服務，協助小慈照顧媽媽，但阿貞自宅因付不出房貸已被查封，過陣子法拍成交後母女將面臨無處棲身困境。

基金會編號：A3727
