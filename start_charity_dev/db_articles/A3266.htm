A3266 抱4歲癌兒 母淚求平安

童咬牙挨針 教人心疼

2014年01月02日

醫院病床上，4歲浩浩讓28歲媽媽李雅萍緊擁在懷裡，媽媽大手牽著浩浩小手，輕聲說：「奉主耶穌聖名禱告，親愛的天父……，把浩浩體內蟲蟲殺死，讓他勇敢，願神帶領一切平安順利。」說罷李雅萍已淚眼婆娑，久久不能自已。報導．攝影╱張嘉恬

未察覺媽媽李雅萍流淚的浩浩，在禱告完後活潑喊著：「哈利路亞，讚美主耶穌！」李雅萍拭淚後說，浩浩1年多前罹患「惡性組織球增多症」，是會吞噬紅、白血球及血小板的癌症。兒童癌症基金會經理李秀玲說，此症在兒童癌症中佔比少，2012年該會服務的500多個案例中佔23名，致病原因不明。

補助不敷4口生活

「沒看過這年齡這麼勇敢的孩子！」醫院吳姓護理師看著濃眉大眼的浩浩對記者說，浩浩因長期化療致血管較硬，打針得反覆扎7或8次才成功，「一般孩子早就大哭大鬧，浩浩常閉眼咬牙忍耐模樣，教人心疼。」吳姓護理師說罷，正讓媽媽換尿布的浩浩從布簾探出頭，俏皮說：「噓……不要說我壞話喔。」
李雅萍說，除了4歲浩浩，她和28歲先生吳聖傑還育有現年6歲長子，家計原由她擔任看護助理與先生賣菜維持，自次子浩浩罹癌後她得全心照顧無法工作，微薄積蓄用盡外還積欠8萬多元債務。兒癌基金會2012年曾協助浩浩每月1萬元為期1年，公所核撥補助1萬8000元也已用盡，僅靠先生賣菜收入每月2萬多元，扣除房貸1萬元後不敷4口生活。

父罹患肝疾難兼差

記者來到李雅萍家，先生吳聖傑正教導長子小恩功課，吳聖傑說，小恩善體人意，「弟弟住院時也常說很想他」。吳說自己為B肝帶原者，每天凌晨3時半起床至下午3時多結束工作，曾想兼差但因肝疾未能過度勞累負擔。
