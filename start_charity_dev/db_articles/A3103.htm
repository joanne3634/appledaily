A3103 嬤父皆罹癌 2兒頓失依

叛逆長子改過 盼早日扛家計

2013年06月20日

62歲的林潘秀蘭虛弱無力地說，自己今年2月發現得兩癌，怎料3月時，38歲兒子林勝平也罹癌，「兒子單親，兩個小孩，一個國三、一個小六，都還要靠他養，疾病偏偏這時找上身」。報導．攝影╱仲芝蓉

林勝平以手壓住腹部，強忍術後神經抽痛說，媽媽得十二指腸癌和壺腹癌第3期，都是腸道癌症，「我以前小腹悶痛、有血便，3月去醫院顧媽媽時順便檢查，才知得直腸癌第3期」。

每月僅靠補助度日

林勝平說，媽開刀切除肝、膽等6個臟器部分組織，手術9小時，「她年紀大風險高，醫生要我們有心理準備，還好術後穩定，我則要治療到10月都無法工作」。
林勝平說，離婚後帶15歲大兒子小琮（音同從）和12歲小兒子小翔回老家同住，「爸69歲還在種田，收入僅能勉強餬口，我原本到外地做建築零工，還能寄錢回來養家」。小琮說，小學時爸爸就教自己和弟弟分擔家務，「農忙時，我周末跟阿公去秧苗場打工，一天可賺1、2千元」。
當地公所急難承辦員說，林家僅領有老農及兒少津貼每月1萬1200元，但林勝平母子每周回診車資每月就要7500元，已撥急難金2萬元，並轉介蘋果慈善基金會，基金會訪視後已撥款紓困。
林勝平的弟弟說，3兄1妹各自成家、孩子還在讀書，「原本輪流到醫院照顧媽媽，沒想到三哥勝平接著病倒，已盡力偶爾買營養品接濟」。

暫停校隊練習顧爸

記者訪視時，小琮表情平靜地告訴記者，其實自己是校內田徑隊，準備了3年終於通過縣賽，能夠參加4月中的全中運，「無奈爸爸那時要開刀，我原想放棄，但教練到醫院來勸說，我直接從醫院去比賽，可能2周沒練習，沒得名」。
小琮告訴記者，剛上國中時叛逆，「感謝爸爸沒有放棄，一直跟我講道理，今年才能順利免試入學上高中，我會更努力早日協助家計」。
