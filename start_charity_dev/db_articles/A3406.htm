A3406 嬤苦守癱兒：快好起來

「我60歲了哪天無法顧他怎辦」

2014年06月30日

晚上7時許，62歲的阿珍嬤孤單地在醫院吃晚餐，今年2月，同住的34歲么子阿松（王松茂）癱瘓，阿珍嬤寸步不離在醫院照顧，她說，兒子腦膜炎開刀出來癱瘓到現在，「連話都不能講了。」報導•攝影╱江品璁

阿珍嬤開收音機放給阿松聽，她說：「復健師教我，要放聲音給他聽，讓他頭腦動，意識比較不會跑掉，不然我叫他都沒反應。」阿珍嬤接著說，阿松未婚，原當搬運零工，跟她一起租屋生活，但今年2月阿松發燒不退，送醫後才發現是腦膜炎致水腦症，「醫生說不樂觀，我很煩惱，想不到5月初又中風，現在只希望能控制住，不要再惡化。」

欠10萬醫費生計囧

阿珍嬤嘆道，阿松本來有交女朋友，「但現在癱成這樣，對方跑得跟飛一樣，但也不能怪她，沒名沒份，不要耽誤人家。」醫院社工說，阿松癱瘓前工作不穩定，阿珍嬤雖60多歲，還在做洗碗工，並兼職找毛巾代工回來做，相當拚命，但兒子癱瘓後皆由阿珍嬤照料，一家無收入，故轉介蘋果基金會，基金會訪視後已撥款暫紓窘迫。
阿珍嬤說，眼前的困難先想辦法度過，「我60多歲了，還有幾個60歲？哪一天，我沒辦法顧他要怎麼辦？送去給人顧又擔心沒錢。」阿珍嬤接著道，阿松累積了10多萬元醫療費，出院後也要人照顧，只能走一步算一步。阿珍嬤靠近阿松喊道：「你有聽到沒，媽媽在跟你講話，你要快點好起來，不然我也沒辦法一直顧，我會老耶！」只見阿松眼睛眨了一下，似有聽懂。

申請托育負擔沉重

阿珍嬤的次子阿賢說，家裡共4手足皆各有家庭，自己跟大哥每月拿幾千元幫弟弟，阿珍嬤說，雖然平常沒與次子同住，但知道次子有時凌晨3時許就被老闆叫起送貨，又有自己的家庭要扛，「我實在不知道怎麼辦才好，只希望阿松能快好起來，不要給人負擔。」
當地居服督導沈玉婷說，阿松與阿珍嬤為經濟弱勢族群，家庭親友支持度往往不夠，阿松若申請政府托育養護補助，須等候床位時間不定，即使通過，仍須補貼耗材等費用，對家屬來說仍是沉重負擔。

基金會編號：A3406
