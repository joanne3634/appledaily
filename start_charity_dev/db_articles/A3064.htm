A3064 高2女堅強 顧病父癌母

「不羨慕別人 希望爸媽健康」

2013年05月06日

「爸，我扶你，慢慢走。」18歲小君扶著中風行動不便的61歲爸爸王松山（阿山），一步步穿過家裡窄小走廊到廁所，耐心幫爸爸脫褲、擦拭便後臀部，她跟記者說：「照顧爸爸是應該做的，只是我更憂心得胃癌的媽媽。」
報導．攝影╱向高彬

小君說，爸爸在4年前曾輕微中風，多年家計靠56歲媽媽薛麗雲做零工和拾荒撐持，未料去年9月媽媽因胃痛查出患胃癌，切除3/5的胃，爸爸在同年10月二度中風，她請假照料病癱雙親，又改念夜校，才能照顧爸媽。

男自慚拖累妻女

讀夜校高二的小君說：「媽愁沒錢付房租生活，還抱病去做洗碗零工，她要我顧爸和讀書，其他她想辦法，但我怕她撐不住。」薛麗雲的麵店老闆娘說，麗雲已做近2年，工作認真，住院時也曾去探望，「知她家現有困難，幫忙讓她做點工，交代她若太累就別撐」。
如今小君家計靠每月三款低收殘障補助共1萬600元，和媽媽掙約7000元微薄工錢，但扣掉房租後，3人僅餘約1萬元難敷醫病家用，小君望著租住的破舊公寓說，自己不會羨慕別人住好吃好，日子苦都沒關係，最希望爸媽健康，一家人在一起，「也珍惜現在還能讀書，若明年畢業沒法再升學，先工作幫家裡」。小君班導陳老師說小君很善良懂事，在校表現好，「有時請假顧爸媽，回來上課一定自動補進度」。
薛麗雲回述，先生曾做過房仲，兩人之前交往同居，育有一女小君，前年才登記結婚。阿山語聲含混地說，多年來沒給妻女過好日子，深覺愧疚，「現在更拖累了她們母女」。阿山與自家弟妹往來少，能幫有限。

獲救濟仍有困難

薛麗雲的二姊說，唯一弟弟已逝，8姊妹裡2個乳癌1個精障，「各家都不好過，麗雲又是最坎坷的，我看她這樣，心裡也很難過」。當地里長及公所已協辦相關救助並撥急難金共3萬元，評估後仍有困難，再轉介蘋果基金會關懷，基金會訪視後已先撥款暫紓窘迫。

【我有話要說】

薛麗雲說：「當年和老公生下女兒後，他幾乎在外生活，對我們母女沒照顧，10年前他才回家。我想他本性不錯，只是一時迷途，就再接納他。對我和女兒來說，有個完整的家，吃苦也無怨悔。」
