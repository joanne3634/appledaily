A3510 孝兒顧癱父 罹癌遲就醫

氣切無法言語 母淚眼相望

2014年11月02日

「阿海，要記得禱告，祈求上帝保守，讓你趕快好起來。」64歲張秀琴含淚叮囑么兒阿海（陳星海），36歲阿海因罹患口腔癌第四期，氣切無法講話，與母兩人淚眼相望。
報導•攝影╱仲芝蓉

張秀琴到醫院探望么兒，再匆忙趕回家，她憂心地說，家裡還有67歲丈夫，年初摔車腦傷中風，「他右半身癱瘓智商剩5歲，沒辦法一人在家，阿海9月住院後，我2、3天才能去探病」。張秀琴嘆氣說，丈夫大小便包尿布、上下床都要人抱，「丈夫個頭比我高又壯，阿海就是為了照顧爸爸才拖延就醫」。
張秀琴說，阿海今年6月就發現右頰長腫塊，雖心裡有數可能不好醫治，但2個大兒子和1女兒都在外成家，就阿海還未婚住家裡，所以只到診所開藥消炎，直到9月腫塊越長越大，已無法張口進食才到醫院檢查，「醫生說他拖延就醫，已惡化成第四期，做化療、放射線療至少半年，家中兩個病人，我怎麼不煩惱」。

地瓜葉配飯果腹

張秀琴的女兒阿萍說，4手足阿海排行最小，「爸媽早年在外地做木工，哥哥們都只有國中畢業就跟爸媽打零工，讓我和阿海讀高中，但鄉下工作機會少，我在加油站工作、哥哥們零工撐家，收入有限，每月湊5000、6000元接濟已盡力」。張秀琴心疼接著說，阿海4年前車禍後右手無法高舉，只能做附近修水溝零工，原本月薪就僅1萬元，現需治病無法工作，「時常就採鄰居空地的地瓜葉配飯，勉強果腹過一餐」。

3口月剩4千度日

當地公所急難承辦員說，醫院社工已協助住院期間請看護照料，村長讓張秀琴接附近小學早晚開關門工作，每月補貼3000元，加上公部門老人津貼、殘障津貼和她丈夫榮民就養金等，每月共約3萬元收入，但扣除兩病殘的營養品灌食費及醫療交通費、房貸支出後，剩約4000元3口生活辛苦。
公所正協助申請急難金，並轉介蘋果慈善基金會，基金會訪視後，也已撥款暫紓困境。阿海同事高先生說，「阿海工作之餘都要顧他癱瘓父親，很辛苦；現他也倒了，希望早日康復」。
