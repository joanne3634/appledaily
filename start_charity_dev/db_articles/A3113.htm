A3113 癌婦含淚化療 精障夫喊加油

二嫂心疼：她為家 累過頭

2013年07月02日

病房窗外雖有陽光，卻沒法讓正做化療的42歲魏麗娟感到和煦光亮，腹內膽道癌如針刺痛，但掛念在她心裡的是家中2個兒子和精障先生，這煎熬才是巨大扯痛。
報導．攝影╱向高彬

跟魏麗娟談話時，記者可感受她眼底掩不住的憂傷。她說，58歲先生阿龍車禍腦傷住精神療養院已6年，由她照顧現年13、11歲的2子，如今她罹癌，醫師告知這病不好醫，「以後怎麼辦？我一個人坐車到醫院化療，看著車窗玻璃裡面的我，一直掉眼淚。」
魏麗娟的二嫂心疼地說，阿龍本做水泥工，傷後傻愣會亂跑，麗娟只好送他安養，自己撐家，「這些年她為家裡累過頭了」。

兒哭：爸媽都病了

如今魏麗娟家靠每月低收補助共1萬3400元，當地公所已撥救助金2萬8000元，但一家4口病殘弱，再轉介蘋果基金會關懷。蘋果基金會訪視後已先撥款紓困。
魏麗娟12年前從印尼嫁來台，她3月時血肚痛，查出患膽道癌，「我跟孩子講媽媽生重病，孩子哭著說，怎麼爸爸病了，媽媽也病。」

「記住爸爸愛你們」

2子說會幫忙顧媽媽，「不讓媽擔心。」孩子的國小班導師誇2人乖巧懂事，常關懷並辦學產基金救助。
記者與魏麗娟到療養院探望阿龍，她說，先生病前老實顧家，「我教孩子要記住爸爸很愛你們。」阿龍見到妻兒露出笑容，魏麗娟跟他解釋好幾遍自己生病之事，這一刻他可能聽懂了，說聲「加油！」

【好友打氣】

趙愛金：「去年我先生生病，那段期間妳（魏麗娟）幫我們好多忙，陪我度過最低潮時候。妳放心，好人有好報，老天會保佑妳的。」
