A3183 養家男突逝 4老小無依

嬤孫無力殮葬 「生活打死結」

2013年09月26日

靈堂前，77歲阿錦婆（周高瑞錦）看著15、13及9歲的3個孫兒，跟著師父為往生的爸爸阿景誦經，她在一旁哽咽地要孫子向爸爸說，在天上保佑全家身體健康、平安順利。說著阿錦婆眼淚止不住簌簌流下，再也說不出話來。
報導•攝影╱韓旭爾

「阿景才44歲，他的命怎麼會這麼薄！」阿錦婆說，8月初時，阿景突發中風，開顱手術急救後，曾一度有好轉，在醫院治療1個多月，結果還是保不住生命。
阿錦婆說過去全靠阿景做零工掙錢養家，如今他去世，「家裡連辦葬禮的錢都沒有，幸好里長轉介葬儀社免費幫忙，才能送他一路好走！」

靠低收補助苦撐

當地里長說，葬禮簡單隆重就好，比較容易解決，未來阿錦婆要撫養3個孫子，政府低收補助僅有7800元，房租就需6000元，「一家老幼怎麼過日子，才令人擔心。」
阿錦婆除了排行老么的阿景外，還有5個子女，次子阿顯說，大家都要養小孩、還房貸，都沒能力，才替媽媽打電話到蘋果基金會求助，基金會訪視後已先撥款紓困。
阿錦婆說，阿景生前與妻感情失和，媳婦2年多前離家失聯，已通報失蹤，這幾年，阿景做工照養一家5口老小，「沒想到他比我這個老人早一步到西天去，他一走，家裡生活打死結，過不下去了。」阿錦婆邊說不禁心酸紅了眼眶，一旁念國三的15歲孫子小鴻安慰說：「阿嬤不要煩惱，我會代替爸爸照顧弟妹。」

床墊凹陷沒錢換

記者再來到阿錦婆家中，進房間一看，讀小四的孫女小鈴睡覺床墊老舊不堪，床面有好幾處大凹陷，小鈴說，晚上睡覺時，拿枕頭墊平，還是很好睡，「爸爸生前常念說，等到賺多點錢，再幫我買新床，說了好幾年，我知道爸爸賺錢很辛苦，可能太累才會中風去世。」
小鴻國中班導師說：「小鴻很有責任感，我知道他家境辛苦，午餐多的水果、甜點常多給幾份，他會帶回家，是個懂得照顧弟、妹的好哥哥。」
