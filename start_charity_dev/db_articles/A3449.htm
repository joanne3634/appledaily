A3449 跛媽獨顧兒 白飯配蛋果腹

2014年08月22日

「媽媽，我扶妳，小心走。」48歲的阿晴因脊椎毛病及雙膝嚴重退化性關節炎，痛到無力行走，得穿護腰、靠拐杖助行，14歲的兒子小勛貼心地在旁攙扶。阿晴說，一走路，「膝蓋痛到像被鐵錘打到，雙腿好像不是自己的！」報導．攝影╱韓旭爾

阿晴坐在椅子上揉著膝蓋說，近幾年來她在市場做清潔雜工，約2年多前發現膝蓋、脊椎都有毛病，要蹲坐、起身都有困難，「醫生說膝蓋軟骨磨損沒法恢復，只能多休息，但我國中都沒念畢業，有工作機會就得把握。」
阿晴市場工作的同事說，好幾次看見阿晴痛到受不了，搬沒幾袋垃圾就坐著休息，「勸她休息，但她說，家裡還有14歲的兒子要養，怎麼累的粗活也得做，看了很心酸。」

國三子過動須服藥

阿晴說，自幼父母離異，她隨媽媽生活，因遭媽媽同居男友家暴，國二時就輟學離家，約14年前與男友生下兒子小勛，小勛才3個月就與男友分手，這些年來，她做零工獨育小勛，「小勛自3歲就診斷出為過動兒，需服藥控制情緒，但他很貼心，知道家裡沒錢，白飯只配荷包蛋就吃得很高興。」
要升國三的小勛說：「我知道媽媽一直為錢很煩惱，三餐只要不會餓肚子就好，吃什麼不重要。」目前阿晴母子每月領低收入戶補助7300元，但房租需5200元，阿晴因經濟壓力，罹患躁鬰症，領有輕度精障證明，長年在身心科就醫服藥。
阿晴1年多前試著在膝蓋注射自費藥劑治療，當時她無力負擔每次2000多元藥費，透過社工轉介蘋果基金會，基金會撥款2萬元協力就醫。

母手術費10幾萬元

社工說，阿晴膝痛好轉後仍做粗活，導致病況惡化，醫生建議進行人工膝關節置換手術，自行負擔費用達10幾萬元，術後有半年需復健不能做粗工，故再轉介蘋果基金會，基金會訪視後已撥款暫紓困境。

基金會編號：A3449
