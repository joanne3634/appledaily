獨生女辭工照料5口陷困

2014年09月23日

60歲的張金花忍著腿傷疼痛，皺眉愁煩說：「今年7月上山抓青蛙不慎滑倒，為賺那100、200元摔斷手腳，女兒單親帶3子，還得暫辭工作回來照料，一切都怪我不小心。」報導•攝影╱仲芝蓉

張金花37歲的女兒陳雅慧告訴記者，離婚後，將14歲次子小程和10歲么子小忠留在鄉下由母親照料，自己帶著17歲讀高三長子小杰在外地小吃店工作，「我打工薪水3萬多元，每月寄1萬、2萬元回來撐家，但母親現在摔傷後起居坐輪椅、換尿布都需人幫忙，我只好返鄉」。

么子想當廚師撐家

住隔壁的表哥王俊傑說，雅慧很孝順，又是獨生女，沒人分擔，「部落裡都是遠親，我們偶爾送自家種的菜、養的雞接濟，但大家都打零工，籌湊3萬多元應急已盡力」。
晚飯過後，小忠收拾碗盤一邊說，阿嬤都在山上打零工，春天的時候挖箭筍，夏天在河裡抓蝦子或到水溝抓青蛙，冬天種高麗菜、蔥還有白菜，阿嬤抓來的青蛙或蝦子會賣給餐廳，有時候自己會跟阿嬤去，晚上7、8時去抓，3小時才抓十幾隻青蛙，賣100、200元，「阿嬤好辛苦，我以後想當廚師，這樣阿嬤和媽媽就不用幫別人打工，換我養家」。讀國三的小程回憶述，那天阿嬤獨自上山，跌到山溝裡還好沒撞到頭，自己打電話回家求救，「那時媽在外地，我只好找鄰居開車上山載阿嬤，阿嬤手腳流好多血，我和弟弟都好緊張」。

「不求富貴盼平安」

陳雅慧說，3個兒子從小都乖巧貼心，尤其長子小杰體諒大人辛勞，兩年前國中畢業就想工作，幫忙分擔家計，「不求他們將來大富大貴，只要平安長大、以後有份正當穩定收入就好，但我堅持至少都要高中畢業。」陳雅慧繼續說，小杰寒暑假都到車廠當技師學徒，現自己暫辭工，只剩小杰薪水1萬2000元撐家。
當地公所急難承辦員說，陳雅慧3子領有單親等補助每月共1萬4900元，加上小杰薪水共2萬6900元，扣照料開銷剩1萬5000元，5口生活辛苦，公所已撥急難金1萬元協助，再轉介蘋果慈善基金會，基金會訪視後已暫先撥款紓困。
