A3185 爸中風 10歲女助復健

難撐５口陷憂鬱 妻瞞斷炊陪醫病

2013年09月29日

「爸，我幫你綁好。」10歲的小嘉在醫院陪伴中風的44歲爸爸許富翔，趁著暑假期間當起小看護幫他穿衣，她知道爸爸病後心情很沮喪，有時講個小笑話逗他開心，「手舉高一點」，在協助復健的過程中，她盼爸爸手腳能趕快恢復力量。報導．攝影╱向高彬

小嘉的爸爸在4月中發病住院，至今仍左半身乏力，她說：「爸爸從躺在床上完全沒力氣，到現在可以慢慢一步一步撐拐杖走」，只是醫師說還要復健半年以上，「爸爸很擔心以後沒力氣作工。」

兼差拼還債致累倒

小嘉的40歲媽媽張淑娥說，先生病後住院這幾個月，由於身邊3個子女才12、10、7歲，需有人顧上學，所以一開始她白天先找看護照顧先生，晚上至工廠做大夜班，但醫護至今已用光微薄積蓄，又因先生病後嚴重憂鬱，她得陪著才安心，因此7月後她就留職停薪，帶著放暑假、兩個較大的女兒到醫院陪顧，只是沒了收入，更愁煩眼前房租家用，「在先生面前不敢多講，怕影響病情。」
張淑娥回述，與先生相親結婚，早年先生跑貨車南北送貨、但事業不順積欠上百萬元債務，這幾年夫妻租屋償債勉強過日，「經濟壓力大，他下班後再兼差，跑個幾小時沒執照的車載客，我曾跟他說別太累，但他想多掙幾百元趕緊還債，沒料到就累倒了。」

「當爸有責任護女」

採訪時，許富翔難掩失落說：「兩個女兒暑假時輪流來醫院陪我，看著她們，做爸爸的有責任保護她們，我必須努力做復健。」
許富翔的務農爸媽已8旬，雖擔憂兒子病況卻也無力幫忙；許富翔的大哥跑計程車租屋育兩子，他說，自家能力不夠，但常打電話關心弟弟。
目前張富翔已通過殘障證明，當地公所社會課也撥急難金2萬元，後續協辦社福補助並轉介蘋果基金會。蘋果基金會訪視後也撥款暫紓窘迫。
