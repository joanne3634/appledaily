A3500 病兄顧癌弟求生機「只剩我能讓他依靠」

2014年10月20日

51歲阿國邊幫罹患口腔癌4期的弟弟阿彬（林國彬）管灌，邊指著阿彬皮瓣移植傷口說：「剛見到弟弟時，他臉頰爛到穿孔，像有2個嘴巴一樣，把我嚇死了，術後還得做化療，要保住性命，還要吃很多苦。」
報導．攝影╱韓旭爾

現年49歲的阿彬語音含糊說，10多年前到大陸經商，後來工廠倒閉，證件又遺失，在大陸逾期居留，這幾年淪落成四處打工餬口的台流（指沒有固定工作的台灣人），3月時口腔潰瘍久不痊癒，到當地醫院就診，醫生診斷為口腔癌，醫療要8萬元人民幣（約40萬元台幣），「我沒有錢醫，本想向公安投案等待遣返，但擔心死在獄中，後來向台商友人求助，我沒能力衣錦還鄉，但如果要死，也一定要落葉歸根。」

為治病落魄回台

阿彬說，7月時友人找關係幫他偷渡回台，返台後投靠哥哥阿國，也向警察自首，檢察官調查他在兩岸都沒犯罪，為了治病才會非法入境，給予緩起訴處分，現在才能安心住院開刀治療。
阿國回述，爸爸早逝，除了阿彬外，還有一個妹妹遠嫁外地，他原本與媽媽同住，做工養家，6年前媽媽中風臥床，他無法穩定工作，照顧癱母壓力讓他罹患憂鬱症，領有中度精障證明，媽媽在1年前已病逝，這些年自己日子也不好過，靠低收補助每月1萬4100元度日，房租就需5000元，「如今阿彬面臨生死關頭，只剩我能讓阿彬依靠，我會盡力照顧弟弟，只盼老天爺能給他一線生機！」

自費就醫開銷大

醫院社工說，阿彬離台超過2年，已除籍，約明年3月才能以健保就醫，目前只能自費治療，院方已協助辦理欠費，使其將來有能力慢慢還，不會影響就醫權利，但阿彬要靠鼻胃管進食，每月需7、8千元，往返就醫交通開銷負擔不輕，故轉介蘋果基金會，基金會訪視後已撥款暫紓困窘。
