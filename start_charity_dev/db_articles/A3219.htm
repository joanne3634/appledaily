A3219 男罹癌當天 娶回腎病妻

「怕留她一人 想把腎給她」

2013年11月07日

45歲阿蓉去年5月因腎臟萎縮得終身洗腎，52歲先生潘萬福也接連罹患食道癌3期，屋裡房間一隅，病妻躺臥進行腹膜透析，癌夫因無法吞嚥喝著營養品，夫妻齊心面對癌病，但恆心毅力卻難抵生計無著的現實。
報導．攝影╱張嘉恬

阿蓉眼中充滿恐懼、心有餘悸說，夫妻皆病心理壓力大，「最近常夢到前夫把我鎖在車裡要開去一起撞電線桿，還用棍子毒打我後在半夜把我丟在山上……在路上看到很像他的人，更怕得要死」，阿蓉全身發抖，用緊捏在手中已掉落紙絮的衛生紙，邊擦拭汩汩流下的淚水邊說著。她說，前段婚姻經法院判決離婚後，經人介紹與現任先生潘萬福交往10多年，直到今年8月，「醫生證實他得食道癌當天，我和他登記結婚」。

月靠補助4700生活

阿蓉望著先生，眼神語調漸趨柔和，心，也卸下防衛，「我們做粗工沒什麼錢，但過馬路他一定牽我的手，10多年來沒1次忘記過」，她也提及有次做工遇到邊坡土石流，先生第一個反應是「保護我。安全感是他給我的最大財富，他沒結過婚，我現在能給的是陪伴和承諾」。
阿蓉說，她每日用腹膜透析洗腎8小時，也因腎臟萎縮導致貧血，曾在醫院照顧先生昏厥2次，目前她得每日陪先生放療，兩人無法工作，每月僅靠她殘津4700元，連支應房租7000元都不夠，還向人周轉了5萬多元。

感謝還來得及結婚

醫院諮商師說：「夫擔心妻想抱病工作，妻子煩惱經濟快『活不下去』，雖獲急難金共3萬元已用盡，醫院協助心理支持，並再轉介蘋果基金會」。
潘萬福說，醫生告知罹癌時，心中無太大衝擊，「只想到以後留阿蓉一個人怎麼辦？再想到還有什麼能為她做？就想把我的腎割給她」。他也說，兩人到戶政事務所登記結婚當天，「我得癌症卻笑得開心，因為有來得及結婚。希望還能有錢有體力拍結婚照，留下美麗回憶」。
