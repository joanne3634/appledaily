A3578 上工36小時 扛家男病癱

振作拼復健「孩子需要爸爸」

2015年01月26日

醫院大廳裡，39歲王志峯做完中風復健，面露疲憊，36歲太太阿華在旁推著他的輪椅，又不時轉頭看旁邊的5歲女兒小慈有沒有跟好，等會兒還要帶女兒去做語言早療。報導•攝影╱向高彬

阿華說，女兒不會說完整詞句，檢查有語言遲緩需做早療，但先生又在去年9月中風。「出事那晚先生突然昏倒，緊急動開腦手術，如今先生左半身沒半點力氣。」阿華說。

長子貼心幫父洗臉

王志峯沮喪地說：「我努力做復健，但進步很慢，坐在輪椅、大小便在尿布裡，我覺得自己像一個廢物。」
王志峯身高近180公分，從前他身強體壯做工，如今癱殘讓他自卑憂鬱，曾要太太推輪椅帶他到附近漁港，推下去一了百了，「我勸他，孩子還需要爸爸陪他們長大。」阿華說。
王志峯回述，自己國中畢業連英文字母都背不熟，做過玻璃廠工人，開小店賣藥燉土虱，「病前做拆裝棚架工人，一度連續36小時沒休息，開車在高速公路打瞌睡差點出事，不敢再做，未料現病倒了。」
相識20年的友人羅先生說，王志峯沒什麼專長，工作不穩定，「他很疼小孩，病後我去看他，他很煩惱怎麼養孩子長大。」
王志峯與阿華婚後生有現年9、5、3歲的2子1女，阿華坦言，兩人曾於2年前辦離婚，其間她在外做工寄錢家用，「先生要我回家重新團圓，我也很想孩子，去年中再回來生活。」
阿華心疼地說，有時她只是出門買點東西，但3歲小兒子一看到，趕緊跑來跟她說：「妳要記得快回來。」讀小三的大兒子小翔說，有段時間媽不住家裡，都是爸煮飯給他們吃，「現在爸爸病了，我拿小臉盆裝溫水給他刷牙洗臉。」

靠母做廚工養6口

王志峯是獨子，自家現與59歲母親同住，他母親說，如今6口家計房貸，靠她做廚工每月2萬餘元薄薪支撐。阿華娘家母照顧失智父難有餘力多幫。目前王志峯重度肢障證明剛核發，當地公所已撥急難金2.5萬元，後續協辦殘障及低收補助，並轉介蘋果基金會，基金會訪視後已撥款暫紓困。
