A3529 童喚癱父 「來看我比賽」

手握棒球套焦慮盼奇蹟

2014年11月26日

小巷弄裡平房，原本是讀小四的小鋒與他單親爸江昌宏的蜷居租處，但小鋒爸9月底中風癱瘓，如今暫於護理之家安置，小鋒由住鄰近的姑姑阿芬照顧；有時小鋒想爸爸，跑回之前父子同睡房間，感覺爸爸陪在身邊。報導•攝影╱向高彬

採訪時，小鋒由阿芬姑姑載去病房探望51歲爸爸江昌宏，小鋒喊了「爸爸」，江昌宏原本迷恍眼神稍有明亮，想伸手拉小鋒。阿芬說，小弟昌宏右半身無力、意識不清，「但他只要看到小鋒，精神就好多了，真是父子連心。」

需語言肢體復健

只是昌宏病倒、小鋒僅9歲，阿芬說，娘家共4姊弟各有家庭，沒法親自照料，大家先籌幾萬元，在昌宏出院後幫付房租和送安養，但愁往後長期安養每月3萬元，「他需要專業的語言和肢體復健，等他可以坐輪椅，我們再接回家照顧。」當地公所正辦急難救助，並待江昌宏通過殘障鑑定以辦低收；醫院社工協助醫護費約1.5萬元並轉介蘋果基金會，基金會訪視後已撥款暫紓困。
阿芬回述，昌宏原在深山老家做農，39歲娶印尼前妻，但夫妻不合，4年前小鋒媽在孩子5歲時離家，後由法院判離婚，這些年昌宏做綁鐵零工每月2萬元，與小鋒租屋生活；「我住他們隔壁，常常就叫他們父子倆一起吃晚飯。」阿芬說：「昌宏他每天早上起來會為孩子準備早餐，並送孩子上學；假日都在家陪小孩。」

「希望快好起來」

小鋒內向話少，採訪時，他拿出棒球手套和球說，自己是棒球校隊，擔任外野手，「這是爸爸買給我的，爸爸叫我要努力練習，等我參加比賽，他要來看。」小鋒班導何老師說：「最近小鋒心情焦慮，已關懷並辦學產基金救助。」
江昌宏的二哥患小兒麻痺，單親做工養兩女，「從前讀小學到初中，都是昌宏幫我背書包上學，我們感情不錯。最近我去看他好幾次，希望快好起來。」大姊阿芬的先生氣喘嚴重、長子重度智障，她靠搭婚喪帳篷零工撐家計，「小鋒是我這姑姑從小疼到大，雖然我們能力並不好，也要盡力把昌宏醫好，不然小鋒很可憐。」
