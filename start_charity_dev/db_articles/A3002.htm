A3002 癌父打工養兒 拖垮病情

獨力拉拔苦撐 盼稚兒學會照顧自己

2013年02月26日

53歲的單親罹癌爸王建成喚著天真的9歲獨生子小宇：「自己洗澡、衣服穿好。」他說：「我食道癌又移轉扁桃腺，只擔心他，要他快點學會照顧自己。」報導．攝影╱向高彬

王建成表情沮喪，嘶啞地說，前年他患食道癌三期，一度病況穩定，原以為躲過癌魔毒掌，未料去年底回診追蹤時，發現扁桃腺有移轉腫瘤，接連兩次切除手術，「當醫師說我扁桃腺癌，又要動刀時，心都涼了，不曉得再切下去，如何活下去。」他掀開上衣，腹部有10多公分長的手術痕。

棲身頂樓加蓋小房

如今王建成術後體弱，醫師建議多休養，不宜勞動，再進行抗癌療程，「我一個大男人就是要做工養孩子，去年我癌病稍穩，只要朋友有送貨零工，我就出門做事，但現在要好好養身體了。」
王建成大姊阿菊說，前年9月曾幫弟弟建成向蘋果基金會求助，當時基金會撥急難金救助，「他就是沒等病好又撐病做工，才又病了。」
也是單親的阿菊說，自己做代工養家，大哥4年前淋巴癌病逝，另有2兄2妹，一起照料9旬母親，兄妹這兩年只能幫建成父子一些，難長期協助。
王建成父子居住在過世大哥家公寓頂樓加蓋小房，空間僅能讓父子容身，他回述，之前做貨運司機，17年前娶大陸妻，妻子曾兩次流產，9年前生下小宇，中年得子讓他開心，但妻子卻出現產後憂鬱症和精神分裂。

「不可以讓爸生氣」

王建成接著說，當時送妻子住院療養都無改善，後來丈人將妻子接回大陸，婚姻由法院判離，「這幾年我做工養孩子，雖有欠銀行卡債，但還能勉強度日。只是我這次一病，不知何時才能健康好轉。」
採訪時，小宇雖調皮，但爸爸說的，他會做好，「爸爸生病了，我不可以讓他生氣。」
小宇班導師葉老師說：「爸爸對他很照顧，我們幫孩子申請獎助學金和早餐券，盡量幫忙。」與王建成認識20多年的前同事饒女士說，建成一心一意要把孩子照顧長大，罹癌後菸酒檳榔都戒了，脾氣改很多，「我常鼓勵他，你這麼疼小孩，老天會保佑你的。」
目前王建成每月領三款低收殘障補助共7300元，難敷父子生活醫病開銷，當地里長表示會再協辦救助。蘋果基金會獲悉訪視後，也已撥急難金暫紓窘迫。

基金會編號：A3002
