A3364 父猝逝 2女伴母拾荒

2014年05月09日

狹小公寓樓梯間，37歲陳俞蓁和兩個女兒整理她拾來瓶罐，她跟記者說，以前是先生阿本帶她撿，但先生3月過世，如今她一人拾荒，「我不太會認路、走不遠，現在撿不到以前一半。」報導•攝影╱向高彬

讀小五的12歲次女小蓉講話像小大人，她說，媽媽有憂鬱症精神問題，「她容易忘東忘西，有時一個人關房間哭，爸爸死後，她常哭說不知道以後怎麼辦。我和姊姊說，我們會照顧妳。」陳俞蓁從菲律賓嫁來14年，採訪時，她易分心恍神，需靠她兩個女兒提醒媽媽。

女做CPR救不回

小蓉的56歲爸爸3月初心肌梗塞猝逝，小蓉回述，那天媽媽回房睡時，發現爸爸癱躺床上沒反應，趕緊叫她來，她一摸爸已沒呼吸心跳，機警地打電話叫119，當時媽媽和讀國一的姊姊嚇哭了，「學校教過急救訓練，119的人講電話教我怎麼幫爸爸做CPR心肺復甦術，我一直做，等到救護員送急救。」她難過地說，根本沒想到害怕，但「還是沒有救活爸爸。」小蓉爸爸後事已由當地慈善救助及公所協力簡單辦妥。
小蓉指著牆角幾根長短柺杖說，那是爸爸生前用的，爸走路跛腳，因為8年前車禍斷腿，也因此沒辦法再做廚師。這幾年爸打零工、撿瓶罐，加低收殘障補助養家，「他喝醉時會罵媽媽。清醒時候，對我們很好。」
13歲的大女兒小瑾告訴記者說，爺奶及大伯都去世，聽爸講以前和4個姑姑不合，已好多年沒見過面，不知聯絡方式，這次葬禮都沒人來，「我們只跟媽媽的表姊家有往來，爸爸喪事，是她們幫忙的。」
陳俞蓁的表姊呂葛麗絲是蘋果基金會今年初服務個案，她本身患鼻咽癌，丈夫呂信良說：「葬喪部分用公所撥的急難金和幾位朋友協助簡單辦完。但我和太太擔心她們往後，代跟蘋果基金會求助。」基金會訪視後已撥款暫紓困。

先協力媽媽就醫

小蓉班導簡老師說：「我們會持續關心孩子家情況。」當地社工說，目前陳家每月領三款低收補助共13400元。
現階段先協力媽媽穩定就醫，再輔導學習工作技能，安定生活。
