A3311 稚女憂癱母 「不好怎辦」

2014年03月05日

「媽媽眼睛為什麼都閉著？」讀小三的小甄好奇問一旁讀小六的姊姊小瑜，小瑜回道：「噓，媽媽在睡覺。」小甄說：「但媽媽已經睡好幾個月了。」姊妹倆討論著媽媽病況，去年底她們36歲的母親王鈴秀中風癱瘓，至今仍昏迷無意識。
報導•攝影╱江品璁

王鈴秀38歲的先生張信鴻說，太太去年11月27日早上，講話突然斷斷續續，那時還沒警覺到是中風前兆，沒想到當晚就右半身癱瘓，12月中大腦又二次出血，昏迷至今，「醫生說，妻子腦血管先天狹窄，又有高血壓病史，才會這麼年輕就發病。」張信鴻說，現在每天在醫院顧太太，平常就喊喊她的名字，看她會不會醒來，但總是失望。
一旁來探視的65歲婆婆阿鳳嬤說，次子在醫院顧媳2個多月，跟著瘦5公斤，「我心疼次子沒日沒夜地照顧，偶爾也來醫院換班，讓他能夠喘息。」

嬤愁沒醫藥費

在旁的小甄擔心問道：「媽媽如果不會好怎麼辦？」阿鳳嬤回：「小孩子不要亂講話，媽媽會好啦。」小甄說：「但是媽媽只是睡覺，都不會說話。」說完後小甄紅了眼眶，阿鳳嬤聽了難過，也流淚說：「媳婦怎麼叫都沒反應，也不知道要拖多久。」
阿鳳嬤說，現在次子在醫院顧媳婦，自己跟先生幫忙照顧孫女，都沒人可工作，「2個孫女才讀小學，媳婦要醫藥費，找嘸錢。」
阿鳳嬤接著說，媳婦是長女，親家公多年前也因中風過世，剩下4個弟妹也都各有家庭，自己則生4子女，與先生住在長子阿勇家，幫忙照顧孫女，2個女兒都已出嫁，長子也有自己的負擔，經濟協助有限。
醫院社工說，王鈴秀原來當品檢員，先生當水泥工，是一般的小家庭，但現在一家頓失經濟來源，故轉介蘋果基金會，基金會獲悉訪視後已撥款紓困。當地里長說，已協辦急難金。

同事嘆很難過

王鈴秀的同事阿琦說，王鈴秀是個很活潑、顧家的人，也沒聽說過有什麼病痛，很意外突然腦中風，「唉，孩子還那麼小，真的很難過。」
