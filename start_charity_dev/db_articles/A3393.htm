A3393：撐家漢倒 離島跨海抗癌

3子女無依「妹妹吵著想爸爸」

2014年06月13日

40歲的郭考政躺在病床上，氣若游絲地說：「以前我很壯、很少生病，沒想到今年2月突然得血癌，高燒不退、嘴破過敏狀況都來，人生真的難預料。」他43歲的妻子阿雲在旁照料，愁容滿面。
報導•攝影╱仲芝蓉

採訪時，郭考政剛抽完骨髓做檢測，他強振精神，不想被癌病打倒，細述他最放心不下的4名兒女，他說，小女兒小伶才讀小二，2個兒子小旭、小承分別讀國三和國一，只有22歲大女兒已出嫁，育一幼子才3歲，「我們是離島居民，平時靠鋪路零工和妻子在學校當廚工，薪水合計約2萬多元，2年前我考了救生員執照，帶浮潛客人賺外快，夏天能多賺5000、6000元，我們都愛孩子，原想多努力，平安拉拔他們長大就好。」
阿雲接著說，今年2月丈夫久咳不癒，島上衛生所醫生說要到大醫院檢查，「我們搭機到台灣，早上9點多的飛機，到醫院都已下午4、5點。」郭考政繼續說，離家一住院就是2個月，子女只好暫由島上二姊和五姊照料，「4月初清明連假，孩子們第一次來探病，我二姊帶他們來，光交通費就近萬元。」

長子悲：爸瘦了

阿雲指著牆上小伶探病時畫的祝福圖，寫著：「祝爸爸早日康復、生日快樂。」2人回憶當時和子女難得的相聚，難掩激動情緒、淚濕眼眶。記者電訪小旭，他說：「去醫院時，看到爸瘦了感覺憔悴，我們很難過，尤其妹妹常吵著想爸爸，我們每天都通電話，我們在姑媽家會幫忙洗衣服、照顧自己，請他不要擔心。」
醫院社工說，郭考政為就醫已在醫院附近租屋，並通過低收入戶，每月可領補助7800元，但扣房租後就剩2800元，雖醫療及骨髓移植可減免，仍難敷流質營養品每月約9000元開銷，故轉介蘋果慈善基金會，基金會訪視後，已撥款暫紓困境。

離島居民怕生病

郭考政的五姊說，手足共8人其中三姊過世，考政是老么，「我們都在島上，生活不寬裕但簡單可過，最怕就是生病。」當地部落文化基金會執行長賴美惠說，島上居民生活都用山泉水、自種地瓜芋頭、捕魚為食，基本生活不成問題，但若需就醫、交通等額外開銷，生活就困難了。
