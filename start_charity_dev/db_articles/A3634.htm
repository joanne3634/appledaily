A3634 癱孫甦醒 爺嬤苦無醫費

母已失聯嬤：不放棄他

2015年04月12日

「謝俊瑋，聽到我叫就舉手回答！」只見17歲的謝俊瑋緩緩舉起右手，臉上露出微笑。「他3年多前還像是植物人，1年前醒了慢慢好起來，我很安慰。」阿嬤洪素女說，但很煩惱沒有復健和醫療費。報導•攝影╱孫敏聿

現年61歲的阿嬤洪素女說，3年多前就讀國二的謝俊瑋，騎機車載學長遭汽車撞倒，學長骨折輕傷，俊瑋呈植物人狀態，肇事者經判決雖須賠償700多萬元，至今僅拿到15萬元，對方疑似脫產，空有債權憑證但對方沒財產、沒收入，無法兌現，還好當時有學生平安保險與強制險等共280萬元。「但3年多來已用盡於看護照料費。」

月耗10萬壓垮生計

洪素女說，俊瑋車禍後須隨時抽痰、鼻胃管灌食，家人缺乏照顧經驗，俊瑋又常發燒須急診送醫，每月花6萬元請專業看護，加上管灌飲品、尿布耗材等，每月至少花費10萬元，2年前改請外籍看護，每月約花2萬5000元。她說，今年1月已請不起外籍看護，又買不起營養品，俊瑋現已能喝粥。「眼看他就快好起來，我不會放棄他。」
一旁60歲的阿公吳清嘉說，他生有3女1子，但長女出嫁，么女與么兒都在外租屋、受僱賣衣服，「每月一人拿2、3000元回來，也沒法再多給。」俊瑋是次女所生，從小由他一手帶大，次女則帶一子在外生活，「次女之後離婚失聯，2年前俊瑋就讀國二的弟弟阿煜沒人照顧，我也接回來顧。」他說，13年前他車禍撞斷腿，即領有輕度肢體身障證明，僅在鄉間打零工維生。「去年中，俊瑋奇蹟般脫離呼吸器，整個人醒了過來，現在只差還不會說話、左半邊肢體還是麻痺，得更努力復健才行。」

「悉心照顧才恢復」

物理治療師林小姐指出，阿公、阿嬤對俊瑋照顧無微不至，「俊瑋能恢復不是奇蹟，是阿公阿嬤和他一起努力的成果。」
當地村幹事表示，吳家去年為中低收入戶，月領補助1萬1600元，已協助送件申請低收入戶資格，盼再增加一些補助，蘋果基金會獲悉訪視後也撥款紓困。
