A3398：嬤罹肝癌 難顧50歲癱兒

嘆「我先走怎麼辦」

2014年06月19日

醫院病房裡，69歲的阿燕婆（麥富燕）努力聽著護理師說明，學習如何幫腦傷成癱的50歲兒子阿雄（藍正雄）翻身、拍背、換尿布及管灌營養品，做簡單復健，阿燕婆表情認真的說：「我要好好學，兒子回家後，才能好好照顧他。」報導．攝影╱韓旭爾

看著兒子癡呆癱軟模樣，阿燕婆無奈說，未婚的阿雄10多年前因跌倒腦受重創，從此右側手腳無力，常發作癲癇，但還可以自己走路、洗澡，只是不能工作，沒想到4月初他又在住家附近摔得頭破血流，經鄰居發現送醫，這次換成左側手腳癱瘓，「雖醒了，意識不清楚，只會說簡單句子，但幸好還記得我是他媽媽。」

靠津貼1萬元度日

阿燕婆說，阿雄管灌營養品及尿布開銷每月需上萬元，出院後還要定期返院回診復健，復康巴士每趟來回就要500多元，「不知該怎麼辦？」蘋果基金會獲悉後，已撥款暫紓阿燕婆母子生活、醫療困窘。
阿燕婆說，丈夫早逝，除了長子阿雄外，還有2個已婚兒子，這些年，她與阿雄相依為命，靠老農年金和阿雄殘障津貼共1萬1400元度日。阿雄住院期間，阿燕婆體力不勝全天照料，晚上做零工的次子阿義下班後會來醫院幫忙照料。阿義說，自己家中還有2個就學孩子要養，哥哥意外後，向人借了4萬多元負擔醫療開銷，「撐到現在，我也沒有辦法了！」阿燕婆嘆口氣接著說，自己3年前還發現罹患2期肝癌，還在追蹤治療，「萬一我比阿雄先走，他該怎麼辦？」

燒柴煮飯省瓦斯費

傍晚記者跟著阿燕婆回到家中，阿燕婆先到住處附近摘取自種蔬菜，之後用木柴燒火煮飯，「家裡不是沒瓦斯爐，只是瓦斯很貴，平時就要能省則省才能過日子。」當地家婦中心社工說，阿燕婆母子因名下祖傳土地超標，無法獲低收，母子倆都有病在身，「會協助申請免費居家服務，每星期3天，分擔阿燕婆照顧重擔。」

基金會編號：A3398
