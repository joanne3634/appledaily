A3207 血癌夫驟逝 「帶著牽掛走」

病發1個月就走 3兒嗷嗷待哺

2013年10月28日

「阿生從發現血癌到去世，不到1個月，走的太快了！」24歲阿雯噙著淚水說，她7月23日生下么女小翎，當時29歲的阿生滿臉喜悅說自己很幸福，「要更努力工作才能養大3個孩子。」報導．攝影╱韓旭爾

靈堂前，阿雯說，8月初，阿生就說頭痛欲裂，掛了急診治療，居然說是血癌，9月就走了。「臨終前，阿生一直對我說：『我愛妳，對不起妳，要堅強一點。』我能感受到阿生是帶著牽掛走的。」
阿雯說，夫妻育有3子女，老大小茹4歲，老么才3個多月，過去阿生做建築零工為生，她在家照看孩子，今年3月時，阿生有出現嘴破、牙齦流血，以為火氣大，醫生說那些都是血癌病徵。

「一輩子都心痛」

阿生死後，喪葬費用約15萬元，全靠國保補助及里長轉介慈善會協助才解決。只是未來家裡沒了經濟支柱，「最小的2個孩子都還要喝奶、包尿布，每月就要1萬元，3個孩子還離不開我身邊，我也沒法工作掙錢。」
目前阿雯公婆雖同住，但54歲的公公貴郎伯中風癱瘓多年，起居需婆婆照料。貴郎伯說：「恨不得代替兒子承受這生死關。」一旁阿生53歲的媽媽素卿婆說：「白髮人送黑髮人，我這個當媽媽的，一輩子都會心痛。」阿生大哥阿濃說，手足3位，他與弟弟都在外地做建築工，收入不穩，負擔爸媽生活費就很吃力，「要照顧弟媳一家，在經濟上能幫的不多。」當地里長說，阿雯現領有育兒津貼共5000元，實在辛苦，蘋果慈善基金會獲悉訪視後，已先提撥急難金暫紓阿雯一家生活。

爸爸去當天使了

採訪時，阿雯4歲女兒小茹在靈堂旁跑跳，全然不知爸爸已去世，只對記者說：「爸爸去當天使了。」一旁的阿雯聽了，臉上的淚再也沒歇過，阿雯說，這幾天她覺得阿生未離開，「靈堂前祭拜滿杯的保力達B，一下子就變少。」阿雯相信阿生在天之靈會保佑她平安順利養大3個孩子。
