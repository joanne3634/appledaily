A3148 兒望癱父 哭喊「快醒」

單親撐家打零工 腦溢血摔車

2013年08月14日

「爸，快清醒。」16歲阿恩在病床邊，望著腦溢血摔車的父親吳萬富亂語躁動，阿恩呼喚59歲的父親，心疼與他單親相依的父親殘癱，又害怕父親會不會突然不在了，一時忍不住心酸嚎啕痛哭。報導•攝影╱向高彬

阿恩說：「爸有高血壓、糖尿病，硬撐打零工，5月底騎車時腦溢血摔車，到現在都意識不清、右邊手腳沒力。」採訪時，見阿恩父親接氧氣機輔助呼吸，仍不停亂吵亂鬧。
阿恩說，父親之前做水泥工，與印尼母親生下他和兩個妹妹，但父親8年前曾因工作不慎從高處墜樓撞碎左肩，「爸媽常為沒錢爭吵， 5年前離婚，媽媽帶兩個妹妹回印尼，之後再沒聯絡」。

被笑沒媽羞愧逃學

阿恩這幾年跟著父親租屋度日，但他說，爸上年紀了，平時每周1、2天零工難敷生計，常見爸跟朋友借錢，拖欠房租被趕搬家是常有的事；他懊悔地說，去年國三時自卑家窮、被同學訕笑沒媽，那時他叛逆常逃學，「爸對這事很生氣，他會罵、也耐心勸我好好讀書，不要像他只能做雜工」。阿恩半年前心性漸定，經親友介紹做機車學徒，每月近萬元工錢貼補家用，並打算今年半工半讀再返校園，「爸很疼我，他出事後我才覺悟，更怕如果他不在了，怎麼辦？」

醫院顧父險憂鬱症

阿恩的姑姑吳秀盆說，這幾年常不時一、兩千接濟他們，現在哥哥病倒，剩阿恩一個大孩子，哪有能力扛家，最近會接阿恩來一起住，「本來他在醫院顧爸爸，但精神體力受不了，我看他都快憂鬱症了，幸社工幫請看護照顧」。
吳萬富的老友阿清說：「這幾年他為了健康，菸酒檳榔全戒了，一心想顧兒子長大。」社工已介入關懷，公所撥急難救助金2萬元，並通過三款低收，並再轉介蘋果基金會，基金會訪視後撥款暫紓困。

【暖流見證】

阿恩的機車行老闆姚鎮維：「這孩子吃苦肯學，再2年就能出師。他現在沒駕照，我請師傅每天載他來上工。我也是窮苦出身，鼓勵他忍耐堅持，才會成長。」
