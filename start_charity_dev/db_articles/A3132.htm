A3132 男抗癌魔 「妻兒要靠我」

精障妻無力顧家 萬元補助養3口

2013年07月26日

46歲蔡嘉興因胃癌術後傷口疼痛，皺著雙眉，記者問及倚靠什麼面對未來，一旁罹精神分裂的42歲前妻阿彩，在旁突大聲說：「太太的愛啦！」蔡嘉興無奈說，阿彩罹精障、16歲兒子患有學習障礙，「我只能盡力和疾病奮鬥」。報導．攝影／張嘉恬

採訪時提及前夫罹癌，阿彩未做回應，但在訪談間始終掛著笑容，並不斷往返廚房與客廳之間，蔡嘉興說，阿彩婚後生下兒子便精神分裂病發，「當時她吵著離婚只好依她，現在有服藥控制病情」。
里長說，蔡嘉興和阿彩法律上雖無婚姻關係，但同住與夫妻無異，目前僅靠低收補助1萬6700元不敷3口生活，「已協辦急難金1萬元」。

兒貼心分擔家務

蔡嘉興去年底罹胃癌，已切除2/3的胃並清除淋巴，今年5月癌細胞擴散至腸子，他說，在加護病房時，「夢到身體冰冷，和人打架，可能是和死神搏鬥吧！既然過了關，老天讓我活下來，就還有責任未了，阿彩和兒子不能自力更生，還要靠我」。採訪期間，16歲兒子小傑始終靜默在旁，問他是否擔心爸爸時，小傑停頓數秒，不知如何回答。蔡嘉興說，小傑雖因學習障礙，課業表現不像一般孩子，但仍會貼心地主動分擔家務。

手足病弱難接濟

蔡嘉興二姊說，4手足中大姊得糖尿病，自己也罹癌，大哥靠零工苦養兩子女，兄姊們只能偶爾協助幾千元，「早年大姊曾助弟買下現居，他再慢慢攤還，現在弟弟無力支付了，大姊也能諒解，只是我們都盡力了」。蘋果基金會訪視後已撥款紓困。

【無法計算的人生】

蔡嘉興說：「小學畢業到布廠做童工，當兵回來26歲結婚，努力賺錢養家，以為盡了做丈夫、父親本分，沒想到太太精神出狀況，我會罹癌，人算不如天算，想再多也沒用，沒米就煮番薯湯，船到橋頭自然直，現在我能做的是配合醫生好好治療。」

基金會編號：A3132
