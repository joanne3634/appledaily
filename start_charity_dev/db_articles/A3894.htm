A3894 癱父忍淚 為子女拼復健

9歲兒打氣：長大會養你

2016年04月01日

49歲石德生半年前摔傷導致脊髓損傷，現無法行走、控制排泄，為避免雙腳萎縮，他拄著四腳助行器，在客廳連續站了1小時，他說，要把握復健黃金期，「為了3個小孩，只能懷抱希望，不能絕望。」報導•攝影╱張嘉恬

石德生讀國一的13歲長女小恩說，爸爸受傷後，醫生說他以後可能不能走路了，「爸爸那時很難過，吃不下飯、不想說話，我陪他撐著助行器慢慢復健。」小恩也說，她和12歲讀小五的妹妹和9歲讀小三的弟弟，常在睡前一起禱告，求主耶穌讓爸爸趕快站起來走路，採訪時，9歲小聖天真說：「把拔不要擔心，我長大會賺錢養你。」桌上玻璃也壓著他在爸爸住院時做的卡片，石德生說，孩子是他復健路上最大力量。

妻：只能往前看

石德生的35歲妻子阿琴說，先生擔任司機載運回收物，她做麵店代班零工補貼家用，半年前先生工作時自高處摔下無法自理生活，她得照料先生起居沒辦法打工，目前靠低收補助及勞保職災補償費每月共3萬多元，扣除房租1萬元後餘約2萬4000元5口生活。
阿琴說，醫生說先生脊損很難再正常行走，但叮嚀要盡力復健，「醫生鼓勵說有人做復健5、6年甚至10多年，可不用輔具自己走的」。她說只能往前看、往前走，「剛開始煩惱錢，我一直哭，小孩看到都說哭最後一次不能再哭，怕影響小孩，只能擦乾眼淚。」

孩子自動做家事

石德生同事說，石德生認真工作、愛妻疼兒，石太太學歷國中畢業，也要兼顧子女，平常只能兼差補貼家用，「石德生也常預支薪水，生活本就辛苦現在更難熬。他是那種默默做事、不和人計較的憨厚老實人，出事後，他也只哭了一天，接下來就認真復健。」
教會高姓傳道師說，石家信仰虔誠，孩子在爸爸受傷後，自動自發幫忙家事，石太太在傷心過後就勇敢堅強，說先生倒了自己才要站更穩，靠信仰面對未來，「教友常家訪代禱，也幫子女進行課輔。」公所社會課表示，已協辦急難金並轉介蘋果慈善基金會S等民間單位，基金會訪視後已先撥款紓困。
