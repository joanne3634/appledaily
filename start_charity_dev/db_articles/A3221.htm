獨育2子 難上工「生活怎過」

2013年11月10日

49歲的盧美珠皺著眉，強忍腰部開刀後的疼痛勉強起身，每個動作都小心翼翼，深怕一點閃失傷得更重，她愁容滿面說：「今年8月底連日大雨，家中漏水，我不小心踩到積水滑倒，怎知這一摔這麼嚴重。」報導•攝影╱仲芝蓉

「我怕開刀後不能工作，拖5天才就醫，原以為多休息就會好，但越來越痛連腳都不能動，醫生說再不開刀會壞死癱瘓。」盧美珠繼續說，丈夫6年前肝炎病逝，留下現讀大三的兒子阿甫和高三女兒小綾，自己在餐廳端盤養家，薪水1萬元加上低收補助1萬1800元共2萬出頭，勉強3口生活，「現至少須休養到明年2月，醫生說要盡量躺著不能外出，但不去工作生活怎過」。

須借便椅才能如廁

阿甫回述，爸爸以前開計程車，病倒後1個月就走了，那時妹妹才小學，媽媽真的很辛苦，「這次她摔傷我在醫院照顧，暑期打工只好請假，開學後學校餐廳20元兩樣青菜，再多添幾碗免費白飯，就這樣過一餐」。
訪視時，盧美珠隨身抱著一根用來做掃把柄的竹子當拐杖使用，她說，醫院幫忙借了輪椅，而家裡廁所是蹲式馬桶，還好鄰居有老人家留下的便椅，就借來用，「能省則省，孩子們在外地讀書，常打電話叮嚀我別再跌倒，很貼心」。
當地公所急難承辦員表示，盧美珠家住偏遠鄉間交通不便，回診叫車來回約2000元，後續每兩周回診一次，低收補助扣車資、再扣除房租2000元，就剩下約5000元3口過日子，公所已撥急難金3萬元協助，再轉介蘋果慈善基金會，基金會經訪視後已撥款紓困。

母甫中風手足難助

盧美珠的二姊說，家中6手足，美珠排倒數第二，自己和大姊已出嫁，3個弟弟的孩子都還在讀書，「美珠滑倒開刀後一個禮拜，大姊也因長年務農負重尾椎骨裂傷開刀，一樣只能臥床，美珠向互助會借5萬元繳醫療費，還欠6萬元，我們好不容易湊齊才幫她繳，沒想到78歲的老母親接著中風，真是屋漏偏逢連夜雨，我們接濟吃的沒問題，就怕生病、受傷要看醫生，尤其坐車的錢，是打零工整天的薪水，實在無餘力負擔」。
