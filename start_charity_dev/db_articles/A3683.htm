A3683 愛家男半癱 掛念3稚兒

妻伴復健積蓄用盡「怎養孩子」

2015年06月16日

「求主為我的家庭、我的病，以主之名帶路，復健順利。」33歲張正平每晚祈禱求康復，他去年底右腦出血中風，如今左半身手腳無力，由34歲太太姜秀珍陪伴住院復健，夫妻倆最掛念3個小孩。報導•攝影╱向高彬

姜秀珍說，先生最疼孩子，「他是小孩第一、老婆第二，就算孩子做錯事，對孩子都是輕聲細語講道理。他每天都說想小孩，擔心以後不能工作，怎麼養孩子。」

娘家幫忙托兒

姜秀珍說，先生中風病前開聯結車，出事那天在高速公路突覺頭暈，撐著開下交流道，助手叫救護車送急診，動開顱手術住院復健至今。原本一家在外租屋，如今姜秀珍陪夫復健，只得將3個11、6、4歲的孩子暫托深山部落娘家照顧。
回想夫妻結緣，姜秀珍說，兩人原是國小同學，但她小學畢業雙親離異，她隨父遷居就此失聯。長大後她曾有一段失敗婚姻，單親撫養現讀小四大兒子，沒想到後來兩人輾轉聯絡，「先生告白說他從小學就喜歡我，我們婚後再生一對小的，他對我大兒子當自己親生一樣疼愛。」
她又說，先生不菸不酒，「他本來是司機助手，婚後努力考到聯結車駕照，近幾年才把家裡債務還清，他卻病倒。」她又說，事後才知車行未投保，雙方仍調解未果；先生病後這半年用罄微薄積蓄，如今房租家用無著。

待做殘障鑑定

採訪時，姜秀珍的娘家小弟帶3孩子到醫院探望，張正平說，看到孩子，病好一大半。姜秀珍說，先生須再復健半年，之後先生養病顧小孩，由她做工撐家。
張正平的生父早歿，他母親說，正平是大兒子，平常她幫單親二兒子帶3幼子，自家緊絀，先前都還是正平拿錢幫家裡，「聽到正平中風，我一路難過哭到醫院看他。」姜秀珍的娘家母親說：「我先幫顧3個外孫，只是小外孫常哭問爸媽什麼時候來接他們，我聽了也很難過。」
教會牧師說，夫妻是虔誠教友，「我們常去探望祈福。」目前張正平待做殘障鑑定，當地公所已協辦急難救助並轉介蘋果基金會，基金會訪視後已先撥款紓困。
