A3859 母打雜養家 女休學顧癌父

3口月靠萬元餬口「想到就心酸」

2016年02月04日

新春將至，56歲阿泉（吳忠泉）卻因肺癌體弱臥床，神情落寞，阿泉說，因工作不穩，經濟狀況不佳，「但過年時至少還能全家在一起四處走走，今年我只能躺在床上過年，想到就心酸。」報導•攝影╱韓旭爾

中午時，16歲小瑾細心幫罹肺癌臥床的爸爸阿泉擦澡、張羅午餐及吃藥，阿泉看著女兒無奈說，小瑾原在外地讀建教班高二，4個月前為顧他而休學，「現在高中沒畢業怎生存？都是我拖累她。」
小瑾說，爸爸5個月前發現罹4期肺癌，無法自理生活，媽媽邊工作邊顧爸爸，很擔心她會累倒，加上學校離家遠，來回要2個多小時，「我上課、工作都心神不寧，我很想多點時間陪在他身旁，才會決定休學。」

「盼年年都能團圓」

小瑾說，希望爸爸能治療順利，「未來很多年的春節，我們一家都能團圓過節。」
小瑾學校老師說，孩子要兼顧課業、工作，原本壓力就不小，如今小瑾又遇家庭變故，贊成她先休學回家陪爸爸，但也鼓勵她將來無論如何一定要完成高職學業才易找到工作。

手足生活苦難伸援

採訪時，小瑾49歲的媽媽阿暖（王秀暖）做完洗碗工回家，阿暖說，夫妻僅育小瑾1女，過往她與丈夫都做零工，家中每月收入僅有2、3萬元，扣除3000元房租3口生活，阿泉病倒後，她做洗碗工、到流浪狗收容場打雜2份工每月才可掙1萬餘元，不夠應付阿泉就醫交通、營養品、尿布花費，「已向人借3萬多元應急，但也找不到人可借了。」
阿泉妹妹說，手足雖有10人，但有7個沒在工作了，大家生活都辛苦，「我在家顧孫子，生活靠子女支應，只能偶爾拿個3、4000元協助哥哥一家付房租。」阿暖在流浪狗收容場工作的同事說，這裡工作時間比較彈性，方便阿暖處理生病丈夫的突發狀況，只是月薪才6000元。
當地公所社會課長說，阿暖一家為中低收入戶，補助僅有阿泉輕殘津貼每月3500元，已核撥1萬元急難金，並協辦低收入戶中，蘋果基金會獲悉訪視後在年關前先提撥急難金紓困。
