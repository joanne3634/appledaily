A3896 男突癱 妻愁4口「全亂了」

獨力難持夫與女「不知家能撐多久」

2016年04月04日

39歲的阿嬌（阮氏嬌）吃力地從輪椅抱起中風、左半身癱瘓的先生阿仁準備復健，言語受損的阿仁只能紅著眼望著太太，阿嬌說：「先生是家中唯一經濟支柱，他倒下後一定很著急我和女兒的生活，無奈有口難言！」報導•攝影╱楊光翔

阿嬌表示，從越南嫁到台灣近20年，和55歲的先生育有讀高一、國二2個女兒，「先生靠做電器維修養家，收入看接單量，約3、4萬元。4口生活少出遊、少外食，長期省吃儉用存了近百萬頭期款，2年前終於貸款買了自己的房子，當時先生提到未來生活將好轉，非常開心，但看到他現在的無助，實在心酸。」

忙照護難找工作

阿嬌接著說，買屋後夫妻利用自宅樓下兼做電器買賣，「增加的收入雖連房貸7000元都不夠，但有自己的小事業，總是個開始；且女兒漸大，各有技藝專長，高一的大女兒小郁美髮造型曾得全國特優，國二的小女兒小方繪畫也很棒，她們是我們最大的成就！沒想到1月時先生突然倒下，生活全亂了。」
阿嬌大女兒小郁導師說：「小郁不僅術科優秀，學科表現亦佳。知悉家中狀況後，已協助申請校內急難金，但幫助有限。」
記者之後隨阿嬌返家，甫進門，讀國二的小方就跑來說：「媽媽，姊姊今天人不舒服，下午還請假回家休息。」讀高一的小郁則向媽媽示意，休息後已好轉不用擔心。
阿嬌聽完女兒說話一臉不捨，轉向記者說：「我只有一個人，實在無法兼顧先生與女兒，又無法找工作，不曉得這個家還能撐多久？」

沒勞保難付醫費

阿嬌鄰居阿桃說：「雖然當厝邊才2年，阿嬌夫婦待人親切，為生活很打拼。我能做的是有時幫忙替她看一下2個女兒，很替他們家計煩惱。」
阿仁大姊阿卿說：「弟弟因自己接工作沒投勞保，雖有私人保險，但僅住院每天理賠500元，根本不夠用，但4個手足都各有家庭，幫忙也有限。」因阿仁身障鑑定需等待半年，現家中僅獲馬上關懷急難金1萬5000元，醫院社工獲悉家屬困境，幫轉介至蘋果基金會，基金會訪視後已撥款暫紓困。
