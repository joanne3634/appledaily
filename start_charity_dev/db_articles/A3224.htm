顧病人無法工作 長子自立工讀

2013年11月13日

「阿智，聽話，不要亂動。」44歲符珈瑄讓極重度多障的15歲小兒子躺床上換尿布，重殘的阿智讓符珈瑄頭疼難照料，如今符珈瑄頭疼事又多一樁，她先生王文星得了食道癌。報導．攝影╱向高彬

阿智換好尿布，拿顆球丟來丟去，又一直要往外衝，符珈瑄一直喊也不聽，她只好出手攔著，「這孩子好動，只聽他爸的話，他才肯乖一點，但他爸這陣子在醫院做治療，不舒服也沒什麼力管。」

4口12坪屋堆雜物

阿智現讀特教高一，每天下課到家後，家人就得緊盯著，好幾回不留神，阿智跑出門搭公車亂晃，後來讓派出所警員找回。
符珈瑄是陸配，嫁來台18年，她說，阿智小時發燒過度，智力如3歲小孩，58歲先生幼時患小兒麻痺、左腿輕殘略跛，8年前遭工廠縮編裁員後，這幾年由先生照料及接送阿智上下課，家計由她在小餐館做服務員，每月約2萬餘元，加上三款低收補助2萬4700元，共4萬多元支應4口家用。
符珈瑄說：「但先生6月時一吃就吐，9月查出食道癌3期，我照顧兩父子沒法做事，補助扣尿布營養品剩1萬多元生活。」17歲大兒子阿傑現讀高二，假日在麵館打工掙零用。
王家住家約12坪，簡陋屋裡堆滿雜物，僅容回身；他大哥家住隔壁，大哥說，沒做工時會幫忙，但1弟2妹各有家庭，能幫有限已盡力。
符珈瑄的同鄉好友阿樺說，自家曾是蘋果基金會3年前受助家庭，當時先生肝衰竭病重，家有2子生活陷困，後來先生去世了，「還好有大家幫忙，我們才能走出困境。看到阿瑄現在困難，幫忙求助，希望幫她們家度過難關。」

鄰搬廢家電供變賣

當地曾里長說，鄰居知符女家境不好，之前常把廢家電拿來讓王文星回收變賣幾百元貼補家用，「有救濟物資都先慰助他們。」當地公所已撥急難金3萬元，蘋果基金會訪視後也提撥「不指定」急難金先紓困境。
在醫院探訪王文星時，他說：「我癌症能不能好，已想開了，人都會走到這條路，只是好走或難走。但就擔心這孩子，以後剩他媽一人怎麼顧？」
