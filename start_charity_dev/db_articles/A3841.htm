A3841 5歲女伴癱母 「不想媽離開」

2016年01月13日

47歲的阿惠洗腎回家虛弱躺在床上，5歲女兒小婷撒嬌依偎身邊想說故事給媽媽聽，只見小婷拿起《龜兔賽跑》的童書說著：「在草地上，有一隻兔子和蝸牛在吵架……」，阿惠聽了面露苦笑。報導．攝影╱韓旭爾

阿惠心酸說，她因糖尿病併發症3年多前右眼全盲、左眼視力約0.1，去年2月腎衰竭洗腎，原本還可照顧自己，不料半年前竟中風，現右手腳癱瘓，大小便都要人幫忙，「這幾年自己被糖尿病糾纏，忙著跑醫院，害得小婷像是沒媽的孩子一樣，沒人教，連龜兔賽跑的故事都沒聽過。」阿惠說著，忍不住閉上眼，淚水從眼皮間隙流下。
小婷見媽媽流淚，天真地安慰說：「這烏龜殻上有2個圈圈，我才以為是蝸牛。」
小婷又說，媽媽都要打針，「我覺得好可怕，但媽媽說不打針就不能陪我長大，我不想媽媽離開我。」

付不出幼稚園學費

一旁小婷52歲的爸爸阿賢說，他與阿惠交往同居20多年，兩人僅小婷一女，阿惠雖多病，但中風前還能在家顧孩子，他外出做建築零工掙錢養家，1年前曾讓小婷上幼稚園，但因付不出學費讀半年就停了，這陣子慈善單位送來二手童書、玩具，小婷都會看圖說故事給自己聽。
阿賢說，阿惠癱後每周復健3次，又要洗腎，每星期跑6次醫院，他因此難分身工作。

靠接濟白米免挨餓

「洗腎有交通車不用花什麼錢，但回診、復健每趟叫車來回就要600多元，每月約1萬元的低收補助一大半都花在上面。」阿賢邊說邊從抽屜拿出阿惠的回診預約單，有腎臟科、眼科、精神科，「幸好最近慈善會接濟白米、罐頭，一家才不會挨餓。」
阿賢的父母已逝，住在隔壁的大哥說，一家4兄弟姊妹各有生活壓力，「經濟上幫不上忙，只能在他們跑醫院時幫忙看一下小婷。」當地派出所承辦急難救助的王姓警員說，地方數家慈善會已陸續協助阿賢一家2萬多元急難金，並再轉介蘋果基金會，基金會訪視後也先撥款紓困。
