A3399：母罹癌父病 女：挺你別怕

2014年06月20日

45歲徐雪美咳得無法停止，10歲兒子小恩趕緊依偎在她身旁，輕柔地拍著媽媽的背。徐雪美前年罹患乳癌經治療一度穩定，但今年4月底復發轉移淋巴，她說：「因為孩子和先生，我才有力量再拼下去。」報導•攝影╱張嘉恬

採訪時徐雪美最大念高一、最小讀小二的5個孩子陸續放學。徐雪美說，7口家計靠補助與先生擔任大樓管理員維持，先生早年婚姻所育的長子在外地讀大一，工讀支應自己房租生活費後協助有限。她說，家裡人口多，粗茶淡飯省一點能度日，但得癌症後有醫療自費，「我會怕，先生對我說：『有恁尪在，免驚啦』（台語：有妳老公在，不用怕），我聽了有被安慰到。」
徐雪美接著說，罹癌後50歲先生阿鴻將上班時段調成大夜班，以便陪她就醫。因未休息，先生頸椎壓迫導致手部痠麻無力負重，一旁阿鴻說：「醫生已下最後通牒說要手術，不然左邊會癱，但手術要長期休養，房租、三餐馬上成問題。」

薄薪獨撐7口家

阿鴻接著說，一度以為過了難關，現太太病況復發他又得手術治療頸椎，他說：「說不累是騙人，但總不能在妻兒面前表現很脆弱，遇到了，只能面對。」阿鴻工作處的大樓住戶王小姐說，阿鴻任職近10年，為人老實客氣，「他一人薄薪養那麼多口本就辛苦，又碰上太太罹癌，雪上加霜，本來住戶想捐款幫助但他婉拒，所以幫忙打電話到蘋果基金會求助」。目前徐雪美全家靠低收等補助，加上阿鴻月薪因生活債務強制扣薪後每月共有約4萬元，但扣除房租1萬1000元後僅約2萬9000元7口生活。蘋果基金會訪視後再度撥款紓困。

「給你無限溫暖」

徐雪美憶及罹癌初期化療掉髮及各種恐懼與生活壓力，讀高一的16歲女兒寫信鼓勵她，其中一段寫道：「……買不起高級的補品給你，可是我可以給你無限期的溫暖及擁抱……，當你無助的時候記得往身後看一看，我們都在你後面支持你、挺你、保護你，愛你，別害怕！」

基金會編號：A3399
