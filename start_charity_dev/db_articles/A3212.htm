A3212 擔一家殘弱 壯男突瘋

中風翁：再顧妻兒也沒幾年

2013年10月31日

72歲的陳勝平嘆了口氣說，去年底長子喉癌病逝，自己太傷心導致腦出血中風，家裡還有64歲癱瘓妻子和37歲精障么子，照料重擔全壓在次子阿雄身上，「今年8月他突然發瘋，強制送醫住精神科病房」。報導．攝影╱仲芝蓉

陳勝平說，夫妻倆生3個兒子、1個女兒已嫁，他回述44歲次子阿雄發瘋時，心有餘悸地說，那天半夜他突然鬼吼鬼叫衝出家門，跑到路上對著空氣揮拳，我喊他都聽不見，把鄰居全吵醒了，「怕他會傷害人，只好報警送醫，還好現可出院，只要按時回診服藥控制」。

「還能走路是命大」

陳勝平說，很擔心阿雄跟么子一樣，「我去年底中風開腦取血塊，復健半年多，現在能走路已是命大，但我這年紀再照顧他們也沒幾年」。他說，13年前么子阿榮出車禍，當時才24歲，留下腦傷癲癇後遺症，之後出現幻聽、自殘，「剛開始就像阿雄現在，後來精神分裂，沒辦法照顧自己」。講到傷心處，妻子楊玉枝在旁掉淚，但她因5年前中風半癱，失語、智力退化，只能從喉嚨發出氣音，難言心中苦悶。
陳勝平繼續說，妻子20多年前車禍左大腿以下截肢，5年前中風右半身癱瘓，連扶拐杖走都沒辦法，只能坐輪椅，「吃飯也需人餵，起居全靠我照顧，還得盯著阿榮阿雄他們吃藥」。

4口月靠萬元度日

當地公所承辦員說，已協助陳家通過低收入戶，每月領補助共2萬1100元，但因他們住得遠，就醫回診每月車資就需7000元，再扣妻子尿布每月2000元、房貸3000元，一家4口就剩不到1萬元過日子，故轉介蘋果慈善基金會，基金會訪視後已撥款紓困。
當地康復之友協會理事長說，民風封閉、工作機會少，居民大多長期貧困，無餘力分勞，「阿雄規律服藥至少半年，情況穩定後可轉介醫院康復之家協助找工作」。陳勝平的女兒說，去年阿雄還到外地打工，年底爸爸中風不得已回來照顧，「自己育3子家計負擔重，只能偶爾拿自種青菜回去探望，陪爸爸說說話，希望老天保佑全家平安」。
