A3543 夫癱殘 婦撐家累垮摔傷

2千元養3口嘆拖累兒

2014年12月15日

53歲余素慧指著額頭疤痕，驚恐回述今年7月做清潔零工時，從鋁梯摔落血流滿面，她無奈嘆道：「丈夫患妄想症已10年，今年6月又中風，右半身癱瘓、失語，我照顧他又去做零工才出意外。」報導•攝影╱仲芝蓉

余素慧額頭縫5針傷疤已癒合，她說，最心疼是拖累12歲兒子小逸。
余素慧繼續說，13年前友人介紹小自己9歲的丈夫阿宏，「我倆都有前段婚姻，那時他當警察很有愛心，常買點心給附近孤苦無依的小朋友吃，原想兩人作伴就好，沒想到意外有了小逸。
原想丈夫工作穩定，相處時雖有時脾氣暴躁，但還能聽我勸，怎料小逸出生後，他越來越失控，工作上也常與人起衝突，只好辭職」。

「媽媽辛苦想幫她」

余素慧說，曾勸他就醫，但他無病識感。直到去年初，他居然在家大聲喊說要放瓦斯，又揚言跳樓，只好強制就醫，確認罹患妄想症。1年多來服藥控制，情況稍穩，沒想到不到幾個月，他居然中風倒下。
採訪時，小逸剛好放學回家，記者問小逸會不會幫忙照顧爸爸，小逸起先低頭不語，過一會兒，才緩緩地說想多幫媽媽，「爸爸很兇，但媽媽都跟我說是因為爸爸生病了，我覺得媽媽很辛苦，想幫她」。
當地長照中心專員說，公所已協助申請低收入戶，每月補助1萬800元，但扣房租及尿布等照料開銷，只剩不到2000元3口生活辛苦，而阿宏未滿65歲難有公費安置，故轉介蘋果慈善基金會，基金會訪視後已撥款暫紓困境。余素慧接著說，半年來陸續向同事、友人賒借10多萬元，「欠的未還實在難再借，真想找地洞鑽」。
余素慧說話時，阿宏只是靜靜看著她，丈夫中風後，情緒反而穩定些，「雖不會說話，但會用叫聲提醒我下雨該收衣服，希望他能這樣漸漸好起來」。

盼兒長大還有希望

余素慧二哥說，兄妹共3人，大哥誤入歧途服刑中、自己負債，妹夫家2弟妹也都各有家計負擔，實難提供經濟協助，只能勸妹妹加油再撐幾年，小逸是她唯一精神支柱，孩子會長大未來還有希望。
