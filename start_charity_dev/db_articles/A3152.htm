A3152 癌蝕父母 3子分工扛家

童心堅強 笑抽家事紙牌任勞任怨

2013年08月19日

「媽，來抽一張牌。」11歲小萍拿著自製寫著「搥背」、「拖地」的家事紙牌，和她37歲媽媽陳麗華玩鬧著，「抽到洗碗。」小萍知道媽媽罹乳癌復發體弱，與兩兄分擔家務，還想到以抽牌方式逗媽媽開心。報導．攝影╱向高彬

小萍用膠帶幫化療中的媽媽黏起掉落枕上頭髮，將升小五的她說，媽常說要獨立，「她不知還能照顧我們多久」，其實讓小萍3兄妹擔憂的不只媽媽，她們56歲爸爸陳飛龍也是口腔癌患者。

夫偶打零工難度日

媽媽陳麗華說，先生之前做電銲工，她做超市店員，與3個現各14、12、11歲的孩子租屋度日，但未料先生3年前以為牙痛發炎還自行拔牙，痛到鮮血直冒掛急診，一查口腔癌四期，手術切骨植皮，當她正忙照料先生，未料竟發現患乳癌，夫妻接連罹癌，「幸好那時恢復不錯，這兩年先生趁有體力，去打零工，我再回到超市做事，日子勉強撐過」。但今年初，陳麗華常感腰背痠痛，本以為太操勞，6月回診查出乳癌復發，侵噬至肝及胸腰椎，近日需手術裝骨釘支撐，「癌病再蔓延，我會下半身癱瘓，但我最怕的是，還有時間陪孩子長大嗎」？
陳麗華回述，先生曾有前段婚姻，「我們在一起後，他全心扛起這個家」；陳飛龍說很擔憂太太，「我不太懂表達，求了串佛珠給她，保佑平安」。

「家就要互相扶持」

記者見孩子乖巧有禮、分工家事，國三長子小揚和小六次子小皇都成熟地說「家是大家的，要互相扶持」。小皇的陳姓班導師說：「以前小皇開朗，孩子媽癌復發後，他笑容少很多，我常鼓勵他，並協辦學產基金救助」。
麗華的超市主管說，麗華做事認真，盼她快康復再回來工作。麗華的多年好友夏小姐代向蘋果基金會求助，「我常陪她到醫院做治療」。目前陳家每月共領低收補助1萬1300元，難敷房租家用，當地公所已撥急難金2.5萬元，蘋果基金會訪視後也撥款暫紓困。
