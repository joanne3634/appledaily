A3462 妻兒喚癱男 「等你回家」

2014年09月08日

「阿強，眼睛快睜開，別睡了，我們等你回家！」50歲的阿雲（許秋雲）帶著2子在病榻呼喚著，她說，53歲的丈夫阿強（傅新強）7月中風倒下後便昏迷不醒。報導．攝影╱韓旭爾

阿雲夫妻倆育有16歲、14歲的2子，阿強做外燴廚師，阿雲則在大賣場做時薪零工月賺1萬餘元貼補家用。阿雲說，阿強有高血壓，長年服藥控制，7月初就常喊著頭暈、頭痛，後來到醫院就診。
「沒想到住院第2天就陷入昏迷，醫生說腦幹的血管塞住了，手術風險很高，只能先用藥物治療，看血栓能不能自行散去。」
阿雲說：「如果他一輩子就這樣躺著，社工說出院後需要送到機構安置，每月要2萬多元，根本不知道去哪找錢？」
就讀美容美髮建教班高一的長子小閎說，從小到大，他和爸爸感情像哥兒們一樣，有心事都會對他講，「我天天來醫院看他，多跟他說話，鬧他說他在裝睡，看爸爸能不能早日清醒。」

「想念爸爸煮的菜」

小閎說，目前就讀建教班，「每學期可打工3個月，多少能夠幫媽媽分擔一點家計。」讀國三的次子小祥說，以前家裡都是爸爸在下廚，「我好想念爸爸煮菜的味道。」

瞞老父僅是小中風

之後記者跟著阿雲到家裡探訪，阿雲特別叮嚀記者不要向她公公說起先生昏迷，因為81歲的公公阿鈞伯原本住在鄉下老家，因婆婆4月才病逝，老人家心情低落，近日來家裡暫住，他以為阿強只是小中風。
阿鈞伯說，阿強做廚師，工作很操勞，前陣子老伴病重，也要阿強幫忙照料，可能太累才中風，「媳婦說，阿強在醫院努力做復健，應該很快就能回家了。」
醫院社工說，阿強一家目前沒有任何社福補助，大約要到年底才能申請殘障證明鑑定，核可後能獲4700元補助，無力負擔2萬多元的安置費用，故轉介蘋果基金會協助，基金會訪視後，已先撥款暫紓阿強家的養護及生活開銷。阿雲說：「阿強目前生命情況穩定，醫生話也沒說死，將來還是有機會能好轉，希望有奇蹟出現，阿強能趕快醒來。」

基金會編號：A3462
