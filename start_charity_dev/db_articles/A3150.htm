A3150 母昏迷弟跛腳 少年無助祈禱

2013年08月16日

「三媽佛祖、上帝公祖、奉旨神將，拜託您保佑媽媽快點醒來。」13歲的小州在病床旁虔誠祈求，因為今年6月，他34歲的媽媽鄭小芳發生車禍至今仍昏迷。報導．攝影╱江品璁

小州51歲父親劉文瑞說，太太懷孕8個月時早產，小州體重僅1500公克，於是就給神明當養子，之後一路平安，且才要升國一體重就80公斤了。
劉文瑞話風一轉難過道，6月18日太太載11歲的次子小豪回家，不料與一名騎機車的大學生相撞，次子左腳骨折，但太太傷到腦幹，醫生說可能會一輩子癱瘓，「兒子小州知道後急得到處拜拜，神明指示要躺高一點，要念經，我們都照做。」

小兒創傷半夜尖叫

一旁的小州播放佛經CD邊對記者說：「我每日輪流播放各經文，媽媽開始會睜眼，但兩眼還是無神。」看護阿柔嫂說，小州來醫院看媽媽，都會交代說經文不能停，「每次他跟媽媽喊話都邊講邊流淚，在旁看了都覺得鼻酸。」
劉文瑞說，自己原做綁鐵工，每個月賺2、3萬，太太則在雞蛋場撿雞蛋，生活可過，「但太太腦傷後，我每日到醫院探望，無力照顧腳傷次子，只好托給二姊照料。」
記者到劉文瑞二姊家訪視，只見讀小5的小豪拄著助行器，緩步移動，「到現在我都忘不了媽媽飛出去5公尺遠，我好害怕，晚上都不敢一個人睡。」劉文瑞的二姊阿秀嫂說，小豪骨折約休養到9月份就可恢復，但心理受創嚴重，半夜常會做噩夢尖叫。」
肇事男大生家長說，對鄭小芳昏迷感到難過，但兒子左眼恐失明，也是受害者，相關責任仍要由法院判決釐清。

保險短期內難補償

社工表示，腦傷的保險理賠，須等最後一次術後半年才可鑑定，短期內難獲補償。劉文瑞的哥哥說，父母已過世，2個姊妹各有家庭，自己當作業員也無力長期協助。當地公所說，已撥急難金2萬元，並轉蘋果基金會協力，基金會訪視後也已撥款暫紓困。
