A3514 重病男悲嘆拖累妻兒

子患癲癇 高二女懂事工讀

2014年11月06日

採訪時，43歲阿鴻從醫院回家，剛下班的39歲妻子阿翠說，先生半年來因遺傳而心臟衰竭，接著又罹患末期腎臟病，「開了切胸骨心臟大手術，以後也要一輩子洗腎，連說話都沒力，還一直說要工作。」報導•攝影／張嘉恬

阿翠說，先生病後「一直說拖累我和2個孩子，情緒一來時，還要我改嫁；我跟他說，只要你意志不消沉，吃苦我不怕。」

請假顧夫丟工作

她接著說，和先生育有現年17歲讀高二的女兒、13歲讀國二的兒子，原本先生做水電零工、她做餐飲服務，先生今年3月病倒，她請了4個多月假，結果工作就丟了。目前她做清潔工每月1萬8000元、先生殘津4700元，扣除房租4500元後僅1萬多元4口生活。
阿翠也說，自己半工半讀完成高職夜間部學業，因7年多前大車禍動過脊椎切骨手術，無法久站拿重，工作平均收入大概都2萬多元。她說，「眼睛睜開都要用錢，現我做清潔工薪水少，也常要搬很重的垃圾，回到家車禍舊傷都在痛，但要養家也不能挑工作了。」

憂父死掉兒大哭

採訪時，正逢阿翠兒女放學回家，阿翠說，13歲的小正有過動症、癲癇，得服藥控制，「兒女都很愛爸爸，爸爸住加護病房時，兒子擔心爸爸死掉，常大哭；女兒說要休學賺錢，我說高中沒畢業怎麼在現在社會生存，要她好好念書，以後才能多賺點錢。」只見癱坐一旁的阿鴻，臉部線條從僵硬轉趨柔軟，眼神流露悲傷，有氣無力說：「唉……，拖累他們啊，唉！」
17歲小君說，自己利用周末在麵攤打工，但因時數有限每月收入不到3000元，而阿鴻母親病逝，父親罹癌重病安置在機構，3個姊姊都出嫁協力有限。阿翠夫妻倆友人袁先生說，阿翠一個女人養家很辛苦，已協助近10萬元。蘋果基金會獲悉訪視後已撥款紓困。
