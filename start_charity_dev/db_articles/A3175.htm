A3175 籌錢開刀 脊突少女重生

不必再遮掩 興奮喊「我是一般人了」

2013年09月18日

18歲的蔡婷安脊椎突出背脊近20公分，升高三的她僅135公分，因目前脊椎已壓迫內臟，今年8月她開刀進行矯正。術後仍須復健的她說：「媽，我撐過來了，謝謝妳沒有放棄我。」報導．攝影╱江品璁

病床上，蔡婷安穿著背架，身上插著引流管導引血水，她喘氣說，開刀前一刻害怕直發抖，不斷禱告，術後覺得全身痠痛，又累又倦，但心裡卻開心吶喊：「我還活著！我要快回去上學，跟同學說我是一般人了！不用再遮掩背脊、要讀大學、要打工，回家要放鞭炮慶祝！」同學彥婷說：「開刀前婷安很害怕，我陪她講電話釋放壓力。」班上46名同學將祝福關懷滿滿貼在她的座位上。

遭譏「胸部長背後」

手術前記者訪視蔡婷安，她說，小學5年級時被同學譏笑胸部長在背後，那時很難過，「我怕他們用奇怪眼神看我，不把我當一般人看待。」
當時43歲的單親媽阿甄紅著眼眶看著女兒說，婷安出生時就進行無肛症、腸道閉鎖等手術，後又脊椎側彎達158度，10幾年來幾度想帶女兒去開刀，「但醫生說手術成功率僅50%，失敗不是癱瘓就是死亡，我只有1個女兒，真的賭不下去。」阿甄難過道，今年5月婷安因肺炎住院，醫生發現脊椎已壓迫內臟，再不手術恐危及性命，「我想跟婷安說對不起，生下她卻害她受苦。」

單親母撐家愁醫費

蔡婷安72歲外公阿彰說，自己生1子1女，7年前女婿過世後女兒阿甄就搬回娘家，沒想到兒子3年前腦瘤失智，安置在機構每月須補貼4千元，「我2年前罹胃癌，太太去年中風過世，孫女脊椎要開刀還要四處籌借，很無奈。」
阿甄說，婷安術後長高10多公分，但醫生說須復健半年以上，她上工時間將減少，「一家為中低收入戶，每月共補助1萬1900元，這次開刀19萬元，自己當廚工月賺1萬4千元負擔不起，向親友籌措許久，只好求助蘋果基金會，也謝謝願幫忙的親友們。」基金會訪視後已撥款紓困。醫院脊椎中心表示，因蔡婷安手術為自費醫療，故無法減免。
安南醫院神經外科主任莊皓宇說，18歲後的病患因骨骼發育完成，須靠手術治療，傷口長、風險高，術後最常見症狀，像胸腰椎無法彎曲，長期腰背僵直疼痛，活動角度減少。

【師生祝福】「等妳回來上課」

蔡婷安高中班導師柯家秀說，校方曾想為婷安改造桌椅，但她卻堅持要與大家一樣，不想被特別照顧，「她很在意自己跟別人不同，但我常告訴她，要拿出勇氣面對，要更開朗，恭喜婷安撐過艱難手術，大家都等妳回來上課。」
