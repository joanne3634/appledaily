A3489 幼子早熟吃苦 癌爸好心疼

2014年10月07日

6歲的小珩黏在爸爸身邊說：「爸爸什麼時候可以不要再去醫院，我想要你天天都能在家陪我。」小珩55歲的爸爸阿光望著他圓圓的眼睛，無奈自己今年5月罹患乙狀結腸癌第二期，未來還有漫長的療程得熬過。
報導•攝影╱仲芝蓉

小珩9歲的哥哥小越在旁拿著水杯，提醒爸爸吃藥，一邊似懂非懂地回應弟弟說：「你只想要爸爸陪你玩，一直玩，以後長大就變成只會玩的人。」

生活剛穩定卻罹病

阿光看著長子小越學大人口氣訓斥弟弟，既好笑又覺心疼，嘆了口氣說，自己44歲才娶妻生子，擔心他們學壞，才耳提面命要他們上進、努力讀書，「我以為還有大半輩子時間可以看他們長大，怎麼料到竟突然罹癌！可能是過去工作習慣外食，吃的不均衡，才讓癌病上身。」
阿光32歲的妻子阿惠是越配，她的國語不流利，顫抖地嗓音和眼眶裡的淚，透露出內心不安，說，丈夫7月開刀，「我好害怕他突然走掉，以後怎麼辦。」
阿光接著說，自己4年前上山幫人種菜打零工，不慎滑倒左腳踝骨折打鋼釘，3年前改開貨車，靠收入3萬多元養4口家；2年前次子上幼稚園後，妻子開始外出做農雜零工、清潔工等，這兩年生活才開始穩定，「怎料今年5月我回院拆鋼釘時，順便接受免費腸癌篩檢，卻發現得病。」

還有高齡母親要顧

當地公所急難承辦員說，公所已撥急難金2萬元，並協助申請低收入戶等補助，再轉介蘋果慈善基金會，基金會訪視後也撥款暫紓困境。
阿光的姊姊說，手足共5人，阿光排行老三，大哥夭折、二弟前年才意外往生，父親罹患肝癌病逝，還有86歲高齡母親由自己和三弟分擔照料，「我和三弟都各有家庭，湊借3萬多元應急，阿光住院時，我也幫忙照顧小越和小珩，他們很聽話飯後會幫忙洗碗，希望阿光早日康復，盡快度過難關。」
阿光回述過往說，曾當導遊小有積蓄開養豬場，未料投資失利倒閉關場，「人生到頭一場空，拖累妻子和兩子跟著吃苦，只能怨嘆自己，沒運沒那個命！」
