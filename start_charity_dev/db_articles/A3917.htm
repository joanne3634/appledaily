A3917 高三生救父 勇捐2/3肝

不捨熬夜工作 扛6口病倒

2016年04月28日

「救爸爸，我覺得很光榮！」讀高三小榕在3月底勇敢捐肝移植救父，在手術成功後，小榕與他52歲父孫泳清正待休養。小榕露出像是勇氣勳章的長手術痕，他說，父恩深重，「我們本來就是一體。」報導•攝影╱向高彬

記者於小榕父子手術前夕到孫家探訪時，記者問孫父是否擔憂，他虛弱地說，不怕手術痛，「只怕若萬一，留下妻兒老母怎麼辦？」一旁小榕接應：「爸放心，我們一定OK！」

盼休養身體再撐家

記者於手術後再至醫院探訪，父子倆仍感傷口疼痛，但綻露笑容，孫泳清說，經10多個小時手術奮戰，「醒來時，太太看著我一直掉眼淚，一句話也講不出來，高興我們都過關了。」他又說，現需終生服用抗排斥藥物，「要感謝我兒子和醫療團隊讓我活下來，等身體養好，再工作把家撐起來。」
46歲媽范姚盈盈說，夫妻婚後生現19、17及14歲的2子1女，小榕排老大、二子讀高二、么女讀國三，並與9旬婆婆共6人同住，先生多年來以開貨車送貨為業，「一天熬夜工作15小時以上，但他有C肝，前年突病倒，查出肝硬化嚴重，沒力做工了。」這兩年家計靠她打農雜零工撐家，「兒子很懂事，先生病後兒子周末都做粗工幫扛家。」
醫院社工表示，目前相關醫費共約40萬元，正評估補助部分醫費。當地公所表示，孫家現僅領補助7256元，待孫泳清核撥殘證後，協辦低收資格，並轉蘋果基金會協力關懷，基金會訪視後已先撥款暫紓困。

做工幫家計不喊累

小榕說，切3分之2肝臟移植給父，「醫師說我是年輕人，好好休養半年，應可長至原本8成，之後可正常生活。」小榕在後天將參加統測考試，有點耽憂體力及課業生疏，但仍會盡力應考，「爸媽希望我再讀四技，多讀點書、也調養身體。」
孫泳清的三哥阿鋐說，家5手足，父及大哥、大姊病歿，「二哥跑計程車、我修車，賺不多，有時接濟幫點小忙。」范姚盈盈娘家僅有一弟阿鈞，他說，「小榕很孝順，之前假日有工，找小榕搬貨掙錢，他從不喊累。」

【孝心克服恐懼甄選「孝行獎」】

小榕的班導師多次至醫院探望，手術當天，全校同學朝會為他家集氣祈福，同學們也寫卡片祝福。
班導說，小榕已獲代表當地參加內政部「全國孝行獎」甄選，「術前我問他怕嗎，他回只怕比對不成、不能救爸。」感謝一切順利，他的孝心克服了恐懼。
