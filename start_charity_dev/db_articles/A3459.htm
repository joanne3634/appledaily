A3459 兒殘妻半盲 撐家夫摔癱

4口靠津貼度日「明天怎過」

2014年09月03日

「做人實在太累，如有下輩子我不想做人了。」46歲的歐致奮悲觀說著，今年6月底，他不慎從2樓摔下導致下半身癱瘓，同時也摔斷了一家4口的經濟支柱。
報導•攝影╱江品璁

歐致奮說，自己與妻子生2個兒子，長子出生時就長短腳，一直到9歲才學會走路，領有中度肢障證明，現在21歲了，去找工作都被人嫌，根本找嘸頭路，也交不到對象，41歲太太5年前罹糖尿病，竟導致右眼視網膜剝離，現在左眼亦看不清楚，「我要扛一家子，但今年6月24日，我做工時，一不注意從2樓滑落，尾椎直接撞在地上，下半身都沒知覺，要包尿布。」

小兒無法抽身工作

一旁19歲次子阿坤說，家境不好，自己又對讀書比較沒興趣，國中畢業後跟爸爸學模板工夫，但技術還沒學到家，爸爸就摔傷，「媽媽今年7月中又因糖尿病導致腎衰竭住院，我一下要顧爸爸，一下要顧媽媽，抽不開身工作。」長子阿旻說，自己找不到工作，曾試著去賣烤香腸，「一根扣掉成本大概只賺5元，鄉下地方生意不好，常常虧錢，最近豬價大漲，實在做不下去只好收掉。」
歐致奮的妻子陳雅惠嘆道：「我自己病一堆，現在先生也倒了，真不知道明天怎麼過。」歐致奮70歲的母親說，自己與先生住在附近，共生2個兒子，都靠粗工為生，歐致奮是長子，「現在看他都坐輪椅，心很痛，煩惱到都睡不著，每天來看他幫他煮飯，很怕他以後不能走了。」

鄰居送米菜度難關

歐致奮難過說，勞保職災給付已申請中，現靠長子的殘障津貼4700元度日，還要負擔房租4000元，要不是鄰居協助一些米、菜，真不知要怎麼度過這難關。歐致奮工作的工頭說，他很拼，不管工作時間多晚，去叫他都會來，「有幫忙他負擔醫療費，就怕他這狀況還要拖很久。」
當地警察蕭健林說，自己在當地服務多年，知道歐致奮做粗工，家境不好，這次摔傷後，家裡頓失經濟來源，已幫忙找當地慈善會關心並轉介蘋果基金會，基金會訪視後已撥款暫紓困窘。當地公所說，已協辦急難金補助。
