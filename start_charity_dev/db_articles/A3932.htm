A3932 單親癌母心疼11歲女 已無爸爸「要跟病魔拚到贏」

2016年05月17日

記者與張秀妹從醫院返家，剛好與讀小五的小珺放學在巷口相遇，母女立即擁抱手牽手；小珺一進家門，就開始洗衣、收摺衣服，她說，在家做倒垃圾、掃地等家務，「媽生病，我多做事，媽可以多休息。」
報導•攝影╱向高彬

11歲小珺是獨生女，父親在她4歲時心肌梗塞猝逝，這些年來她與52歲媽張秀妹相依度日，她說，媽病前做清潔工，年初常喊吞口水痛，左耳聽不太到，「媽去醫院檢查，回家時表情難過，跟我說她得癌症，要做很久治療，又說不能做工、沒錢怎麼辦？她最近常跟我說，早點長大、學會照顧自己。」
小珺又說，最近媽去做治療，回來時連爬樓梯都沒力，很累躺床上休息，「我跟媽說，我可以照顧妳。我去幫媽買飯，可是她吃不太下，她本來就很瘦，最近瘦更多，我有點擔心。」

女兒是這輩子唯一希望

張秀妹聽女兒貼心話語，眼眶微濕地說，女兒是這輩子唯一希望，「醫師說我鼻咽癌三期，但我個性倔強，我告訴自己，不要把自己當病人，治療再痛苦都要忍，用意志力對抗，為了女兒，一定跟病魔拚到贏。」
原本張秀妹做清潔工，每月約1.9萬元，加上單親補助2000元，勉敷租屋家用，但張秀妹2月罹癌，如今她停工醫病，但房租就5500元，難敷生計。目前當地公所辦理急難低收補助。醫院社工評估困苦，再轉介蘋果基金會關懷。蘋果基金會訪視後，也撥款暫紓急窘。
張秀妹回述，早年第一段婚姻因前夫家暴離異；後來她與做廚師的亡夫阿宗再婚，生一女小珺，但阿宗早歿，之後她做雜工養小珺，期間曾有短暫感情但分手，只是無奈癌魔突襲，「現在只想趕快病好，我繼續把寶貝女兒養大。」

友讚心地善良會度難關

張秀妹娘家爸媽歿，另有2姊1弟4妹，她二妹阿娟說：「彼此住得遠，有時我去看她們，雖然能力有限，我們姊妹每月合力幫她約5000元。」小珺的班導師說，正協辦學產基金和行天宮急難金救助。張秀妹的同事兼好友阿玲說，秀妹心地善良，「以前她都會幫同事多做，她剛發病時，換我們多幫她，讓她提早下班去休息治療。我自己是乳癌患者，常以過來人經驗鼓勵她抗癌，有信心，一定可以好的。」
