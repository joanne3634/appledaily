A3123 婦扛百萬債 顧腦傷女

精障家貧 自責害女燒炭受苦

2013年07月12日

「痛就叫出來，悶在心裡會更痛！」50歲的單親媽媽阿莉正幫腰、臀及手部3度燙傷的24歲女兒小熙換藥。阿莉說，小熙個性內向，心裡有苦都不說，「都怪家裡經濟給她太多壓力，逼得她想不開！」
報導．攝影╱韓旭爾

阿莉說，2年前小熙大學畢業覓職不順，今年3月才到便利商店上班，不料工作1個多月就在家燒炭自殺未遂，身體9%面積遭燙傷，腦部也有損傷。陽光基金會社工說，已協助製作壓力衣，她腦損及心理問題也需追蹤治療。

月入1萬欠債百萬

想到女兒需面對痛苦又艱辛復健過程，阿莉語帶哽咽地說，15年前喪夫時，小熙9歲，么女小儀才3歲，扛家壓力讓她罹患躁鬱症，持有輕度精障證明，需服藥控制，只在家做成衣代工，每月才賺約1萬元，「這些年來用現金卡借錢應急，已欠上百萬元，女兒們靠學貸、打工才能讀大學。」

家計沉重萌生死意

阿莉嘆氣接著說，2年來小熙丟了上百封履歷表都石沈大海，「我透過鄰居介紹便利店工作，雖知她做得不愉快，但總認為適應就好了！」便利店店長說，小熙工作認真，只是沉默寡言，「想不到她會做出如此舉動。」採訪時，問小熙怎會想不開？小熙表情呆滯說：「忘了！」停了半分鐘又說：「知道媽媽為錢煩惱，但我只能眼巴巴看著，無能為力。」
阿莉老鄰居阿琴說，阿莉工作努力，但錢賺得不多，「她打電話給我，2次總有1次是要借錢。」阿莉讀大一的么女小儀說，她雖利用課餘打工，「只夠付自己學費、生活費。」當地里長說，阿莉一家領有低收補助1萬600元，未來也會提供物資協助。蘋果基金會獲悉訪視後已先撥款紓困。
林子堯醫師表示，小熙在家計紓困後，仍應透過藥物及心理療法，改善負面思考，防止悲劇再度發生。
