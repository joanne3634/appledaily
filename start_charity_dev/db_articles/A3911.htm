為雙親辭工苦撐「命運折磨人」

2016年04月20日

63歲辜許麗華癱坐輪椅哽咽地說：「65歲丈夫已經中風5年，他住護理之家費用、尿布錢都靠兒子一人負擔，今年2月我又中風，命運為什麼這樣折磨人？」
報導•攝影╱仲芝蓉

辜許麗華39歲的次子阿鉦握著媽媽的手，輕聲安慰她不要多想，趕緊把握時間復健，能進步一點都好，接著對記者說：「我原在外地電子廠工作，媽那天在房裡昏倒，一早83歲外婆敲門沒回應才發現，趕緊叫救護車，醫生說今年8月前是黃金復健期，但外婆去年也曾小中風，根本無力照顧媽，我只好辭工回來。」

安養費6千開銷重

辜許麗華傷心地邊流淚邊告訴記者，自己原有2子1女，大兒子7年前心肌梗塞，才34歲，送醫都來不及突然就走了，么女嫁得遠，「早年我和丈夫做工程小包，結果沒賺到錢，連房子都被查封，那時阿鉦和他哥哥懂事讀高職夜校，白天都打工自己賺學費。」
當地公所急難承辦員說，辜許麗華殘障證明申請中，後續將協助申請低收入戶，及提高她丈夫安養費的差額補助，但每月仍有至少6000元自費負擔，家中目前僅領有老人津貼7463元，公所已申請「馬上關懷」急難金2萬元協助，並轉介蘋果慈善基金會，基金會訪視後已撥款暫紓困境。
辜許麗華大弟說，姊夫有2弟1妹都少來往，自己從前也做鐵工，和小弟共湊約5萬元幫忙已盡力。

僅靠鄰居米菜接濟

面對父母接連病倒，阿鉦說：「5年前為了爸每月安養差額，我去借信貸40萬元，好不容易今年初才還完，媽就接著病倒，生活靠鄰居接濟米、菜，還有公所、功德會救濟金，真不知怎麼還，希望媽快好起來，我能靠自己工作養家。」採訪結束前，阿鉦的外婆拉著記者說：「這外孫真的很懂事，可憐家裡這樣，女孩子聽到都跑掉。」
當地功德會王理事長說，附近工作機會少，大多是零工收入不穩，辜許麗華原靠撿回收變賣，所騎的三輪機車也常故障，一修就要5、6000元，也靠阿鉦寄錢回來，「這年輕人很有擔當，但家境實在可憐，功德會已申請急難金5000元幫忙。」
