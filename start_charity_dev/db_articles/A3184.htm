A3184 一家3腦萎 精障兒扛家

伴癱母入睡 「好像小時候」

2013年09月27日

「啊……啊……啊。」65歲的阿霞嬤見記者訪視，用盡力氣號叫，想把心中的苦楚統統告訴記者，一旁47歲的長子陳榮財不捨道，媽媽約15年前小腦萎縮症病發，至今除意識清醒外，其他功能都退化。
報導•攝影╱江品璁

陳榮財說，20年前，媽媽突然不會扣鈕釦，走路也常跌倒，看遍內外科都找不到原因，只覺得很像過世外祖母曾有的病兆，直到15年前，媽媽吃東西常嗆咳，甚至從鼻子跑出食物，才確診為罕病小腦萎縮症。

4口每月萬元苦撐

陳榮財嘆道，媽媽5年前病情惡化，罹肝癌多年的爸爸也在今年1月過世，沒想到3、4年前，44歲的妹妹跟42歲的弟弟也出現口齒不清、跌倒等徵兆，去年底原在外工作的他們全搬回家住，今年5月確診也罹小腦萎縮症，目前雖可自理生活，但無法工作，不知道惡化速度有多快，「現在媽媽四肢癱瘓，須包尿布、灌食營養品，我們4口月靠補助2萬7700元過日，扣除媽媽的尿布及灌食營養品1萬2000元，辛苦生活。」
弟弟陳榮仁說3年前開始出現肢體不協調，「我知道發病了，不怪媽媽，她也不願這樣。」

弟妹罹病憂惡化快

妹妹陳雪芳說話咬字已漸受影響，她緩慢的說：「看到媽媽惡化情況，我害怕未來，只能默默流淚。」小腦萎縮協會劉宣汶社工說，小腦萎縮症有遺傳性與不明原因散發性2類型，阿霞嬤一家為遺傳性，未來腦神經會逐步退化，只剩大腦保有意識，目前無特效藥可治療。

里長慨嘆皆高材生

陳榮財朋友楊山明說，陳榮財19年前曾任國代助理，那時打選戰，壓力太大，竟罹精神分裂症，「他成績優秀，但無奈變成這樣，已借了他十幾萬周轉。」
陳榮財苦笑道，現在吃藥控制，已穩定多了，「我現在常抱著媽媽入睡，好像回到了小時候，只是這回角色顛倒了！ 」
當地里長感嘆道，陳榮財台大畢業，弟妹也是政大高材生，但陳榮財領有輕度精障證明，母親和弟妹相繼病發，全家都倒了。陳榮財的小阿姨說，家中共4手足，大哥、大姊和自己都已發病，「我跟女兒相依為命，因為我現在講話也開始不清楚了，無法幫上忙很無奈。」
