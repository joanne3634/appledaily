A3607 父爺癱瘓 19歲兒從軍扛家

婦照護兩頭燒「要更堅強」

2015年03月08日

傍晚，40歲的阿蓮（白秀蓮）幫脊損癱瘓的丈夫阿明按摩復健，緊接著到醫院呼吸照顧病房探視中風的公公利福伯，阿蓮無奈說，最近每天來回家裡、醫院，「家裡一下子來了這麼多不幸，只能告訴自己要更堅強。」報導．攝影╱韓旭爾

阿蓮說，她與44歲丈夫阿明（萬正明）育有19歲、17歲2子女，阿明原開怪手養家。

「1次受傷全搞垮」

去年10月底，阿明工作時從怪手跌落造成頸椎損傷，「不料12天後，阿明都還沒有出院，同住64歲的公公竟突然中風。」
採訪時，阿明躺在床上，一臉愁容說，開了快30年的怪手，「從沒發生意外，沒想到第一次受傷就把我搞垮了，只能努力復健，求不要拖累家人。」阿明嘆氣又說，家裡原本有3兄妹，但弟、妹都因病去世，爸爸只剩他這個兒子，「爸爸身體一向很健康，不知是不是自己受傷讓他太擔心，才會引發中風。」
阿蓮說，她分不了身工作，阿明又沒勞保理賠，幸好19歲兒子念祖懂事，丈夫意外後，願意當志願役士兵賺錢擔家計，「只是兒子的人生規劃都變了。」

月薪難撐家中醫費

念祖說，他去年6月高職畢業，申請提前接受4個月軍事訓練，打算結訓後準備體大考試，想當體育老師。爸爸受傷時，他正在部隊受訓，請假趕到醫院看爸爸，當時聽醫生說爸爸下半輩子都只能躺床上，隔天回部隊就決定簽下志願役，「遇到事情總要想辦法解決，家裡沒人賺錢也不行，所以我得扛下來。」
阿蓮說，目前念祖每月2萬8000元的月薪扣除零用及返家交通費，約可給家裡2萬元，僅能支付公公在呼吸病房費用，先生的復健交通費和尿布等開銷，每月近1萬元仍無法負擔。當地服務社工說，阿蓮家的政府補助僅有每月3500元，故轉介蘋果基金會，基金會訪視後已先撥款紓困。
阿蓮17歲的女兒小涵在變故後，也懂得節約度日，上學的早、午餐只花60元就解決。讀高二的小涵說，她高中畢業後，也想學哥哥當職業軍人，盡力讓家裡變得更好。
