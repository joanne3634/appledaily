A3198 苦顧老父憨兒 癌男抱病上工

2013年10月15日

「那天趕不上末班車，錢不夠，還好謝先生願載我。」丁小姐說，知悉單親計程車司機謝金報罹鼻咽癌，老父失智又罹糖尿病，高一兒子患學習障礙，代向蘋果基金會求助。51歲謝金報嘆，老母4年前病逝、妹妹子宮癌、姊姊失聯，目前房租6000元由教會協助，但每月靠低收殘津1萬9000元不敷開銷，今年6月基金會已撥急難金紓困，但他癌病控制不佳、老父不良於行，開計程車收入銳減，他說：「對社會關懷，我會抱感恩心，有生之年盡力助人。」
報導．攝影╱張嘉恬

【上午7:00 兒陪化療】

謝金報說，讀啟智學校的兒子小哲頭腦不靈光，但懂得擔憂他體力不支，假日會陪他到醫院化療（圖），「小哲生母在他3歲離家，後得知她也已因癌病逝。」篤信基督的他說：「命運難抵抗，但禱告讓我有力量。」

【下午5:00 忙照料父】

「孩子都不理我……」謝金報結束化療回家聽見他78歲老父嚷嚷著。他替老父注射胰島素（圖），再將速食包加熱和入白飯當作晚餐，他說，老父2月跌斷腿癒後不佳，早年腦部積水引發失智，現行動不便有身心障礙證明，他祈禱：「讓我生命延續顧好老父與兒子是最大心願。」

【晚上10:00 硬撐開車】

與謝金報一同排班的司機阿龍說：「他很古意（台語：老實），無線電通話常聽他載到錢不夠的乘客，差2、30元他也不計較。」但目前謝金報化療後耗弱，且常在開車前鼻孔要塞棉花避免膿水流下（圖），「他勉強晚上10時後到捷運站排班但月收入才3、4千」。
