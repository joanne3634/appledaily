A3884 扛家男罹癌苦撐 5口擠10坪房

2016年03月18日

「阿公，嘴巴還痛痛嗎？」7歲小歆一放學便跳到50歲外公阿昱（蔡敬昱）病榻又抱又親，阿昱原因口腔癌腫瘤疼痛緊蹙眉頭，外孫女讓他露出難得的微笑。阿昱口齒不清說：「對家人的愛和不捨，是我忍痛抗癌的動力。」
報導．攝影╱韓旭爾

阿昱50歲妻子阿勤（陳立勤）說，阿昱4年多前曾罹口腔癌，去年8月追蹤時，醫生還說沒問題，怎知才過1個月阿昱右臉頰就腫起，癌症復發且是第4期。阿勤哽咽又說，過往阿昱做水電零工養家，她在麵攤做雜工日掙700元貼補家用，如今阿昱體弱連走路都要人攙扶，她只能放下工作照顧他。

打地舖讓小孩睡床

阿勤夫妻育有31歲、18歲及16歲3女，小歆是單親、入監服刑的長女所生，小歆一出生就由夫妻撫養，另2女讀高一。記者採訪時，見阿勤家租屋處約10坪。18歲次女小渝說，家裡物質不及別人，但爸媽對孩子的關心和愛護從沒少過，小渝指著鋪在客廳地上床墊說，家裡只有一間房，以往姊妹、小歆睡房間，爸媽在客廳打地舖，現爸爸生病才改睡房間，「每次看到爸爸痛到大哭、用頭撞牆，真的好難過。」
阿勤讀高一的三女小平說，她讀建教班每學期在餐廳實習3個月，她打工時收入扣掉住宿、生活開銷，每月能資助家用2、3000元。她常鼓勵爸爸一定要堅持下去把病治好，「我們全家人要在一起長長久久，少一個都不行。」

孫女：怕阿公走掉

小歆導師說：「小歆雖小，但只要聊到阿公，就露出擔憂神情，很怕阿公會走掉。」導師說，學校已盡力幫小歆申請助學金，減輕家長經濟壓力。阿昱說，他10手足已4人往生，其他5人少聯絡，經濟上沒法幫忙。阿勤娘家哥哥說，一家3兄妹，「我生病零工機會少，還要顧80歲媽媽，沒能力幫忙。」
癌症希望基金會社工說，阿勤5口家領有低收、民間資源每月約2萬7000元，房租4000元，但阿昱流質營養品開銷每月就需1萬餘元，故轉介蘋果基金會，基金會訪視後已撥款暫紓困境。

基金會編號：A3884
