A3095 忍病省醫費 單親媽突癱

16歲女撐家 「沒如此害怕過」

2013年06月10日

讀高一的小佳放學進門放下書包後，馬上坐在42歲、中風媽媽阿蘭（黃秀蘭）旁邊和她說話，過會兒餵媽媽吃飯、擦澡，小佳說：「我一定要盡力照顧媽媽，讓她快點好起來！」
報導．攝影╱韓旭爾

阿蘭跟記者說，離婚後靠著採果、噴農藥等粗活賺錢撫養16歲小佳，「自己有高血壓，但手頭緊，沒按時看醫生，不料一病就這麼嚴重。」

「照顧媽是應該的」

一旁16歲的小佳說，3月底時，回家發現媽媽倒地抽搐、口吐白沫，她驚慌打電話叫救護車，「媽媽昏迷了4天，我從沒如此害怕過，直到她醒過來，睜眼看見我就叫『寶貝』，我高興到哭出來。」小佳說，現媽媽左半身癱瘓，而且「問她事情，有時要想很久，又記不清楚。」
提起過往生活，阿蘭說，這些年她努力做工，「至少還能過日子，不像現在手腳不能動，想做也沒機會。」家婦中心翁姓社工說，小佳以前常在外玩到很晚，媽媽生病後，懂事不少。小佳說：「我比誰都清楚，媽媽有多辛苦，照顧她是應該的。」

月僅領補助1900元

阿蘭的大姊阿琴說，3年多前小佳爸爸去世後，留給母女倆一棟房屋的2樓，阿蘭中風後與她同住，母女就睡在客廳裡。阿琴單親育有3子，娘家父母也同住，她說：「我自己生活很緊，看到妹妹癱躺床上，很心疼，卻也無能為力。」當地公所社會課說，目前阿蘭領有單親補助每月1900元，因小佳繼承亡父林地，地處偏遠、難出售，一時又難獲低收核可，但眼前阿蘭復健的醫療交通、尿布費用全是重擔，蘋果基金會訪視後已撥款暫紓困。

【逆境中的力量】

阿蘭說：「中風癱瘓後曾情緒低落，但女兒的話卻讓我有活下去的動力，小佳曾說：『媽媽，妳以前都叫我寶貝，現在換妳當我的寶貝，我會照顧妳，有妳作伴就是我最大的幸福。』」
