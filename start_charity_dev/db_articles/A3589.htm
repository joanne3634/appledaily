A3589 顧癌父癱嬤 3童幫灌食

6口月僅1.1萬度日「不敢想以後」

2015年02月08日

「阿嬤妳渴嗎？我給妳喝水。」11歲的小瑜掃完地，再幫77歲中風癱瘓的阿嬤鐘林柒從鼻胃管餵開水。「阿嬤等下要洗澡囉。」14歲哥哥小榕和13歲姊姊小芸，則準備端熱水和毛巾幫阿嬤洗澎澎。報導•攝影╱孫敏聿

「爸爸得癌症，媽媽陪他去化療的時候，家事和阿嬤都交給我們。」13歲的小芸說，爸爸鐘瑞曲是修車工人，媽媽做水果包裝臨時工，一天賺6百元，「我們都是阿嬤帶大的，小時候她幫我們換尿布，現在換我們幫她是應該的。」
一旁的小榕說，阿嬤中風5年，原本還能推她坐輪椅去散步，每周日去隔壁的教堂禮拜，這2年阿嬤陷入昏睡，還要抽痰、灌牛奶，「我們3個不敢抽痰，都靠媽媽，不然阿嬤痰卡住可能會死掉。」
小榕35歲的母親林慧玲陪同丈夫就醫返家，接著說，16年前從印尼嫁來台灣，6年前公公過世，接著婆婆中風，「每個月尿布、營養品就要2萬元，房貸要繳8千元，少了我去賺錢，6口只靠我先生3萬多元修車收入支撐，未料去年竟肺癌病倒。」

導師辦減免助學

51歲鐘瑞曲因肺部惡性腫瘤壓迫神經，說話低沉模糊又喘，「我媽去年9月住院一個月，欠醫藥費3萬多沒錢給，大哥打零工、小弟也在外地討生活，孩子又還小，我都不敢想以後了。」
小芸國二導師說，小芸父親罹癌後，她常在聯絡簿寫道，很擔心爸爸和阿嬤身體，功課也一落千丈，「我申請免費第8節和每周六課輔，也減免制服費。」

醫母耗百萬積蓄

鄰居韓北文表示，51歲的鐘瑞曲獨挑母親照顧責任5年，花光近百萬積蓄，「我也只能偶爾幫忙1、2千元。」里幹事表示，林慧玲一家6口靠低收7800元與婆婆老人年金3500元，共1萬1300元難度日，去年12月份領慈濟基金會2萬元急難救助金。基金會獲悉訪視後也先撥款暫紓困。
