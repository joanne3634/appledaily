A3015 撐家嬤罹癌 拾野菜養孫

為病夫欠百萬「壓到喘不過氣」

2013年03月13日

「阿嬤，這樣有卡爽快嗎？」15歲的小輝體貼地替阿嬤按摩、搥背，60歲的阿淺嬤（陳余淺）欣慰道：「乖孫。」但想到去年12月被診斷出罹患子宮內膜癌，又不禁愁上心頭。 報導．攝影╱江品璁

讀國三的小輝幫阿淺嬤按摩說：「4歲時媽媽就跟爸爸離婚，8年前爸爸欠下大筆債務後失蹤，我跟大哥小豪、二哥小憲從小就由阿公阿嬤帶大，他們就是我的爸媽。」提起失蹤多年的兒子，阿淺嬤難掩失落道：「沒有消息，就是好消息，算我前世欠他，今生還他，只盼能把孫子顧到長大成人。」
阿淺嬤道，6年前發生輕微中風，傷及語言神經，但為了家計，「復健後到農會當清潔工每月賺1萬2，將3孫拉拔長大，怎料又得了子宮內膜癌，今年1月底已動手術把卵巢、子宮摘除。」
61歲的先生阿傳伯說，自己12年來長期洗腎，3年前還引起腎性副甲狀腺機能亢進，進行副甲狀腺切除術，無法工作，「太太一人撐起全家，現在卻罹癌病倒，實在很不甘。」阿淺嬤嘆道，先生的醫病開銷壓得全家喘不過氣，只好拿房子去抵押借款，陸續欠了百萬元，每個月要還1萬多元，「現在我罹癌辭去工作，每月靠低收、家扶、殘障津貼共2萬4200元維持，只好去山上撿野菜，回家炒就是一餐。」

「顧公嬤一輩子」

讀高三的長孫小豪說：「今年準備大學學測時，阿嬤剛好罹癌，很難靜下心讀書，本想立刻轉讀夜校打工，但阿嬤要我別擔心，要我用功準備7月份指考，盼能考上國立大學。」小輝接著說：「好想快點長大，我會負起責任照顧阿公阿嬤一輩子。」
阿淺嬤的前同事阿雯嫂說，阿淺嬤打掃認真，寒暑假時三名孫子都會到農會幫忙，難得有這麼孝順的孩子。

嘆「就是放不下」

阿淺嬤含淚說：「已經活了一甲子，又罹癌剩沒多少日子就要回去了，但就是放不下我這3個乖孫跟先生。」
小輝葉姓班導師說，小輝曾在100學年度獲頒教育部全國孝親楷模獎，這次阿淺嬤罹癌，學校也幫忙申請獎助學金。阿淺嬤的女兒阿雅說，5年前中風傷及腦幹，全靠先生做鐵窗撐家計，上面還有一個大哥阿宏是靠打零工為生。當地公所已協辦「馬上關懷」急難金。蘋果基金會訪視後，也撥款暫紓困窘。
