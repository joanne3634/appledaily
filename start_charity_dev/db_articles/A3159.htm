A3159 顧癱父殘母 女愁醫費

停工斷收入 背債苦撐家

2013年08月27日

「爸，我是淑敏，認得我嗎？」27歲洪淑敏跟因心肌梗塞昏迷造成缺氧性腦病變的爸爸洪金章喊著，洪金章氣切難出聲，他發楞看著淑敏約半分鐘，很慢地點了頭，淑敏難過：「爸每年記得幫我過生日，現在卻變這樣。」報導．攝影╱向高彬

洪淑敏說，今年59歲的爸爸有心臟病、高血壓，4月底在家發病，「靠呼吸器維持近3個月，7月中才脫離呼吸器，醫師說要肢體復健，等好些再評估心臟手術」。

弟闖禍甫入獄

只是洪淑敏擔憂的，除癱瘓爸爸，還有患僵直性脊椎炎、憂鬱及恐慌症的55歲媽媽吳惠月。她說，媽媽約10年前脊椎發病、曾裝骨釘且為肢障，這幾年更為成天闖禍的23歲弟弟阿堅，煩出精神問題，阿堅4月初入獄，豈料爸爸4月底病癱，「寫信給弟說爸病了，他回說很擔心」。
吳惠月回述，先生做過鋪地磚工，在她精神出狀況後，先生怕她出意外，改零賣茶葉好顧她，這幾年靠女兒淑敏在專櫃賣鞋支應租屋家用，但如今淑敏忙顧病殘父母，沒法工作，往後復健生計仍有難題。
吳惠月的小妹麗年說，最近已借20萬元給姊應急，「這陣子她恐慌到連洗澡都會怕，我也常去陪她」，她又說，3月二姊病逝、4月娘家老母病逝、二姊夫病倒，5月三姊夫病逝，「最近家族不平順，辛苦外甥女淑敏了」。

「這家真可憐」

洪父的老友阿郎伯說，去醫院探望好幾回，「他沒顧好身體病倒，這家真可憐」。當地公所已撥急難金1.5萬元。蘋果基金會也撥款暫紓急窘。

【我有話要說】

淑敏媽媽吳惠月：「以前做工掙的、勞保退休金，都為不懂事的兒子賠掉，還背債約40萬元，有時想，不如沒這兒子。只是我和先生病倒，現在靠女兒扛家，她為家犧牲很多，苦了她了。」
