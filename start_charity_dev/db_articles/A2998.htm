A2998 父中風臥床 稚女難眠

撐家支柱倒 妻憂：不知何時會好

2013年02月21日

8歲小蘋趴在爸爸懷裡，難過地說：「爸爸趕快好起來，晚上沒有你陪我，我都睡不著。」但小蘋的爸爸因中風傷腦，聽著小蘋的話，只是眨眼、無意識地揮著手。
報導．攝影╱仲芝蓉

記者訪視時，小蘋和36歲的媽媽阿蓮（阮氏蓮）一起用輔具幫爸爸固定雙手，避免爸爸誤把鼻胃管扯出來，阿蓮嘆氣說，48歲的丈夫阿龍（孫龍文）去年12月突然頭暈，就醫發現腦中風，接連開刀2次取出腦中血塊和積水，但他右半身癱瘓臥床、意識不清醒不會說話，「很怕他無法恢復」。阿蓮繼續說，和阿龍生有1子1女分別是10歲小鈺、8歲小蘋，小蘋個性外向開朗，阿龍很疼她，「以前每天晚上小蘋都要爸爸哄才睡，現在阿龍不知何時會好，想到就難過」。

1月4700元養4口

阿蓮說，阿龍早年傷耳聽覺障礙，長年都做清潔工撐家，自己也在子女陸續上學後到附近辣椒廠打零工，雖然收入只夠過生活，一家4口平安也就不奢求別的，「醫生說中風後半年是黃金復健期，要他持續就醫做復健，但我們家住得遠，坐車來回要花2小時，每周至少復健3次，我根本沒辦法分身去工作」。
當地公所急難救助承辦員說，阿龍發病時因血管爆裂情況危急，開刀時的醫療手術費高達16萬元已由醫院社工協助減免，而他後續吞嚥困難，只能以鼻胃管灌食流質營養品，每月開銷至少9000元，但阿龍一家4口目前僅領有殘障津貼每月4700元，正協助申請「馬上關懷」及低收入戶等補助，並轉介蘋果慈善基金會，基金會訪視後，已先撥款暫紓困境。

同事湊錢買營養品

阿蓮71歲的婆婆說，丈夫死得早，2子1女靠自己做看護零工養大，「小兒子和女兒都各有家計負擔，生活也過得辛苦，還好阿蓮肯吃苦，只希望阿龍加油，盡快康復」。
阿蓮的老闆說，阿蓮在辣椒廠工作3年多，工作認真努力，她為貼補家計，下班回家安頓好子女後，晚上9時還到市場幫忙包豬肉、剪檳榔，「無奈她丈夫生病需照顧，同事們已湊幾千元買營養品幫忙，希望她能堅強撐下去」。

【暖流見證】學校補助幫度難關

小蘋學校高校長說，小蘋兄妹倆在校表現都乖巧良好，他們的學雜費等開銷已由當地政府補助減免，學校正另外協助申請「學產基金」及民間慈善會等補助，盼助小蘋家早日度過難關。
