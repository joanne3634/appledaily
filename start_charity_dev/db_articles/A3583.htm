小六長女貼心顧家「只能靠我了」

2015年02月01日

52歲張炳聰癱躺病床上，為防止他動手拔掉鼻胃管、氣切管，雙手被綁著。他48歲妻子阿美帶著9歲唐氏症次女小鈺，聲聲呼喚他，只見他像見到陌生人般，毫無表情。報導•攝影╱張嘉恬

阿美說，先生張炳聰靠油漆零工維持家計，去年（2014）10月底不慎從高處墜落，導致顱內出血，目前仍意識模糊、四肢癱瘓，阿美得在旁照料，但家中尚有11歲長女，夜間只好聘請看護協力。
記者陪同阿美和小鈺返家後，只見9歲小鈺，突然一股腦兒摟住正翻動鍋鏟的媽媽還說了一聲：「媽咪我愛妳！」接著阿美紅著眼眶說，很多同鄉從越南嫁來台灣，「每個人過的生活不一樣，每個人命運不同，我和丈夫錢賺得不多，還生了個唐寶寶，他現在又出事情，以後日子不知會怎樣，但我想有家庭就要有責任。」

「抱著媽媽一起哭」

過了半晌，阿美就讀小六的長女小瑛放學回家，她放下書包換了便服，馬上替妹妹洗澡，她說爸爸發生意外後，常看媽媽在偷哭，「每次媽從醫院回家，我問媽說爸什麼時候會好，她說醫生不敢保證，她也不知道，然後就忍不住哭出來，我就抱住媽媽和她一起哭。」小瑛還說，媽媽在哭的時候總對她說：「以後只能靠我了，所以我要認真讀書。」

打零工賺薄薪苦撐

協助阿美唐氏症次女小鈺早療、安親的社工表示，小鈺爸零工收入微薄，小鈺媽為補貼家用在五金工廠打零工，平時4 口收支緊絀，「小鈺媽非常認真工作，天天加班，連周末也不休息。」社工也說，小鈺爸出意外，每月看護及抽痰管、尿布等耗材費就要2萬多元，困境雪上加霜。
張炳聰的謝姓同事說，打零工收入微薄，「大家都是做1天工賺1天錢，也沒勞保，生活清苦，他家裡又有唐氏症小孩，照顧起來又累一點，他出事，大家盡力幫了10萬元，但已用盡在張炳聰初期專人看護費與醫療耗材費」。張炳聰雖有4手足，但也靠零工苦撐家計，協助有限，蘋果基金會獲悉訪視後已撥款暫紓困。
