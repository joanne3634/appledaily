A3533 撐家男抱憾逝 留下5口老幼殘

2014年12月01日

「阿益住院2個禮拜就走了，太突然！」41歲阿芳（林素芳）紅著眼眶說，她4月生下么兒小軒，當時阿益滿臉笑意、抱著孩子說，多1個兒子，肩上責任更重，「要更努力工作栽培3個兒子讀書。」如今已天人永隔！報導•攝影╱韓旭爾

阿芳說，阿益才33歲，9月下旬他因右手無力又癲癇發作，診斷罹患腦膿瘍，手術前醫生就說阿益有先天性心臟病，開刀風險高，但不開也不行，結果術後因心臟衰竭往生，「臨終前，他一直對我說，很抱歉闖不過這一關，要我堅強點，代替他孝順父母，好好撫養3個孩子長大。」

「爸來夢裡看我」

阿芳說，她能感受阿益當時心裡滿是不捨和掛念。她與阿益都是再婚，9歲、7歲2子是阿益與前妻所生，夫妻所生的小軒才8個多月，過去阿益做水泥零工月賺2、3萬元養家，她在家照看孩子。
同住還有54歲的婆婆，64歲公公及31歲小叔、29歲小姑。婆婆是重度智障，公公泰山伯右腿不良於行，平時在自家2分大農地種紅豆，年收成僅賺4、5萬元，小叔車禍後右手萎縮已失業很久，小姑在髮廊工作，每月收入約1萬多元，阿益死後，喪葬費靠國保補助及慈善會幫忙才完成。
採訪時，9歲長子小義拿出爸爸生前使用的手機說，裡面有爸爸的照片，「我想他時，就拿起來看一看。」7歲次子小廷說：「爸爸有來夢裡看我，帶我去買玩具。」一旁的阿芳聽了，不停哽咽掉淚，她說，阿益在天之靈一定會保佑她平安順利養大孩子。

翁：淚往肚子吞

泰山伯也說：「我的眼淚都往肚子吞，怕阿益的靈魂看到，行嘸開腳（台語，意謂捨不得離開），不願跟著菩薩到西天去。」阿益的妹妹阿月說，兄妹都只讀到國中，「實在沒能力幫去世的大哥分擔孩子教養花費。」當地村長說，阿芳現就靠政府社福補助1萬700元生活，家裡8口有5人是老殘幼，實在辛苦，阿芳說：「小軒還要我照顧，我只能盡力找家庭手工，加減賺錢貼補家用。」蘋果慈善基金會獲悉訪視後，已先撥款暫紓阿芳一家困境。

基金會編號：A3533
