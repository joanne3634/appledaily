A3551 殘父盼裝義肢 陪女出嫁

15歲女立志跑船「帶爸媽環遊世界」

2014年12月24日

55歲邱鑽榮癱臥病床，49歲妻子陳俐臻在旁照料，記者見邱鑽榮臀部包覆紗布，沒法直立坐起身子，他下半身僅餘一條乾瘦右腿。報導•攝影╱向高彬

邱家讀國三的獨生女小玲說，爸爸癱瘓13年，從她有記憶來，爸爸都癱臥床上，近1年為臀部褥瘡反覆潰爛所苦，9月中引發敗血症，醫師從左臀以下削肉全腿截肢，在右腹做人工肛門。探訪時，邱鑽榮很無奈地對記者說：「就是按奈啦（台語：就這樣啦），不能走、一隻腳又沒了。」

「從小爸沒力抱我」

陳俐臻回述，先生受傷前做電銲工，當年921地震，他到災區重建修屋墜樓癱瘓，傷後用罄約80萬元理賠及積蓄，試過所有中西醫療法都無法再站立，先生一直難接受、曾想輕生；到前幾年，隨著女兒漸大，他突然想通了，想要好好陪女兒長大，但這兩年先生的大姊、母親接連過世，「現在他又截肢，這陣子情緒又陷入低潮，不太講話。」
小玲說：「從小爸爸沒力氣抱我，現在我長大了，我會幫爸爸跑腿，有時揹他坐代步車，讓他出門透透氣。」她說，盼明年上當地第一志願高中，「我家靠海邊，以後我想跑商船，看很多不同國家，還想帶他們環遊世界。」小玲的班導師說，小玲開朗孝順，「學習態度一直很認真。」老師並推薦小玲參與教育部慈孝家庭楷模，獲獎表揚。

3口家晚餐僅百元

邱家3口居住的舊平房僅約8坪，陳俐臻說，目前每月家計靠低收、家扶及張榮發基金會補助共約1萬7000多元，之前夫妻還能做家庭代工，每月可多5000元，「我和先生不吃早餐，午餐吃饅頭夾蛋，3人晚餐錢不到100元簡單度日。」「先生說希望過些日子，裝一隻義肢給他，以後女兒出嫁時，他坐輪椅看起來還是兩隻腳；哪天他走時，人是完整的。」
邱鑽榮的姊夫說，鑽榮受傷前很樂觀，「現在很自卑，我們常去看看鼓勵他。」陳俐臻的弟弟說，「10多年來，姊姊揹姊夫回診抱上抱下，不喊一聲苦。她說先生為家受傷，她有責任顧他一輩子。」
