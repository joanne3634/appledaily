A3791 癌母牽掛「怕傻女兒沒媽」

4口3病弱 么女打工貼補家計

2015年11月05日

晚上9時，54歲癌母劉秀玲幫重度智障的二女兒小如刷牙，邊跟她說：「以後妳去教養院了，媽媽不能幫妳刷牙，妳要自己刷乾淨，不然髒髒。」29歲小如嗯啊沒什麼回應，也不知她聽懂沒。報導•攝影╱向高彬

劉秀玲幫小如擦臉，再催促讓她進房就寢，但小如抱薄被側身躲房門旁，想等劉秀玲陪她睡。劉秀玲跟記者嘆道：「家裡一波未平、一波又起，是老天爺給我功課吧。」

鬱夫出門會迷路

劉秀玲與58歲丈夫阿文育3女，32歲長女已嫁，20歲小女兒小瑄在外地讀夜間部大三，她說，二女兒小如幼時學講話慢，5歲經幼稚園老師提醒到醫院查出是重度智障，至今心智像4歲童，連上廁所都需幫忙，小如一直由她照料，本來夫妻想照顧小如到老，但先生3年前精神出問題，靠她做洗碗工扛租屋家計，生活過得勉強，未料7月她發現罹患結腸癌三期，體力弱，「只好準備把女兒送教養院公費安置。只是我若萬一，最牽掛這傻女兒，以後沒媽了。」
劉秀玲又說，先生阿文憂鬱病況嚴重，「他知我得癌後，一度找出過期駕照想去換照再跑車，但他出門會迷路，哪還能做事。」採訪時，阿文眼神渙散地說，「我們……這個家靠太太，家裡不能沒有她。」如今劉家4口每月低收殘障補助共2萬2300元，扣房租8千元，剩不到1萬5千元，難敷租屋醫病家用，蘋果基金會獲悉訪視後已先撥款暫紓困。
阿文的雙親歿，與兩兄弟少聯絡。劉秀玲的大姊說，娘家共7手足，大弟胰臟癌歿，兩弟顧8旬病母，姊妹們有時拿1、2千元接濟秀玲，「我跟她說，先安心治療才要緊。」

半工半讀不喊苦

記者至小女兒小瑄打工處探訪時，她說從高一就打工，現在每月掙1萬餘元，省吃儉用約3千元給媽媽，「以前不懂事，只想自由，現在卻是無限想家。我白天上班、晚上上課，一點也不辛苦，只要能幫到媽媽，一切都值得。」
