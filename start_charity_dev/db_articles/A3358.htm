A3358 癌父驟逝 4女失依靠

生前身兼母職 後事困窘

2014年05月04日

殯儀館靈堂裡，4個女孩哀傷看著爸爸余志宏遺照，盼爸在天之靈一路好走。讀高一長女小琳說，罹癌爸於4月26日病逝，「他病前交代我要好好代他照顧妹妹。」如今余志宏7日周三將出殯，家裡還愁喪葬費。
報導•攝影╱向高彬

記者3月中在醫院初訪42歲余志宏時，他雖同時罹下咽癌及食道癌，但氣色仍好，4個女兒在一旁陪伴照顧，讀國三的次女小萱還特別跟記者說，爸爸是「無敵鐵金剛，什麼都會，沒什麼難得倒他。」小琳說，爸媽離婚5年，「爸身兼母職照顧我們4個女生，內衣褲和生理期用品，都是他幫我們準備。」
那時女兒貼心的言談，讓余志宏生前感性地說：「想到4個寶貝，我怎麼可以丟下她們。」但言猶在耳，余志宏上月底突陷昏迷，女兒還特地到大廟為他祈福、願折壽救父，沒料到他仍不敵癌魔溘逝。

不發訃聞沒白包

余志宏的59歲父親阿雄伯白髮送黑髮，他愁容說：「志宏病後這1年，已花了不少錢，如今他走了，祖母長輩仍在，依習俗不發訃聞、沒白包，喪葬費讓我很傷腦筋，還在找人籌借。」
余志宏生前跟記者回述，他與前妻阿芳生有現17、15、14、13歲的4女，做工地雜工養家，但去年中他做工摔傷，緊接發現罹兩癌；他59歲父親阿雄伯曾非婚再生一個讀小五的兒子，做粗工養小他近30歲的小弟，「爸年初摔車脊椎受傷開刀，又長骨刺，行走仍疼痛，醫師說還需休養一段時間。」當時經當地公所轉介蘋果基金會，基金會訪視後曾撥急難金3萬元。
余志宏的前妻阿芳近日從外地回來協辦後事，她說：「眼前要先辦好志宏的後事，再重新找工作負擔4個女兒求學開銷。」

低收補助難養7口

當地公所社會課表示，目前阿雄伯暫無法工作，現每月低收、家扶及張榮發基金會補助共約3萬元，扣除房租水電後，剩1.5萬元難敷7口生計。
