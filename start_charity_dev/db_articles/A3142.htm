A3142 靠每月2600元補助苦撐 11歲女守癌母 「換我照顧妳」

2013年08月06日

往年暑假，11歲小祐愛跟著她的養母李芳樺一起做清潔打掃工，在旁當小幫手，但今年暑假，她在家照顧52歲罹結腸癌的媽媽、陪她赴醫院回診。報導．攝影╱向高彬

小祐將升小六，她說，本來媽媽擔憂化療後虛弱沒力照顧，想送她去寄養家庭安置，但她堅持不肯，還說：「我長大了，我來照顧妳，我愛妳，我要跟妳在一起」，讓媽媽聽了眼淚直流。

穿二手衣鞋不抱怨

由於小祐鼓勵，讓李芳樺更鼓起勇氣抗癌，李芳樺說，離婚3年來靠清潔零工、撿拾瓶罐回收，單親撫養女兒，她一天工錢700元，每月約15天工，女兒知她掙錢不易，假日陪做工拿刷子刷、拿抹布擦，「平常她穿人家的二手衣鞋，不會跟人家比好壞，都說有就好。母女常煮一鍋菜湯雜燴吃3餐。我病後，她懂事很多，最近說，以後半工半讀學美髮，就有穩定工作，還可幫我剪漂亮頭髮」。
本來母女勉強能餬口度日，但李芳樺愁言：「5月初拉血，到醫院檢查才知是結腸癌三期，醫師說癌細胞快侵犯到子宮、卵巢，手術一併切除，和裝人工肛門」，如今李芳樺抗癌沒力做工，母女每月僅有3款低收補助2600元，難敷房租生計。小祐的學校已協辦救助金3000元，當地公所社會課撥救助金2萬元，公所經評估抗癌漫長艱困，再轉介蘋果基金會，基金會訪視後也撥款暫紓急窘。

「養女是我的生命」

李芳樺回述，年輕時曾接觸學佛，一度想出家，人生起伏經歷工作不順、前夫不好、婚姻受挫、自身不孕、領養小祐等事，她認為一切事物有因果，「病了，也坦然接受，女兒還小，雖非親生，但抱來養，她叫我媽媽，我就有責任，她是我的生命。」李芳樺的大哥說，自家兄妹或已歿、或自家困難，難多幫阿樺。
在醫院採訪時，李芳樺的近30年好友廖小姐來探望，她說，阿樺從小父早逝家貧，沒讀書不識字，阿樺離婚後養女兒，「先前工作不穩，付不出房租，當時我和姊姊幫她，看她這一年平順些，結果竟罹癌，唉，苦命人越遇到苦命事」。

基金會編號：A3142
