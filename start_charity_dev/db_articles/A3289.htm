擠工寮燒柴生活「想不到未來」

2014年01月28日

63歲湯雪娥駛著車齡18年老舊手排貨卡，將高冷蔬菜從部落載下山趕早市擺攤，記者看她隨崎嶇山路熟練進退檔，因無法疾駛被閃大燈仍自在從容，就像在夫殘、子病、孫年幼，常出事端的家中，一直扮演支撐者角色。
報導．攝影╱張嘉恬

湯雪娥結束擺攤，在前往醫院探視兒子的途中對記者說，67歲先生因關節病變、蜂窩性組織炎長年行走有礙，去年10月底才動清創手術，40歲兒子范安倫接著因壞死性筋膜炎、肝硬化昏迷就醫，現逐漸甦醒但仍無法自理生活。她說，她和老伴、兒子、36歲媳婦阿惠及13歲和4歲兩孫女6口同住部落工寮，平時靠向親友借地務農為生，兒子、媳婦也在部落打零工補貼家用。

「部落阿信」不喊苦

阿惠說：「先生已脫離險境，但日後需人照料，我想不到未來。」目前湯雪娥全家6口僅靠她賣菜及補助每月不到2萬元，蘋果基金會獲悉後已撥款紓困。
湯雪娥離開醫院，趕在傍晚前回部落替先生打理晚餐，記者陪同她返家途中，一名鄰居說：「她是他們家的水泥柱，部落裡的阿信！照顧老公、擔心兒子，每天部落、市區來回好幾趟，看起來不累但心一定是累的！」她說：「看別人悠閒度日，我不羨慕，每個人有不同的命，過日子，總是開心比難過好。」
服務湯雪娥13歲孫女的當地尤社工表示，孫女常在睡前禱告求爸爸快好、快回家、快能講話，想聽爸爸聲音，尤社工也說，范安倫雖尚有21歲長女在外租屋打工，每月已盡力協助2至3千元，18歲次女已嫁育1幼子有自家負擔協力有限。另一名17歲長子因有學習障礙，目前靠在部落替人揹行李的零工薄薪支應自己的生活費。

「生活本就靠自己」

湯雪娥說，她尚有長女與次子，但都辛苦過日。入夜傍晚，她一邊準備燒熱水，一邊說：「老公不能賺錢、兒子又重病，朋友說我像男人撐起這個家，我只覺得生活本來就要靠自己。」
