A3505 日做18小時 父過勞重癱

兒打工攤家計 「等爸爸回來」

2014年10月27日

「南無大悲觀世音菩薩。」50歲的陳綵瀅念著佛號，邊安撫病床上的54歲丈夫林國清，只見林國清雖無法講話，卻雙拳緊握、睜大眼睛，陳綵瀅無奈的說：「今年8月丈夫心肌梗塞昏迷後，就一直這樣了。」報導•攝影╱江品璁

陳綵瀅說，丈夫在水公司當契約工，1年前又找了滑石粉工廠的兼差，「他清晨6時就出門，晚上12時回家，一天休息不到5小時，想不到今年8月9日突然心肌梗塞，導致腦缺氧昏迷，到現在都沒有意識。」

母哭喊：快回來

陳綵瀅嘆道，上月，丈夫出加護病房後都認不得人，表情一直很驚恐，「病倒後好像失了魂，都不認得家人了，他是很信神明的人，我很希望神明可以給他一道光，讓他可以跟著光一起回來。」醫院社工說，林國清病情一度危急，但在醫生拔管時奇蹟存活，現重癱後續有長期安置問題，僅靠妻子當清潔工每月2萬多元、母親老農津貼7千元，難敷5口開銷，故轉介蘋果基金會，基金會訪視後已撥款暫紓困窘。
之後記者到林國清家中訪視，他88歲老母金月嬤哭著對記者說：「兒子很打拼，我叫他別做了，他都要我別煩惱，很孝順，知道我最近身體都怪怪，還說要帶我去看醫生，結果我還沒倒，他就先倒在那了，國清啊，你就快回來啊。」

盼家境變好出遊

18歲讀高三的長子小敬說，自己跟弟弟暑假打工分擔家計，「爸爸曾說，等家境變好全家一起去日本泡溫泉、喝清酒，「我告訴他，一定要好起來實現約定。」17歲讀高二的次子小霖說，「我和哥哥會半工半讀完成學業，全家人都在等爸爸回來。」
林國清工廠同事王柏森說，林國清是臨時工，但認真、勤奮，他很少講家裡的事，他孩子讀高中正是花錢年齡，所以兼兩份工，「真的很不捨這樣一個好同事。」林國清大哥說，家中共4兄弟，但都各有家庭負擔，協助有限。當地里長說，已協辦急難金補助。
