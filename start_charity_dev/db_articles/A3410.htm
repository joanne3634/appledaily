太太自責：我不是好媽媽 拖累大家

2014年07月07日

「媽，小心燙。」現讀醫專1年級的小萱餵母親喝湯，只見阿春僅能張開嘴吸著湯汁，小萱說，媽媽有糖尿病，沒想到去年10月中風癱瘓，今年3月又褥瘡、腎衰竭住院一個月，家裡沒人照顧，只好先送機構。報導•攝影╱江品璁

一旁阿春46歲的先生阿輝說，自己多年來受椎間盤移位所苦，都要穿束腰固定，無法做粗重零工，只好跟太太一起擺流動攤販賣涼麵，每月約2萬多，加上低收入戶補助1萬4100元，扣掉房租5千元，養1兒1女生活還可過。但去年8月，讀大二的兒子小宇騎機車載同學，搶黃燈撞上休旅車。阿輝嘆道，兒子車禍沒多久，換太太癱瘓，現在每月光是養護費和耗材就要2萬多元，「剩我一人賣涼麵，負擔變得很大」。

車禍醫費兩頭燒

兒子小宇說，自己本有打工，但車禍後左右腳骨折，「找工作都受限，我也很想幫家裡減輕負擔。」阿輝說：「他同學手腳都摔斷，對方家長要我們負責，協調會開了2次都無法和解，車主也要我們賠錢，根本沒錢可賠，只好走法院，等判決結果。」
這時，病床上的阿春掉淚說：「本來家境就不好，又多我這筆費用，我不是一個好媽媽，也不是一個好老婆，拖累大家。」
記者跟著阿輝回家，他趕著調製涼麵所需醬料。他說，以前客人買涼麵還會配個飲料，「但現在景氣差，很多客人都只單買小份涼麵，每月能有1萬多元收入就不錯了。」

「為兒我要堅強」

阿輝指著一旁精神科藥物說：「你看這些都是我在吃的，這半年來，我會不由自主流眼淚，會想不開，醫生說我有憂鬱症，但想到2個孩子還沒出社會，太太又還癱著，我也要堅強。」
阿輝的二姊阿蕙說，家裡5手足，三姊妹皆已出嫁，「我先生今年也才因大腸癌過世，大家都有家庭經濟難題。」教友黃明順說，知道阿輝家出現困難，故轉請蘋果基金會關懷，基金會訪視後已先撥款暫紓困窘。
