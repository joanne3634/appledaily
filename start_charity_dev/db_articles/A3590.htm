盼伴稚兒長大「撐到牽女兒步禮堂」

2015年02月09日

42歲蔣再粒罹患食道癌半年，他虛弱躺在床上，和40歲妻子卓麗娟四目相對，夫妻倆看著身旁圍繞的13、12、11及8歲4個孩子，想到無法掌握的病況、沒有把握的未來，頓時間，兩人都紅了眼眶。報導•攝影／張嘉恬

卓麗娟說，他和蔣再粒同居15年，生了3男1女，夫妻倆都做粗工為生，一直沒餘錢籌辦婚禮，直到蔣再粒去年6月得食道癌，才至戶政事務所登記結婚。「醫生跟他說是癌症後，他馬上提結婚，因為他想要真正家的感覺。」
卓麗娟接著說，先生在妻兒面前裝堅強，內心脆弱，「表面上是硬漢，晚上我都聽到他邊咳痰邊偷哭，他最疼8歲女兒，我只能鼓勵他，要撐到牽女兒的手步入禮堂。」

長子：我會顧弟妹

蔣再粒懷裡的女兒小慈接著對記者說：「把拔剛生病還可以講話時有次對我說，如果他死掉，怎麼辦。我跟把拔說，你會活很久啦。」只見蔣再粒將小慈往懷裡摟得更緊了。
卓麗娟還說，經濟壓力加上擔憂先生病情，她被診斷罹患焦慮與恐慌症，常發作心悸、氣喘，還曾路倒多次。一旁她13歲長子小辰則說，爸爸在房間休息時，媽媽常常一個人在客廳想事情，想著想著就流眼淚，「我安慰媽咪說，我說我是老大，會顧好弟弟妹妹，要她放心。」
與蔣再粒夫妻共事多年的友人阿成說，夫妻一直都很勤奮打拼，「常追著我問還有哪裡有工得做，夫妻協力養家，雖沒什麼錢，但一家人幸福和樂，現在一家之主病了，全家亂了腳步。」

手工月入僅2千元

目前蔣家每月僅靠低收補助共1萬6300元，雖然卓麗娟做手工補貼開銷，但每月最多僅2000元，但蔣再粒管灌營養品及氣切傷口耗材費就共需2萬元。蔣的父親肝癌病逝，母親罹腸癌現由出嫁妹妹照料，哥哥則罹鼻咽癌自顧不暇，協力有限，基金會獲悉訪視後已撥款紓困。
