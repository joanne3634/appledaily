A3269 童窩睡沙發 顧重病媽

孝順兄妹分工「還好有2寶貝」

2014年01月07日

12歲小凱依偎著單親媽媽楊芷維，替她按摩手腳，還真情流露：「媽咪妳是爸爸也是媽媽，好辛苦，我好愛妳。」47歲楊芷維罹患類風濕性關節炎，雙腳無力行走，1年前雙手也萎縮，她說：「還好有2個寶貝孩子，我才看得到未來。」
報導．攝影╱張嘉恬

傍晚時分，她讀小五11歲女兒小瑜下課進門問：「媽咪，今天心情好嗎？我講笑話給妳聽！」只見癱躺沙發的楊芷維，用腰撐身體力不從心，換靠手肘也無法坐起，得由子女合拉雙手才勉強起身，採訪時寒流報到，她瑟縮身軀說：「最怕冬天，關節反覆發炎痛到睡不著，這病只能吃藥打針控制，走路也靠助行器。」

無法工作存款耗盡

楊芷維早年做粗工曾摔傷脊椎，5年前罹類風濕性關節炎手腳漸萎縮，「半年前併發纖維肌痛症，什麼事都做不了，存款用完積欠房租，被錢追著跑，因憂鬱症通過精障鑑定，我身體整組都壞了」。她說，1年多前不堪貧、病吃了過量安眠藥，還是讀國一12歲兒子小凱打119救了她。
楊芷維憶及，醒來後小瑜問她是不是不要她，「我才整個清醒，還有2個懂事孩子，是丟不掉的責任」。
社工表示，因媽媽無力做家務，小凱主動製表和妹妹分工，「小瑜為照料媽媽上廁所，每晚陪睡在客廳，媽媽睡長沙發，她睡短沙發」。小瑜導師說：「她很樂觀，熱心助人」。已協辦午餐費減免並申請獎助學金。楊芷維昔日工地徐姓同事說：「她有難大家互相，我盡力協助她9萬元，還能做的是幫買飯、載送就醫。」

補助不敷4口開銷

楊芷維早年離婚，後與男友生有小凱、小瑜，但因男友家暴10年前分手。她和前夫所育長女已嫁、次女小婷婚變，小婷的3歲女兒也與楊芷維同住。20歲小婷說，因國中畢業學歷有限，目前在外地租屋打工收入微薄，「讓媽媽照顧女兒不應該，我正找固定月薪工作，希望能有些分擔」。目前楊芷維每月靠低收、家扶與慈濟補助共2萬3850元，扣除房租8000元後不敷4口開銷，蘋果基金會獲悉訪視後已撥款紓困。
