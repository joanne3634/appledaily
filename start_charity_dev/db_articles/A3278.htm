A3278 孝子摔斷腿 洗腎母苦撐

「不捨媽為我奔波」

2014年01月15日

54歲單親媽阿揚洗完腎，急忙搭電梯走回同家醫院的兒子病房，因長期洗腎臉色暗沉、憔悴的阿揚氣喘吁吁說，兒子去年10月底因車禍重傷，無法自理生活，需要她在旁照顧。報導•攝影╱張嘉恬

病房裡，阿揚現年20歲兒子易晏廷無法起身只好在病床上大號，媽媽阿揚清理後拉開布簾，只見易晏廷一臉困窘，不捨又無奈說：「真的很難過，自己的疏忽，讓病弱的媽媽為我奔波辛勞。」

子不休假累到昏倒

易晏廷說，去年10月底為閃避後方急速超車的車輛，不慎自撞路邊障礙物，右腿股骨、髕骨、脛骨骨幹骨折、韌帶斷裂，右手掌也骨折，需休養復健至少一年以上。
阿揚說早年腎臟功能不佳，但懷孕10多天經檢查才知病況嚴重，醫生曾建議中止妊娠否則終身洗腎，「早年我有段婚姻沒有孩子，後來未婚懷孕，覺得是上天賜給我，當時孩子父親也離開了，我只剩肚裡的寶貝，所以決定冒險生下」。
阿揚說，早年她靠打字員收入加上手工苦撐家計，日漸病重無力工作，兒子便在升高二時休學，「他從沒怨我給他的出生背景、家世，或讓他必須中斷學業，還為多賺錢，在便利商店時曾排滿一個月大夜班不休假，累到昏倒」。
易晏廷昔日餐廳店長王小姐說，他凡事以媽優先，「休假就是陪媽洗腎，同事邀他唱歌吃飯，他也說賺錢要繳房租、養家，工作很有責任感，總願替別人多分擔一些，很難得的年輕人」。

母當年冒險生下兒

易晏廷說，從有記憶他就跟著媽媽在醫院洗腎，「媽在我小六時跟我說她冒生命危險生我時，就決心一輩子都會帶著她、養她」，他也說，媽雖有4個手足，但都要顧自家生計協助有限，但會在臉書上為他打氣，「精神鼓勵也是重要的支持力量」。
易晏廷平時餐廳工作收入每月約2萬多元，支應房租9100元與母子生活已吃緊，因他在休假日發生自撞車禍，無法符合勞保職災及強制險認定，目前每月僅有阿揚殘津4700元，不敷母子生活醫療，蘋果基金會獲悉訪視後已撥款紓困。
