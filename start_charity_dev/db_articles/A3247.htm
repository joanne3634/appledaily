A3247 7口擠破屋 「怕被活埋」

女休學顧罹癌外公炒菜得撐傘

2013年12月13日

醫院病房裡，67歲友安伯（曹友安）口齒不清地說，他嚼了40年檳榔，去年中發現罹患舌癌，「今年4月右臉頰出現硬塊又化膿，癌症擴散了，醫師說癌症治療不易，鼓勵我一定要堅持下去。」報導．攝影╱韓旭爾

57歲老伴阿美婆（羅阿美）說，夫妻育有2女，多年來與單親長女阿君、3個外孫及弟弟13歲女兒小琪同住，家裡還有80歲老父，因阿君有嚴重憂鬱症，無力工作，3外孫還在讀書，7口家生計全靠他們夫妻做零工維持。怎料阿君在去年11月意外去世，「1年多來家裡變得亂糟糟！」阿美婆次女曹羅以說她在外地教會擔任義務性質傳教工作，對家裡經濟幫不上忙。

感謝姑姑給個家

提起小琪身世，阿美婆說，小琪是弟弟13年前非婚所生女兒，「當年生母把小琪帶到家裡、丟下就走，弟弟不承認小琪是自己孩子，孩子總要有人照顧，我就當家裡多一個小女兒。」阿美婆說，為了小琪，她與弟弟多次爭吵，多年無聯絡。
記者到阿美婆家中探訪，阿美婆18歲外孫女小馨正帶15、12歲妹、弟和小琪3人摺衣物。小馨說，去年底為處理媽媽喪事，加上外公婆常在醫院，她只能先休學幫忙家務並與外婆輪流在醫院顧外公。今年讀國一的小琪說：「感謝姑姑給我溫暖的家。」記者觀察居住環境，廚房屋頂一角已塌陷，小馨說：「下雨天炒菜，要有人幫我撐傘擋雨才行。」讀高一的妹妹小亞說，牆壁四處都有龜裂，「擔心地震一來，全家被活埋。」

夫妻情操感動人

附近教會王姓師母說，這些年來從未看過阿美婆弟弟來關心小琪，阿美婆夫妻對養育小琪、3外孫從不抱怨，令人感動。家婦中心社工說，阿美婆一家領有老人年金、父親老農年金共1萬4000元，但友安伯就醫交通及流質營養品每月就要上萬元，轉介蘋果基金會協助，基金會訪視後已先撥款紓困。
