A3904 跛父心衰竭 自責孩子吃苦

2016年04月13日

來到39歲單親爸陳威宗與兩兒女租住屋裡，他頭髮斑白，樣貌看來比實際年齡老上好幾歲，他瘸跛右腿，垂頭喪氣地說話，「我已半年沒工，房租付不出來，快養不了小孩了。」
報導•攝影╱向高彬

讀國一的大兒子小楷疼惜地看著爸爸陳威宗，跟記者說，爸走遠一點就喘，有時腳沒力會跌倒，「他也沒力氣拿很重的東西，現在每天是我去倒垃圾和曬衣服。」陳威宗聽見孩子話語說：「很難過，兒子才13歲，女兒才8歲，讓小孩跟著我吃苦了。」
陳威宗掀開衣服和褲管，露出背腿手術痕，右腿略萎，他回述，18歲時曾發生嚴重車禍頸椎及右腿骨折，那時復健近2年，才可緩步行走，「婚後孩子出生，但我書讀不多、沒專長，經姊姊朋友介紹曾做殯葬工作，撿骨、洗大體、扛棺材、搭棚架的粗工都做。」

單親養家負債自卑

陳威宗又說，近4年他做墳地砍草及清潔零工，加上孩子低收補助勉撐房租家用，欠約40萬元生活不足債務，「與前妻3年前離婚，孩子全給我，再沒她消息。」去年初他發覺右腿乏力，不時抽搐，醫師研判舊傷後遺症，又未料去年中他常胸悶，查出心臟衰竭，需長期服藥控制，「我心煩又自卑，除了接小孩就不想出門，醫師說我有憂鬱症。」
陳威宗大姊阿英說，娘家雙親已歿多年，威宗排家中老二，另有一少往來的已婚小妹，近大半年見弟弟「有不好的想法，交代我幫他顧孩子，我鼓勵他，孩子已沒媽疼，不能再沒爸，這樣很可憐。」只是「先前我開早餐店，多少幫點忙，但我有骨刺，生意不好收店了，沒力多幫弟了。」

補助不敷房租家用

陳威宗小女兒小亭的班導師說：「社工曾與我聯絡孩子家情況，我幫忙申請獎助學金，持續關心。」當地社福中心社工說，目前陳家每月領低收補助5390元，但房租就7000元，「我們正協助殘障鑑定，並幫找手工工作貼補家用，因公部門資源有限，經評估轉介蘋果基金會，此時若可幫他減輕身心壓力，相信有機會漸回常軌。」基金會訪視後已先撥款暫紓困。
