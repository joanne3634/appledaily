A3180 妻喚失憶夫 「快想起孩子」

11歲童擔心媽 貼心幫家務

2013年09月23日

「啊啊啊！」49歲的陳文祥大聲叫嚷，今年8月他因二度中風神志不清，常鼓譟不安，一旁6歲次子小瑞，被父親的吵鬧聲嚇到，張大眼睛說：「爸爸跟以前不一樣了。」
報導．攝影╱江品璁

「你叫什麼名字？」面對妻子詢問，陳文祥雙眼無神搖了搖頭，36歲的洪小喬說，先生今年3月曾中風，半身癱瘓，8月又罹水腦症、腦出血，「一句話都沒講就倒下，醫生說血塊壓迫大腦，記憶受損，人都認不得了，僅一半機率能好轉，整個家也跟著垮。」
洪小喬說婚後生2子，6年前罹血栓致心臟受損，領有輕度重器障證明，為維持家計，還是去做工，「但先生中風失智，每周須復健5天，無法上工，僅靠低收補助共1萬8100元維持，扣掉房租5000元，快過不下去，只好求助蘋果基金會。」基金會訪視後已撥款紓困。

「爸像孩子一樣」

11歲的長子小榤（榤音同傑）說，爸爸每次生病不看醫生，都去藥局買成藥，「現在爸爸還會流口水，像小孩子一樣，媽媽要照顧全家好辛苦，我只好幫忙整理家事。」洪小喬說，先生失智後把家人都忘了，「就算他再也不能走，我也沒關係，只要他能想起我是誰和孩子的事，不要都認不得人。」洪小喬邊說邊流淚，小榤也紅了眼眶拿面紙替母親拭淚，一旁次子小瑞歪頭疑惑問媽媽：「妳怎麼眼睛都紅紅的？」令人心酸。

師幫申請獎學金

陳文祥大姊阿珠說，家裡3兄弟姊妹，大哥已過世，弟弟中風當然煩惱，「但我現在無業，只能偶爾接濟1000∼2000元。」小榤的吳姓導師說，小榤成績本不錯，但他爸中風後小榤常請假，數學跟不上，已加強輔導並申請獎助學金。當地村長說已協辦馬上關懷急難金2萬元。

基金會編號：A3180
