A3408：3兄弟病殘 月靠4千過活

撐家大弟罹癌倒下 嘆「對不起媽」

2014年07月02日

「看建雄這樣，做哥哥很不忍心，因為舌癌連飯都不能好好吃，只能用灌的。」建雄55歲的大哥呂榮雄難過地說，53歲的大弟建雄去年底發現罹患舌癌且擴散到淋巴；同住的47歲小弟文雄則腦瘤開刀後手腳無力、視力認知也受損。
報導•攝影╱韓旭爾

呂榮雄說，爸爸早逝，媽媽在3年多前也因中風往生，一家有3男2女共5手足，大姊和小妹出嫁多年，3兄弟中，他和大弟只有國中畢業，做綁鐵粗工收入不穩定，女孩看不上眼，兩人都未婚。而47歲小弟文雄原本當遠洋漁夫，收入較多，曾結婚育有3子女，「但他30歲時因腦膜瘤開刀，造成視力、認知受損，左側手腳無力、癲癇不時發作，領有重殘證明」，他病倒不久，妻子就帶孩子離家，失聯多年。

兄傷腦弟重殘

指著自己腦袋上的手術傷疤，呂榮雄說，他5年前車禍傷腦，造成右手腳無力，無法再做粗工，小弟文雄又重殘，這些年來，3兄弟都靠建雄撐家，「3兄弟就這樣相互支持，簡單生活。」沒想到，去年9月建雄舌根潰瘍久不痊癒，拖了2個月到大醫院檢查，竟是舌癌。
建雄神情疲憊、啞著嗓子說，自己做綁鐵工，難免抽菸吃檳榔，如今罹患舌癌才體會生不如死的感覺，「媽媽臨終前特別放心不下大哥和小弟，她說，兄弟是一輩子的事，交代我要好好照顧兩人，如今我罹患癌症，沒把握還能活多久？要對不起媽媽了。」

管灌品開銷大

這半年來，榮雄因照料化療的大弟，文雄一人在家，而文雄本無法走路，三餐、大小便都要人幫，榮雄說：「說也奇怪，前陣子文雄居然能拿著拐杖走路，癲癇也少發作，我想是上帝同情我太累，賜恩典讓文雄的病況好轉。」文雄說：「我要學著自己顧自己，大哥要顧兩個人太累了！」
當地部落營造協會理事長說，榮雄3兄弟生活僅賴文雄重殘津貼每月4700元，目前建雄的管灌品每月就要1萬元，因此轉介蘋果基金會協助。榮雄大姊說，她與小妹各有家庭負擔，經濟上沒能幫上忙。基金會接案訪視後，已先撥款暫紓呂家困境。
