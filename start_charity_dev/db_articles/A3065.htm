A3065 7口老殘憨 婦愁出一身病

「沒人賺錢」 不敢開刀 吞藥止痛

2013年05月07日

43歲單親媽阿玉去年3月才置換右髖關節，現左髖關節也因缺血性壞死得手術，罹患中度精障的她，擔憂7口老病幼生計，說話顯得焦慮，「爸爸出車禍、媽媽換膝關節，大女兒和兒子智能有障礙，家裡沒人能賺錢。」
報導．攝影／張嘉恬

採訪時阿玉正一字一句閱讀《蘋果日報》，她說，7年前因罹患躁鬱、恐慌症等精神疾病持有中度精障證明，長年服藥影響她言語表達，「有點結巴所以讀報練習。」

兒女憨傻找嘸工

阿玉看著桌上堆如小山的藥袋說，一天得服20多顆藥，因6年前得了肝硬化、狹心症，5年前又切除全子宮，「現等著換左髖關節，痛到從客廳走到廁所要半小時，因無法負擔手術材料費，先吃藥止痛」。
阿玉說，同住的84歲老父去年11月車禍致四肢胸部挫傷，66歲母親也換了雙膝關節仍需休養，「哥哥揹債每月僅剩9000元要撐3口，單親姊姊工傷還要養3個孩子，都自身難保。」提及家計緊絀，阿玉說話不斷重複且語氣急促。
阿玉的先生9年前病逝，2人育有現年20歲長女、18歲次女及17歲長子，她無奈說，長女經診斷有學習障礙，高中畢業曾做包裝工但因動作慢被開除，輕度智能障礙的長子曾於洗車店、電鍍廠工作，但「人家洗車他在玩方向盤，不然就是把電鍍廠藥水抹在身上。」

憂沒人照顧幼娃

阿玉的18歲次女小瑤則為單親媽，採訪時小瑤正哄著懷中1歲幼女，她說，雖曾因不理智而結婚生子，但現女兒年幼、母重病、姊弟憨傻，「我是家裡較容易找工作的，應該多承擔。只是高齡祖父母現也需養傷，若我去工作，沒人能替我顧孩子。」
阿玉說，目前家計仰賴低收殘津等補助共1萬8000元，雖然父親每月尚領榮民月退俸1萬2500元，但需支應工傷無力工作的單親姊姊一家4口，家計入不敷出，家婦中心社工表示，阿玉家7口僅1人有工作能力，故轉介蘋果基金會，基金會訪視後已撥款暫紓困境。

【專家意見】可通報社福服務

中華民國智障者家長總會祕書長林惠芳表示，因阿玉家庭功能薄弱，可由鄰里長、社工或像蘋果基金會等社福團體通報，由社會局身心障礙者成人個案管理服務介入，協助長女身心障礙鑑定、長子至庇護工廠就業及18歲次女幼兒社區托育服務。目前阿玉家況需民間協力，銜接家庭功能重建。
