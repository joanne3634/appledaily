A3534 養憨女幼子 老運將苦撐

月賺嘸萬元沒錢就醫夜半痛哭

2014年12月02日

採訪時，61歲阿山伯（簡崐山）正在房間裡餵34歲、極重度智障的女兒阿如吃飯，只見阿山伯把一匙飯菜餵進阿如嘴巴，阿如嚼了幾口，沒往肚子吞，卻直接往外吐，食物殘渣散滿地。報導•攝影╱韓旭爾

阿山伯無奈說，阿如不會講話，病痛不會表達，「通常她不肯吃飯，就是身體不舒服。」阿山伯說，阿如前陣子接連2天沒吃飯，掛3科門診才查出罹患卵巢囊腫，住院1星期摘除卵巢。

17年舊車生意差

阿山伯說，阿如是他與亡妻所生，阿如到5歲不會說話，鑑定為極重度智障，至今連爸爸都不會叫。20年前妻子病逝，他父代母職，才發現阿如狀況多到說不完，大小便失禁、一不注意就跑出門，好幾次差點走失……，他覺得自己撐不下去了，才在13年前再娶老婆分擔照顧責任，並生下現年11歲么兒小璟，但老婆受不了壓力，7年前離家出走，最後經法院協調離婚。
這幾年，阿山伯利用照顧2子女空檔時間開計程車賺錢養家，他說：「1天跑7、8小時，扣掉油錢和修車錢，1個月賺不到1萬元。」阿山伯的計程車友人說：「阿山伯車子是17年的老爺車，搶不到生意。」
長期照顧憨女及經濟壓力，阿山伯在4、5年前罹患憂鬱症，長年在身心科就醫服藥。阿山伯說，有次半夜阿如全身過敏，想帶她去掛急診，但身上卻只剩300元，只能看著阿如無助嚎啕大哭，他差點就要想不開，「後來想到小璟還要靠我養，我走了，小璟人生也毀了，才沒做出傻事。」

「面臨雙老困境」

阿山伯讀小六的兒子小璟說：「爸爸很可憐，要吃安眠藥才能睡覺，我放假時會幫忙照顧姊姊。」
當地里幹事說，阿山伯因名下有祖傳持分土地超標，不符低收資格，補助僅有阿如殘障津貼及單親補助共7000元，他開車收入不多，阿如又常就醫，蘋果基金會獲悉訪視後已撥款紓困。
高醫大社工系主任陳政智表示，阿山伯一家面臨憨兒及父親雙雙老化的「雙老家庭」困境，需有民間資源介入，以扶助生活不足，並得以申請居家照顧，以減輕身心壓力。
