How it works?

1
填寫基本資料
詢問您的個人屬性和平時參與的公益行為，透過資料分析的方式，幫助我們更了解捐款人的特質和關注的議題，用以推薦未來的待援家庭給您。
2
閱讀暖流報導
每回合瞭解 10 個急難家庭，告訴我們您的捐款意願高低（0～100分），若您有意貢獻更多，可多做幾個回合，提供我們更豐富的資訊。
3
推薦未來的暖流版文章
推薦系統將依據 Step 1 & 2 的資料做出判斷，在適當的時機告知您可能較有意願捐款的家庭；若您有意幫助，您可以回覆對該篇文章的捐款意願，或直接按下「前往捐款」按鈕，透過蘋果日報慈善基金會捐助急難家庭。

FAQ

  • add 如何透過蘋果日報慈善基金會捐款？

    連結至《蘋果日報慈善基金會》首頁， 您可以下載捐款表格，進行信用卡或郵政劃撥、現金及支票……等任何形式的捐款。若要指定捐助對象，請填上「捐款編號」，即每篇個案報導最後所附的號碼。或致電 - (02) 6601-6999 - (02) 6601-6999、0809-008585、0809-008585 或電郵至 hope85@appledaily.com.tw 與蘋果日報慈善基金聯絡。

  • add 蘋果基金會會開立發票嗎？

    對於捐款人的支持皆製發收據，每次捐款立開收據者，在您捐款入帳後15日內寄發；採「年度合開一張」者，基金會統一於每年2-3月間寄發，以利報稅扣繳列舉憑証。

  • add 若指定捐款個案，錢會全數轉交給個案嗎？

    是的，基金會均全數轉交指定對象。

  • add 若不指定捐款對象，透過基金會轉贈的捐款將會作為何種用途？

    如蒙您「不指定」捐款，即共同匯入「急難救助金」，支持本會服務個案第一時間紓困及弱勢社福團體服務計畫。所有相關捐助手續費皆由《蘋果日報》支付。

  • add 請問如何查看基金會的帳務或是財報？

    基金會每年由會計師進行查帳，並將財報暨會務報告公告於《蘋果日報》，及基金會網頁之「財務報告」，基金會免付費電話是0809008585。

  • add 基金會董事成員組成

    蘋果日報慈善基金會的董事成員包括了社福界及財會法律背景之專業人士。

  • add 若捐款會全數捐出，如何支付工作人員薪水與行政支出？

    基金會工作人員及行政支出皆由蘋果日報支付，大眾捐助皆全數用於弱勢服務。
